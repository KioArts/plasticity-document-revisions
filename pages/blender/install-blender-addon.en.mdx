# Setup Blender Bridge

To use the Blender Bridge, install the add-on in Blender and enable the server in Plasticity. Setup is a one time job and from then on, the two apps connect automatically whenever both are running.

## Install the Blender add-on

1. Download the latest version of the add-on:
   [plasticity-blender-addon-main.zip](https://github.com/nkallen/plasticity-blender-addon/archive/refs/heads/main.zip)

2. Launch Blender.

3. Open **Edit > Preferences > Add-ons**.

4. Click **Install...** and select the downloaded zip file. Do not unzip it first because Blender reads the archive directly.

5. Tick the checkbox next to **Object: Plasticity** when it appears in the list. It may take a moment to load on the first install.

Once enabled, the **Plasticity** panel appears in the sidebar of the 3D Viewport. Press <kbd>N</kbd> in the viewport if the sidebar is hidden.

## Enable the server in Plasticity

1. Launch Plasticity.

2. Open **Preferences** from the **P** menu in the top left of the screen.

3. In the left column, select **Server**.

4. Tick **Enabled**.

The bridge is now active on both sides. You're ready to go.

Check out [how to use the Blender Bridge](/blender/how-to-use) to walk through your first refresh and explore the rest of the workflow.
