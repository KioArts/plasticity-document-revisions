# Blender Bridge einrichten

Um die Blender Bridge zu verwenden, installiere das Add-on in Blender und aktiviere den Server in Plasticity. Die Einrichtung ist einmalig – danach verbinden sich beide Anwendungen automatisch, sobald sie laufen.

## Blender‑Add‑on installieren

1. Lade die neueste Version des Add-ons herunter:  
   [plasticity-blender-addon-main.zip](https://github.com/nkallen/plasticity-blender-addon/archive/refs/heads/main.zip)

2. Starte Blender.

3. Öffne **Edit > Preferences > Add-ons**.

4. Klicke auf **Install...** und wähle die heruntergeladene ZIP-Datei aus. Entpacke sie nicht vorher, da Blender das Archiv direkt liest.

5. Aktiviere das Kontrollkästchen neben **Object: Plasticity**, sobald es in der Liste erscheint. Beim ersten Installieren kann das Laden einen Moment dauern.

Nach der Aktivierung erscheint das **Plasticity**‑Panel in der Seitenleiste des 3D‑Viewports. Drücke <kbd>N</kbd>, falls die Seitenleiste ausgeblendet ist.

## Server in Plasticity aktivieren

1. Starte Plasticity.

2. Öffne **Preferences** über das **P**‑Menü oben links.

3. Wähle in der linken Spalte **Server**.

4. Aktiviere **Enabled**.

Die Bridge ist nun auf beiden Seiten aktiv. Du bist startklar.

Sieh dir [die Verwendung der Blender Bridge](/blender/how-to-use) an, um deinen ersten Refresh durchzugehen und den restlichen Workflow zu entdecken.
