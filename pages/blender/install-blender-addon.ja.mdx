# Blender Bridge のセットアップ

Blender Bridge を使用するには、Blender にアドオンをインストールし、Plasticity でサーバーを有効にします。セットアップは一度行えば完了し、その後は両方のアプリが起動しているだけで自動的に接続されます。

## Blender アドオンのインストール

1. 最新バージョンのアドオンをダウンロードします：  
   [plasticity-blender-addon-main.zip](https://github.com/nkallen/plasticity-blender-addon/archive/refs/heads/main.zip)

2. Blender を起動します。

3. **Edit > Preferences > Add-ons** を開きます。

4. **Install...** をクリックし、ダウンロードした zip ファイルを選択します。Blender は zip を直接読み込むため、事前に解凍しないでください。

5. リストに **Object: Plasticity** が表示されたらチェックを入れます。初回インストール時は読み込みに少し時間がかかる場合があります。

有効化すると、3D ビューポートのサイドバーに **Plasticity** パネルが表示されます。サイドバーが非表示の場合は <kbd>N</kbd> を押してください。

## Plasticity でサーバーを有効にする

1. Plasticity を起動します。

2. 画面左上の **P** メニューから **Preferences** を開きます。

3. 左側のリストから **Server** を選択します。

4. **Enabled** にチェックを入れます。

これで両側のブリッジが有効になりました。準備完了です。

最初の更新操作やワークフロー全体を確認するには、[Blender Bridge の使い方](/blender/how-to-use) をチェックしてください。
