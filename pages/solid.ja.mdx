### ブーリアンとカット

- ソリッドボディを結合・減算するためのツール。

| 名前 | 説明 |
| :--- | :--- |
| [Boolean](/solid/boolean) | 2 つのソリッドボディを結合、減算、または交差させます。 |
| [Cut](/solid/cut) | 別のボディを切削ツールとして使用してソリッドボディを切断します。 |

### インスタンス

- ソリッドおよびシートのインスタンスを作成・実体化するツール。

| 名前 | 説明 |
| :--- | :--- |
| [Create Instance](/solid/create-instance) | 選択したジオメトリの再利用可能なインスタンスを作成します。 |
| [Create Solid and Sheet Instance](/solid/create-solid-and-sheet-instance) | ソリッドとシートの再利用可能なインスタンスを作成します。 |

### サーフェス作成

- サーフェスおよびソリッドを作成するツール。

| 名前 | 説明 |
| :--- | :--- |
| [Alternative Duplicate](/solid/alternative-duplicate) | 選択したサーフェス断片からソリッドを生成する別の方法です。 |
| [Bridge Surface](/solid/bridge-surface) | 2 つの既存サーフェス間に滑らかなブリッジサーフェスを作成します。 |
| [Constrained Surface](/solid/constrained-surface) | 既存ジオメトリに拘束されたサーフェスを作成します。 |
| [Extrude](/solid/extrude) | カーブまたはフェースを指定方向に押し出してソリッドまたはシートを作成します。 |
| [Loft](/solid/loft) | 複数のカーブまたはプロファイルを通過させてサーフェスを作成します。 |
| [Patch](/solid/patch) | 開いた境界を新しいサーフェスパッチで埋めます。 |
| [Pipe](/solid/pipe) | カーブパスに沿ってパイプソリッドを作成します。 |
| [PolySpline](/solid/polyspline) | メッシュオブジェクトを変換してポリサーフェスを作成します。 |
| [Revolve](/solid/revolve) | カーブを軸の周りに回転させてソリッドまたはシートを作成します。 |
| [Square](/solid/square) | XNurbs を使用して境界カーブ間に未トリムサーフェスを作成します。 |
| [Sweep](/solid/sweep) | プロファイルカーブをパスに沿ってスイープし、ソリッドまたはシートを作成します。 |
| [Xnurbs](/solid/xnurbs) | 境界カーブから高品質なサーフェスを生成します。 |
