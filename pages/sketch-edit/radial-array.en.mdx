# Radial Array

<Reference shortcut="" command="Radial Array" button="Toolbar Bottom Right" />

You can duplicate curves in a radial array.

1. Select the curve you want to duplicate.
2. Click on the **Radial Array** icon in the Command bar bottom-right.
3. Click on the point you want to set as the center.
4. The duplication preview will be displayed.
5. Specify the options:
   - **Degrees**: Specify the angle for duplication.
   - **Number**: Specify the number of duplications. You can also use `Shift`+Mouse wheel to adjust.
   - **Repeat**: Specify the number of repetitions in a radial pattern.
   - **Repeat distance**: Specify the distance between each repetition in a radial pattern.
6. Confirm with **OK** in the command dialog or right click.
