# カーブに沿って複製する

カーブをカーブに沿って複製できます。

<Reference shortcut="" command="Curve Array" button="" />

1. 移動したいカーブを選択します。
2. ショートカット`F`でコマンドパレットを表示して、**Curve Array**と入力して選びます。
3. 沿わせたいカーブを選択します。
4. オプションを指定します。
   - **Number**: 複製する数を指定します。`Shift`+マウスホイールの回転でも指定できます。
   - **Twist**: カーブに沿って捻る角度を度数で指定します。
   - **Scale**: カーブに沿って拡大縮小する倍率を指定します。
   - **Alignment**: 
      - **Parallel**: 軌道に関わらず、平行に複製します。
      - **Normal**: 軌道に沿った角度で複製します。
7. コマンドダイアログのOKまたは右クリックで確定します。
