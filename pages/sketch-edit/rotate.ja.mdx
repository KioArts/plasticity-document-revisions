# 回転する 

 カーブまたは複数のポイント、CVを回転できます。

<Reference shortcut="R" command="Rotate" button="コマンドバー左下" />

1. 移動したい対象を選択します。
2. ショートカット`R`を押すか、コマンドバー左下のアイコンをクリックします。
3. 表示された回転ギズモを使って回転します。
   - `R` : **Screen space** : 見た目の平面に限定します。
   - `F` : **Freestyle** : 最初のクリックで始点、次のクリックで終点を指定します。
   - `V` : **Pivot** : ギズモの位置をクリックして指定します。
   - `W` : **Worldspace** : ギズモの座標軸を元に戻します。
4. コマンドダイアログのOKまたは右クリックで確定します。
