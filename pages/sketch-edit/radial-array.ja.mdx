# 円形状配列で複製する

カーブを円形状配列で複製できます

<Reference shortcut="" command="Radial Array" button="ツールバー右下" />

1. 複製したいカーブを選択します。
2. コマンドバー右下の**Radial Array**アイコンをクリックします。
3. 中心にしたい点をクリックします。
4. 複製プレビューが表示されます。
5. オプションを指定します。
   - **Degrees**：複製する角度を指定します。
   - **Number**: 複製する数を指定します。`Shift`+マウスホイールの回転でも指定できます。
   - **Repeat**: 放射状に繰り返す回数を指定します。
   - **Repeat distance**: 放射状に繰り返す間隔を指定します。
7. コマンドダイアログの**OK**または右クリックで確定します。
