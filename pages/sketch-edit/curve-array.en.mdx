# Duplicate along curve

You can duplicate curves along another curve.

<Reference shortcut="" command="Curve Array" button="" />

1. Select the curve you want to move.
2. Use the shortcut `F` to display the command palette and select **Curve Array**.
3. Select the curve to use as the path.
4. Specify the options:
   - **Number**: Specify the number of duplications. You can also use `Shift`+Mouse wheel to adjust.
   - **Twist**: Specify the angle of twist along the curve in degrees.
   - **Scale**: Specify the scaling factor along the curve.
   - **Alignment**:
     - **Parallel**: Duplicate the curves in parallel, regardless of the path.
     - **Normal**: Duplicate the curves with angles aligned to the path.
5. Confirm with **OK** in the command dialog or right click.
