# Duplicate in rectangular array

You can duplicate curves in a rectangular array.

<KinouGaiyou shortcut="" command="Rectangular Array" button="Command Bar Bottom Right" />

1. Select the curve you want to duplicate.
2. Click on the **Rectangular Array** icon in the Command bar bottom-right.
3. The duplication preview will be displayed.
4. Click on the desired direction for duplication.
5. Specify the options:
   - **Distance 1**: Specify the distance.
   - **Number**: Specify the number of duplications. You can also use `Shift`+Mouse wheel to adjust.
6. If you want to duplicate in another direction, click on **Select endpoint 2** in the dialog.
    1. Click on the desired direction for duplication.
    2. Specify the options:
       - **Distance 2**: Specify the distance.
       - **Repeat**: Specify the number of repetitions. You can also use `Shift`+Mouse wheel to adjust.
7. Confirm by clicking **OK** in the command dialog or right-clicking.
