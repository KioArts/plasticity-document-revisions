# 長方形配列で複製する

カーブを長方形配列で複製できます

<KinouGaiyou shortcut="" command="Rectangular Array" button="コマンドバー右下" />

1. 複製したいカーブを選択します。
2. コマンドバー右下の**Rectangular Array**アイコンをクリックします。
3. 複製プレビューが表示されます。
4. 複製したい方向をクリックします。
5. オプションを指定します。
   - **Distance 1**: 距離を指定します。
   - **Number**: 複製する数を指定します。`Shift`+マウスホイールの回転でも指定できます。
6. もうひとつの方向にも配列で複製したいときは、ダイアログの**Select endpoint 2**をクリックします。
    1. 複製したい方向をクリックします。
    2. オプションを指定します。
         - **Distance 2**: 距離を指定します。
         - **Repeat**: 複製する回数を指定します。`Shift`+マウスホイールの回転でも指定できます。
7. コマンドダイアログの**OK**または右クリックで確定します。
