# Freestyle Mirror

<Reference shortcut="" command="Freestyle Mirror" button="" />

You can mirror curves based on a freely defined axis.

1. Select the curve to be mirrored.
2. Display the command palette by pressing the shortcut `F`, then type and select **Freestyle Mirror**.
3. Specify the mirroring direction and options:
   - Use the first click as the start point and the second click as the end point to define the axis for mirroring.

4. Confirm with OK in the command dialog or right click.
