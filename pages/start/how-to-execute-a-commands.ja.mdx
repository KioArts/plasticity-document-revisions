# コマンドの実行方法

Plasticityでコマンドを実行する方法は、画面下部のアイコン、キーボードショートカット、コマンドパレットの3種類があります。

## コマンドバー
- コマンドバーはスクリーンの下部にあります。
![Command Bar](./img/command-bar.png)
<div class="dl">
- 左側: 
   - 移動や回転など使用頻度の高いコマンドが並んでいます。
- 右側:
   - 選択したオブジェクトに応じて表示されるコマンドが変わります。
</div>

## キーボードショートカット
- よく使われるコマンドにはキーボードショートカットが割り当てられています。具体的なショートカットは各コマンドのページを参照してください。

## コマンドパレット
- ボタンやショートカットのないコマンドはコマンドパレットから実行できます。作業領域で<kbd>F</kbd>キーを押すとコマンドパレットが表示されます。
- コマンド名を入力して実行します。さらに、任意のコマンドにショートカットを割り当てたり、お気に入りのコマンドをピン留めすることもできます。

### コマンドの実行
1. <kbd>F</kbd>キーを押します。
2. コマンドパレットが表示されます。
3. コマンド入力欄にコマンド名を入力します。
4. 入力した単語を含むコマンドが表示されます。入力ごとに候補が絞られていきます。
5. 上下の矢印キーと<kbd>Enter</kbd>で目的のコマンドを選択します。
6. コマンドが実行されます。

![Command Palette](./img/command-palette.gif)

### ショートカットの割り当てとお気に入りの登録
1. <kbd>F</kbd>キーを押します。
2. コマンドパレットが表示されます。
3. コマンド入力欄にコマンド名を入力します。
4. 入力した単語を含むコマンドが表示されます。入力ごとに候補が絞られていきます。
5. 目的のコマンドが表示されたら、右クリックしてメニューを選択します。
![Command Pallet Right-click](./img/command-palette-right-click.png)

<div class="dl">
- Add to favorites:
   - コマンドがお気に入りに追加され、入力フィールドの下にボタンが表示されます。
   - また、ボタンの右クリックメニューから削除することができます。
- Assign Shortcut:
   - ショートカットが割り当てられます。「Press a key to assign shortcut...」と表示されたら、割り当てたいキーを押します。
- Remove Shortcut:
   - ショートカットの割り当てが解除されます。
</div>

<Callout type="warning" emoji="⚠️">
すでに使用されているショートカットを割り当てた場合、警告メッセージなしに上書きされます。重複を避けるために、<kbd>Alt</kbd>、 <kbd>Ctrl</kbd>、 <kbd>Shift</kbd>などのキーを組み合わせてください。
</Callout>
