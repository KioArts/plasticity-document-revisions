# File Menu

The file menu is opened by clicking on the P in the upper left corner.

![File Menu](/img/file-menu.png)

| Command Name              | Shortcut                  | Note                                                                      |
| ------------------------- | ------------------------- | ------------------------------------------------------------------------- |
| New                       | <kbd>Ctrl-N</kbd>         | Clear the scene, then New Document                                        |
| Open                      | <kbd>Ctrl-O</kbd>         |                                                                           |
| Import/Append             | <kbd>Ctrl-Shift-O</kbd>   | See [supported import file formats](#supported-import-file-formats) below |
| Save                      | <kbd>Ctrl-S</kbd>         |                                                                           |
| Export/Save as            | <kbd>Ctrl-Shift-S</kbd>   | See [supported export file formats](#supported-export-file-formats) below |
| Quick save version        |                           | Save the same filename with the timestamp                                 |
| Save as startup scene     |                           | Save the current document as the startup scene                            |
| Preferences               | <kbd>Ctrl-, (Comma)</kbd> | See [Preferences](./preferences) page                                     |
| Manage licenses           |                           | Manage the license token and devices                                      |
| Plasticity Version: 0.0.0 |                           | Installed Plasticity version                                              |
| Quit                      | <kbd>Ctrl-Q</kbd>         |                                                                           |


## Supported import file formats [#supported-import-file-formats]
- **Indie and Studio** 
    - Plasticity file (*.plasticity)
    - STEP file (*.stp;*.step)
    - Parasolid file (*.x_t;*.x_b)
    - Wavefront OBJ file (*.obj)
    - Image files (*.png;*.jpg;*.jpeg)
- **Studio only**
    - IGES files (*.igs;*.iges) 
    - SAT files (*sat)
    - Rhino files (*.3dm) 

## Supported export file formats [#supported-export-file-formats]
- **Indie and Studio** 
    - Plasticity file (*.plasticity)
    - STEP file (*.stp;*.step)
    - Parasolid file (*.x_t;*.x_b)
    - [Wavefront OBJ file](./export-obj) (*.obj)
    - STL file (*.stl)
- **Studio only**
    - IGES file (*.igs;*.iges)
    - SAT file (*sat)
