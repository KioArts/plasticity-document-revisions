# ショートカットとコマンドパレット

Plasticityで機能を実行する方法は、画面下部のアイコン、キーボードショートカット、コマンドパレットの3種類があります。

## 画面下部のアイコン

- 画面下部 左側：使用頻度の高い移動や回転などの機能が並んでいます。
- 画面下部 右側：選択したものによって使用可能な機能が並んでいます。

## キーボードショートカット

- 使用頻度が高い機能にはキーボードショートカットが割り当てられています。
- 参照 [ショートカット一覧](shortcuts.ja.md)

## コマンドパレットとお気に入り

- アイコンもショートカットも用意されていない機能はコマンドパレットから実行できます。
- 作業エリア内でショートカット`F`を押すとコマンドパレットが表示され、機能名を入力して実行できます。
- また、コマンドパレットで機能に任意のショートカットを割り当てたり、お気に入りの機能をピンすることができます。

![F-Menu](./img/f-menu.png)

1. デフォルトのショートカット（グレーの文字）
2. 右クリックメニュー
    - お気に入りへの登録
    - ショートカットを登録する
    - デフォルトのショートカットに戻す
3. 登録したショートカット（紫色）
4. お気に入りに登録した機能



### 機能を実行する

1. ショートカット`F`を押します。
2. コマンドパレットが表示されます。
3. コマンド入力欄に機能の名称を入力します。
4. 入力した単語を含む機能が表示されます。
5. 入力を進めると機能の候補が絞られていきます。
6. 目的の機能が表示されたら、上下の矢印キーと`Enter`で確定します。
7. 機能が実行されます。

### ショートカットの割り当てと、お気に入りの登録

1. ショートカット`F`を押します。
2. コマンドパレットが表示されます。
3. コマンド入力欄に機能の名称を入力します。
4. 入力した単語を含む機能が表示されます。
5. 入力を進めると機能の候補が絞られていきます。
6. 目的の機能が表示されたら、右クリックしてメニューを選択します。
   - **Add to favorites**: お気に入りに登録します。入力欄の下にアイコンが追加されます。アイコンの右クリックメニューから解除できます。
   - **Assign Shortcut**: ショートカットを割り当てます。 **Press a key to assign shortcut...**と表示されたら割り当てたいキーを押します。
   - **Remove Shortcut**: ショートカットの割り当てを解除します。


<Callout type="warning" emoji="⚠️">
  すでに割り当てられているショートカットを入力すると上書きされます。このとき警告のメッセージは表示されません。<br />
  `Alt` `Ctrl` `Shift`などを組み合わせて重複を回避する必要があります。
</Callout>
