# Befehle für Festkörper

| Befehlsname                                                 | Tastenkombination                     | Anmerkung                                                                                      |
| ------------------------------------------------------------ | ---------------------------- | ---------------------------------------------------------------------------------------- |
| [Boolean](/solid/boolean)                                     | <kbd>Q</kbd>                 | Boolesche Körper und/oder Flächen.                                                       |
| [Cut](/solid/cut)                                             | <kbd>C</kbd>                 | Körper und Fläche schneiden.                                                              |
| **Solid and Sheet Commands**                                 |                              |                                                                                          |
| [Alternative Duplicate](/solid/alternative-duplicate)         | <kbd>Alt+D</kbd>           | Alternative Duplikat.                                                                     |
| [Constrained Surface](/solid/constrained-surface)             |                              | Eine eingeschränkte Fläche erstellen.                                                    |
| [Deform](/solid/deform)                                      |                               | Vereinheitlichter Befehl für Kurve verformen, Körper und Fläche verformen.                |
| [Deform Solid and Sheet](/solid/deform-solid-and-sheet)       |                               | Objekte auf unregelmäßigen und regelmäßigen Oberflächen wickeln.                         |
| [Extrude](/solid/extrude)                                     | <kbd>E</kbd>                 | Flächen, Regionen und Kurven extrudieren.                                                |
| [Fillet](/solid/fillet)                                       | <kbd>B</kbd>                 | Vereinheitlichter Befehl für Kante abrunden, Eckpunkt abrunden, Schale abrunden.         |
| [Fillet Shell](/solid/fillet-shell)                           | <kbd>B</kbd>                 | Kanten abrunden oder fasen.                                                              |
| [Hollow](/solid/hollow)                                       |                              | Einen Körper hohl machen.                                                                 |
| [Loft](/solid/loft)                                           | <kbd>L</kbd>                 | Fläche, Kante und Kurve loften.                                                          |
| [Patch](/solid/patch)                                         |                              | Löcher in der Fläche / geschlossene Kurven zu einer Fläche füllen.                       |
| [Pipe](/solid/pipe)                                           | <kbd>P</kbd>                 | Ein Rohr aus Kante und Kurve erstellen.                                                  |
| [Raise Degree](/solid/raise-degree)                           | <kbd>Shift+S</kbd>         | Vereinheitlichter Befehl für Grad der Kurve erhöhen, Grad der Fläche erhöhen.            |
| [Raise Surface Degree](/solid/raise-surface-degree)           | <kbd>Shift+S</kbd>         | Kontrollelemente der Fläche erhöhen.                                                      |
| [Remove Fillets from Shell](/solid/remove-fillets-from-shell) |                              | Fasen von Körpern und Flächen entfernen.                                                 |
| [Revolve](/solid/revolve)                                     |                              | Region oder Kurve drehen.                                                                |
| [Sweep](/solid/sweep)                                         | <kbd>Shift+P</kbd>         | Region oder Kurve fegen.                                                                  |
| [Square](/solid/square)                                       |                               | Flächen mit expliziter Kontrolle über Grade und Spannweiten erstellen.                  |
| [Slide](/solid/slide)                                        |                               | Vereinheitlichter Befehl für Slide Curve CV, Slide Surface CV.                           |
| [Slide Surface CV](/solid/slide-surface-cv)                   | <kbd>Shift+G</kbd>           | Verschieben Sie ein CV auf einer Oberfläche entlang des CV-Hulls.                        |
| [Thicken](/solid/thicken)                                     |                              | Dicke zu einer ausgewählten Fläche hinzufügen.                                           |
| [Thicken Face](/solid/thicken-face)                           |                              | Dicke zu einer ausgewählten Fläche hinzufügen.                                           |
| [XNurbs](/solid/xnurbs)                                       |                               | Hochwertige und komplexe Oberflächen erstellen.                                          |
| **Sheet Commands**                                           |                              |                                                                                          |
| [Bridge Surface](/solid/bridge-surface)                       |                              | Zwei Flächen verbinden.                                                                    |
| [Extend](/solid/extend)                                       |                              | Vereinheitlichter Befehl für Kurve erweitern, Fläche erweitern.                          |
| [Extend Sheet](/solid/extend-sheet)                           |                              | Eine oder mehrere Außenkanten der Fläche erweitern.                                      |
| [Find Boundary Edges](/solid/find-boundary-edges)             |                              | Kanten einer Öffnung in der Fläche auswählen.                                            |
| [Join](/solid/join)                                           | <kbd>J</kbd>                 | Vereinheitlichter Befehl für Kurven verbinden, Flächen verbinden.                       |
| [Join Sheets](/solid/join-sheets)                             | <kbd>J</kbd>                 | Flächen mit passenden Kanten verbinden.                                                  |
| [Reverse Sheet](/solid/reverse-sheet)                         |                              | Die Richtung der Normalen auf einer Fläche umkehren.                                    |
| [Unjoin](/solid/unjoin)                                       | <kbd>Alt+J</kbd>           | Vereinheitlichter Befehl für Kurven trennen, Flächen trennen, Schalen trennen.          |
| [Unjoin Faces](/solid/unjoin-faces)                           | <kbd>Alt+J</kbd>           | Ausgewählte Flächen von Körper oder Fläche trennen.                                      |
| [Unjoin Shells](/solid/unjoin-shells)                         | <kbd>Alt+J</kbd>           | Alle Flächen von ausgewählten Körpern oder Flächen in einzelne Flächen trennen.         |
| [Untrim](/solid/untrim)                                       | <kbd>Alt+T</kbd>           | Eine Fläche basierend auf der ursprünglichen Form erstellen.                            |
| **Face Commands**                                            |                              |                                                                                          |
| [Delete Face](/solid/delete-face)                             |<kbd>Shift+Backspace</kbd> oder <kbd>Shift+Delete</kbd> oder <kbd>Shift+X</kbd> | Fläche löschen.                                                                        |
| [Draft Face](/solid/draft-face)                               |                              | Fläche neigen.                                                                           |
| [Match Face](/solid/match-face)                               |                              | Flächen an eine andere Fläche anpassen.                                                  |
| [Offset Face](/solid/offset-face)                             | <kbd>Shift+O</kbd>         | Ausgewählte Flächen versetzen.                                                           |
| [Rebuild](/solid/rebuild)                                     |                              | Vereinheitlichter Befehl für Kurve neu aufbauen, Fläche neu aufbauen.                   |
| [Rebuild Face](/solid/rebuild-face)                           |                              | Fläche neu aufbauen.                                                                     |
| [Unwrap Face](/solid/unwrap-face)                             |                              | Fläche entfalten.                                                                        |
| **Edge Commands**                                            |                              |                                                                                          |
| [Delete Redundant Topology](/solid/delete-redundant-topology) |                              | Überflüssige Kanten entfernen.                                                            |
| [Extend Edge](/solid/extend-edge)                             |                              | Kanten erweitern.                                                                         |
| [Imprint](/solid/imprint)                                     | <kbd>Shift+I</kbd>         | Vereinheitlichter Befehl für Körper abdrucken, Kurve Körper abdrucken.                   |
| [Imprint Body Body](/solid/imprint-body-body)                 | <kbd>Shift+I</kbd>         | Kanten auf dem Zielkörper oder der Fläche erstellen, wo sie mit einem Werkzeugkörper oder einer Fläche schneiden. |
| [Imprint Curve Body](/solid/imprint-curve-body)               | <kbd>Shift+I</kbd>         | Kanten auf dem Zielkörper oder der Fläche erstellen, wo sie auf eine Werkzeugkurve projizieren. |
| [Isoparam](/solid/isoparam)                                   | <kbd>Ctrl+R</kbd>          | Isoparametern hinzufügen.                                                                 |
| [Offset Curve](/solid/offset-curve)                           | <kbd>O</kbd>                 | Vereinheitlichter Befehl für versetzte ebene Kurve, versetzte Region, versetzte Flächenumrandung, versetzte Kante. |
| [Offset Edge](/solid/offset-edge)                             | <kbd>O</kbd>                 | Ausgewählte Kanten versetzen.                                                             |
| [Offset Face Loop](/solid/offset-face-loop)                   | <kbd>O</kbd>                 | Die Umrisskanten ausgewählter Flächen versetzen.                                         |
