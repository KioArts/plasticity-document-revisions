# 拡大縮小する

<Reference shortcut="S" command="Scale" button="コマンドバー左下" />

ソリッドまたはシートを拡大縮小できます。

1. 移動したいソリッドまたはシートを選択します。
2. ショートカット`S`を押すか、コマンドバー左下のアイコンをクリックします。
3. 表示された拡大縮小ギズモを使って動かします。
   - `S` **Uniform**
      - X、Y、Z軸方向に同じだけ拡大縮小します。
   - `F` **Freestyle**
      - 最初のクリックで始点、次のクリックで終点を指定します。
   - `W` **Worldspace**
      - ギズモの座標軸を元に戻します。
   - そのほかのオプションは[ソリッド編集の共通オプション](common-options)を参照してください。
4. コマンドダイアログのOKまたは右クリックで確定します。
