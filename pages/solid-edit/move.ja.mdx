# 移動する

<Reference shortcut="G" command="Move" button="コマンドバー左下" />

ソリッドまたはシート、フェイス、エッジを移動できます。

1. 移動する対象としてソリッドまたはシート、フェイス、エッジを選択します。
2. ショートカット`G`を押すか、コマンドバー左下のアイコンをクリックします。
3. 表示された移動ギズモを使って動かします。
   - `F` **Freestyle**
      - 移動する方向と長さを、始点と終点の2回のクリックで指定します。
   - `W` **Worldspace**
      - ギズモの座標軸を元に戻します。
   - そのほかのオプションは[ソリッド編集の共通オプション](common-options)を参照してください。
4. コマンドダイアログの**OK**または右クリックで確定します。
