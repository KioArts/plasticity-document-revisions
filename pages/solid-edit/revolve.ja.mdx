# リボルブする

リージョンまたはカーブをリボルブする

1. リボルブ対象となるリージョンまたはカーブを選択します。
2. コマンドバー左下のアイコンをクリックします。
3. 回転の軸を指定します。最初に始点をクリック、次に終点をクリックします。2点をつなぐ線が軸になります。
4. オプションを指定します。
    - `Shift-T`: **Thickness**:  {/* ??? unknown function */}
    - `A`: **Angle**: カーソルを移動して回転角度を指定します。
5. コマンドダイアログの**OK**または右クリックで確定します。
