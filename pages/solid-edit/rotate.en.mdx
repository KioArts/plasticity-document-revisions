# Rotate

<Reference shortcut="R" command="Rotate" button="Bottom-left of Command Bar" />

You can rotate solids, sheets, and faces.

1. Select a solid, sheet, or face as the object to rotate.
2. Press the shortcut `R` or click the button at the bottom-left of the Command Bar.
3. Use the displayed Rotation Gizmo to rotate.
   - `F` **Freestyle**
      - Specify the starting point with the first click and the ending point with the next click.
   - `W` **World space**
      - Releases the rotation restrictions.
   - Refer to [Solid Editing Common options](common-options) for other options.
4. Confirm with **OK** in the command dialog or right click.
