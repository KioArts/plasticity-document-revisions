# Rotate

You can rotate solids, sheets, and faces.

<KinouGaiyou shortcut="R" command="Rotate" button="Bottom-left of Command Bar" />

1. Select a solid, sheet, or face as the object to rotate.
2. Press the shortcut `R` or click the icon at the bottom-left of the Command Bar.
3. Use the displayed Rotation Gizmo to rotate:
   - `X`: **X axis**: Restricts rotation to the X-axis.
   - `Y`: **Y axis**: Restricts rotation to the Y-axis.
   - `Z`: **Z axis**: Restricts rotation to the Z-axis.
   - `R`: **Screen space**: Restricts rotation to the visible plane.
   - `F`: **Freestyle**: Specify the starting point with the first click and the ending point with the next click.
   - `V`: **Pivot**: Restricts rotation in the direction of the clicked object.
   - `W`: **World space**: Releases the rotation restrictions.
4. Confirm by clicking **OK** in the command dialog or right-clicking.
