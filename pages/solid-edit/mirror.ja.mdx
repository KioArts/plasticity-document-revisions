# ミラーする

ソリッドまたはシート、フェイスをミラーできます。

<KinouGaiyou shortcut="ALt+X" command="Mirror" button="" />

1. ミラー対象となるソリッドまたはシート、フェイスを選択します。
2. ショートカット`Alt`+`C`を押すか、コマンドバー左下のアイコンをクリックします。
3. ミラーの方向とオプションを指定します。
    - `X` : **Position X** : X軸のプラスの方向にミラーします。
    - `Y` : **Position Y** : Y軸のプラスの方向にミラーします。
    - `Z` : **Position Z** : Z軸のプラスの方向にミラーします。
    - `Shift`+`X` : **Negative X** : X軸のマイナスの方向にミラーします。
    - `Shift`+`Y` : **Negative Y** : Y軸のマイナスの方向にミラーします。
    - `Shift`+`Z` : **Negative Z** : Z軸のマイナスの方向にミラーします。
    - `F` : **Freestyle** : 最初のクリックで始点、次のクリックで終点で指定した軸でミラーします。
    - `Q` : **Union(merge)** : ミラーしたふたつを結合します。
    - **Cut**: ミラーの平面でミラー対象を分割してからミラーします。
4. コマンドダイアログの**OK**または右クリックで確定します。
