# Cut

You can cut solids or sheets.
The cutting direction is perpendicular to the construction plane.

<KinouGaiyou shortcut="C" command="Cut Solid With Curve" button="" />

1. Press the shortcut `C` or click the icon at the bottom-left of the Command Bar.
2. Make sure that **Select target bodies** in the dialog is highlighted in purple, and select the solid or sheet to be cut.
3. Make sure that **Select cutters** in the dialog is highlighted in purple, and select the curve or face to be used as the cutter.
   - `S`: **Screen space**: Sets the cutting direction to align with the visible direction.
4. Confirm with **OK** in the command dialog or right click.
