# 自由な軸でミラーする

<Reference shortcut="" command="Freestyle Mirror" button="" />

ソリッドまたはシート、フェイスを自由な軸を基準にミラーできます。

1. ミラー対象となるソリッドまたはシート、フェイスを選択します。
2. ショートカット`F`でコマンドパレットを表示して、**Freestyle Mirror**と入力して選びます。
3. ミラーの方向とオプションを指定します。
    - 最初のクリックで始点、次のクリックで終点で指定した軸でミラーします。
    - **Cut**: ミラーの平面でミラー対象を分割してからミラーします。
    - `Q` : **Union(merge)** : ミラーしたふたつを結合します。
4. コマンドダイアログの**OK**または右クリックで確定します。
