# 回転する 

<Reference shortcut="R" command="Rotate" button="コマンドバー左下" />

ソリッドまたはシート、フェイスを回転できます。

1. 回転対象としてソリッドまたはシート、フェイスを選択します。
2. ショートカット`R`を押すか、コマンドバー左下のアイコンをクリックします。
3. 表示された回転ギズモを使って回転します。
   - `F` **Freestyle**
      - 最初のクリックで始点、次のクリックで終点を指定します。
   - `W` **Worldspace**
      - ギズモの座標軸を元に戻します。
   - そのほかのオプションは[ソリッド編集の共通オプション](common-options)を参照してください。
4. コマンドダイアログのOKまたは右クリックで確定します。
