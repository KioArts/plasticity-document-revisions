# Freestyle Mirror

You can mirror solids, sheets, and faces based on a freely chosen axis.

<KinouGaiyou shortcut="" command="Freestyle Mirror" button="" />

1. Select the solid, sheet, or face to be mirrored.
2. Open the command palette by pressing the shortcut `F`, then type **Freestyle Mirror** and select it.
3. Specify the mirroring direction and options:
   - Mirror along the axis defined by the first click as the start point and the second click as the end point.
   - **Cut**: Divide the mirror target with the mirror plane before mirroring.
   - `Q`: **Union(merge)**: Merge the two mirrored entities together.
4. Confirm by clicking **OK** in the command dialog or right-clicking.
