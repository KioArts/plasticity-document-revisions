# ブーリアンする

ソリッドまたはシートをブーリアンできます。

<KinouGaiyou shortcut="Q" command="Boolean" button="" />

1. ショートカット`Q`を押すか、コマンドバー左下のアイコンをクリックします。
2. ダイアログの**Select target bodies**が紫色で囲まれているのを確認し、ブーリアン対象となるソリッドまたはシートを選択します。
3. ダイアログの**Select tool bodies**が紫色で囲まれているのを確認し、ツールとなるソリッドまたはシートを選択します。
4. ブーリアン操作を指定します。デフォルトは**Difference**です。
    - `Shift`+`Q`: **Slice**: ブーリアン対象とツールの交点にソリッドを作成し、余りのソリッドも残します。
    - `Q`: **Union**: ブーリアン対象とツールを結合します。
    - `W`: **Difference**: ブーリアン対象からツールを切り取ります。
    - `Shift`+`E`: **Intersect**: ブーリアン対象とツールの交点にソリッドを作成します。
    - `T`: **Keep Tools**: ブーリアン後もツールを残します。
    - `G`: **Move**: ツールを移動できます。
        - `G` : **Screen space** : 見た目の平面に限定します。
        - `X` : **X axis** : X軸方向に限定します。
        - `Y` : **Y axis** : Y軸方向に限定します。
        - `Z` : **Z axis** : Z軸方向に限定します。
        - `Shift`+`Z` : **Z plane** : Z平面に限定します。
        - `Shift`+`X` : **X plane** : X平面に限定します。
        - `Shift`+`Y` : **Y plane** : Y平面に限定します。
    - `R`: **Rotate**: ツールを回転できます。
        - `R`: **Screen space**: 回転軸をもとに戻します。
   - `S`: **Scale**: ツールを拡大縮小できます。
        - `S` : **Uniform** : X、Y、Z軸方向に同じだけ拡大縮小します。   
5. コマンドダイアログの**OK**または右クリックで確定します。
