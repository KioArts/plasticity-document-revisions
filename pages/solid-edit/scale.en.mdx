# Scale

<Reference shortcut="S" command="Scale" button="Bottom-left of Command Bar" />

You can scale solids or sheets.

1. Select the solid or sheet you want to scale.
2. Press the shortcut `S` or click the button at the bottom-left of the Command Bar.
3. Use the displayed Scale Gizmo to resize.
   - `S` **Uniform**
      - Scales uniformly in the X, Y, and Z directions.
   - `F` **Freestyle**
      - Specify the starting point with the first click and the ending point with the next click.
   - `W` **World space**
      - Releases the scaling restrictions.
   - Refer to [Solid Editing Common options](common-options) for other options.
4. Confirm with OK in the command dialog or right click.
