# Extrude（押出し）

<Reference shortcut="E" command="Extrude" button="コマンドバー右" />



フェイスまたはリージョンを押し出したときはソリッドが作成されます。カーブのときはシートが作成されます。

1. ショートカット`E` を押すか、ボタンをクリックします。
2. 押し出すフェイス、リージョンまたはカーブを選択します。
3. 黄色いドットを動かし距離を指定します。
   - **Distance**
      - 距離と反対側の距離を指定します。
   - **Degrees**
      - 押し出しの角度を指定します。
   - **Thickness**
      - 壁の厚さを指定します。
   - `F` **Freestyle**
      - クリックして始点と終点を指定し、移動の方向と距離を指定します。
   - `V` **Pivot**
      - クリックしてギズモの位置を指定します。
   - `D` **Distance 1**
      - マウスの動きで距離を指定します。
   - `A` **Angle 1** 
      - マウスの動きで押し出しの角度を指定します。
   - `Shift-T` **Thickness**
      - マウスの動きで壁の厚さを指定します。
   - `Tab` **Lock distances**
      - 距離と反対側の距離を同じにします。
4. \<オプション\>押し出したオブジェクトでブーリアンできます。
   - **Select target bodies**: ブーリアンされるソリッドかシートを指定します。
      - そのほかのオプションは[ブーリアンオプション](boolean-options)を参照してください。
5. ソリッドまたはシートが作成されます。
