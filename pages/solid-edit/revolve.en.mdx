# Revolve

You can revolve regions or curves.

<Reference shortcut="" command="Revolve" button="" />

1. Select the region or curve to be revolved.
2. Click the icon at the bottom-left of the Command Bar.
3. Specify the axis of revolution. Click the start point first, then the end point. The line connecting the two points will be the axis.
4. Specify the options.
   - `Shift`+`T`: **Thickness**: {/* ??? unknown function */}
   - `A`: **Angle**: Move the cursor to specify the rotation angle.
5. Confirm with **OK** in the command dialog or right click.
