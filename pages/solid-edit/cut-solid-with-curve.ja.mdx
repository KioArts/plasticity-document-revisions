# カットする

<Reference shortcut="C" command="Cut Solid With Curve" button="" />

ソリッドまたはシートをカットできます。
カットする方向は構築平面に対して垂直な方向です。

1. ショートカット`C`を押すか、コマンドバー左下のアイコンをクリックします。
2. ダイアログの**Select target bodies**が紫色で囲まれているのを確認し、カット対象となるソリッドまたはシートを選択します。
3. ダイアログの**Select cutters**が紫色で囲まれているのを確認し、カッターとなるカーブまたはフェイスを選択します。
    - `S`: **Screen space**: カットする方向を見た目の方向にします。
4. コマンドダイアログのOKまたは右クリックで確定します。
