### Sketching Fundamentals

- Learn the essentials of sketching accurate curves, snapping to points, and using temporary construction guides. See [Sketching Essentials](/tool/sketching-essentials) for a full overview.

### Curves and Lines

- Tools to create, bridge, and manipulate curves.

| Name | Description |
| :--- | :--- |
| [Line](/tool/line) | Draw straight lines between two points. |
| [Spline Curve](/tool/spline-curve) | Draw a smooth curve by placing points that the curve passes through. |
| [Control Point Curve](/tool/control-point-curve) | Draw a curve using control points that influence its shape without lying on it. |
| [Bridge Curve](/sketch/bridge-curve) | Connect two existing curves with a smooth bridging curve. |
| [Split Curve At Point](/sketch/split-segment) | Split a curve into two separate curves at a specified point. |
| [Insert CV At Point](/sketch/insert-knot) | Add a control vertex to an existing curve for finer shape control. |
| [Trim](/sketch/trim) | Trim curves at their intersections to clean up geometry. |
