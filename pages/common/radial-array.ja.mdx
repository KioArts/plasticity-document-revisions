## 説明

角度と数を指定して、オブジェクトを円形パターンで複製します。放射状およびインスタンスによる複製も可能です。

<Callout type='info'>
**注意:** 画像およびメッシュ（.OBJ、.STL、.FBX など）は配列でき、複製角度やコピー数を調整できます。インスタンスオブジェクトの配列もサポートされています。
</Callout>

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. コマンドバー右側の <ButtonIcon type="button-radial-array" /> ボタンをクリックします。
3. 中心として設定したい点をクリックします。
4. 複製プレビューが表示されます。
5. コマンドダイアログおよびキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
   - **モード:**
         - **トータル:** 指定した角度値に基づき、すべての複製を均等に配置する総距離を指定します。
         - **間隔:** 各複製の間隔距離を指定します。
   - **角度:** 複製の角度を指定します。
   - **数 2:** 複製数を指定します。
   - **数 1:** 放射状パターンの繰り返し数を指定します。
   - **ステップ 1:** 放射状パターンの各繰り返し間の距離を指定します。
   - **インスタンス作成:** 独立したコピーではなくインスタンスを作成します。（[インスタンス作成](/solid/create-instance) を参照）
   - <Kst>キーボードショートカット</Kst>
   - <kbd>Shift + ホイール上</kbd> **コピー追加:** 複製数を増やします。
   - <kbd>Shift + ホイール下</kbd> **コピー削減:** 複製数を減らします。
   - <kbd>I</kbd> **インスタンス切り替え:** ミラーされた半分をインスタンスとして作成するか、独立コピーとして作成するかを切り替えます。
6. コマンドダイアログで OK を押すか、右クリックで確定します。
