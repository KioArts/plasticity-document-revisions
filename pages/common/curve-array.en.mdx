# Curve Array

<Reference shortcut="" command="Curve Array" button="" />

You can duplicate curves, solids, sheets in a curve array.

1. Select the object you want to duplicate.
2. Use **"Curve Array":** in the Command Palette.
3. Select the curve to use as the path.
4. Set the options in the command dialog and shortcut key.
  - **Number:** Specify the number of duplications.
  - **Twist:** Specify the angle of twist along the curve in degrees.
  - **Scale:** Specify the scaling factor along the curve.
  - **Alignment:**
      - **Parallel:** Duplicate the Curves in parallel, regardless of the path. 
      - **Normal:** Duplicate the Curves with angles aligned to the path.
   - <kbd>Shift</kbd>-Wheel up **Add copy:** Increases the number of duplications.
   - <kbd>Shift</kbd>-Wheel down **Subtract copy:** Reduces the number of duplications.
4. Confirm with OK in the command dialog or right-click.
