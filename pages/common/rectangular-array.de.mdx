# Rectangular Array

<Reference shortcut="" command="Rectangular Array" button="Im vorgeschlagenen Befehlsmenü"  buttonIcon='button-rectangular-array'/>

Duplizieren Sie Objekte in einem linearen Muster, indem Sie die Anzahl der Duplikate festlegen. Optional kann eine zweite Richtung definiert werden, um ein rechteckiges Array zu erstellen. Die Option zum Erstellen von Instanzen ist verfügbar.

<Callout>
Bilder und Meshes (.OBJ, .STL, .FBX usw.) können angeordnet werden, mit Steuerelementen für Duplizierungsrichtung, Anzahl der Kopien und Duplizierungsmodi. Das Erstellen von Instanzen wird jedoch nicht unterstützt.
</Callout>
## Grundlegende Nutzung

1. Wählen Sie das Objekt aus, das Sie duplizieren möchten.
2. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-rectangular-array" /> rechts von der Befehlsleiste.
3. Die Duplikationsvorschau wird angezeigt.
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Binden Sie an die X-, Y- oder Z-Achse](/tool/sketching-essentials#constrain-to-an-axis).
4. Klicken Sie auf die gewünschte Richtung.
5. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - **Modus:**
         - **Ausdehnung:** Geben Sie den Abstand der Ausdehnung an, bei dem alle Duplikate gleichmäßig angeordnet werden. 
         - **Abstand:** Geben Sie den Abstand für jedes Duplikat an.
   - <Kst>Befehlsdialog</Kst>
      - **Abstand 1:** Geben Sie den Abstand an.
      - **Anzahl:** Geben Sie die Anzahl der Duplikate an.
      - **Make instances:** Erstellen Sie Instanzen anstelle separater Kopien. (See [Create Instances](/solid/create-instance))
   - <Kst>Tastenkürzel</Kst>
   - <kbd>Shift + Mausrad nach oben</kbd> Kopie hinzufügen: Erhöht die Anzahl der Duplikate.
   - <kbd>Shift + Mausrad nach unten</kbd> Kopie reduzieren: Verringert die Anzahl der Duplikate.
6. Wenn Sie in eine andere Richtung duplizieren möchten, klicken Sie im Dialog auf **Select endpoint 2**.
   1. Klicken Sie mit der rechten Maustaste auf die gewünschte Duplikationsrichtung.
   2. Legen Sie die Optionen im Befehlsdialog fest.
      - <Kst>Befehlsdialog</Kst>
         - **Distance 2:** Geben Sie den Abstand an.
         - **Repeat:** Geben Sie die Anzahl der Wiederholungen an.
         - **Make instances:** Erstellen Sie Instanzen anstelle separater Kopien. (See [Create Instances](/solid/create-instance))
      - <Kst>Tastenkombinationen</Kst>
      - <kbd>Umschalt + Mausrad hoch</kbd> **Add copy:** Erhöht die Anzahl der Duplikate.
      - <kbd>Umschalt + Mausrad runter</kbd> **Subtract copy:** Verringert die Anzahl der Duplikate.
7. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
