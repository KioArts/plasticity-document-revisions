## Beschreibung

Dupliziere Objekte in einem linearen Muster, indem du die Anzahl der Duplikate angibst. Optional kannst du eine weitere Richtung definieren, um eine rechteckige Anordnung zu erstellen. Die Option, Instanzen zu erstellen, ist ebenfalls verfügbar.

<Callout type='info'>
**Hinweis:** Bilder und Meshes (.OBJ, .STL, .FBX usw.) können angeordnet werden, mit Steuerungen für Duplikationsrichtung, Kopienanzahl und Duplikationsmodi. Das Anordnen von Instanzobjekten wird ebenfalls unterstützt.
</Callout>

## Grundlegende Verwendung

1. Wähle das Objekt aus, das du duplizieren möchtest.
2. Klicke auf die Schaltfläche <ButtonIcon type="button-rectangular-array" /> rechts in der Befehlsleiste.
3. Die Duplikationsvorschau wird angezeigt.
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Achse:** [Auf die X-, Y- oder Z-Achse beschränken](/tool/sketching-essentials#constrain-to-an-axis).
4. Klicke auf die gewünschte Richtung.
5. Lege die Optionen im Befehlsdialog und in den Tastenkombinationen fest.
      - **Modus:**
         - **Bereich:** Gibt die Distanz an, über die alle Duplikate gleichmäßig angeordnet werden.
         - **Abstand:** Gibt den Abstand zwischen den einzelnen Duplikaten an.
   - <Kst>Befehlsdialog</Kst>
      - **Distanz 1:** Gibt die Distanz an.
      - **Anzahl:** Gibt die Anzahl der Duplikate an.
      - **Instanzen erstellen:** Erstellt Instanzen statt unabhängiger Kopien. (Siehe [Instanzen erstellen](/solid/create-instance))
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>Shift + Mausrad hoch</kbd> **Kopie hinzufügen:** Erhöht die Anzahl der Duplikate.
   - <kbd>Shift + Mausrad runter</kbd> **Kopie entfernen:** Verringert die Anzahl der Duplikate.
6. Wenn du in eine weitere Richtung duplizieren möchtest, klicke im Dialog auf **Endpunkt 2 auswählen**.
   1. Rechtsklicke auf die gewünschte Richtung für die Duplikation.
   2. Lege die Optionen im Befehlsdialog fest.
      - <Kst>Befehlsdialog</Kst>
         - **Distanz 2:** Gibt die Distanz an.
         - **Wiederholen:** Gibt die Anzahl der Wiederholungen an.
         - **Instanzen erstellen:** Erstellt Instanzen statt unabhängiger Kopien. (Siehe [Instanzen erstellen](/solid/create-instance))
      - <Kst>Tastenkombinationen</Kst>
      - <kbd>Shift + Mausrad hoch</kbd> **Kopie hinzufügen:** Erhöht die Anzahl der Duplikate.
      - <kbd>Shift + Mausrad runter</kbd> **Kopie entfernen:** Verringert die Anzahl der Duplikate.
      - <kbd>I</kbd> **Instanzen umschalten:** Schaltet um, ob die erzeugte Hälfte als Instanz oder als unabhängige Kopie erstellt wird.
7. Bestätige mit OK im Befehlsdialog oder per Rechtsklick.
