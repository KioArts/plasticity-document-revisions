# Rectangular Array

<Reference shortcut="" command="Rectangular Array" button="Rechts von der Befehlsleiste" />

Duplizieren Sie Objekte in einem linearen Muster, indem Sie die Anzahl der Duplikate angeben. Optional können Sie eine andere Richtung definieren, um ein rechteckiges Array zu erstellen.

1. Wählen Sie das Objekt aus, das Sie duplizieren möchten.
2. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-rectangular-array" /> rechts von der Befehlsleiste.
3. Die Duplikationsvorschau wird angezeigt.
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Binden Sie an die X-, Y- oder Z-Achse](./sketching-essentials#constrain-to-an-axis).
4. Klicken Sie auf die gewünschte Richtung.
5. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - **Distance 1:** Geben Sie den Abstand an.
   - **Number:** Geben Sie die Anzahl der Duplikationen an.
   - <kbd>Umschalt</kbd> + Mausrad hoch **Add copy:** Erhöht die Anzahl der Duplikate.
   - <kbd>Umschalt</kbd> + Mausrad runter **Subtract copy:** Verringert die Anzahl der Duplikate.
6. Wenn Sie in eine andere Richtung duplizieren möchten, klicken Sie im Dialog auf **Select endpoint 2**.
   1. Klicken Sie mit der rechten Maustaste auf die gewünschte Duplikationsrichtung.
   2. Legen Sie die Optionen im Befehlsdialog fest.
      - **Distance 2:** Geben Sie den Abstand an.
      - **Repeat:** Geben Sie die Anzahl der Wiederholungen an.
      - <kbd>Umschalt</kbd> + Mausrad hoch **Add copy:** Erhöht die Anzahl der Duplikate.
      - <kbd>Umschalt</kbd> + Mausrad runter **Subtract copy:** Verringert die Anzahl der Duplikate.
7. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
