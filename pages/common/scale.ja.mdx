# Scale（拡大縮小）

<Reference shortcut="S" command="Scale" button="コマンドバー左" />

カーブ、複数のポイント、複数のCV、ソリッド、シートを拡大縮小できます。

{/* 1. 拡大縮小したいオブジェクトを選択します。
2. ショートカット <kbd>S</kbd> を押すか、ボタンをクリックします。
3. 表示された拡大縮小ギズモを使用して大きさを変更します。 
   - <kbd>S</kbd> **Uniform**
      - X、Y、Z方向に均等に拡大縮小します。
   - <kbd>F</kbd> **Freestyle**
      - 最初のクリックで拡大縮小の始点を、次のクリックで終点を指定します。
    - その他のオプションは[共通オプション](common-options) を参照してください。
4. コマンドダイアログのOKまたは右クリックで確定します。
 */}
