# Radial Array

<Reference shortcut="" command="Radial Array" button="Right of Command Bar"  buttonIcon='button-radial-array'/>

Duplicate objects in a circular pattern by specifying the angle and number. Radial and instance duplication are also possible.

- Images and meshes (.OBJ, .STL, .FBX, etc.) can be arrayed, with controls for duplication angle and number of copies. However, instancing is not supported.

## Basic Usage

1. Select the object to duplicate.
2. Click the button <ButtonIcon type="button-radial-array" /> on the Right of Command Bar.
3. Click on the point you want to set as the center.
4. The duplication preview will be displayed.
5. Set the options in the Command Dialog and Keyboard Shortcuts.
   - <Kst>Command Dialog</Kst>
   - **Mode:**
         - **Total:** Specify the total distance at which all duplications will be evenly arranged at the given degree value. 
         - **Spacing:** Specify the spacing distance for each of the duplications.
   - **Degrees:** Specify the angle for duplication.
   - **Number 2:** Specify the number of duplications.
   - **Number 1:** Specify the number of repetitions in a radial pattern.
   - **Step 1:** Specify the distance between each repetition in a radial pattern.
   - **Make instances:** Create instances instead of independent copies. (See [Create Instances](/solid/create-instances))
   - <Kst>Keyboard Shortcuts</Kst>
   - <kbd>Shift + Wheel up</kbd> **Add copy:** Increases the number of duplications.
   - <kbd>Shift + Wheel down</kbd> **Subtract copy:** Reduces the number of duplications.   
6. Confirm with OK in the Command Dialog or right-click.
