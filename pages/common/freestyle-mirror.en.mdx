# Freestyle Mirror

<Reference shortcut="" command="Freestyle Mirror" button="" />

You can mirror based on a freely defined axis: Curves, Solids, Sheets, faces.

1. Select the objects you want to mirror.
2. Open the Command Palette by pressing the shortcut <kbd>F</kbd>, then type **Freestyle Mirror** and select it.
3. Specify the mirroring direction.
   - Specify the axis of the mirror with the first click as the starting point and the second click as the ending point.
4. Specify the options.
   - **Cut**
      - Divide the mirror target with the mirror plane before mirroring.
   - <kbd>Q</kbd> **Union(merge)**
      - Merge the two mirrored entities together.
4. Confirm with OK in the command dialog or right click.
