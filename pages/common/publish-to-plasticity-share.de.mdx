## Beschreibung

Lädt eine (.plasticity)-Datei in die Cloud hoch und erzeugt eine eindeutige private URL, die sich in einem neuen Browser-Tab öffnet. Der Link kann für eine Nur-Lesen-Ansicht mit anderen geteilt werden. Sobald der Link geteilt wird, ist er öffentlich zugänglich. Jedes hochgeladene Dokument unterstützt mehrere Revisionen; diese können auf der Seite [**Plasticity Share**](https://share.plasticity.xyz/me/) verwaltet und gelöscht werden.

<Callout>
1. Es wird nur die Mesh-Darstellung des Modells veröffentlicht – keine CAD-Geometrie oder Quelldateien werden einbezogen, und die Datei kann nicht heruntergeladen werden.  
2. Die Funktion ist standardmäßig deaktiviert. Aktiviere sie über **Einstellungen > Web Publishing**.  
3. „Publish to Web“ exportiert auch Materialinformationen aus dem Assets-Menü.  
4. Mesh-Daten sind **nur zur Ansicht**; direktes Modellieren oder Bearbeiten der veröffentlichten Meshes ist nicht möglich.  
5. Unabhängig vom Lizenztyp erhält jeder Benutzer **5 GB** Cloud-Speicher.  
</Callout>

## Grundlegende Verwendung

1. Wähle das Objekt aus, das du im Web veröffentlichen möchtest.  
2. Öffne die Befehlspalette, gib **Auf Plasticity Share veröffentlichen** ein und führe den Befehl aus.  
3. Ein Pop-up erscheint mit der Frage „Bist du sicher, dass du ins Web teilen möchtest?“ — klicke auf **Ja**.  
4. Ein neuer Browser-Tab öffnet sich und zeigt das veröffentlichte Objekt an.  
5. Eine eindeutige URL wird für das geteilte Objekt erzeugt.
