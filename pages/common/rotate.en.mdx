# Rotate

<Reference shortcut="R" command="Rotate" button="Left of Command Bar" />

You can rotate curves, multiple points, multiple  control vertices (CVs), solids, sheets, and faces.<br />
 To Rotate objects freely, see [**Freestyle Rotate**](#freestyle-rotate) described below.

1. Select the objects you want to rotate.
2. Press <kbd>R</kbd>.
3. Use the displayed Rotation Gizmo to rotate.
4. Set the options in the command dialog and shortcut key.
   - **Angle:** Specify the last specified angle using the rotation Gizmo as a numerical value
   - **Axis:** Specify the rotation axis using the X, Y, or Z vector.
   - <kbd>X</kbd> **X-axis:** Limit to the X-axis direction.
   - <kbd>Y</kbd> **Y-axis:** Limit to the Y-axis direction.
   - <kbd>Z</kbd> **Z-axis:** Limit to the Z-axis direction.
   - <kbd>G</kbd> **Screen space:** Limit to the visible plane.
   - <kbd>F</kbd> **Freestyle:** Click the starting point and endpoint of the rotation axis, and then click the reference point. Move the cursor to rotate the object, and then click to set the desired angle.
   - <kbd>V</kbd> **Pivot:** Click to specify the position of the Gizmo.
   - <kbd>W</kbd> **Worldspace:** Change the Gizmo's coordinate axis to world coordinates.
5. Confirm with OK in the command dialog or right-click.

## Freestyle Rotate [#freestyle-rotate]

This command is equivalent to the <kbd>F</kbd> option of the **Rotate** command.

1. Select the objects you want to rotate.
2. Use **Freestyle Rotate** in the Command Palette.
3. Click the starting point and endpoint of the rotation axis, and then click the reference point.
4. Move the cursor to rotate the object, and then click to set the desired angle. 
5. Set the options in the command dialog and shortcut key.
   - The option is the same as the **Rotate** command.
6. Confirm with OK in the command dialog or right-click.
