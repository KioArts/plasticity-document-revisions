# Scale

<Reference shortcut="S" command="Scale" button="Left of Command Bar" />

You can scale Curves, multiple points, multiple CVs, solids, sheets.<br />
To scale objects freely, see [**"Freestyle Scale"**](#freestyle-scale) described below.

1. Select the objects you want to scale.
2. Press <kbd>S</kbd>.
3. Use the displayed Scale Gizmo to resize.
4. Set the options in the command dialog and Shortcut Key.
   - **XYZ:** Specify the scale of the X, Y, and Z axes.
   - <kbd>X</kbd> **X axis:** Limit to the X-axis direction.
   - <kbd>Y</kbd> **Y axis:** Limit to the Y-axis direction.
   - <kbd>Z</kbd> **Z axis:** Limit to the Z-axis direction.
   - <kbd>Shift-X</kbd> **X plane:** Limit to the X-plane direction.
   - <kbd>Shift-Y</kbd> **Y plane:** Limit to the Y-plane direction.
   - <kbd>Shift-Z</kbd> **Z plane:** Limit to the Z-plane direction.
   - <kbd>S</kbd> **Uniform:** Scales uniformly in the X, Y, and Z directions.
   - <kbd>F</kbd> **Freestyle:** Specify the scaling axis by clicking on the starting and ending points, and then click with the desired scaling ratio.
   - <kbd>V</kbd> **Pivot:** Click to specify the position of the Gizmo.
   - <kbd>W</kbd> **Worldspace:** Change the gizmo's coordinate axis to world coordinates.
4. Confirm with OK in the command dialog or right click. 

## Freestyle Scale

This command is equivalent to the <kbd>F</kbd> option of the **"Scale"** command.

1. Select the objects you want to scale.
2. Use **"Freestyle Scale"** in the Command Palette.
3. Specify the scaling axis by clicking on the starting and ending points.
4. Move the cursor to scale the object, and then click the desired scaling ratio.
5. Set the options in the command dialog and Shortcut Key.
   - The option is the same as the **"Scale"** command.
6. Confirm with OK in the command dialog or right click.
