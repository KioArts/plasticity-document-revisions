# Rectangular Array（長方形配列で複製）

<Reference shortcut="" command="Rectangular Array" button="推奨コマンドメニューで" buttonIcon='button-rectangular-array' />

複製数を指定して、オブジェクトを直線状に複製できます。必要に応じて別の方向を指定することで、長方形配列を作成できます。インスタンスを作成するオプションも利用可能です。

- 画像およびメッシュ（.OBJ、.STL、.FBX など）は、複製方向、コピー数、複製モードを制御して並べることができます。ただし、インスタンス化はサポートされていません。

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. コマンドバー右のボタン <ButtonIcon type="button-rectangular-array" /> をクリックします。
3. 複製プレビューが表示されます。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
4. 複製したい方向をクリックします。
5. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - **モード:**
         - **範囲:** すべての複製が均等に配置される範囲距離を指定します。
         - **間隔:** 各複製の間隔距離を指定します。
   - <Kst>コマンドダイアログ</Kst>
      - **距離 1:** 距離を指定します。
      - **数:** 複製の数を指定します。
      - **インスタンスを作成:** 独立したコピーの代わりにインスタンスを作成します。（インスタンスの作成
 を参照） (See [Create Instances](/solid/create-instance))
   - <Kst>キーボードショートカット</Kst>
   - <kbd>Shift + ホイールアップ</kbd> コピー追加: 複製の数を増やします。
   - <kbd>Shift + ホイールダウン</kbd> コピー削除: 複製の数を減らします。
5. \<オプション\> もうひとつの方向にも配列で複製したいときは、ダイアログの**Select endpoint 2**をクリックします。
   1. 複製したい方向を右クリックします。
   2. コマンドダイアログでオプションを設定します。
      - <Kst>コマンドダイアログ</Kst>
         - **Distance 2:** 距離を指定します。
         - **Repeat:** 複製する回数を指定します。
         - **インスタンスを作成:** 独立したコピーの代わりにインスタンスを作成します。（インスタンスの作成
 を参照） (See [Create Instances](/solid/create-instance))
      - <Kst>キーボードショートカット</Kst>
      - <kbd>Shift</kbd> + ホイール上回転 **Add copy:** 複製の数を増やします。
      - <kbd>Shift</kbd> + ホイール下回転 **Subtract copy:** 複製の数を減らします。
6. コマンドダイアログのOKをクリックするか、右クリックで確定します。
