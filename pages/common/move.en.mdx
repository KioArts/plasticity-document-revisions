# Move

<Reference shortcut="G" command="Move" button="Left of Command Bar"  buttonIcon='button-move'/>

Move objects using Move Gizmo. Options for single-axis constraints or [**freestyle movement**](#freestyle-move) are available. The pivot is also adjustable.

## Basic Usage

1. Select the objects you want to move.w
2. Press <kbd>G</kbd>.
3. Use the displayed Move Gizmo to move.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - <Kst>Command Dialog</Kst>
      - **XYZ:** Specify the amount of movement of the X, Y, and Z axes.
      - **Orientation:** Set the orientation of the move gizmo.
         - <ButtonIcon type="button-normal"/> **Normal:** Sets the orientation to align with the object's surface normal.
         - <ButtonIcon type="button-pivot"/> **Pivot:** Sets the orientation to match the previously defined pivot orientation.
         - <ButtonIcon type="button-construction-plane"/> **Construction Plane:** Sets the orientation to align with the active construction plane.
         - <ButtonIcon type="button-world"/> **World:** Sets the orientation to align with the global (world) axis.
      - **Pivot:** Set the position of the pivot point.
         - <ButtonIcon type="button-bbox"/> **Bbox:** Positions the pivot at the center of the bounding box enclosing the selected objects.
         - <ButtonIcon type="button-median"/> **Median:** Positions the pivot at the average location of the pivots of the selected objects.
         - <ButtonIcon type="button-active"/> **Active:** Positions the pivot at the last selected (active) object's pivot.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](/tool/sketching-essentials#constrain-to-an-axis).
      - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z plane:** Constrain to the X, Y, or Z-plane.
      - <kbd>G</kbd> **Screen space:** Limit to the visible plane.
      - <kbd>F</kbd> **Freestyle:** Click to specify the starting point and endpoint, which will determine the direction and distance of movement.
      - <kbd>V</kbd> **Pivot:** Click to specify the position of the Gizmo.
      - <kbd>Alt + V</kbd> **Remove pivot:** Remove the configured pivots.
      - <kbd>W</kbd> **Orientation:** Change the Gizmo's coordinate axis to different orientations.
4. Confirm with OK in the Command Dialog or right-click.

<Callout type="info" emoji="ðŸ’¡">
**Tips**
- To move selected objects to the origin, press: <kbd>G</kbd> (Move) > <kbd>F</kbd> > right-click > right-click.
- To move an object to the origin based on a selected point, press: <kbd>G</kbd> (Move) > <kbd>F</kbd> > Click the object's point > right-click.
</Callout>

## Freestyle Move[#freestyle-move]

This command is equivalent to the <kbd>F</kbd> option of the **Move** command.

1. Select the objects you want to move.
2. Use the **Freestyle Move**  command in the Command Palette.
3. Click the starting point and endpoint to determine the direction and distance of movement.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - The option is the same as the **Move** command.
4. Confirm with OK in the Command Dialog or right-click.

# Move Control Point

<Reference shortcut="G" command="Move" button="Left of Command Bar"  buttonIcon='button-move'/>

Move Control Points using Move Gizmo. Options for single-axis constraints or [**freestyle movement**](#freestyle-move) and Proportional Editing are available. The orientation of the Move Gizmo and Pivot placement is also adjustable.

## Basic Usage

1. Select the objects you want to move.w
2. Press <kbd>G</kbd>.
3. Use the displayed Move Gizmo to move.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - <Kst>Command Dialog</Kst>
      - **XYZ:** Specify the amount of movement of the X, Y, and Z axes.
      - **Orientation:** Set the orientation of the move gizmo.
         - <ButtonIcon type="button-normal"/> **Normal:** Sets the orientation to align with the object's surface normal.
         - <ButtonIcon type="button-pivot"/> **Pivot:** Sets the orientation to match the previously defined pivot orientation.
         - <ButtonIcon type="button-construction-plane"/> **Construction Plane:** Sets the orientation to align with the active construction plane.
         - <ButtonIcon type="button-world"/> **World:** Sets the orientation to align with the global (world) axis.
      - **Pivot:** Set the position of the pivot point.
         - <ButtonIcon type="button-bbox"/> **Bbox:** Positions the pivot at the center of the bounding box enclosing the selected objects.
         - <ButtonIcon type="button-median"/> **Median:** Positions the pivot at the average location of the pivots of the selected objects.
         - <ButtonIcon type="button-active"/> **Active:** Positions the pivot at the last selected (active) object's pivot.
      - **Proportional:** A feature that provides multiple proportional editing options, influencing how CVs (control vertices) are affected when moved.
         - **None:** No proportional effect; only the selected CVs move.
         - **All:** Affects all CVs on the surface. Specify the falloff value of U and V for better proportional falloff control.
         - **Selected:** Applies proportional editing to the selected CVs, affecting their movement. Specify the falloff value of U and V for better proportional falloff control. 
      - **Falloff U:** Specify the falloff amount in V direction.
      - **Falloff V:** Specify the falloff amount in U direction.
      - **Mirror:** Apply a mirror effect relative to the active [**construction plane**](/plasticity-essentials/construction-plane). The mirroring axes adapt to either the global construction plane or a custom-defined construction plane. 
         - **None:** No mirror plane is specified. 
         - **X/Y/Z:** Mirror in either X, Y, and Z axes.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](/tool/sketching-essentials#constrain-to-an-axis).
      - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z plane:** Constrain to the X, Y, or Z-plane.
      - <kbd>G</kbd> **Screen space:** Limit to the visible plane.
      - <kbd>F</kbd> **Freestyle:** Click to specify the starting point and endpoint, which will determine the direction and distance of movement.
      - <kbd>V</kbd> **Pivot:** Click to specify the position of the Gizmo.
      - <kbd>Alt + V</kbd> **Remove pivot:** Remove the configured pivots.
      - <kbd>W</kbd> **Orientation:** Change the Gizmo's coordinate axis to different orientations.
4. Confirm with OK in the Command Dialog or right-click.

<Callout type="info" emoji="ðŸ’¡">
**Tips**
- To move selected objects to the origin, press: <kbd>G</kbd> (Move) > <kbd>F</kbd> > right-click > right-click.
- To move an object to the origin based on a selected point, press: <kbd>G</kbd> (Move) > <kbd>F</kbd> > Click the object's point > right-click.
