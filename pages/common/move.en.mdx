# Move

<Reference shortcut="G" command="Move" button="Left of Command Bar" />

You can move Curves, Points, CVs, Solids, Sheets, Faces, Edges.

1. Select the objects you want to move.
2. Press <kbd>G</kbd>.
3. Use the displayed Move Gizmo to move.
4. Set the options in the command dialog and Shortcut Key.
   - **XYZ:** Specify the amount of movement of the X, Y, and Z axes.


   - <kbd>F</kbd> **Freestyle**
      - Click to specify the starting point and the endpoint, determining the direction and distance of movement.
   - Other options: Refer to [Common options](common-options) .
4. Confirm with OK in the command dialog or right click.
