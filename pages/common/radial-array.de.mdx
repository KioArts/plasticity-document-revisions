# Radial Array

<Reference shortcut="" command="Radial Array" button="Rechts von der Befehlsleiste" buttonIcon='button-radial-array' />

Duplizieren Sie Objekte in einem kreisförmigen Muster, indem Sie Winkel und Anzahl festlegen. Radiale Duplizierung und Instanz-Duplizierung sind ebenfalls möglich.

- Bilder und Meshes (.OBJ, .STL, .FBX usw.) können angeordnet werden, mit Steuerelementen für den Duplizierungswinkel und die Anzahl der Duplikate. Die Instanzierung wird jedoch nicht unterstützt.

## Grundlegende Nutzung

1. Wählen Sie das Objekt zum Duplizieren aus.
2. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-radial-array" /> rechts von der Befehlsleiste.
3. Klicken Sie auf den Punkt, den Sie als Zentrum festlegen möchten.
4. Die Duplikationsvorschau wird angezeigt.
5. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - <Kst>Befehlsdialog</Kst>
   - **Modus:**
         - **Gesamt:** Geben Sie die Gesamtdistanz an, bei der alle Duplikate gleichmäßig auf dem angegebenen Gradwert verteilt werden.
         - **Abstand:** Geben Sie den Abstand für jedes der Duplikate an.
      - **Degrees:** Geben Sie den Winkel für die Duplizierung an.
      - **Number 2:** Specify the number of duplications.
      - **Number 1:** Specify the number of repetitions in a radial pattern.
      - **Step 1:** Specify the distance between each repetition in a radial pattern.
      - **Make instances:** Create instances instead of independent copies. (See [Create Instances](/solid/create-instance))
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Umschalt</kbd> + Mausrad hoch **Add copy:** Erhöht die Anzahl der Duplizierungen.
      - <kbd>Umschalt</kbd> + Mausrad runter **Subtract copy:** Verringert die Anzahl der Duplizierungen.   
6. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
