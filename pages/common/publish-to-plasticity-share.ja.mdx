## 説明

(.plasticity) ファイルをクラウドにアップロードし、新しいブラウザタブで開く一意のプライベート URL を生成します。リンクは閲覧専用として誰とでも共有できます。リンクを共有した時点で公開状態になります。アップロードされた各ドキュメントは複数のリビジョンをサポートしており、[**Plasticity Share**](https://share.plasticity.xyz/me/) ページで管理および削除できます。

<Callout>
1. 公開されるのはモデルのメッシュ表現のみで、CAD ジオメトリやソースファイルは含まれず、ファイルのダウンロードもできません。  
2. この機能はデフォルトで無効になっています。**設定 > Web Publishing** から有効化してください。  
3. Web への公開では、Assets メニューのマテリアル情報もエクスポートされます。  
4. メッシュデータは **閲覧専用** であり、公開されたメッシュを直接編集することはできません。  
5. ライセンス種別に関係なく、すべてのユーザーに **5 GB** のクラウドストレージが提供されます。  
</Callout>

## 基本的な使い方

1. Web に公開したいオブジェクトを選択します。  
2. コマンドパレットで **Plasticity Share へ公開** と入力して実行します。  
3. 「Web に共有してもよいですか？」というポップアップが表示されるので、**Yes** をクリックします。  
4. 新しいブラウザタブが開き、公開されたオブジェクトが表示されます。  
5. 共有されたオブジェクト用の一意の URL が生成されます。

## Web ビューポートインターフェース

<Center width="550px">
![Web Viewport Breakdown](/img/web-viewport-breakdown.png)
</Center>

| 番号 | 要素 | 説明 |
| :--: | ---- | ---- |
|  1   | すべてのドキュメント | クラウドサーバー上に公開されたすべてのドキュメントを表示します。 |
|  2   | サインアウト | 現在の Web セッションからサインアウトします。サインアウト時には「Not signed in. Share a document from Plasticity and refresh this tab to sign in.」と表示されます。 |
|  3   | メッシュデータ | Plasticity で選択したオブジェクトが可視化用にメッシュデータへ変換されたものです。 |

<Callout> 
Web ビューポートの操作は Plasticity のデフォルト操作と同じです：**中クリック**で回転、**右クリック**でパン。右側パネルでは [**シェーダーモード**](/plasticity-essentials/shader-mode)、[**レンダーモード**](/plasticity-essentials/render-mode)、[**カメラ設定**](/plasticity-essentials/camera-settings)、[**グリッドと単位**](/plasticity-essentials/grid-and-units) にアクセスできます。 
</Callout>
