# Move（移動する）

<Reference shortcut="G" command="Move" button="コマンドバー左" />

カーブ、ポイント、CV、ソリッド、シート、フェイス、エッジを移動できます。
また、始点と終点して移動できる[**Freestyle Move**](#freestyle-move)コマンドもあります。

1. 移動したいオブジェクトを選択します。
2. <kbd>G</kbd>を押します。
3. 表示された移動ギズモを使用して移動します。
4. コマンドダイアログとショートカットキーでオプションを設定します。
   - **XYZ:** X、Y、Z軸方向の移動量を指定します。
   - <kbd>X</kbd> **X axis:** X軸方向に限定します。
   - <kbd>Y</kbd> **Y axis:** Y軸方向に限定します。
   - <kbd>Z</kbd> **Z axis:** Z軸方向に限定します。
   - <kbd>Shift-X</kbd> **X plane:** X平面に限定します。
   - <kbd>Shift-Y</kbd> **Y plane:** Y軸平面に限定します。
   - <kbd>Shift-Z</kbd> **Z plane:** Z軸平面に限定します。
   - <kbd>G</kbd> **Screen space:** 見た目の平面に限定します。
   - <kbd>F</kbd> **Freestyle:** 始点と終点をクリックして、移動の方向と距離を指定します。
   - <kbd>V</kbd> **Pivot:** クリックしてギズモの位置を指定します。
   - <kbd>WV</kbd> **Worldspace:** 限定を解除します。
5. コマンドダイアログのOKまたは右クリックで確定します。

## Freestyle Move（自由に移動）

1. 移動したいオブジェクトを選択します。
2. コマンドパレットで**"Freestyle Move（自由に移動）"**を実行します。
3. 始点と終点をクリックして、移動の方向と距離を指定します。
4. コマンドダイアログとショートカットキーでオプションを設定します。
   - **XYZ:** X、Y、Z軸方向の移動量を指定します。
   - <kbd>X</kbd> **X axis:** X軸方向に限定します。
   - <kbd>Y</kbd> **Y axis:** Y軸方向に限定します。
   - <kbd>Z</kbd> **Z axis:** Z軸方向に限定します。
   - <kbd>Shift-X</kbd> **X plane:** X平面に限定します。
   - <kbd>Shift-Y</kbd> **Y plane:** Y軸平面に限定します。
   - <kbd>Shift-Z</kbd> **Z plane:** Z軸平面に限定します。
   - <kbd>G</kbd> **Screen space:** 見た目の平面に限定します。
   - <kbd>F</kbd> **Freestyle:** 始点と終点をクリックして、移動の方向と距離を指定します。
   - <kbd>V</kbd> **Pivot:** クリックしてギズモの位置を指定します。
   - <kbd>WV</kbd> **Worldspace:** 限定を解除します。
5. コマンドダイアログのOKまたは右クリックで確定します。

<Callout type="info" emoji="ℹ️">
   **"Freestyle Move"** コマンドは **"Move"** コマンドの<kbd>F</kbd>オプションと同じです。
</Callout>
