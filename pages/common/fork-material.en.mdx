# Fork Material

<Reference shortcut="Shift + M" command="Fork Material" button="" />

Allows you to duplicate and edit the materials for the selected objects.

## Basic Usage

1. Select the object you want to fork the material. 
2. Press <kbd>Shift + M</kbd>.
3. The material will be duplicated and editable.
