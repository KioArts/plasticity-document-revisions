# Freestyle Move

<Reference shortcut="" command="Freestyle Move" button="" />

You can move based on a freely defined axis: curves, points, CVs, solids, sheets, faces, edges.

1. Select the objects you want to move.
2. Open the command palette by pressing the shortcut <kbd>F</kbd>, then type **Freestyle Move** and select it.
3. Specify the move direction and distance.
   - Click to specify the starting point and the endpoint of movement.
   - Options: Refer to [Common options](common-options) .
4. Confirm with OK in the command dialog or right click.
