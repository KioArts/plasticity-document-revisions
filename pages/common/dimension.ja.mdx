## 説明

矩形カーブのサイズやシリンダーの直径など、寸法値を変更します。

## 基本的な使い方

1. 寸法を変更したいオブジェクトを選択します。
    | 寸法変更の対象                           | 選択モード |
    | ---------------------------------------- | ---------- |
    | カーブ矩形の2辺の長さ                     | エッジ     |
    | カーブの円または円弧の直径               | エッジ     |
    | シリンダーの直径                         | フェース   |
    | 球の直径                                 | ソリッド   |
    | 立方体の3辺の長さ                         | ソリッド   |
    | ソリッドのフィレットのサイズ             | フェース   |
    | ソリッドの平面間距離                     | フェース   |
2. <kbd>=</kbd> を押すか、コマンドパレットで **寸法** コマンドを使用します。
3. 寸法値が表示されます。
4. <kbd>Tab</kbd> を押して寸法入力モードに入ります。
5. 寸法値を入力します。
6. 複数の寸法がある場合、<kbd>Tab</kbd> を押して切り替えながら入力します。
7. <kbd>Enter</kbd> を押して確定します。
