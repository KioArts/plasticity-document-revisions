# Rotate（回転）

<Reference shortcut="R" command="Rotate" button="コマンドバー左" />

カーブ、複数のポイント、複数のCV、ソリッド、シート、フェイスを回転できます。<br />
自由な軸で回転するには、下記の [**Freestyle Rotate**](#freestyle-rotate) を参照してください。

1. 回転したいオブジェクトを選択します。
2. <kbd>R</kbd>を押します。
3. 表示された回転ギズモを使用して回転します。
4. コマンドダイアログとショートカットキーでオプションを設定します。
   - **Angle:** 回転ギズモで最後に指定した角度を数値で指定します。
   - **Axis:** 回転軸をX、Y、Zのベクトルで指定します。
   - <kbd>X</kbd> **X-axis:** X軸方向に限定します。
   - <kbd>Y</kbd> **Y-axis:** Y軸方向に限定します。
   - <kbd>Z</kbd> **Z-axis:** Z軸方向に限定します。
   - <kbd>G</kbd> **Screen space:** 見た目の平面に限定します。
   - <kbd>F</kbd> **Freestyle:** 回転軸の始点と終点をクリックし、参照点をクリックします。カーソルを動かして回転し、希望の角度でクリックします。
   - <kbd>V</kbd> **Pivot:** クリックしてギズモの位置を指定します。
   - <kbd>W</kbd> **Worldspace:** ギズモの座標軸をワールド座標にします。
5. コマンドダイアログのOKまたは右クリックで確定します。

## Freestyle Rotate（自由な軸で回転）[#freestyle-rotate]

このコマンドは **"Rotate"** コマンドの<kbd>F</kbd>オプションと同じです。

1. 回転したいオブジェクトを選択します。
2. コマンドパレットで **"Freestyle Rotate"** を実行します。
3. 回転軸の始点と終点をクリックし、参照点をクリックします。
4. カーソルを動かして回転し、希望の角度でクリックします。
5. コマンドダイアログとショートカットキーでオプションを設定します。
   - オプションは **"Rotate"** コマンドと同じです。
6. コマンドダイアログのOKまたは右クリックで確定します。
