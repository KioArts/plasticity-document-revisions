## 説明

複製数を指定して、オブジェクトをカーブに沿って複製します。ツイスト、スケール、アライメント、インスタンス作成などのオプションが利用できます。

<Callout type='info'>
**注意:** 画像およびメッシュファイル（.OBJ、.STL などの形式）はカーブに沿って配列でき、ツイスト、スケール、アライメント、位置分布などのパラメータを調整できます。インスタンスオブジェクトの配列もサポートされています。
</Callout>

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. コマンドパレットで **カーブ配列** コマンドを使用します。
3. パスとして使用するカーブを選択します。
4. コマンドダイアログおよびキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **数:** 複製数を指定します。
      - **ツイスト:** カーブに沿ったツイスト角度（度）を指定します。
      - **スケール:** カーブに沿ったスケール係数を指定します。
      - **アラインメント:**
         - **Normal（ノーマル）:** カーブのローカル法線フレームに合わせて向きを調整しながら複製します。単純なパスには適していますが、複雑な曲率ではねじれが発生する場合があります。
         - **Parallel（パラレル）:** カーブの回転を無視し、元の向きを維持したまま複製します。パスに沿って回転させたくない場合に最適です。
         - **Transport（トランスポート）:** カーブに沿って安定したフレームを滑らかに伝播させ、ねじれや反転を起こさずに複製します。Normal や Parallel が破綻しやすい複雑・有機的なパスに最適です。

      - **範囲:** 選択中の方式に応じて、複製範囲を調整する値を指定します。
      - **方式:** 複製範囲の制御方法を定義します。
         - **距離:** Plasticity の設定に基づき、距離値で複製範囲を調整します。
         - **比率:** `0.00`〜`1.00` の比率で複製範囲を調整します。（デフォルト: `1.00`）
      - **インスタンス作成:** 独立したコピーではなくインスタンスを作成します。（[インスタンス作成](/solid/create-instance) を参照）
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Shift + ホイール上</kbd> **コピー追加:** 複製数を増やします。
      - <kbd>Shift + ホイール下</kbd> **コピー削減:** 複製数を減らします。
      - <kbd>I</kbd> **インスタンス切り替え:** ミラーされた半分をインスタンスとして作成するか、独立コピーとして作成するかを切り替えます。
4. コマンドダイアログで OK を押すか、右クリックで確定します。
