# Curve Array（カーブに沿って複製する）

<Reference shortcut="" command="Curve Array" button="" />

コピー数を指定して、オブジェクトをカーブに沿って複製できます。ねじり（ツイスト）、スケール、整列、インスタンス作成のオプションが利用できます。

<Callout>
画像およびメッシュ（.OBJ、.STL、.FBX など）は、ねじり、スケール、整列、カーブ上での配置を制御して並べることができます。ただし、インスタンス化はサポートされていません。
</Callout>

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. コマンドパレットで**Curve Array**を実行します。
3. 沿わせたいカーブを選択します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **Number:** 複製する数を指定します。
      - **Twist:** カーブに沿って捻る角度を度数で指定します。
      - **Scale:** カーブに沿って拡大縮小倍率を指定します。
      - **Alignment**
         - **Parallel:** 軌道に関わらず、平行に複製します。 
         - **Normal:** 軌道に沿った角度で複製します。
      - **Extent:** 現在選択されている方法に応じて、複製の範囲を調整する値を指定します。
      - **Method:** 複製の範囲をどのように制御するかを定義します。
         - **Distance:** Plasticityの設定に基づき、メートル値で複製範囲を調整します。
         - **Ratio:** 0.00〜1.00の比率値で複製範囲を調整します。（初期値：1.00）
      - **Make instances:** Create instances instead of independent copies. (See [Create Instances](/solid/create-instance))
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Shift</kbd> + ホイール上回転 **Add copy:** 複製の数を増やします。
      - <kbd>Shift</kbd> + ホイール下回転 **Subtract copy:** 複製の数を減らします。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
