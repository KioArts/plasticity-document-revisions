# Curve Array（カーブに沿って複製する）

カーブ、ソリッド、シートをカーブに沿って複製できます。

<Reference shortcut="" command="Curve Array" button="" />

1. 移動したいオブジェクトを選択します。
2. ショートカット<kbd>F</kbd>でコマンドパレットを表示して、**Curve Array**と入力して選びます。
3. 沿わせたいカーブを選択します。
   - **Number**
      - 複製する数を指定します。<kbd>Shift</kbd>+マウスホイールの回転でも指定できます。
   - **Twist**
      - カーブに沿って捻る角度を度数で指定します。
   - **Scale**
      - カーブに沿って拡大縮小倍率を指定します。
   - **Alignment** 
      - **Parallel**
         - 軌道に関わらず、平行に複製します。
      - **Normal**
         - 軌道に沿った角度で複製します。
4. コマンドダイアログのOKまたは右クリックで確定します。
