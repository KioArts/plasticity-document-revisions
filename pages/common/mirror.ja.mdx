# Mirror

<Reference shortcut="Alt + X" command="Mirror" button="Left of Command Bar"  buttonIcon='button-mirror'/>

Mirror Gizmoを使用してオブジェクトをミラーリングします。  
ミラープレーンに沿ってオブジェクトを分割し、半分を統合します。  
[**フリースタイルミラーリング**](#freestyle-mirror)も可能です。

## 基本的な使い方

1. ミラーするオブジェクトを選択します。  
2. <kbd>Alt + X</kbd> を押すか、ボタンをクリックします。  
3. 表示されたMirror Gizmoを使用してミラーリングを行います。  
4. ミラーの方向とオプションを指定します。  
   - <Kst>コマンドダイアログ</Kst>  
      - **カット:** 有効にすると、ミラーリング前にミラープレーンで対象オブジェクトを分割します。  
      - **結合:** 有効にすると、ミラーリングされた2つのオブジェクトを結合します。  
   - <Kst>キーボードショートカット</Kst>  
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **位置 X/Y/Z:** X/Y/Z軸の正の方向にミラーリングします。  
      - <kbd>Shift + X</kbd>/<kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **負の X/Y/Z:** X/Y/Z軸の負の方向にミラーリングします。  
      - <kbd>F</kbd> **フリースタイル:** 開始点と終了点をクリックして、ミラープレーンを指定します。  
      - <kbd>Q</kbd> **結合 (マージ):** ミラーリングされた2つのオブジェクトを結合します。  
5. フラットまたはカーブしたサーフェスをクリックして、ミラープレーンを指定することもできます。  
   - **カーブしたサーフェスを選択した場合、ミラープレーンはそのサーフェスの法線に垂直で接触する平面になります。**  
6. コマンドダイアログでOKを押すか、右クリックで確定します。


## Freestyle Mirror[#freestyle-mirror]

このコマンドは、**Mirror** コマンドの <kbd>F</kbd> オプションと同等です。

1. ミラーリングするオブジェクトを選択します。  
2. コマンドパレットで **フリースタイルミラー** コマンドを使用します。  
3. 始点と終点をクリックしてミラープレーンを指定します。  
4. コマンドダイアログとキーボードショートカットでオプションを設定します。  
   - オプションは **Mirror** コマンドと同じです。  
5. コマンドダイアログのOKを押すか、右クリックで確定します。
