## 説明

ミラーギズモを使用してオブジェクトをミラーリングします。ミラー平面に沿って切断し、両半分を結合します。[**フリースタイルミラーリング**](#freestyle-mirror)も可能です。

## 基本的な使い方

1. ミラーリングしたいオブジェクトを選択します。
2. <kbd>Alt + X</kbd>を押すか、ボタンをクリックします。
3. 表示されたミラーギズモを使用してミラーリングします。
4. ミラーリングの方向とオプションを指定します。
   - <Kst>コマンドダイアログ</Kst>
      - **ミラー平面で切断:** 有効にすると、ミラーリング前にミラー対象がミラー平面で分割されます。
      - **両半分を結合:** 有効にすると、2つのミラーリングされたオブジェクトが結合されます。
      - **インスタンスを作成:** 有効にすると、ミラーリングされた半分は独立したコピーではなく、元のオブジェクトのインスタンスとして作成されます。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **位置 X/Y/Z** X/Y/Z軸の正方向にミラーリングします。
      - <kbd>Shift + X</kbd>/<kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **負の X/Y/Z** X/Y/Z軸の負方向にミラーリングします。
      - <kbd>I</kbd> **インスタンス作成を切り替え:** ミラーリングされた半分をインスタンスとして作成するか、独立したコピーとして作成するかを切り替えます。
      - <kbd>F</kbd> **フリースタイル:** 開始点と終了点をクリックしてミラー平面を指定します。
      - <kbd>Q</kbd> **結合（マージ）:** 2つのミラーリングされたオブジェクトを結合します。
5. 平面または曲面をクリックしてミラー平面を指定することもできます。
   - 曲面が選択された場合、ミラー平面はその曲面の法線に対して垂直で、曲面に接触する平面になります。
6. コマンドダイアログのOKまたは右クリックで確定します。

## フリースタイルミラー[#freestyle-mirror]

このコマンドは**ミラー**コマンドの<kbd>F</kbd>オプションと同等です。

1. ミラーリングしたいオブジェクトを選択します。
2. コマンドパレットで**フリースタイルミラー**コマンドを使用します。
3. 開始点と終了点をクリックしてミラー平面を指定します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - 上記の[ミラー](/common/mirror#basic-usage)と同じオプション。
   - <Kst>キーボードショートカット</Kst>
      - 上記の[ミラー](/common/mirror#basic-usage)と同じオプション。
5. コマンドダイアログのOKまたは右クリックで確定します。
