# Mirror（ミラー）

<Reference shortcut="Alt-X" command="Mirror" button="Left of Command Bar" />

カーブ、ソリッド、シート、フェイスをミラーできます。
{/* 
1. ミラーしたいオブジェクトを選択します。
2. ショートカットキー <kbd>Alt-X</kbd> を押すか、ボタンをクリックします。
3. 表示されたミラーギズモを使ってミラーします。
4. ミラーの方向とオプションを指定します。
   - <kbd>X</kbd> **Position X**
      - X軸の正の方向にミラーします。
   - <kbd>Y</kbd> **Position Y**
      - Y軸の正の方向にミラーします。
   - <kbd>Z</kbd> **Position Z**
      - Z軸の正の方向にミラーします。
   - <kbd>Shift-X</kbd> **Negative X**
      - X軸の負の方向にミラーします。
   - <kbd>Shift-Y</kbd> **Negative Y**
      - Y軸の負の方向にミラーします。
   - <kbd>Shift-Z</kbd> **Negative Z**
      - Z軸の負の方向にミラーします。
   - <kbd>F</kbd> **Freestyle**
      - 1回目のクリックを始点、2回目のクリックを終点としてミラーの軸を指定します。
   - <kbd>Q</kbd> **Union(merge)**
      - ミラーした2つを結合します。
   - **Cut**
      - ミラー対象をミラー平面で分割してからミラーします。
5. コマンドダイアログのOKまたは右クリックで確定します。 */}
