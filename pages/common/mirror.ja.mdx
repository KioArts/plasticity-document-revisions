# Mirror（ミラー）

<Reference shortcut="Alt + X" command="Mirror" button="コマンドバー右"  buttonIcon='button-mirror'/>

Mirror Gizmoを使用してオブジェクトをミラー反転できます。ミラープレーンに沿ってスライスし、半分を結合します。[フリースタイルミラーリング](#freestyle-mirror)も可能です。

## 基本的な使い方

1. 移動したいオブジェクトを選択します。  
2. <kbd>G</kbd> を押します。  
3. 表示される移動ギズモを使用して移動します。  
4. コマンドダイアログとキーボードショートカットでオプションを設定します。  
   - <Kst>コマンドダイアログ</Kst>  
      - **XYZ:** X、Y、Z軸の移動量を指定します。  
      - **オリエンテーション:** 移動ギズモの向きを設定します。  
         - <ButtonIcon type="button-normal"/> **ノーマル:** オブジェクトの表面法線に合わせて向きを設定します。  
         - <ButtonIcon type="button-pivot"/> **ピボット:** 以前に設定したピボットの向きに合わせます。  
         - <ButtonIcon type="button-construction-plane"/> **作業平面:** アクティブな作業平面に合わせて向きを設定します。  
         - <ButtonIcon type="button-world"/> **ワールド:** グローバル（ワールド）軸に合わせて向きを設定します。  
      - **ピボット:** ピボットポイントの位置を設定します。  
         - <ButtonIcon type="button-bbox"/> **バウンディングボックス:** 選択したオブジェクトを囲む境界ボックスの中心にピボットを配置します。  
         - <ButtonIcon type="button-median"/> **中央値:** 選択したオブジェクトのピボットの平均位置に配置します。  
         - <ButtonIcon type="button-active"/> **アクティブ:** 最後に選択した（アクティブな）オブジェクトのピボットに配置します。  
   - <Kst>キーボードショートカット</Kst>  
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z軸:** [X、Y、Z軸に制限](/tool/sketching-essentials#constrain-to-an-axis)。  
      - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X、Y、Z平面:** X、Y、Zのいずれかの平面に制限します。  
      - <kbd>G</kbd> **スクリーンスペース:** 表示される平面に制限します。  
      - <kbd>F</kbd> **フリースタイル:** クリックして開始点と終了点を指定し、移動の方向と距離を決定します。  
      - <kbd>V</kbd> **ピボット:** クリックしてギズモの位置を指定します。  
      - <kbd>Alt + V</kbd> **ピボット削除:** 設定されたピボットを削除します。  
      - <kbd>W</kbd> **オリエンテーション:** ギズモの座標軸を異なる向きに変更します。  
5. コマンドダイアログのOKを押すか、右クリックで確定します。

<Callout type="info" emoji="💡">
**ヒント**  
- 選択したオブジェクトを原点に移動するには、<kbd>G</kbd>（移動） > <kbd>F</kbd> > 右クリック > 右クリック を押します。  
- 選択したポイントを基準にオブジェクトを原点に移動するには、<kbd>G</kbd>（移動） > <kbd>F</kbd> > オブジェクトのポイントをクリック > 右クリック を押します。  
</Callout>

## Freestyle Mirror（自由な平面でミラー）[#freestyle-mirror]

このコマンドは **Mirror** コマンドの<kbd>F</kbd>オプションと同じです。

1. ミラーしたいオブジェクトを選択します。
2. コマンドパレットで **Freestyle Mirror** を実行します。
3. 始点と終点をクリックしてミラー平面を指定します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - オプションは **Mirror** コマンドと同じです。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。

# Move Control Point

<Reference shortcut="G" command="Move" button="Left of Command Bar"  buttonIcon='button-move'/>

移動ギズモを使用して、表面上に配置された制御点を移動します。単軸制約や[**フリースタイル移動**](#freestyle-move)、および比例編集のオプションが利用可能です。移動ギズモの方向とピボットの配置も調整できます。

## 基本的な使い方

1. 移動したいオブジェクトを選択します。
2. <kbd>G</kbd> を押します。
3. 表示された移動ギズモを使用してオブジェクトを移動します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **XYZ:** X、Y、Z軸の移動量を指定します。
      - **方向:** 移動ギズモの向きを設定します。
         - <ButtonIcon type="button-normal"/> **法線:** オブジェクトの表面法線に合わせて向きを設定します。
         - <ButtonIcon type="button-pivot"/> **ピボット:** 以前に定義されたピボットの向きに合わせます。
         - <ButtonIcon type="button-construction-plane"/> **作業平面:** アクティブな作業平面に合わせて向きを設定します。
         - <ButtonIcon type="button-world"/> **ワールド:** グローバル（ワールド）軸に合わせて向きを設定します。
      - **ピボット:** ピボットポイントの位置を設定します。
         - <ButtonIcon type="button-bbox"/> **バウンディングボックス:** 選択したオブジェクトを囲むバウンディングボックスの中心にピボットを配置します。
         - <ButtonIcon type="button-median"/> **中央値:** 選択したオブジェクトのピボットの平均位置にピボットを配置します。
         - <ButtonIcon type="button-active"/> **アクティブ:** 最後に選択した（アクティブな）オブジェクトのピボットに配置します。
      - **プロポーショナル:** 複数のプロポーショナル編集オプションがあり、CV（制御頂点）の移動に影響を与えます。
         - **なし:** プロポーショナル効果なし。選択したCVのみが移動します。
         - **すべて:** サーフェス上のすべてのCVとその動きを影響させます。UおよびVのフォールオフ値を指定し、より良い調整を行います。
         - **選択:** 選択したCVにのみプロポーショナル編集を適用し、影響を与えます。UおよびVのフォールオフ値を指定して調整します。 
      - **フォールオフ U:** U方向のフォールオフ量を指定します。
      - **フォールオフ V:** V方向のフォールオフ量を指定します。
      - **ミラー:** アクティブな[**作業平面**](/plasticity-essentials/construction-plane)に対してミラー効果を適用します。ミラー軸は、グローバル作業平面またはカスタム作業平面に適応します。
         - **なし:** ミラープレーンなし。
         - **X/Y/Z:** X、Y、Z軸に沿ってミラーします。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z軸:** [X、Y、Z軸に制約](/tool/sketching-essentials#constrain-to-an-axis)。
      - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X、Y、Z平面:** X、Y、Z平面に制約。
      - <kbd>G</kbd> **スクリーンスペース:** 可視平面内に制限。
      - <kbd>F</kbd> **フリースタイル:** クリックして開始点と終了点を指定し、移動の方向と距離を決定。
      - <kbd>V</kbd> **ピボット:** クリックしてギズモの位置を指定。
      - <kbd>Alt + V</kbd> **ピボット削除:** 設定されたピボットを削除。
      - <kbd>W</kbd> **方向:** ギズモの座標軸を異なる向きに変更。
4. コマンドダイアログでOKを押すか、右クリックで確定。

<VideoPlayer fileName="move-control-point"/>

<Callout type="info" emoji="🗒️">
**注意**  
- サーフェスCVをプロポーショナル編集する際、最後にアクティブだったCVの選択がピボット位置を決定します。これに注意してください。プロポーショナル編集の動作に影響を与えます。
</Callout>
