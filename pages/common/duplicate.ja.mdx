## 説明

オブジェクトを複製し、移動ギズモを使用して移動します。単一軸の制限またはフリースタイル移動が可能です。ピボットは調整可能です。

<Callout type="info">
**注意:** 複数のリージョンを複製する場合、結合された形状として複製されます。
</Callout>

## 基本的な使い方

1. 複製したいオブジェクトを選択します。
2. <kbd>Shift + D</kbd>を押します。
3. 表示された移動ギズモを使用して移動します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **XYZ:** X、Y、Z軸の移動量を指定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z軸:** [X、Y、またはZ軸に制限](/tool/sketching-essentials#constrain-to-an-axis)します。
      - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X、Y、Z平面:** X、Y、またはZ平面に制限します。
      - <kbd>G</kbd> **スクリーンスペース:** 見える平面に制限します。
      - <kbd>F</kbd> **フリースタイル:** クリックして開始点と終了点を指定し、移動の方向と距離を決定します。
      - <kbd>V</kbd> **ピボット:** クリックしてギズモの位置を指定します。
      - <kbd>W</kbd> **ワールド座標:** ギズモの座標軸をワールド座標に変更します。
4. コマンドダイアログのOKまたは右クリックで確定します。
