# Duplicate（複製）

<Reference shortcut="Shift-D" command="Duplicate" button="コマンドバー左" />

リージョン、カーブ、シート、フェイスを複製できます。

1. 複製したいオブジェクトを選択します。
2. ショートカット<kbd>Shift-D</kbd>を押すか、ボタンをクリックします。
3. 表示された移動ギズモを使って移動します。
   - <kbd>F</kbd> **Freestyle**
      - 1回目のクリックを始点、2回目のクリックを終点として、移動の方向と距離を指定します。
   - その他のオプションについては、[共通オプション](common-options)を参照してください。
4. コマンドダイアログのOKまたは右クリックで確定します。

<Callout type="info">
   - 複数のリージョンを複製すると、結合されたシェイプとして複製されます。
</Callout>
