# Complete Edge

<Reference shortcut="" command="Complete Edge" button="In Suggested Commands Menu" buttonIcon="button-complete-edge"/>

Extend the selected Edge until it intersects with another Edge.

## Basic Usage

1. Select Edges you want to extend.
2. Use the **"Complete Edge"** command in the Command Palette or click the <ButtonIcon type="button-complete-edge"/> button in the [Suggested Commands](/plasticity-essentials/command-bar#suggested-commands) menu.
3. The Edge is extended.
