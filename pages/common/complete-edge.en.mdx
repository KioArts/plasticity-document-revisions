## Description

Extend the selected Edge until it intersects with another Edge.

## Basic Usage

1. Select Edges you want to extend.
2. Use the **Complete Edge** command in the Command Palette or click the <ButtonIcon type="button-complete-edge"/> button in the Suggested Commands menu.
3. The Edge is extended.
