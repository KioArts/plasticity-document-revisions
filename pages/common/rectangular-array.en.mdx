# Rectangular Array

<Reference shortcut="" command="Rectangular Array" button="Right of Command Bar" />

You can duplicate Curves, Solids, Sheets in a rectangular array.

1. Select the object you want to duplicate.
2. Click on the button.
3. The duplication preview will be displayed.
4. Click on the desired direction.
   - **Distance 1**: Specify the distance.
   - **Number**: Specify the number of duplications. You can also use <kbd>Shift</kbd>+Mouse wheel to adjust.
5. \<Option\> If you want to duplicate in another direction, click on **Select endpoint 2** in the dialog.
    1. Click on the desired direction for duplication.
       - **Distance 2**: Specify the distance.
       - **Repeat**: Specify the number of repetitions. You can also use <kbd>Shift</kbd>+Mouse wheel to adjust.
6. Confirm with OK in the command dialog or right click.
