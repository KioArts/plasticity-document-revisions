### スケッチの基本

- 正確なカーブのスケッチ、点へのスナップ、一時的な作図補助線の使用の基本を学びます。完全な概要は[スケッチの基本](/tool/sketching-essentials)を参照してください。

### カーブとライン

- カーブの作成、接続、操作のためのツール。

| 名前 | 説明 |
| :--- | :--- |
| [ライン](/tool/line) | 2点間に直線を引く。 |
| [スプライン曲線](/tool/spline-curve) | カーブが通過する点を配置して滑らかな曲線を描く。 |
| [コントロールポイント曲線](/tool/control-point-curve) | カーブ上に乗らずに形状に影響するコントロールポイントを使用して曲線を描く。 |
| [カーブをブリッジ](/sketch/bridge-curve) | 2つの既存のカーブを滑らかなブリッジカーブで接続する。 |
| [点でカーブを分割](/sketch/split-segment) | 指定した点でカーブを2つの別々のカーブに分割する。 |
| [点にCVを挿入](/sketch/insert-knot) | より細かい形状制御のために既存のカーブにコントロール頂点を追加する。 |
| [トリム](/sketch/trim) | カーブを交差点でトリムしてジオメトリを整理する。 |

### 円ツール

- 異なる構築方法で円を作成するためのツール。

| 名前 | 説明 |
| :--- | :--- |
| [中心円](/tool/center-circle) | 中心点と半径を定義して円を描く。 |
| [2点円](/tool/two-point-circle) | 直径上の2点を定義して円を描く。 |
| [3点円](/tool/three-point-circle) | 3つの指定した点を通る円を描く。 |
| [接線円](/tool/tangent-circle) | 既存のカーブまたはエッジに接する円を描く。 |
| [楕円](/tool/ellipse) | 中心と2つの半径を定義して楕円を作成する。 |
