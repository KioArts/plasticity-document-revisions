# Solid Commands

| Command Name                                                 | Shortcut                     | Note                                                                                     |
| ------------------------------------------------------------ | ---------------------------- | ---------------------------------------------------------------------------------------- |
| [Boolean](/solid/boolean)                                     | <kbd>Q</kbd>                 | Boolean Solids and/or Sheets.                                                           |
| [Cut](/solid/cut)                                             | <kbd>C</kbd>                 | Cut Solid and Sheet.                                                                    |
| **Solid and Sheet Commands**                                 |                              |                                                                                          |
| [Alternative Duplicate](/solid/alternative-duplicate)         | <kbd>Alt+D</kbd>           | Alternative Duplicate.                                                                    |
| [Constrained Surface](/solid/constrained-surface)             |                              | Create a constrained surface.                                                           |
| [Deform](/solid/deform)                                      |                               | Unified command for Deform Curve, Deform Solid and Sheet.                               |
| [Deform Solid and Sheet](/solid/deform-solid-and-sheet)       |                               | Wrap objects on irregular and regular surfaces.                                        |
| [Extrude](/solid/extrude)                                     | <kbd>E</kbd>                 | Extrude Faces, Regions and Curves.                                                      |
| [Fillet](/solid/fillet)                                       | <kbd>B</kbd>                 | Unified command for Fillet Curve, Fillet Vertex, Fillet Shell.                          |
| [Fillet Shell](/solid/fillet-shell)                           | <kbd>B</kbd>                 | Fillet or chamfer the Edges.                                                            |
| [Hollow](/solid/hollow)                                       |                              | Hollow out a Solid.                                                                     |
| [Loft](/solid/loft)                                           | <kbd>L</kbd>                 | Loft Face, Edge and Curve.                                                              |
| [Measure Continuity](/tool/measure-continuity)               |                            | Measure continuity across surface edges.                                                  |
| [Toggle Surface Curvature](/solid/toggle-surface-curvature) |                            | Measure curvature across surface edges.                                                  |
| [Patch](/solid/patch)                                         |                              | Fill Sheet holes / Closed Curves to a Sheet.                                            |
| [Pipe](/solid/pipe)                                           | <kbd>P</kbd>                 | Create a Pipe from Edge and Curve.                                                      |
| [Create Instance](/solid/create-instance)                                     | | Unified command for Create Solid and Sheet Instance, Create Curve Instance.                                |
| [Create Solid and Sheet Instance](/solid/create-solid-and-sheet-instance)                                     | | Duplicate multiple uneditable objects.                                |
| [Realize Instances](/solid/realize-instances)                 | | Unified command for Realize Solid and Sheet Instances, Realize Curve Instances.                                |
| [Realize Solid and Sheet Instances](/solid/realize-solid-and-sheet-instances)     | | Convert Solid and Sheet instances to real solid geometry.                                |
| [Raise Degree](/solid/raise-degree)                           | <kbd>Shift+S</kbd>         | Unified command for Raise Curve Degree, Raise Surface Degree.                           |
| [Raise Surface Degree](/solid/raise-surface-degree)           | <kbd>Shift+S</kbd>         | Increase surface Control-Vertices.                                                      |
| [Remove Fillets from Shell](/solid/remove-fillets-from-shell) |                              | Remove Solid and Sheet fillets.                                                         |
| [Revolve](/solid/revolve)                                     |                              | Revolve Region or Curve.                                                                |
| [Sweep](/solid/sweep)                                         | <kbd>Shift+P</kbd>         | Sweep Region or Curve.                                                                  |
| [Square](/solid/square)                                       |                               | Create surfaces with explicit control of degrees and spans.                            |
| [Slide](/solid/slide)                                        |                              | Unified command for Slide Curve CV, Slide Surface CV.                                   |
| [Slide Surface CV](/solid/slide-surface-cv)                   | <kbd>Shift+G</kbd>           | Slide a CV on a Surface along the CV Hull.                                              |
| [Thicken](/solid/thicken)                                     |                              | Add thickness to a selected Sheet.                                                      |
| [Thicken Face](/solid/thicken-face)                           |                              | Add thickness to a selected Face.                                                       |
| [XNurbs](/solid/xnurbs)                                       |                               | Create high-quality and complex surfaces.                                              |
| **Sheet Commands**                                           |                              |                                                                                          |
| [Align](/solid/align)                                         |                              | Align two Sheets.                                                                    |
| [Bridge Surface](/solid/bridge-surface)                       |                              | Bridge two Sheets.                                                                      |
| [Extend](/solid/extend)                                       |                              | Unified command for Extend Curve, Extend Sheet.                                         |
| [Extend Sheet](/solid/extend-sheet)                           |                              | Extend one or more outside Edges of the Sheet.                                          |
| [Find Boundary Edges](/solid/find-boundary-edges)             |                              | Select Edges of a Sheet opening.                                                        |
| [Join](/solid/join)                                           | <kbd>J</kbd>                 | Unified command for Join Curves, Join Sheets.                                           |
| [Join Sheets](/solid/join-sheets)                             | <kbd>J</kbd>                 | Join Sheets with matching Edges.                                                        |
| [Reverse](/solid/reverse)                                     |                              | Unified command for Reverse Curve, Reverse Sheet.                                       |
| [Reverse Sheet](/solid/reverse-sheet)                         |                              | Reverse the direction of normal on a sheet.                                             |
| [Unjoin](/solid/unjoin)                                       | <kbd>Alt+J</kbd>           | Unified command for Unjoin Curves, Unjoin Faces, Unjoin Shells.                         |
| [Unjoin Faces](/solid/unjoin-faces)                           | <kbd>Alt+J</kbd>           | Separate selected Faces from Solid or Sheet.                                            |
| [Unjoin Shells](/solid/unjoin-shells)                         | <kbd>Alt+J</kbd>           | Separate all Faces from selected Solids or Sheets into individual Faces.                |
| [Untrim](/solid/untrim)                                       | <kbd>Alt+T</kbd>           | Create a Sheet based on the original shape.                                             |
| **Face Commands**                                            |                              |                                                                                          |
| [Delete Face](/solid/delete-face)                             |<kbd>Shift+Backspace</kbd> or <kbd>Shift+Delete</kbd> or <kbd>Shift+X</kbd> | Delete Face.                              |
| [Draft Face](/solid/draft-face)                               |                              | Tilt Face.                                                                              |
| [Match Face](/solid/match-face)                               |                              | Match Faces to another Face.                                                            |
| [Offset Face](/solid/offset-face)                             |                              | Offset the selected Faces.                                                               |
| [Rebuild](/solid/rebuild)                                     |                              | Unified command for Rebuild Curve, Rebuild Face.                                        |
| [Rebuild Face](/solid/rebuild-face)                           |                              | Rebuild Face.                                                                           |
| [Unwrap Face](/solid/unwrap-face)                             |                              | Unwrap Face.                                                                            |
| **Edge Commands**                                            |                              |                                                                                          |
| [Delete Redundant Topology](/solid/delete-redundant-topology) |                              | Remove excess Edges.                                                                      |
| [Complete Edge](/common/complete-edge)                        |                              | Extend Edges.                                                                             |
| [Imprint](/solid/imprint)                                     | <kbd>Shift+I</kbd>         | Unified command for Imprint Body Body, Imprint Curve Body.                                |
| [Imprint Body Body](/solid/imprint-body-body)                 | <kbd>Shift+I</kbd>         | Create Edges on the target Solid or Sheet where it intersects with a tool Solid or Sheet. |
| [Imprint Curve Body](/solid/imprint-curve-body)               | <kbd>Shift+I</kbd>         | Create Edges on the target Solid or Sheet where it projects onto a tool Curve.            |
| [Isoparam](/solid/isoparam)                                   | <kbd>Ctrl+R</kbd>          | Add isoparams.                                                                            |
| [Offset Curve](/solid/offset-curve)                           | <kbd>O</kbd>                 | Unified command for Offset Planar Curve, Offset Region, Offset Face Loop, Offset Edge.   |
| [Offset Edge](/solid/offset-edge)                             | <kbd>O</kbd>                 | Offset the selected Edges.                                                               |
| [Offset Face Loop](/solid/offset-face-loop)                   | <kbd>O</kbd>                 | Offset the outline edges of selected Faces.                                              |
