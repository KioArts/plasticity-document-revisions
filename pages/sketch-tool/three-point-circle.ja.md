# Three-Point Circle

- **円周上の3点で指定した円**を作成できます。
- ショートカット : なし
- ボタン : ツールバー

1. ツールバーのアイコンをクリックします。
2. 円周上の最初の位置でクリックします。
3. カーソルを移動して円周上の2点目でクリックします。
   - `K` : **Knife** : フェイスにエッジを追加します。
   - `X` : **X axis** : X軸方向に限定します。
   - `Y` : **Y axis** : Y軸方向に限定します。
   - `Z` : **Z axis** : Z軸方向に限定します。
4. カーソルを移動して円周上の3点目でクリックします。
5. 円が作成されます。
