# Center Rectangle（中心からの四角形）

<Reference shortcut='' command='Center Rectangle' button='ツールバー' />

中心点を指定した四角形を作成できます。

1. ツールバーのボタンをクリックします。
2. 四角形の中心点でクリックします。
   | ショートカットキー | オプション | 説明                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | 長さを数値で指定します。 |
   | `C`          | **Center / Corner**| **Corner Rectangle**に切り替えます。 |
   | `S`          | **Square** | 正方形に限定します。 |
3. 対角線上の点でクリックします。
4. コマンドダイアログの**OK**または右クリックで確定します。
5. 四角形が作成されます。
