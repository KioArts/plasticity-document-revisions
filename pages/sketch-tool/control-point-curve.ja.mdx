# Control Point Curve（制御点カーブ）

<Reference shortcut="" command="Control Point Curve" button="ツールバー" />

制御点カーブを作成できます。

1. ツールバーのボタンをクリックします。
2. 制御点カーブの最初の位置でクリックします。
3. 次の位置でクリックを繰り返して、制御点カーブを作成します。
   | ショートカットキー | オプション | 説明                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |**Control /  Through-point**| **Spline Curve**に切り替えます。 |
   | `Ctrl-Z`  | **Undo** | 直前のクリックを取り消します。 |
4. コマンドダイアログの**OK**または右クリックで確定します。
5. 制御点カーブが作成されます。
