# Three Point Rectangle（3点指定の四角形）

<Reference shortcut='' command='Three Point Rectangle' button='ツールバー' />

3点を指定した四角形を作成できます。

1. ツールバーのボタンをクリックします。
2. 四角形の一辺の始点でクリックします。
   - `Tab` 
      - 長さを数値で指定します。
   - そのほかのオプションは[スケッチツールの共通オプション](common-options)を参照してください。
3. 四角形の一辺の終点でクリックします。
4. 四角形の3点目でクリックします。
5. コマンドダイアログのOKまたは右クリックで確定します。
6. 四角形が作成されます。
