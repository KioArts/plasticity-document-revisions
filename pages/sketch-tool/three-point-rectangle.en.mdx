# Rectangle with Three Points

You can create a rectangle specified by three points.

<Reference shortcut='' command='Three Point Rectangle' button='Toolbar' />

1. Click the button on the toolbar.
2. Click at the start point of one side of the rectangle.
   | Shortcut Key | Option | Description                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | Specify length a numerical value. |
3. Click at the end point of the same side of the rectangle.
4. Click at the third point of the rectangle.
5. Confirm with **OK** in the command dialog or right click.
6. A rectangle is created.
