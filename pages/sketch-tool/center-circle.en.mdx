# Center Circle 

<Reference shortcut="" command="Center Circle" button="Toolbar" />

This function allows you to create a circle by specifying the center and diameter.

1. Click the button on the toolbar.
2. Click at the point for the circle's center.
3. Move the cursor and click on a point on the circumference.
   | Shortcut Key | Option | Description                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | Specify diameter with a numerical value. |
   | `V`          | **Vertical/ Horizontal**  | Changes the inclination with respect to the construction plane. |
4. Confirm with **OK** in the command dialog or right click.
5. The circle is created.
