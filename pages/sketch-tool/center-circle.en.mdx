# Circle from Center 

This function allows you to create a circle by specifying the center and diameter.

<KinouGaiyou shortcut="" command="Center Circle" button="Toolbar" />

1. Click on the icon in the toolbar.
2. Click at the position for the circle's center.
3. Move the cursor and click at the position for the diameter.
   - `Tab` : Specify the diameter with a numerical value.
   - `K` : **Knife** : Adds an edge to the face.
   - `V` : **Vertical/ Horizontal** : Changes the inclination with respect to the construction plane.
   - `X` : **X axis** : Limits to the X-axis direction.
   - `Y` : **Y axis** : Limits to the Y-axis direction.
   - `Z` : **Z axis** : Limits to the Z-axis direction.
4. Confirm by clicking on the **OK** button in the command dialog or by right-clicking.
5. The circle is created.
