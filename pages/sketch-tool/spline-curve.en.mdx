# Spline Curve

You can create a spline curve.

<KinouGaiyou shortcut="" command="Spline Curve" button="Toolbar" />

1. Click the icon on the toolbar.
2. Click at the initial position of the spline curve.
3. Repeat clicking at the next position to create the spline curve.
   - `K` : **Knife** : Adds an edge to the face with the created curve.
   - `Tab` : **Control / Through-point** : Switches to Control Point Curve.
   - `Cmd+Z` : **Undo** : Cancels the previous click.
   - `X` : **X axis** : Constrains to the X-axis.
   - `Y` : **Y axis** : Constrains to the Y-axis.
   - `Z` : **Z axis** : Constrains to the Z-axis.
4. Confirm with **OK** in the command dialog or right-click.
5. A spline curve is created.
