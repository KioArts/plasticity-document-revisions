# スプラインカーブ

スプラインカーブを作成できます。

<KinouGaiyou shortcut="" command="Spline Curve" button="ツールバー" />


1. ツールバーのボタンをクリックします。
2. スプラインカーブの最初の位置でクリックします。
3. 次の位置でクリックを繰り返して、スプラインカーブを作成します。
   | ショートカットキー | オプション | 説明                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |**Control /  Through-point**|**Control Point Curve**に切り替えます。 |
   | `Ctrl-Z`  | **Undo** | 直前のクリックを取り消します。 |
4. コマンドダイアログの**OK**または右クリックで確定します。
5. スプラインカーブが作成されます。
