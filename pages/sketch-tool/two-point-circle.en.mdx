# Two-Point Circle

<Reference shortcut='' command='Two-Point Circle' button='Toolbar' />

You can create a circle specified by two points of the diameter.

1. Click the button on the toolbar.
2. Click on the first point on the circumference.
3. Move the cursor and click on the second point on the circumference.
   | Shortcut Key | Option | Description                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `V`          | **Vertical/ Horizontal**  | Changes the inclination with respect to the construction plane. |
4. A circle is created.
