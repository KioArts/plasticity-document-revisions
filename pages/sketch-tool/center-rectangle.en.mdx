# Rectangle from Center

This function allows you to create a rectangle by specifying the center point.

<Reference shortcut='' command='Center Rectangle' button='Toolbar' />

1. Click the button on the toolbar.
2. Click at the point for the rectangle's center.
   | Shortcut Key | Option | Description                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | Specify length a numerical value. |
   | `C`          |**Center / Corner**| Switches to **Corner Rectangle**. |
   | `S`          |**Square**| Limits to a square. |
3. Click at a point on the diagonal.
4. Confirm with **OK** in the command dialog or right click.
5. The rectangle is created.
