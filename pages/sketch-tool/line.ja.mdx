# Line（線分）

<Reference shortcut="Shift-A" command="Line" button="ツールバー" />

一連の接続された線分を作成できます。

1. ツールバーのボタンをクリックします。
2. 線分の最初の位置でクリックします。
3. 次の位置でクリックを繰り返して、一連の線分を作成します。
   - `Tab` 
      - 長さと角度を数値で指定します。
   - `Ctrl-Z` **Undo**
      - 直前のクリックを取り消します。
4. コマンドダイアログの**OK**または右クリックで確定します。
5. 線分が作成されます。
