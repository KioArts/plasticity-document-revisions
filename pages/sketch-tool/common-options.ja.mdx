# スケッチツールの共通オプション

これらのオプションは、スケッチツール実行時にキーボードショートカットキーで指定できます。

| ショートカットキー | オプション | 説明                                   |
| ------------ | ------ | --------------------------------------------- |
| `K`            | **Knife**  | 作成されたカーブでフェイスにエッジを追加します。 |
| `X`            | **X axis** | X軸方向に限定します。|
| `Y`            | **Y axis** | Y軸方向に限定します。 |
| `Z`            | **Z axis** | Z軸方向に限定します。 |
