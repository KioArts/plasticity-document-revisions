# 2点指定の円

直径の2点で指定した円を作成できます。

<KinouGaiyou shortcut='' command='Two-Point Circle' button='ツールバー' />

1. ツールバーのアイコンをクリックします。
2. 直径の最初の位置でクリックします。
3. カーソルを移動して直径の2点目でクリックします。
   - `K` : **Knife** : フェイスにエッジを追加します。
   - `V` : **Vertical/ Horizontal** : 構築平面に対する傾きを変えます。
   - `X` : **X axis** : X軸方向に限定します。
   - `Y` : **Y axis** : Y軸方向に限定します。
   - `Z` : **Z axis** : Z軸方向に限定します。
4. 円が作成されます。
