# 2点指定の円

直径の2点で指定した円を作成できます。

<Reference shortcut='' command='Two-Point Circle' button='ツールバー' />

1. ツールバーのボタンをクリックします。
2. 直径の最初の位置でクリックします。
3. カーソルを移動して直径の2点目でクリックします。
  | ショートカットキー | オプション | 説明                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `V`         |**Vertical/ Horizontal**| 構築平面に対する傾きを変えます。                             |
4. 円が作成されます。
