# Rectangle from Diagonal

This function allows you to create a rectangle by specifying two points on the diagonal.

<Reference shortcut='' command='Corner Rectangle' button='Toolbar' />

1. Click the button on the toolbar.
2. Click at the start point on the diagonal of the rectangle.
   | Shortcut Key | Option | Description                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | Specify length with a numerical value. |
   | `C`          |**Center / Corner** | Switch to **Center Rectangle**.|
   | `S`          |**Square** | Limits to a square.|
3. Click at the end point on the diagonal.
4. Confirm with **OK** in the command dialog or right click.
5. The rectangle is created.
