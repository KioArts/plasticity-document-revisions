# Control Point Curve

<Reference shortcut="" command="Control Point Curve" button="Toolbar" />

This function allows you to create a control point curve.

1. Click the button on the toolbar.
2. Click at the start point of the control point curve.
3. Repeat clicking at the next point to create the control point curve.
   | Shortcut Key | Option | Description                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |**Control / Through-point**| Switches to **Control spline curve****. |
   | `Ctrl-Z`      |**Undo**| Cancels the last click. |
4. Confirm with **OK** in the command dialog or right click.
5. The control point curve is created.
