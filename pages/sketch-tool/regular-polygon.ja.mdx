# 多角形

多角形を作成できます。

<Reference shortcut='' command='Regular Polygon' button='ツールバー' />

1. ツールバーのボタンをクリックします。
2. 多角形の中心位置でクリックします。
3. カーソルを動かして外接円上の点でクリックします。
   | ショートカットキー | オプション | 説明                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | 外接円の直径を数値で指定します。 |
   | `Shift`+`ホイール上回転` |**Add a vertex**| 多角形の頂点を増やします。 |
   | `Shift`+`ホイール下回転` |**Subtract a vertex**| 多角形の頂点を減らします。 |
4. コマンドダイアログの**OK**または右クリックで確定します。
5. 多角形が作成されます。
