# Line Segment

You can create a series of connected line segments.

<KinouGaiyou shortcut="Shift+A" command="Line" button="Toolbar" />

1. Click the icon on the toolbar.
2. Click at the initial position of the line segment.
3. Repeat clicking at the next position to create a series of line segments.
   - `Tab` ï¼š Specify length and angle with numbers.
   - `K` : **Knife** : Add edges to the face with the created curve.
   - `Cmd+Z` : **Undo** : Cancel the last click.
   - `X` : **X axis** : Limit to the X-axis direction.
   - `Y` : **Y axis** : Limit to the Y-axis direction.
   - `Z` : **Z axis** : Limit to the Z-axis direction.
4. Confirm with **OK** in the command dialog or right click.
5. A line segment is created.
