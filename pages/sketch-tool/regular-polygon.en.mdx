# Polygon

This function allows you to create a regular polygon.

<Reference shortcut='' command='Regular Polygon' button='Toolbar' />

1. Click the button on the toolbar.
2. Click at the point for the polygon's center.
3. Move the cursor and click on a point on the circumscribed circle.
   | Shortcut Key | Option | Description                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | Specify diameter of the circumscribed circle with a numerical value. |
   | `Shift-Wheel up` | **Add a vertex**| Increases the vertices of the polygon. |
   | `Shift-Wheel down` | **Subtract a vertex**| Reduces the vertices of the polygon. |
4. Confirm with **OK** in the command dialog or right click.
5. The regular polygon is created.
