# 対角線からの四角形

対角線上の2点を指定した四角形を作成できます。

<KinouGaiyou shortcut='' command='Corner Rectangle' button='ツールバー' />

1. ツールバーのボタンをクリックします。
2. 四角形の対角線上の始点でクリックします。
   | ショートカットキー | オプション | 説明                                   |
   | ------------ | ------ | --------------------------------------------- |
   | `Tab`        |        | 長さを数値で指定します。 |
   | `C`          |**Center / Corner**| Center Rectangleに切り替えます。 |
   | `S`          |**Square**| 正方形に制限します。 |
3. 対角線上の終点でクリックします。
4. コマンドダイアログの**OK**または右クリックで確定します。
5. 四角形が作成されます。
