# ライセンスタイプと機能

Plasticity には **Indie** と **Studio** の 2 つのエディションがあります。以下は、機能、ライセンス条件、想定される利用ケースを比較した構造化された概要です。

## ライセンス比較

| 機能 | Indie エディション | Studio エディション |
| ---- | :----------------: | :------------------: |
| 価格 | $175 + 現地税 | $299 + 現地税 |
| 商用利用 | 可 | 可 |
| ノードロックライセンス | 最大 2 台まで | 最大 4 台まで |
| 3D モデリングツール | 基本ツールセット | 高度なフルツールセット： [Align](/solid/align), [PolySplines](/solid/polyspline), [xNURBS](/solid/xnurbs), [Square](/solid/square), [Rebuild Face (Explicit Control)](/solid/rebuild-face) |
| インポート / エクスポート対応 | [詳細を見る](/plasticity-essentials/import-export) | [詳細を見る](/plasticity-essentials/import-export) |
| ソフトウェアアップデート | 12 か月分を含む | 12 か月分を含む |
| ベータアクセス | なし | 12 か月間含まれる |
| Blender Bridge | 含まれる | 含まれる |

## 想定ユーザー

- **Indie エディション**  
  個人クリエイター、学生、ホビーユーザー、小規模チーム向け。基本機能を備えたモデリング環境を必要とするユーザーに適しています。

- **Studio エディション**  
  プロフェッショナルユーザーや制作チーム向け。高度なフルツールセット、拡張されたデバイスアクティベーション、パイプライン統合のためのベータアクセスが必要な環境に適しています。
