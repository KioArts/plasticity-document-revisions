# スケッチコマンド[#sketch-commands]

| コマンド名                                                         | ショートカット              | 機能                                                                                                    |
| ----------------------------------------------------------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| [Deform](/sketch/deform)                                           |                            | 曲線、ソリッド、およびシートを変形させるための統一コマンド。                                                  |
| [Deform Curve](/sketch/deform-sketch)                              |                            | 既存の曲線を別の表面に巻き付ける。                                                                                |
| [Extend](/sketch/extend)                                           |                            | 曲線を延長するための統一コマンド。                                                                                |
| [Extend Curve](/sketch/extend-curve)                               |                            | 曲線を延長する。                                                                                                 |
| [Fillet](/sketch/fillet)                                           | <kbd>B</kbd>               | フィレット曲線、フィレット頂点、フィレットシェルのための統一コマンド。                                             |
| [Fillet Curve](/sketch/fillet-curve)                               | <kbd>B</kbd>               | 2つの曲線セグメントの間にフィレットまたは面取りを行う。                                                          |
| [Fillet Vertex](/sketch/fillet-vertex)                             | <kbd>B</kbd>               | 接続された頂点のフィレットまたは面取りを行う。                                                                  |
| [Offset Curve](/sketch/offset-curve)                               | <kbd>O</kbd>               | オフセット平面曲線、オフセット領域、オフセットフェースループ、オフセットエッジのための統一コマンド。                      |
| [Offset Planar Curve](/sketch/offset-planar-curve)                 | <kbd>O</kbd>               | 平面曲線をオフセットする。                                                                                      |
| [Offset Region](/sketch/offset-region)                             | <kbd>O</kbd>               | 領域のアウトライン曲線をオフセットする。                                                                        |
| [Offset Vertex](/sketch/offset-vertex)                             | <kbd>O</kbd>               | 頂点の両側に沿って新しい頂点を挿入する。                                                                         |
| [Raise Degree](/sketch/raise-degree)                               | <kbd>Shift+S</kbd>       | 曲線の次数を上げるための統一コマンド。                                                                           |
| [Raise Curve Degree](/sketch/raise-curve-degree)                   | <kbd>Shift+S</kbd>       | 制御点と曲線の次数を増加させる。                                                                                |
| [Rebuild](/sketch/rebuild)                                         |                            | 曲線とフェースを再構築するための統一コマンド。                                                                   |
| [Rebuild Curve](/sketch/rebuild-curve)                             |                            | 制御点の位置を最適化して再構築する。                                                                             |
| [Reverse](/sketch/reverse)                                         |                            | 曲線とシートを反転させるための統一コマンド。                                                                     |
| [Reverse Curve](/sketch/reverse-curve)                             |                            | 曲線の方向を逆にする。                                                                                         |
| [Subdivide Curve](/sketch/subdivide-curve)                         | <kbd>Alt+S</kbd>         | 曲線の制御点とスパンを増やす。                                                                                  |
| [Slide](/sketch/slide)                                             |                            | Slide Curve CV、Slide Surface CVの統一コマンド。                                                 |
| [Slide Curve CV](/sketch/slide-curve-cv)                           | <kbd>Shift+G</kbd>         | CVをサーフェス上でCVハルに沿ってスライドさせます。                                                                  |
| [Text](/sketch/text)                                               |                            | テキストから曲線を作成する。                                                                                    |
| [Toggle Points](/sketch/toggle-points)                             |                            | 制御点を隠すまたは表示する。                                                                                     |
| **Project Commands**                                               |                           |                                                                                                            |
| [Alternative Duplicate](/sketch/alternative-duplicate)             | <kbd>Alt+D</kbd>       | 現在の構造平面に曲線を投影する。                                                                               |
| [Create Outline](/sketch/create-outline)                           |                          | オブジェクトのアウトラインを投影して3D曲線を作成する。                                                         |
| [Duplicate Curve and Project](/sketch/duplicate-curve-and-project) |                          | 曲線を複製して投影する。                                                                                        |
| [Project](/sketch/project)                                         | <kbd>I</kbd>             | ボディボディをプロジェクト、カーブボディをプロジェクト、カーブカーブをプロジェクトするための統一コマンド。                |
| [Project Body Body](/sketch/project-body-body)                     | <kbd>I</kbd>             | 2つのソリッドまたはシート間の交差線を表す新しい曲線を作成する。                                                |
| [Project Curve Body](/sketch/project-curve-body)                   | <kbd>I</kbd>             | ソリッドまたはシートに曲線を投影して新しい曲線を作成する。                                                     |
| [Project Curve Curve](/sketch/project-curve-curve)                 | <kbd>I</kbd>             | 2つの独立した曲線から生成された面の交点を投影して新しい曲線を作成する。                                       |
| [Project Outline](/sketch/project-outline)                         | <kbd>Alt+D</kbd>       | ソリッドまたはシートのアウトラインを投影する。                                                                  |
| **Join and Split Commands**                                        |                            |                                                                                                            |
| [Bridge](/sketch/bridge)                                           |                          | ブリッジ曲線、ブリッジ頂点、ブリッジエッジのための統一コマンド。                                               |
| [Bridge Curve](/sketch/bridge-curve)                               |                          | 2つの曲線をつなぐ。                                                                                             |
| [Bridge Edge](/sketch/bridge-edge)                                 |                          | 2つのエッジをつなぐ。                                                                                          |
| [Bridge Vertex](/sketch/bridge-vertex)                             |                          | 2つの頂点をつなぐ。                                                                                            |
| [Cut Curve](/sketch/cut-curve)                                     |                          | 曲線を切り取る。                                                                                               |
| [Join](/sketch/join)                                               | <kbd>J</kbd>             | 曲線を結合し、シートを結合するための統一コマンド。                                                               |
| [Join Curves](/sketch/join-curves)                                 | <kbd>J</kbd>             | 接続ポイントで曲線を結合する。                                                                                 |
| [Split Segment](/sketch/split-segment)                             |                          | 曲線を2つのセグメントに分割する。                                                                              |
| [Trim](/sketch/trim)                                               | <kbd>T</kbd>             | 曲線をトリムする。                                                                                             |
| [Unjoin](/sketch/unjoin)                                           | <kbd>Alt+J</kbd>       | 曲線、面、シェルを分離するための統一コマンド。                                                                   |
| [Unjoin Curves](/sketch/unjoin-curve)                              | <kbd>Alt+J</kbd>       | 結合された曲線を分離する。                                                                                     |
| [Insert CV At Point](/sketch/insert-knot)                          |                        | スプライン曲線に制御点を挿入します。                                                                            |
| **Vertex Commands**                                                |                          |                                                                                                            |
| [Convert Vertex](/sketch/convert-vertex)                           |                          | 頂点を制御点に変換する。                                                                                       |
| [Delete Redundant Topology](/sketch/delete-redundant-topology)     |                          | 余分なポイントを削除する。                                                                                     |
