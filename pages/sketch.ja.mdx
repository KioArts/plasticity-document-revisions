### ブリッジと接続

- カーブ、エッジ、頂点を接続するためのツール。

| 名前 | 説明 |
| :--- | :--- |
| [Bridge](/sketch/bridge) | 2 つの開いたエッジまたはカーブを滑らかに接続します。 |
| [Bridge Curve](/sketch/bridge-curve) | 2 つのカーブを新しい接続カーブでつなぎます。 |
| [Bridge Edge](/sketch/bridge-edge) | サーフェス上の 2 つの開いたエッジを接続します。 |
| [Bridge Vertex](/sketch/bridge-vertex) | 2 つの頂点を新しいカーブで接続します。 |

### カットとトリム

- カーブを切断、トリム、分割するためのツール。

| 名前 | 説明 |
| :--- | :--- |
| [Cut Curve](/sketch/cut-curve) | 指定した点または交点でカーブを切断します。 |
| [Trim](/sketch/trim) | カーブ同士の交点でトリムし、ジオメトリを整理します。 |
| [Split Segment](/sketch/split-segment) | 指定した点でカーブのセグメントを分割します。 |

### 変形とスライド

- カーブやコントロールポイントを変形・スライドするためのツール。

| 名前 | 説明 |
| :--- | :--- |
| [Deform](/sketch/deform) | ケージまたはコントロール構造を使ってジオメトリを変形します。 |
| [Deform Curve](/sketch/deform-curve) | カーブの形状をインタラクティブに変形します。 |

### アライン

| 名前 | 説明 |
| :--- | :--- |
| [Align](/sketch/align) | 選択したジオメトリを基準点に整列します。 |
| [Align Vertex](/sketch/align-vertex) | 頂点を別の頂点、点、または参照ジオメトリに整列します。 |
