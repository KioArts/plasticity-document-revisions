## スケッチのコマンド - リージョン・カーブ・ポイント・CV[#sketch-commands]
| コマンド名                                                        | ショートカット     | 機能                                   |
| ----------------------------------------------------------------- | ------------------ | -------------------------------------- |
| [Fillet](sketch/fillet)                                           | <kbd>B</kbd>       | カーブをフィレットまたは面取りする     |
| [Offset Curve](sketch/offset-curve)                               | <kbd>O</kbd>       | カーブをオフセットする                 |
| [Subdivide Curve](sketch/subdivide-curve)                         | <kbd>Shift-S</kbd> | カーブのCVを増やす                     |
| [Extend](sketch/extend)                                           |                    | カーブを延長する                       |
| [Rebuild 🚧](sketch/rebuild)                                     |                    | カーブを再構築する                     |
| [Text](sketch/text)                                           |                    | テキストを作成する                     |
| **投影のコマンド**                                                |                    |                                        |
| [Project](sketch/project)                                         | <kbd>I</kbd>       | カーブをソリッドまたはシートに投影する |
| [Duplicate Curve and Project](sketch/duplicate-curve-and-project) |                    | カーブを複製して投影する               |
| **結合・分割のコマンド**                                          |                    |                                        |
| [Join](sketch/join)                                               | <kbd>J</kbd>       | 一致した点でカーブをつなぐ             |
| [Bridge](sketch/bridge)                                       |                    | カーブをつなぐ                         |
| [Trim](sketch/trim)                                               | <kbd>T</kbd>       | カーブを切り捨てる                     |
| [Cut Curve 🚧](sketch/cut-curve)                                 |                    | カーブを切り離す                       |
| [Split Segment](sketch/split-segment)                         |                    | カーブをふたつのセグメントに分割する   |
| **ポイントのコマンド**                                            |                    |                                        |
| [Delete Redundant Topology 🚧](sketch/delete-redundant-topology) |                    | 余剰なポイントを削除する               |
| [Convert Vertex 🚧](sketch/convert-vertex)                       |                    | ポイントをCVに変換する
