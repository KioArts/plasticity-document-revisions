### Skizziergrundlagen

- Lernen Sie die Grundlagen des präzisen Skizzierens von Kurven, des Einrastens an Punkten und der Verwendung temporärer Konstruktionshilfslinien. Siehe [Skizzier-Grundlagen](/tool/sketching-essentials) für eine vollständige Übersicht.

### Kurven und Linien

- Werkzeuge zum Erstellen, Verbinden und Bearbeiten von Kurven.

| Name | Beschreibung |
| :--- | :--- |
| [Linie](/tool/line) | Geraden Linien zwischen zwei Punkten zeichnen. |
| [Spline-Kurve](/tool/spline-curve) | Eine glatte Kurve durch das Platzieren von Punkten zeichnen, durch die die Kurve verläuft. |
| [Kontrollpunkt-Kurve](/tool/control-point-curve) | Eine Kurve mit Kontrollpunkten zeichnen, die ihre Form beeinflussen, ohne auf ihr zu liegen. |
| [Kurvenbrücke](/sketch/bridge-curve) | Zwei bestehende Kurven mit einer glatten Überbrückungskurve verbinden. |
| [Kurve am Punkt teilen](/sketch/split-segment) | Eine Kurve an einem bestimmten Punkt in zwei separate Kurven teilen. |
| [CV am Punkt einfügen](/sketch/insert-knot) | Einen Kontrollpunkt zu einer bestehenden Kurve hinzufügen für feinere Formkontrolle. |
| [Trimmen](/sketch/trim) | Kurven an ihren Schnittpunkten trimmen, um die Geometrie zu bereinigen. |
