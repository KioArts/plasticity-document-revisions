# 選択モード

選択したいオブジェクトに応じて選択モードを切り替えます。
選択モードの切り替えは、画面左上のボタンか、キーボードショートカットでできます。

![Selection mode buttons](/img/selection-mode-buttons.png)

| 選択モード                                     | ショートカット       | ボタン                                |
| ---------------------------------------------- | -------------------- | ------------------------------------- |
| コントロールポイント選択モード                 | <kbd>1</kbd>         | (1)                                   |
| エッジ選択モード                               | <kbd>2</kbd>         | (2)                                   |
| フェイス選択モード                             | <kbd>3</kbd>         | (3)                                   |
| ボディ選択モード                               | <kbd>4</kbd>         | (4)                                   |
| 全種類選択可能なモード                         | <kbd>Tab</kbd>       | (5)                                   |
| コントロールポイント選択モードのみを切り替える | <kbd>Shift + 1</kbd> | <kbd>Shift</kbd>を押しながらボタン (1) |
| エッジ選択モードのみを切り替える               | <kbd>Shift + 2</kbd> | <kbd>Shift</kbd>を押しながらボタン (2) |
| フェイス選択モードのみを切り替える             | <kbd>Shift + 3</kbd> | <kbd>Shift</kbd>を押しながらボタン (3) |
| ボディ選択モードのみを切り替える               | <kbd>Shift + 4</kbd> | <kbd>Shift</kbd>を押しながらボタン (4) |

## オブジェクトを選択する

### マウスで選択する

- クリック: 選択できます。
- 右から左に向かってドラッグして選択： 緑色の選択範囲に触れたオブジェクトが選択されます。
- 左から右に向かってドラッグして選択： 紫色の選択範囲に完全に含まれるオブジェクトのみが選択されます。

![Mouse Drag selection](/img/mouse-drag-selection.gif)

### キーボードショートカットで選択する

| 操作                                             | ショートカット                   | コマンド名            |
| ------------------------------------------------ | -------------------------------- | --------------------- |
| すべてのオブジェクトを選択する                   | <kbd>A</kbd>                     | **Select all**        |
| すべてのカーブを選択する                         |                                  | **Select all Curves** |
| すべてのオブジェクトの選択を解除する             | <kbd>Esc</kbd>                   | **Deselect all**      |
| 選択を反転する                                   | <kbd>Alt + A</kbd>               | **Invert Selection**  |
| オブジェクトを選択不可にする                     |                                  | **Lock Selected**     |
| 隣接するオブジェクトを選択に追加する             | <kbd>Ctrl + Shift + =</kbd>      | **Select Adjacent**   |
| エッジをループ選択する                           | <kbd>Alt</kbd> + クリック        |                       |
| エッジをリング選択する                           | <kbd>Alt + Ctrl</kbd> + クリック |                       |
| 同一平面上のすべてのカーブ、リージョンを選択する | <kbd>Alt</kbd>クリック           |                       |
| すべての平行なフェイスを選択する                 | <kbd>Alt + Ctrl</kbd> + クリック |                       |

| 操作                                                       | ショートカット      | コマンド名                       |
| ---------------------------------------------------------- | ------------------- | -------------------------------- |
| 選択しているオブジェクトに含まれるエッジのみを選択する     | <kbd>Ctrl + 2</kbd> | **Selection: convert to edges**  |
| 選択しているオブジェクトに含まれるフェイスのみを選択する   | <kbd>Ctrl + 3</kbd> | **Selection: convert to faces**  |
| 選択しているオブジェクトを含んでいるソリッドのみを選択する | <kbd>Ctrl + 4</kbd> | **Selection: convert to solids** |
