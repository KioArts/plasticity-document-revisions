# アウトライナー

## オブジェクトの整理

- 作成されたオブジェクトは、ソリッド、シート、カーブに分けられます。
- ドラッグ＆ドロップで並び順を変更できます。
![Outliner 1](/img/outliner-01.png)

## グループ

- アイコンをクリックするか、<kbd>Ctrl + G</kbd> を押すとグループを作成できます。
- 選択していたオブジェクトは作成したグループに移動します。
- グループから外すには、オブジェクトを選択して<kbd>Alt + G</kbd>を押します。
- すべてのグループを折りたたむことができます。 
![Outliner 2](/img/outliner-02.png)

## ステイタスの変更

- オブジェクトやグループのステイタスを変更することができます。
    - 一時的に非表示にしたり、再び表示したりすることができます。
    - 一時的に無効にしたり、再び有効にしたりできます。
    - ロックまたはアンロックできます。
![Outliner 3](/img/outliner-03.png)

### アウトライナーでの操作

- <ButtonIcon type="button-outliner-hide" double="true" />: 表示・非表示を切り替えます。
- <ButtonIcon type="button-outliner-disable" double="true" /> : 使用可否を切り替えます。
- <ButtonIcon type="button-outliner-disable-selection" double="true"  /> : ロックを切り替えます。

### ショートカットキーでの操作

| 操作                                 | ショートカットキーでの操作     | コマンド名      |
| ------------------------------------ | ------------------------------ | --------------- |
| 選択しているオブジェクトを隠す       | <kbd>H</kbd>                   | Hide selected   |
| 選択されていないオブジェクトを隠す   | <kbd>Shift + H</kbd>           | Hide unselected |
| すべての表示する                     | <kbd>Alt + H</kbd>             | Unhide all      |
| 非表示を反転する                     | <kbd>Ctrl + H</kbd>            | Invert Hidden   |
| 同じ平面のすべてのカーブを隠す       | Select Regions -> <kbd>H</kbd> |                 |
| 選択しているオブジェクトをロックする |                                | Lock selected   |


## マテリアルカラー[#material-color]

- オブジェクトのマテリアル色がドットで表示されます。
- ドットをドラッグ＆ドロップすることで、色をコピーすることができます。
- マテリアル色はグループにも設定できます。同様の方法で色をコピーすることができます。
- マテリアルの詳細は、 [**Set Material**](../common/set-material)、[**Remove Material**](../common/remove-material)を参照してください。
