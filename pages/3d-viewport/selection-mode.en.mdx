# Selection Mode

Switch the selection mode according to the object you want to select.
To switch the selection mode, use the buttons in the upper left corner of the screen or keyboard shortcuts.

![Selection mode buttons](/img/selection-mode-buttons.png)

| Selection mode                           | Shortcut             | Button                     |
| ---------------------------------------- | -------------------- | -------------------------- |
| Control point selection mode             | <kbd>1</kbd>         | (1)                        |
| Edge selection mode                      | <kbd>2</kbd>         | (2)                        |
| Face selection mode                      | <kbd>3</kbd>         | (3)                        |
| Body selection mode                     | <kbd>4</kbd>         | (4)                        |
| All selectable mode                      | <kbd>Tab</kbd>       | (5)                        |
| Toggle Control Point selection mode | <kbd>Shift + 1</kbd> | <kbd>Shift</kbd> + click (1) |
| Toggle Edge selection mode          | <kbd>Shift + 2</kbd> | <kbd>Shift</kbd> + click (2) |
| Toggle Face selection mode          | <kbd>Shift + 3</kbd> | <kbd>Shift</kbd> + click (3) |
| Toggle Body selection mode         | <kbd>Shift + 4</kbd> | <kbd>Shift</kbd> + click (4) |

## Selecting Objects

### Selecting with the mouse

- **Click:** Select the object.
- **Drag from right to left:** Select objects that touch the green selection.
- **Drag from left to right:** Select objects completely contained in the purple selection.

![Mouse Drag selection](/img/mouse-drag-selection.gif)
