# 一般機能

| 機能                     | ショートカット                                |
| ------------------------ | --------------------------------------------- |
| コピー                   | <kbd>Ctrl + C</kbd>                             |
| 貼り付け                 | <kbd>Ctrl + visible</kbd>                       |
| 削除                     | <kbd>X</kbd> or <kbd>Delete</kbd>             |
| フェイスを削除           | <kbd>Shift + X</kbd> or <kbd>Shift + Delete</kbd> |
| 取り消し                 | <kbd>Ctrl + Z</kbd>                             |
| やり直し                 | <kbd>Ctrl + Shift + Z</kbd>                       |
| 最後のコマンドを繰り返す | <kbd>Shift + R</kbd>                            |
| [設定](./preferences)    | <kbd>Ctrl + , (Comma)</kbd>                     |
| 終了                     | <kbd>Ctrl + Q</kbd>                             |

## オブジェクトの表示、非表示

オブジェクトを一時的に見えなくしたり、再度表示したりできます。

| 操作                                     | ショートカット                   | コマンド名          |
| ---------------------------------------- | -------------------------------- | ------------------- |
| 選択オブジェクトを非表示にする           | <kbd>H</kbd>                     | **Hide selected**   |
| 非選択オブジェクトを非表示にする         | <kbd>Shift + H</kbd>               | **Hide unselected** |
| 非表示を解除する                         | <kbd>Alt + H</kbd>                 | **Unhide all**      |
| 非表示を反転する                         | <kbd>Ctrl + H</kbd>                | **Invert Hidden**   |
| 同一平面上のすべてのカーブを非表示にする | リージョンを選択して<kbd>H</kbd> |                     |

## オブジェクトのグループ化

オブジェクトをグループ化できます。

| 操作                               | ショートカット    | コマンド名           |
| ---------------------------------- | ----------------- | -------------------- |
| 選択オブジェクトをグループ化する   | <kbd>Ctrl + G</kbd> | **Group Selected**   |
| 選択オブジェクトをグループから外す | <kbd>Alt + G</kbd>  | **Ungroup Selected** |
