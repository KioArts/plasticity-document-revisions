# Object Manupulation

## Object Selection

### Selection Modes

Switch the selection mode according to the object you want to select.
To switch the selection mode, use the buttons in the upper left corner of the screen or keyboard shortcuts.

![Selection mode buttons](/img/selection-mode-buttons.png)

| Selection mode                           | Shortcut             | Button                     |
| ---------------------------------------- | -------------------- | -------------------------- |
| Control point selection mode             | <kbd>1</kbd>         | (1)                        |
| Edge selection mode                      | <kbd>2</kbd>         | (2)                        |
| Face selection mode                      | <kbd>3</kbd>         | (3)                        |
| Solid selection mode                     | <kbd>4</kbd>         | (4)                        |
| All selectable mode                      | <kbd>Tab</kbd>       | (5)                        |
| Toggle only control point selection mode | <kbd>Shift - 1</kbd> | <kbd>Shift</kbd>-click (1) |
| Toggle only edge selection mode          | <kbd>Shift - 2</kbd> | <kbd>Shift</kbd>-click (2) |
| Toggle only face selection mode          | <kbd>Shift - 3</kbd> | <kbd>Shift</kbd>-click (3) |
| Toggle only solid selection mode         | <kbd>Shift - 4</kbd> | <kbd>Shift</kbd>-click (4) |

### Selecting Objects

#### Select with mouse

- Click: You can select an object.
- Dragging from right to left: Objects touching the green selection will be selected.
- Drag to select from left to right: Only objects that are completely contained in the purple selection will be selected.

![Mouse Drag selection](/img/mouse-drag-selection.gif)

## Show and hide objects

Objects can be temporarily hidden or made visible again.

| Operation                         | Shortcut                      | Command className   |
| --------------------------------- | ----------------------------- | ------------------- |
| Hide selected objects             | <kbd>H</kbd>                  | **Hide selected**   |
| Hide unselected objects           | <kbd>Shift-H</kbd>            | **Hide unselected** |
| Unhide                            | <kbd>Alt-H</kbd>              | **Unhide all**      |
| Invert Hidden                     | <kbd>Ctrl-H</kbd>             | **Invert Hidden**   |
| Hide all curves on the same plane | Select regions - <kbd>H</kbd> |                     |

## Grouping objects

You can group objects.

| Operation                            | Shortcut          | Command className    |
| ------------------------------------ | ----------------- | -------------------- |
| Group selected objects               | <kbd>Ctrl-G</kbd> | **Group Selected**   |
| Remove selected objects from a group | <kbd>Alt-G</kbd>  | **Ungroup Selected** |
