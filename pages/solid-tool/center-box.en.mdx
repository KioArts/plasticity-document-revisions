# Center Box

<Reference shortcut='' command='Center Box' button='Toolbar' />

Allows you to create a rectangular box from a specified center point.

1. Click the button on the toolbar.

2. Click at the center point of the base surface.
   | Shortcut Key | Option     | Description                             |
   | ------------ | ---------- | --------------------------------------- |
   | `C`          | **Center/Corner** | Switches to **corner box**.|
   | `S`          | **Square** | Limits to a square. |
3. Click at the end point of the perimeter.
4. Move the cursor and click at the desired height.
   | Shortcut Key | Option     | Description                             |
   | ------------ | ---------- | --------------------------------------- |
   | `H`          | **Height** | Moves the cursor to specify the height. |
5. Click **OK** in the command dialog or right-click to confirm.
6. A rectangular box is created.
