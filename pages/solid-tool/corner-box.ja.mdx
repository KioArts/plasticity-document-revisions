# 対角線からの直方体

対角線上の2点を指定した四角形から直方体を作成できます。

<Reference shortcut='Shift-B' command='Corner Box' button='ツールバー' />

1. ツールバーのボタンをクリックします。
2. 底面の対角線上の始点でクリックします。
   | ショートカットキー| オプション | 説明 |
   | ------------ | ---------- | --------------------------------------- |
   | `C`          | **Center/Corner** | **Center box**に切り替えます。|
   | `S`          | **Square** | 正方形に限定します。 |
3. 対角線上の終点でクリックします。
4. カーソルを動かして目的の高さでクリックします。
   | ショートカットキー| オプション | 説明 |
   | ------------ | ---------- | --------------------------------------- |
   | `H`          | **Height** | カーソルを動かして高さを指定します。 |
5. コマンドダイアログの**OK**または右クリックで確定します。
6. 直方体が作成されます。
