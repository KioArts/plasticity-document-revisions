# Corner Box（対角線からの直方体）

<Reference shortcut='Shift-B' command='Corner Box' button='ツールバー' />

対角線上の2点を指定した四角形から直方体を作成できます。

1. ツールバーのボタンをクリックします。
2. 底面の対角線上の始点でクリックします。
   - `C` **Center/Corner**
      - Center Boxに切り替えます。
   - `S` **Square**
      - 正方形に限定します。
3. 対角線上の終点でクリックします。
4. カーソルを動かして目的の高さでクリックします。
   - `H` **Height**
      - カーソルを動かして高さを指定します。
   - そのほかのオプションは[ソリッド編集のブーリアンオプション](boolean-options)を参照してください。
5. コマンドダイアログのOKまたは右クリックで確定します。
6. 直方体が作成されます。
