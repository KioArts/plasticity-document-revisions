# Sphere（球）

<Reference shortcut='' command='Sphere' button='ツールバー' />

中心と直径を指定した球を作成できます


1. ツールバーのボタンをクリックします。
2. 球の中心でクリックします。
3. カーソルを動かして目的の直径でクリックします。
   - そのほかのオプションは[ソリッドツールの共通オプション](common-options)と[ソリッド編集のブーリアンオプション](boolean-options)を参照してください。
4. コマンドダイアログのOKまたは右クリックで確定します。
5. 球が作成されます。
