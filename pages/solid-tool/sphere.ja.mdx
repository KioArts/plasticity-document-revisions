# 球

中心と直径を指定した球を作成できます

<KinouGaiyou shortcut='' command='Sphere' button='ツールバー' />

1. ツールバーのボタンをクリックします。
   - **Select target bodies**: ブーリアンの対象（ソリッド、シート）を選択します。
   - **Radius**: 直径を指定します。
2. 球の中心でクリックします。
   - `Shift`+`Q`: **Slice**: ブーリアン対象との交点にソリッドを作成し、余りのソリッドも残します。
   - `Q`: **Union**: ブーリアン対象と結合します。
   - `W`: **Difference**: ブーリアン対象から切り取ります。
   - `Shift`+`E`: **Intersect**: ブーリアン対象との交点にソリッドを作成します。
3. カーソルを動かして目的の直径でクリックします。
4. 黄色いドットをドラッグして直径の変更ができます。
5. コマンドダイアログの**OK**または右クリックで確定します。
6. 球が作成されます。
