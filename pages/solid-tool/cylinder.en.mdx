# Cylinder

Allows you to create a cylinder from a circle specified by its center and diameter.

<KinouGaiyou shortcut='Shift-C' command='Cylinder' button='Toolbar' />

1. Click the button on the toolbar.
2. Click at the point for the circle's center on the base surface.
3. Move the cursor and click on a point on the circumference.
4. Move the cursor and click at the desired height.
   | Shortcut Key | Option | Description        |
   | ------------ | ------ | ------------------ |
   | `F`          |**Radius**| Move the cursor to specify the radius. |
   | `D`          |**Height**| Move the cursor to specify the height. |
5. Click **OK** in the command dialog or right-click to confirm.
6. A cylinder is created.
