# インスタンス作成

<Reference shortcut="" command="インスタンス作成" button="コマンドパレット" buttonIcon="button-instance" />

インスタンスコマンドは選択されたオブジェクトに応じて実行されます：
- **カーブインスタンス作成:** カーブが選択されている場合は、[**カーブインスタンス作成**](/sketch/create-curve-instance) を参照してください。  
- **ソリッド・シートインスタンス作成:** ソリッドまたはシートが選択されている場合は、[**ソリッド・シートインスタンス作成**](/solid/create-solid-and-sheet-instance) を参照してください。
