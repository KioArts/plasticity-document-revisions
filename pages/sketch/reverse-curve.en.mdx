# Reverse Curve

<Reference shortcut="" command="Reverse" button=""/>

Inverse the direction of a curve. 

## Basic Usage

1. Select the curve to reverse its direction.
2. Type **Reverse** in the Command Palette.
3. Now the direction of the curve has been reversed.
