# Project Curve Body [#project-curve-body]

<Reference shortcut="I" command="Project" button="Im Menü Vorgeschlagene Befehle"  buttonIcon='button-project'/>

Erstellen Sie eine neue Kurve, indem Sie eine Kurve auf ein Solid oder Sheet projizieren. Wählen Sie die Projektionsmethode normal oder Richtungsvektor aus. Bidirektional ist ebenfalls verfügbar.

## Grundlegende Nutzung

1. Wählen Sie das Solid oder Sheet aus, auf das projiziert werden soll.
2. Drücken Sie <kbd>I</kbd> oder klicken Sie auf die Schaltfläche.
3. Wählen Sie die Kurve aus, die projiziert werden soll.
4. Legen Sie die Optionen im Befehlsdialog fest:
   - <Kst>Befehlsdialog</Kst>
      - **Method:**
            - **Normal:** Projiziert in Richtung der positiven Z-Achse basierend auf der Konstruktionsebene.
            - **Vector:** Projiziert in eine lineare Richtung. **Bidirectional** und **Hide occlusion** werden aktiviert.  
            ![Project Curve Body Method Demonstration](/img/project-curve-body-method.png)
      - **Bidirectional:** Wenn aktiviert, erfolgt die Projektion auch in die negative Richtung der angegebenen Achse.
            ![Project Curve Body Bidirectional Demonstration](/img/project-curve-body-bidirectional.png)
      - **Hide occluded:** Wenn aktiviert, erfolgt die Projektion nicht auf versteckten Faces in Richtung der positiven Achse.
            ![Project Curve Body Hide Occluded Demonstration](/img/project-curve-body-hide-occluded.png)
5. Rechtsklick oder Klick auf OK im Befehlsdialog zur Bestätigung.
6. Die projizierte Kurve wird erstellt.
