# Rebuild Curve [#rebuild-curve]

<Reference shortcut="" command="Rebuild Curve" button="Rechts von der Befehlsleiste" buttonIcon='button-rebuild' />

Durch Optimieren der Positionen der Control Vertices(CVs) neu aufgebaut, ohne die Form des Curve zu ändern.

## Grundlegende Nutzung

1. Wählen Sie die Kurve aus, die Sie neu erstellen möchten.
2. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-rebuild" />.
3. Stellen Sie die Optionen im Befehlsdialog ein.
  - <Kst>Befehlsdialog</Kst>
    - **Methode:**
      - **Anpassen:** Neu erstellen innerhalb der Toleranzen.
        - **Toleranz:** Geben Sie die Toleranzen an.
        - **Ecken behalten:** Wenn aktiviert, werden die Ecken der Kurve nicht verändert.
      - **Punkte** Neu erstellen innerhalb der angegebenen Anzahl von CVs.
        - **Punkte:** Geben Sie die Anzahl der CVs an.
      - **Explizite Kontrolle:** Neu erstellen innerhalb der angegebenen Anzahl von Graden und Spannen.
        - **Grad & Spannen:** Geben Sie den Wert für Grad und Anzahl der Spannen an.
4. Bestätigen Sie mit OK im Befehlsdialog oder durch Rechtsklick.
