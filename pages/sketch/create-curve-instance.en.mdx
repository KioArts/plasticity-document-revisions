# Create Curve Instance

<Reference shortcut="" command="Create Instance" button="In Command Palette"  buttonIcon="button-instance"/>

Create reusable duplicates of curves. An **instance**, similarly to instancing solid and sheet objects, stores transformation data (position, pivot point, rotation, and scale), color data and references the original curve. Any modification made to the master curve is instantly reflected in all of its instances.

Because curve instances are read-only copies of their source, they can't be edited independently. They also hide the CV (control-vertex) layout, which makes them visually distinct from the original curve.

<Callout>
1. Use the **Create Instance** command on curves when you need many identical details such as consistent, symmetrical sketches for building a vehicle's construction framework or related products.
2. Instances work with array commands such as [**Curve Array**](/common/curve-array), [**Radial Array**](/common/radial-array), and [**Rectangular Array**](/common/rectangular-array).  
3. Instances can also be used with the [**Mirror**](/common/mirror) command, enabling live symmetry where changes to the master geometry updates all mirrored instances automatically.
4. Cannot use [**Place**](/common/place) and [**Copy with Placement**](/common/copy-with-placement) commands on created instances for now.
</Callout>

## Basic Usage

1. Select the Curve you want to use as the parent object..
2. Open the Command Palette, type ** Create Instance**, and execute the command.
3. Move the newly created instance to the desired location.
4. Confirm with OK in the Command Dialog or right-click.

## Instance Groups

The process of instancing groups is similar to instancing Curves.

When a group is selected and it contains primary sketches, an instance of the group will be created and placed outside of the group folder under its own dedicated category in the Outliner.

<Center width="600px">
![!Instancing a Group Part 1](/img/instance-group-curve.png)
</Center>

| Number | Description |
| :----: | ----------- |
| 1 | Main group with components inside. |
| 2 | Parent objects that will be instanced. |
| 3 | Instance of main group allocated outside of the main group under the category **"Instances"**. |

### Basic Usage

1. Select a Group in the Outliner to instantiate.
2. Search for *Create Instance* in the Command Palette and execute it.
3. Move the instanced group.
4. Confirm with OK in the Command Dialog or right-click.
