# Offset Planar Curve

<Reference shortcut="O" command="Offset Planar Curve" button="" />

Verschieben Sie eine planare Kurve, indem Sie die Entfernung angeben, um neue Kurven zu erstellen. Die Option für symmetrisches Versetzen von beiden Seiten ist verfügbar.

1. Wählen Sie die Kurven aus, die Sie versetzen möchten.
2. Drücken Sie <kbd>O</kbd>.
3. Bewegen Sie den gelben Pfeil, um die Entfernung festzulegen.
4. Legen Sie die Optionen im Befehlsdialog und den Tastenkombinationen fest.
   - **Distance:** Geben Sie die Entfernung an.
   - <kbd>V</kbd> **Gap fill:** 
        - **Round:** Rundet konvexe Ecken. 
        - **Linear:** Verbindet konvexe Ecken mit geraden Linien. 
        - **Natural:** Natürlich
   - <kbd>Tab</kbd> **Lock distances:** Verschiebung mit gleichem Abstand auf beiden Seiten.
   - <kbd>F</kbd> **Freestyle:** Es funktioniert ähnlich wie der unten beschriebene [**Freestyle Offset Planar Curve**](#freestyle-offset-planar-curve).
   - <kbd>D</kbd> **Distance:** Geben Sie die Entfernung mit der Mausbewegung an.
5. Bestätigen Sie mit OK im Befehlsdialog oder Rechtsklick.

## Freestyle Offset Planar Curve

Verschieben Sie eine planare Kurve, indem Sie die Entfernung mit der Cursorbewegung angeben, um neue Kurven zu erstellen. Die Option für symmetrisches Versetzen von beiden Seiten ist verfügbar.

Dieser Befehl kann nicht nur über die **Freestyle**-Option im **Offset Curve**-Befehl ausgeführt werden, sondern auch durch Verwendung des **Freestyle Offset Planar Curve** im Befehlspalette.

1. Wählen Sie die Kurven aus, die Sie versetzen möchten.
2. Drücken Sie <kbd>O</kbd>.
3. Drücken Sie <kbd>F</kbd>, um den Freestyle-Modus zu betreten.
4. Bewegen Sie den Cursor zum Versatzpunkt und klicken Sie darauf.
   - Wenn Rasterschnappfunktionen aktiviert sind, rastet der Versatzpunkt am Raster ein.
   - Wenn Objektschnappfunktionen aktiviert sind, rastet der Versatzpunkt am Objekt ein.
   - **Distance:** Geben Sie die Entfernung an.
   - **Gap fill:** 
        - **Round:** Rundet konvexe Ecken. 
        - **Linear:** Verbindet konvexe Ecken mit geraden Linien. 
        - **Natural:** Natürlich    
   - <kbd>Tab</kbd> **Lock distances:** Verschiebung mit gleichem Abstand auf beiden Seiten.
5. Bestätigen Sie mit OK im Befehlsdialog oder Rechtsklick.
