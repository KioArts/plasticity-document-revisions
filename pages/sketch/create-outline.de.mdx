## Beschreibung

Projiziert den Umriss von Volumenkörpern oder Flächen entlang der Normalenrichtung auf die aktuelle Konstruktionsebene und erzeugt dabei eine Kurve. Während der Befehl **[Umriss projizieren](/sketch/project-outline)** eine projizierte Kurve in 2D erzeugt, erstellt dieser Befehl eine 3D-Kurve. Die Kurven werden exakt an der ursprünglichen Grenze des Objekts positioniert.

## Grundlegende Verwendung

1. Legen Sie die gewünschte Konstruktionsebene für die Projektion fest.
2. Wählen Sie die zu projizierenden Volumenkörper oder Flächen aus.
3. Verwenden Sie den Befehl **Umriss erstellen** in der Befehlspalette.
4. Die projizierten Kurven werden erstellt.
5. Optional können Sie die Kurven mit dem Verschieben-Gizmo verschieben.
6. Klicken Sie mit der rechten Maustaste oder klicken Sie im Befehlsdialog auf OK, um zu bestätigen.
