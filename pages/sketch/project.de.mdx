# Projekt

<Reference shortcut="I" command="Project" button="Im vorgeschlagenen Befehlsmenü"  buttonIcon='button-project'/>

Der Projekt-Befehl wird basierend auf dem ausgewählten Objekt ausgeführt:
   - **Projekt-Kurve-Kurve:** Wenn zwei Kurven ausgewählt sind, siehe [**Projekt-Kurve-Kurve**](project-curve-curve).
   - **Projekt-Körper-Körper:** Wenn zwei Festkörper oder Flächen ausgewählt sind, siehe [**Projekt-Körper-Körper**](project-body-body).
   - Andernfalls: siehe [**Projekt-Kurve-Körper**](project-curve-body).
