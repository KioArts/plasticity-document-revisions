# Subdivide Curve

<Reference shortcut="Alt + S" command="Subdivide Curve" button="" />

選択したカーブのコントロールポイント（CV）とカーブセグメントの数を増やします。 

## 基本的な使い方

1. 1つ以上のカーブを選択します。
2. <kbd>Alt + S</kbd>を押すか、コマンドパレットで **Subdivide** と入力します。
3. CVの数が増加し、それらが選択されます。

<Callout type="info" emoji="💡">
    **Tip**
    Plasticityでは、増加したカーブセグメントはCV（コントロールポイント）を伴う単一のカーブとして表示されます。さらに、インタラクティブパネルには、選択したカーブに関する情報（例えば、次数、スパンの数、クラス分類など）が表示されます。
    ![Interactive Panel](/img/interactive-panel-subdivide-raise-degree-curve.png)
</Callout>
