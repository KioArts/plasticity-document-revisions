# Subdivide Curve

<Reference shortcut="Alt + S" command="Subdivide Curve" button="" />

選択したカーブのコントロールポイント（CV）とカーブセグメントの数を増やします。 

## 基本的な使い方

1. 1つ以上のカーブを選択します。
2. <kbd>Alt + S</kbd>を押すか、コマンドパレットで **Subdivide** と入力します。
3. CVの数が増加し、それらが選択されます。

<Callout type="info" emoji="💡">
**Tip**
プラスティシティでは、増加したカーブセグメントは、CVを持つ単一のカーブとして表示されます。また、 [詳細表示 ](/plasticity-essentials/user-interface-overview) パネルは、選択されたカーブに関する情報を提供し、その次数、スパンの数、および分類が含まれます。
