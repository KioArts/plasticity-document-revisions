# Instance

<Reference shortcut="" command="Instance" button="Command Palette" buttonIcon="button-instance" />

The Instance command runs based on the selected object:
     - **Instance Curve:** If Curves are selected, see [**Instance Curve**](/sketch/instance-curve). 
     - **Instance Solid and Sheet** Solids or Sheets are selected, see [**Instance Solid and Sheet**](instance-solid-and-sheet).
