## 説明

2 本のカーブを G0 / G1 / G2 / G3 のいずれかの連続性でブリッジします。テンション調整、オートトリム、曲率表示に対応しています。

## 基本的な使い方

<Callout type="info">
ポイントを選択する前に、[Object Snaps](/plasticity-essentials/plasticity-interface/snap#object-snap) の Control‑Point、Edge、Curve、Face、Axis を有効にしてください。  
開始点と終了点は、カーブ・エッジ・フェイス上の任意の位置にスナップできます。
</Callout>

1. ツールバーの <ButtonIcon type="button-bridge-curve" /> ボタンをクリックします。
2. 開始点をクリックします。
3. 終了点をクリックします。  
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
     - <Kst>コマンドダイアログ</Kst>
          - **Parameter（パラメータ）：**
               - **Value 1/2:** 開始点／終了点の相対位置。
          - **Sense（方向反転）：**
               - **Sense 1/2:** 開始点／終了点の方向を反転します。
          - **Tension 1：**
               - **Value 1/2:** 開始点／終了点の 1〜3 番目の CV に影響するテンション。
          - **Tension 2：**
               - **Value 1/2:** 開始点／終了点の 2・3 番目の CV に影響するテンション。
          - **Tension 3：**
               - **Value 1/2:** 開始点／終了点の 3 番目の CV に影響するテンション。
          - **Start curvature（開始側の曲率）：**
               - **G0, G1, G2, G3:** 開始点の[カーブ連続性](/cad-essentials/continuity-curve-and-surface)を指定します。
          - **End curvature（終了側の曲率）：**
               - **G0, G1, G2, G3:** 終了点の[カーブ連続性](/cad-essentials/continuity-curve-and-surface)を指定します。
          - **Trim:** ブリッジを超えて伸びているカーブを自動トリムします。
          - **Show curvature:** 曲率表示を切り替えます。詳細は [Toggle Curvature](/tool/toggle-curvature)。
     - <Kst>キーボードショートカット</Kst>
     - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z 軸:** [軸に制限する](/tool/sketching-essentials#constrain-to-an-axis)。
     - <kbd>Tab</kbd> **連続性切り替え:** G0 → G1 → G2 → G3 を順に切り替えます。
     - <kbd>D</kbd> **G1 テンション:** G1 連続性のテンションを調整します。
     - <kbd>Q</kbd> **Trim:** 選択中のカーブのトリムをオン／オフします。
          
5. コマンドダイアログの OK または右クリックで確定します。
