# Fillet

<Reference shortcut="B" command="Fillet" button="In Suggested Commands Menu"   buttonIcon='button-fillet'/>

The Fillet command runs based on the selected object:
     - **Fillet Curve:** If Curves are selected, see [**Fillet Curve**](fillet-curve).
     - **Fillet Vertex:** If verticies are selected, see [**Fillet Vertex**](fillet-vertex).
     - **Fillet Shell:** If Edges are selected, see [**Fillet Shell**](/solid/fillet-shell) in the Solid Commands section.
