# Unjoin Curves [#unjoin-curve]

<Reference shortcut="Alt + J" command="Unjoin" button="Im vorgeschlagenen Befehlsmenü"  buttonIcon='button-unjoin' />

Trennt verbundene Curves in einzelne.

## Grundlegende Nutzung

1. Wählen Sie die Curves aus, die Sie trennen möchten.
2. Drücken Sie <kbd>Alt + J</kbd>.
3. Die Curves werden getrennt.
