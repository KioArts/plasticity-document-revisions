## Description

Project a curve or edge onto the construction plane to create a new curve. 
This command shares the same functionality as [**Alternative Duplicate**](alternative-duplicate).

## Basic Usage

1. Select the curves or edges you want to project.
2. Set up your construction plane you wish the curves to be projected on.
3. Press <kbd>Alt + D</kbd> to execute the command or type **Alternative Duplicate** in the Command Palette.
4. The curve is created. Use the displayed Move Gizmo to move.
5. Right-click or click OK in the Command Dialog to confirm.
