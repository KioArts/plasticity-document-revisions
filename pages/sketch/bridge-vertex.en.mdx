## Description

Bridges two Vertices that can be G0/G1/G2/G3 continuous. Tension is adjustable and measuring curvature is supported.

## Basic Usage

1. Select two Vertices you want to bridge.
2. Use the **Bridge**  command in the Command Palette; or press <kbd>L</kbd> alternatively.
3. Set the options in the Command Dialog and Keyboard Shortcuts. 
     - <Kst>Command Dialog</Kst>
          - **Tension 1:**
               - **Value 1/2:** The tension value affected 1st Control-Vertex(CV) to 3rd CV of the starting point/ending point.
          - **Tension 2:**
               - **Value 1/2:** The tension value affected 2nd CV and 3rd CV of the starting point/ending point.
          - **Tension 3:**
               - **Value 1/2:** The tension value affected 3rd CV of the starting point/ending point.
          - **Start curvature:**
               - **G0, G1, G2, G3:** Specify the [continuity of the  Curve](/cad-essentials/continuity-curve-and-surface) at the starting point.
          -  **End curvature:**
               - **G0, G1, G2, G3:** Specify the [continuity of the  Curve](/cad-essentials/continuity-curve-and-surface) at the ending point.
     - <Kst>Keyboard Shortcuts</Kst>
          - <kbd>Tab</kbd> **Cycle continuity:** Toggle between G0, G1, G2, and G3 continuity. 
          - <kbd>D</kbd> **G1 Tension:** Adjust the tension for G1 continuity.       
4. Confirm with OK in the Command Dialog or right-click.
