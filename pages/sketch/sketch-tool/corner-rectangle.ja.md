# Corner Rectangle

- 対角線上の2点を指定した四角形を作成できます。
- ショートカット : なし
- ボタン : ツールバー

1. ツールバーのアイコンをクリックします。
2. 四角形の対角線上の始点でクリックします。
   - `Tab` ： 数値で長さを指定します。
   - `K` : **Knife** : フェイスにエッジを追加します。
   - `C` : **Center / Corner** : Center Rectangleに切り替えます。
   - `S` : **Square** : 正方形の補助線を引きます。
3. 対角線上の終点でクリックします。
4. コマンドダイアログの**OK**または右クリックで確定します。
5. 四角形が作成されます。
