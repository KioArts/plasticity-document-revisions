## Description

Converts standard Vertices into Control-Vertices.

## Basic Usage

1. Select the Vertices you want to convert.
2. Use the **Convert Vertex** command in the Command Palette.
3. The Vertices are now Control Vertices.

<Callout type="info">
You can also convert a Vertex by double-clicking on it.
</Callout>
