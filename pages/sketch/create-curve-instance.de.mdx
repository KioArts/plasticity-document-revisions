## Beschreibung

Erstellt wiederverwendbare Duplikate von Kurven. Eine **Instanz** speichert, ähnlich wie bei der Instanzierung von Volumenkörpern und Flächenobjekten, Transformationsdaten (Position, Pivot-Punkt, Rotation und Skalierung) sowie Farbdaten und verweist auf die ursprüngliche Kurve. Jede Änderung an der Masterkurve wird sofort in allen ihren Instanzen widergespiegelt.

Da Kurveninstanzen schreibgeschützte Kopien ihrer Quelle sind, können sie nicht unabhängig bearbeitet werden. Sie verbergen auch das CV-Layout (Kontrollscheitelpunkte), was sie optisch von der Originalkurve unterscheidet.

<Callout>
1. Verwenden Sie den Befehl **Instanz erstellen** für Kurven, wenn Sie viele identische Details benötigen, die konsistent und symmetrisch sein müssen.
2. Instanzen funktionieren mit Array-Befehlen wie [**Kurven-Array**](/common/curve-array), [**Radiales Array**](/common/radial-array) und [**Rechteckiges Array**](/common/rectangular-array).
3. Instanzen können auch mit dem Befehl [**Spiegeln**](/common/mirror) verwendet werden, was eine Live-Symmetrie ermöglicht, bei der Änderungen an der Mastergeometrie alle gespiegelten Instanzen automatisch aktualisieren.
4. Die Befehle [**Platzieren**](/common/place) und [**Mit Platzierung kopieren**](/common/copy-with-placement) können derzeit nicht auf erstellte Instanzen angewendet werden.
</Callout>

## Grundlegende Verwendung

1. Wählen Sie die Kurve aus, die Sie als übergeordnetes Objekt verwenden möchten.
2. Öffnen Sie die Befehlspalette, geben Sie **Instanz erstellen** ein und führen Sie den Befehl aus.
3. Verschieben Sie die neu erstellte Instanz an die gewünschte Position.
4. Bestätigen Sie mit OK im Befehlsdialog oder durch Rechtsklick.

</Callout>

## Instanzgruppen

Der Prozess der Instanzierung von Gruppen ist ähnlich wie die Instanzierung von Kurven.

Wenn eine Gruppe ausgewählt ist und primäre Skizzen enthält, wird eine Instanz der Gruppe erstellt und außerhalb des Gruppenordners unter einer eigenen dedizierten Kategorie im Outliner platziert.

<Center width="600px">
![!Instancing a Group Part 1](/img/instance-group-curve.png)
</Center>

| Nummer | Beschreibung |
| :----: | ----------- |
| 1 | Hauptgruppe mit enthaltenen Komponenten. |
| 2 | Übergeordnete Objekte, die instanziert werden. |
| 3 | Instanz der Hauptgruppe, die außerhalb der Hauptgruppe unter der Kategorie **„Instanzen"** zugewiesen wird. |

### Grundlegende Verwendung

1. Wählen Sie eine Gruppe im Outliner zum Instanzieren aus.
2. Suchen Sie in der Befehlspalette nach **Instanz erstellen** und führen Sie den Befehl aus.
3. Verschieben Sie die instanzierte Gruppe.
4. Bestätigen Sie mit OK im Befehlsdialog oder durch Rechtsklick.
