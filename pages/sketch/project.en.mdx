# Project

<Reference shortcut="I" command="Project" button="In Suggested Commands Menu"  buttonIcon='button-project'/>

The Project command runs based on the selected object:
     - **Project Curve Curve:** If two Curves are selected, see [**Project Curve Curve**](project-curve-curve).
     - **Project Body Body:** If two Solids or Sheets are selected, see [**Project Body Body**](project-body-body).
     - Otherwise: see [**Project Curve Body**](project-curve-body).
