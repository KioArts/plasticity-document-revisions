## Description

Create a new Curve by projecting a Curve onto a Solid or Sheet. Choose projection method normal or direction vector. Bidirectional is also available.

## Basic Usage

1. Select the Solid or Sheet to project onto.
2. Press <kbd>I</kbd> or click the button.
3. Select the Curve to project.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - <Kst>Command Dialog</Kst>
      - **Method:**
            - **Normal:** Projects in the positive Z-axis direction based on the construction plane.
            - **Vector:** Projects in a linear direction. **Bidirectional** and **Hide occlusion** will be enabled.     
            <img src="/img/project-curve-body-method.svg" width="500" />
      - **Bidirectional:** When enabled, the projection will also occur in the negative direction of the specified axis.      
      <img src ="/img/project-curve-body-bidirectional.svg" width="500" />
      - **Hide occluded:** When enabled, the projection will not occur on hidden Faces in the positive direction of the specified axis.
      <img src ="/img/project-curve-body-hide-occluded.svg" width="500" />
   - <Kst>Command Dialog</Kst>
      - <kbd>S</kbd> **Screen space:** Project the new Curve using the the current camera view as a projection method.
5. Right-click or click OK in the Command Dialog to confirm.
6. The projected Curve will be created.
