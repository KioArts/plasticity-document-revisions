# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="Im vorgeschlagenen Befehlsmenü"  buttonIcon='button-offset-curve'/>

Der Offset-Kurve-Befehl wird basierend auf dem ausgewählten Objekt ausgeführt:
   - **Offset-Scheitelpunkt:** Wenn Scheitelpunkte ausgewählt sind, siehe [Offset-Scheitelpunkt](offset-vertex).
   - **Offset-Planare Kurve:** Wenn Kurven ausgewählt sind, siehe [Offset-Planare Kurve](offset-planar-curve).
   - **Offset-Region:** Wenn Regionen ausgewählt sind, siehe [Offset-Region](offset-region).
   - **Offset-Flächen-Schleife:** Wenn Flächen ausgewählt sind, siehe [Offset-Flächen-Schleife](/solid/offset-face-loop) im Abschnitt „Solid-Befehle“.
   - **Offset-Kante:** Wenn Kanten ausgewählt sind, siehe [Offset-Kante](/solid/offset-edge) im Abschnitt „Solid-Befehle“.
