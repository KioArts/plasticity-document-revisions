# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="Rechts der Befehlsleiste" />

Einheitlicher Befehl für **Offset Planar Curve**, **Offset Region**, **Offset Face Loop** und **Offset Edge**. Der Befehl wird basierend auf dem ausgewählten Objekt ausgeführt.
   - Wenn Kurven ausgewählt sind: [**Offset Planar Curve**](./offset-planar-curve).
   - Wenn Regionen ausgewählt sind: [**Offset Region**](./offset-region).
   - Wenn Faces ausgewählt sind: siehe [**Offset Face Loop**](../solid/offset-face-loop) im Abschnitt Solid Commands.
   - Wenn Kanten ausgewählt sind: siehe [**Offset Edge**](../solid/offset-edge) im Abschnitt Solid Commands.
