# Unjoin

<Reference shortcut="Alt + J" command="Unjoin" button="Rechts von der Befehlsleiste"  buttonIcon='button-unjoin'/>

Der Unjoin-Befehl wird basierend auf dem ausgewählten Objekt ausgeführt:
   - **Unjoin-Kurven:** Wenn Kurven ausgewählt sind, siehe [**Unjoin-Kurven**](unjoin-curve).
   - **Unjoin-Flächen:** Wenn Flächen ausgewählt sind, siehe [**Unjoin-Flächen**](/solid/unjoin-faces) im Abschnitt „Solid-Befehle“.
   - **Unjoin-Schalen:** Wenn Festkörper oder Flächen ausgewählt sind, siehe [**Unjoin-Schalen**](/solid/unjoin-shells) im Abschnitt „Solid-Befehle“.
