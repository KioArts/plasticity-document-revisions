# Toggle Curvature

<Reference shortcut="" command="Toggle Curvature" button=""/>

The Measure Curvature command runs based on the selected object:
     - **Toggle Curve Curvature:** If Curves are selected, see [Toggle Curve Curvature](/sketch/toggle-curve-curvature).
     - **Toggle Surface Curvature:** If Surface Edges are selected, see [Toggle Surface Curvature](/solid/toggle-surface-curvature) in the Solid Commands section.
