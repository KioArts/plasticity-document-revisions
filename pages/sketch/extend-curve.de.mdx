# Extend Curve

<Reference shortcut="" command="Extend Curve" button="" />

Verlängere eine Curve, indem du die Entfernung angibst. Wähle aus Verlängerungsformen wie Natural, Linear, Soft, Reflective und Arc.

1. Wähle den Endpunkt der Curve, die du verlängern möchtest.
2. Verwende den **Extend** Befehl in der Befehlspalette.
3. Drücke <kbd>D</kbd>, bewege den Cursor und klicke auf den Endpunkt.
4. Stelle die Optionen im Befehlsdialog ein.
   - **Shape:**
      - **Natural** / **Liner** /  **Soft** /  **Reflective** / **Arc**
      - Sieh dir das Video an, um die Unterschiede zwischen jeder Option zu sehen.
   - **Distance:** Gib die Entfernung an.
5. Bestätige mit OK im Befehlsdialog oder Rechtsklick.
