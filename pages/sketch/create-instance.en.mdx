# Create Instance

<Reference shortcut="" command="Create Instance" button="Command Palette" buttonIcon="button-instance" />

The Instance command runs based on the selected object:
     - **Create Instance Curve:** If Curves are selected, see [**Create Instance Curve**](/sketch/instance-curve). 
     - **Create Instance Solid and Sheet** Solids or Sheets are selected, see [**Create Instance Solid and Sheet**](/solid/instance-solid-and-sheet).
