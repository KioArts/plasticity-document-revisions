# Create Instance

<Reference shortcut="" command="Create Instance" button="Command Palette" buttonIcon="button-instance" />

The Instance command runs based on the selected object:
     - **Create Curve Instance:** If Curves are selected, see [**Create Curve Instance**](/sketch/create-curve-instance). 
     - **Create Solid and Sheet Instance:** Solids or Sheets are selected, see [**Create Solid and Sheet Instance**](/solid/create-solid-and-sheet-instance).
