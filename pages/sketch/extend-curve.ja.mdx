# Extend Curve（カーブを延長する）

<Reference shortcut="" command="Extend Curve" button="" />

カーブを延長できます。

## 基本的な使い方

1. 延長したいカーブ端の頂点を選択します。
2. コマンドパレットで **Extend** を実行します。
3. <kbd>D</kbd>を押し、カーソルを動かして、延長先の位置でクリックします。
4. コマンドダイアログでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
      - **Shape:** 拡張されたカーブの形状を指定します。**Natural**、**Linear**、**Soft**、**Reflective**、**Arc** のオプションは、押し出し時のカーブの形状や曲率に直接影響を与えます。  
            - **Natural:** 拡張の開始直前に元のカーブが持っていた形状をそのまま継続することで、カーブを「自然に」延長します。  
            - **Linear:** 境界点から直線的にフラットにカーブを延長し、現在の接線方向を維持しながら曲率を変更しません。  
            - **Soft:** 拡張部分を既存のカーブと徐々にブレンドし、滑らかな遷移を実現しながら接線と曲率の両方を維持します。  
            - **Reflective:** カーブの曲率と接線を反映させ、対称的でシームレスな拡張を行い、カーブの対称性を維持するのに最適です。  
            - **Arc:** 境界点から円弧状にカーブを延長します。  

            ![Extend Curve Shape](/img/extend-curve-shape.png) 
            
      - **Distance:** 距離を指定します。
5. コマンドダイアログでOKを押すか、右クリックで確定します。
