## Description

Project the curves onto the current construction plane along the normal direction, thereby generating a new Curve. The resulting projection flattens the original geometry onto the plane while preserving the curve's overall shape and continuity.

<Center width="450px">
![Alternative Duplicate](/img/alternative-duplicate.svg)
</Center>

## Basic Usage

1. Set the desired construction plane for projection. See [Construction Planes](/plasticity-essentials/plasticity-interface/construction-plane) for more information.
2. Select the Curves you want to project.
3. Press <kbd>Alt + D</kbd> to execute the command.
4. The projected Curves will be created on the current construction plane.
5. Right-click or click **OK** in the Command Dialog to confirm.

<Callout type='info'>
**Tip:** After projection, you can reposition the resulting Curves using the [Move](/common/move) Gizmo.
</Callout>
