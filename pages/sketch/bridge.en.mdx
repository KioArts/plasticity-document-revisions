# Bridge

<Reference shortcut="" command="Bridge" button="" />

The Bridge command runs based on the selected object:
     - **Bridge Curve:** If two curves are selected or nothing is selected, see [**Bridge Curve**](bridge-curve).
     - **Bridge Vertex:** If two vertices are selected, see [**Bridge Vertex**](bridge-vertex).
     - **Bridge Edge:** If two edges are selected, see [**Bridge Edge**](bridge-edge).
