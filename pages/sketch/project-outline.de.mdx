## Beschreibung

Projiziert die Umrisse von Volumenkörpern oder Flächen entlang ihrer Normalenrichtung auf die aktive Konstruktionsebene und erzeugt dabei eine passende neue Kurve.

## Grundlegende Verwendung

1. Legen Sie die gewünschte Konstruktionsebene für die Projektion fest.
2. Wählen Sie die Volumenkörper oder Flächen aus, auf die projiziert werden soll.
3. Drücken Sie <kbd>Alt + D</kbd>, um den Befehl auszuführen.
4. Die projizierten Kurven werden auf der aktuellen Konstruktionsebene erstellt.
5. Optional können Sie die Kurven mit dem Verschieben-Gizmo verschieben.
6. Klicken Sie mit der rechten Maustaste oder klicken Sie im Befehlsdialog auf OK, um zu bestätigen.
