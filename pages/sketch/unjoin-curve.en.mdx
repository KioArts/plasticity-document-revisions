## Description

Separates joined curves into individual ones. Use this command to break apart previously joined curves when you need to edit, delete, or manipulate them independently.

## Basic Usage

1. Select the Curves you want to separate.
2. Press <kbd>Alt + J</kbd>.
3. The Curves are separated.
