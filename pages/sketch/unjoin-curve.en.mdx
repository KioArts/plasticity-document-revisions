# Unjoin Curves [#unjoin-curve]

<Reference shortcut="Alt + J" command="Unjoin" button="In Suggested Commands Menu"  buttonIcon='button-unjoin' />

Separate joined Curves into individual ones.

## Basic Usage

1. Select the Curves you want to separate.
2. Press <kbd>Alt + J</kbd>.
3. The Curves are separated.
