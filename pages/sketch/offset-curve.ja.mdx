# Offset Curve（カーブをオフセットする）

<Reference shortcut="O" command="Offset Curve" button="推奨コマンドメニューで" buttonIcon='button-offset-curve' />

Offset カーブ コマンドは選択されたオブジェクトに基づいて実行されます：
   - **Offset 頂点:** 頂点が選択されている場合は、[Offset 頂点](offset-vertex) を参照してください。
   - **Offset 平面カーブ:** カーブが選択されている場合は、[Offset 平面カーブ](offset-planar-curve) を参照してください。
   - **Offset 領域:** 領域が選択されている場合は、[Offset 領域](offset-region) を参照してください。
   - **Offset 面ループ:** 面が選択されている場合は、ソリッドコマンドのセクションにある [Offset 面ループ](/solid/offset-face-loop) を参照してください。
   - **Offset エッジ:** エッジが選択されている場合は、ソリッドコマンドのセクションにある [Offset エッジ](/solid/offset-edge) を参照してください。
