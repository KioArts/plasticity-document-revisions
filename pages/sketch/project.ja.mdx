# Project（投影）

<Reference shortcut="I" command="Project" button="推奨コマンドメニューで"  buttonIcon='button-project'/>

プロジェクト コマンドは選択されたオブジェクトに基づいて実行されます：
   - **プロジェクト カーブ カーブ:** 2つのカーブが選択されている場合は、[**プロジェクト カーブ カーブ**](project-curve-curve) を参照してください。
   - **プロジェクト ボディ ボディ:** 2つのソリッドまたはシートが選択されている場合は、[**プロジェクト ボディ ボディ**](project-body-body) を参照してください。
   - それ以外の場合： [**プロジェクト カーブ ボディ**](project-curve-body) を参照してください。
