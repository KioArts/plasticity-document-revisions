## Description

Toggles the visibility of Control-Vertex (CV) and Vertex points on curves, and CV points on faces (sheets) and bodies (solids).

## Basic Usage

1. Select the geometry: 
    - **Curves**: Select the curve or a CV point to hide the CV layout. To restore visibility, select the curve and execute the command again.
    - **Sheets**: Select the face or a CV point to toggle visibility. To restore the default state, select the face and execute the command again.
    - **Solids**: Select a face or CV point to toggle an individual face, or select the solid to affect all faces. To restore the default state, select the desired faces or the solid and execute the command again.
2. Type **Toggle Points** in the Command Palette and left-click it.
3. The points are hidden or revealed.

<Callout type="default">
**Tip:** For quicker access, assign a keyboard shortcut to this command. Right-click it in the Command Palette, select Assign Shortcut, and press your desired key combination.
</Callout>
