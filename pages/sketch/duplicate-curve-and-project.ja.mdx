## Beschreibung

Projiziert eine Kurve oder Kante auf die Konstruktionsebene, um eine neue Kurve zu erstellen.
Dieser Befehl hat die gleiche Funktionalität wie [**Alternatives Duplizieren**](alternative-duplicate).

## Grundlegende Verwendung

1. Wählen Sie die Kurven oder Kanten aus, die Sie projizieren möchten.
2. Richten Sie die Konstruktionsebene ein, auf die die Kurven projiziert werden sollen.
3. Drücken Sie <kbd>Alt + D</kbd>, um den Befehl auszuführen, oder geben Sie **Alternative Duplicate** in der Befehlspalette ein.
4. Die Kurve wird erstellt. Verwenden Sie das angezeigte Verschieben-Gizmo zum Verschieben.
5. Klicken Sie mit der rechten Maustaste oder klicken Sie im Befehlsdialog auf OK, um zu bestätigen.
