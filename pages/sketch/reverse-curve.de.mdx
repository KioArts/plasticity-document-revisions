# Reverse Curve

<Reference shortcut="" command="Reverse" button=""/>

Richten Sie die Richtung einer Kurve um. 

## Grundlegende Nutzung

1. Wählen Sie die Kurve aus, deren Richtung Sie umkehren möchten.
2. Geben Sie **Reverse** in die Befehlsleiste ein.
3. Jetzt wurde die Richtung der Kurve umgekehrt.
