# Unjoin

<Reference shortcut="Alt + J" command="Unjoin" button="Right of Command Bar"  buttonIcon='button-unjoin'/>

The Unjoin command runs based on the selected object:
     - **Unjoin Curves:** If Curves are selected, see [**Unjoin Curves**](unjoin-curve).
     - **Unjoin Faces:** If Faces are selected, see [**Unjoin Faces**](/solid/unjoin-faces) in the Solid Commands section.
     - **Unjoin Shells:** If Solids or Sheets are selected, see [**Unjoin Shells**](/solid/unjoin-shells) in the Solid Commands section.
