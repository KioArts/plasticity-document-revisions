# Slide

<Reference shortcut="Shift+G" command="Slide" button="" />

Slide CVs along the direction derived from the CV hull.

## Basic Usage

1. Select the CV you want to slide.
2. Press <kbd>Shift + G</kbd> to activate the Slide command.
3. Set the options in the Command Dialog and Keyboard Shortcuts.
   - <Kst>Command Dialog</Kst>
      - **Distance:** 
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>Ctrl + Z</kbd> **Undo** 
      - <kbd>U</kbd> **Positive U**
      - <kbd>Shift + U</kbd> **Negative U**
      - <kbd>N</kbd> **Normal**
      - <kbd>V</kbd> **Positive V**
      - <kbd>Shift + V</kbd> **Negative V**
3. Gizmo handles will appear with specified direction handles. Move the CV with the handles to manipulate their position.
4. Confirm with OK in the Command Dialog or right-click.
