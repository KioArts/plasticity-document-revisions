# Realize Instance

<Reference shortcut="" command="Realize Instance" button="Command Palette" buttonIcon="button-instance" />

The Realize Instance command runs based on the selected object:
     - **Realize Curve Instances:** If Curves are selected, see [**Realize Curve Instances**](/sketch/realize-curve-instances). 
     - **Realize Solid and Sheet Instances** Solids or Sheets are selected, see [**Realize Solid and Sheet Instances**](/solid/realize-solid-and-sheet-instances).
