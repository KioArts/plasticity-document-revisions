# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="In Suggested Commands Menu"  buttonIcon='button-offset-curve'/>

The Offset Curve command runs based on the selected object:
     - **Offset Vertex:** If Vetices are selected, see [Offset Vertex](offset-vertex).
     - **Offset Planar Curve:** If Curves are selected, see [Offset Planar Curve](offset-planar-curve).
     - **Offset Region:** If Regions are selected, see [Offset Region](offset-region).
     - **Offset Face Loop:** If Faces are selected, see [Offset Face Loop](/solid/offset-face-loop) in the Solid Commands section.
     - **Offset Edge:** If Edges are selected, see [Offset Edge](/solid/offset-edge) in the Solid Commands section.
