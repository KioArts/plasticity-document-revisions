# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="Right of Command Bar" />

This command automatically calls the following command depending on the selected object.

- When curves are selected: **"Offset Planar Curve"** described below.
- When Regions are selected: **"Offset Region"** described below.
- When Faces are selected: see [**"Offset Face Loop"**](../solid/offset-curve#offset-face-loop) in Solid's **"Offset Curve"**.
- When Edges are selected: see [**"Offset Edge"**](../solid/offset-curve#offset-edge) in Solid's **"Offset Curve"**.
