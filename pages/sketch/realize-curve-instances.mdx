# Realize Curve Instances

<Reference shortcut="" command="Realize Instances" button="In Command Palette"  buttonIcon="button-instance"/>

Convert selected Curve instances into editable sketch objects, turning them into a true Curve that can be modified as needed.

## Basic Usage

1. Select the Curve instances you want to convert.
2. Open the Command Palette, type **Realize Instances**, and execute the command.
3. The instances have been converted into editable objects.
