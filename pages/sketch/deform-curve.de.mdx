# Deform Curve

<Reference shortcut="" command="Deform" button="" />

Wickeln Sie vorhandene Solids- und Sheet-Objekte auf eine regelmäßige oder unregelmäßige Oberfläche.

## Grundlegende Nutzung

1. Wählen Sie die Objekte aus, die Sie umhüllen möchten.
2. Geben Sie **Deform** in die Befehlsleiste ein.
3. Wählen Sie eine Referenzfläche als Neutralebene aus.
4. Wählen Sie die Zielfläche aus, auf die die Objekte deformiert werden sollen.
5. Stellen Sie die Optionen im Befehlsdialog ein.
    - <Kst>Befehlsdialog</Kst>
          - **Skalierung:** Skalierung in **Z,Y,X Wert**
          - **Offset:** Offset in **Z,Y,X Wert**
          - **Flip:** 
                - **Spiegeln:** Spiegeln Sie die deformierten Objekte horizontal zur Oberfläche.
                - **UV:** Drehen Sie die U- und V-Ausrichtung der deformierten Objekte um.
                - **Normal:** Drehen Sie die Normalen der deformierten Objekte um.
          - **Werkzeuge behalten:** Führen Sie die Deform-Operation durch, während die Werkzeugobjekte intakt bleiben.
6. Bestätigen Sie mit OK im Befehlsdialog oder durch Rechtsklick.

<Callout type="info" emoji="ℹ️">
Scale und Offset verwenden ein **UVN**-Koordinatensystem anstelle eines **XYZ**-Koordinatensystems, wenn der parametrische Wert angepasst wird. Dies ermöglicht es Ihnen, die Ausrichtung der deformierten Objekte an einem bestimmten Punkt in Bezug auf die Zieloberfläche festzulegen. Für weitere Informationen zum UVN-Koordinatensystem klicken Sie [hier](/cad-essentials/uvn-coordinate-system).
</Callout>

### Kurvenverformung mit dem Befehl "Fläche entpacken"

Um unerwünschte Verformungen oder ungenaue Platzierungen zu vermeiden, verwenden Sie den Befehl "Fläche entpacken" aus der Befehlsübersicht auf dem Zielobjekt. Dies erzeugt eine korrekt abgebildete Oberfläche, die anschließend als genaue Referenzfläche verwendet werden kann.
