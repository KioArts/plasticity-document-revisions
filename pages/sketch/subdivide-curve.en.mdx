# Subdivide Curve

<Reference shortcut="Alt + S" command="Subdivide Curve" button="" />

Increase the number of Control Vertices(CVs) by doubling its count on selected curves each time its being ran. 

## Basic Usage

1. Select one or more Curves.
2. Press <kbd>Alt + S</kbd> or type **Subdivide** in the Command Palette.
3. The number of CVs is increased, and they are selected.

<Callout type="info" emoji="ðŸ’¡">
**Tip**     
- Using the Raise Degree command on a single-span curve adds one control vertex (CV) per execution, whereas the Subdivide command doubles the number of CVs each time it is applied. 
</Callout>
