# Subdivide Curve

<Reference shortcut="Alt + S" command="Subdivide Curve" button="" />

Increase the number of Control Vertices(CVs) and Curve Segments on the selected curves. 

## Basic Usage

1. Select one or more Curves.
2. Press <kbd>Alt + S</kbd> or type **Subdivide** in the Command Palette.
3. The number of CVs is increased, and they are selected.

<Callout type="info" emoji="ðŸ’¡">
    **Tip**
In Plasticity, the increased curve segments are displayed as a single curve with CVs. Additionally, the [Detail Display](/plasticity-essentials/user-interface-overview) panel provides information about the selected curve, including its degree, the number of spans, and its classification.
