# Cut Curve（カーブを切断）

<Reference shortcut="C" command="Cut Curve" button="コマンドバー左" />

カーブを、カーブまたはフェイスで切断できます。

<Callout type="info">カット方向は構築平面に対して垂直な方向です。</Callout>

1. ショートカット<kbd>C</kbd>を押すか、ボタン<ButtonIcon type="button-cut" />をクリックします。
2. もう一度<kbd>C</kbd>を押して、**Cut Curve**コマンドに切り替えます。
3. ターゲットとカッターを指定します。
   - **Select target bodies**: カットするカーブを選択します。
   - **Select cutters**: カッターとして使用するカーブまたはフェイスを選択します。
   - <kbd>S</kbd> **Screen space:** カット方向を見た目の方向からにします。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
