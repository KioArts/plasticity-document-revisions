# スケッチとソリッド 共通のコマンド

| コマンド名                                            | ショートカット              | 機能                                   |
| ----------------------------------------------------- | --------------------------- | -------------------------------------- |
| [Copy with Placement](/common/copy-with-placement)   | <kbd>Ctrl+Shift+C</kbd>     | オブジェクトを参照点付きでコピーします。           |
| [Curve Array](/common/curve-array)                   |                             | カーブに沿ってオブジェクトを複製します。           |
| [Dimension](/common/dimension)                       | <kbd>=</kbd>                | 寸法値を変更します。                               |
| [Duplicate](/common/duplicate)                       | <kbd>Shift+D</kbd>          | オブジェクトを複製します。                         |
| [Measure](/common/measure)                           | <kbd>Ctrl+=</kbd>           | 寸法値を測定します。                               |
| [Mirror](/common/mirror)                             | <kbd>Alt+X</kbd>            | オブジェクトを反転します。                         |
| [Move](/common/move)                                 | <kbd>G</kbd>                | オブジェクトを移動します。                         |
| [Paste with Placement](/common/paste-with-placement) | <kbd>Ctrl+Shift+V</kbd>     | オブジェクトを参照点付きで貼り付けます。           |
| [Place](/common/place)                               | <kbd>Ctrl+D</kbd>           | 指定された軸に沿ってオブジェクトを複製します。     |
| [Radial Array](/common/radial-array)                 |                             | 放射状のパターンでオブジェクトを複製します。       |
| [Rectangular Array](/common/rectangular-array)       |                             | 長方形のパターンでオブジェクトを複製します。       |
| [Remove Material](/common/remove-material)           | <kbd>Alt+M</kbd>            | 材料を削除します。                                 |
| [Rotate](/common/rotate)                             | <kbd>R</kbd>                | オブジェクトを回転します。                         |
| [Scale](/common/scale)                               | <kbd>S</kbd>                | オブジェクトを拡大縮小します。                     |
| [Section Analysis](/common/section-analysis)         |                             | オブジェクトを通して仮想のスライスを作成する。            | 
| [Set Material](/common/set-material)                 | <kbd>M</kbd>                | 材料を設定します。                                 |
