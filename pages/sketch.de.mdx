# Skizzierbefehle

| Befehlsname                                                       | Tastenkombination          | Anmerkung                                                                                                    |
| ----------------------------------------------------------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| [Deform](/sketch/deform)                                           |                            | Vereinheitlichte Befehle für das Deformieren von Kurven, Festkörpern und Flächen.                                |
| [Deform Curve](/sketch/deform-sketch)                              |                            | Wickeln Sie vorhandene Kurven auf eine andere Oberfläche.                                                        |
| [Extend](/sketch/extend)                                           |                            | Vereinheitlichte Befehle für das Verlängern von Kurven und Flächen.                                             |
| [Extend Curve](/sketch/extend-curve)                               |                            | Eine Kurve verlängern.                                                                                         |
| [Fillet](/sketch/fillet)                                           | <kbd>B</kbd>               | Vereinheitlichte Befehle für Fasenkurven, Fasenpunkte und Fasenhülsen.                                        |
| [Fillet Curve](/sketch/fillet-curve)                               | <kbd>B</kbd>               | Fasen oder Fasen zwischen zwei Kurvensegmenten.                                                                 |
| [Fillet Vertex](/sketch/fillet-vertex)                             | <kbd>B</kbd>               | Fasen oder Fasen an verbundenen Punkten.                                                                        |
| [Offset Curve](/sketch/offset-curve)                               | <kbd>O</kbd>               | Vereinheitlichte Befehle für das Offset einer ebenen Kurve, eine Region, eine Flächenkontur, eine Kante.         |
| [Offset Planar Curve](/sketch/offset-planar-curve)                 | <kbd>O</kbd>               | Eine ebene Kurve offsetieren.                                                                                   |
| [Offset Region](/sketch/offset-region)                             | <kbd>O</kbd>               | Die Umrisskurven einer Region offsetieren.                                                                      |
| [Offset Vertex](/sketch/offset-vertex)                             | <kbd>O</kbd>               | Neue Punkte auf beiden Seiten eines Punktes entlang der Kurve einfügen.                                         |
| [Raise Degree](/sketch/raise-degree)                               | <kbd>Shift+S</kbd>       | Vereinheitlichte Befehle zum Anheben des Grades von Kurven und Flächen.                                         |
| [Raise Curve Degree](/sketch/raise-curve-degree)                   | <kbd>Shift+S</kbd>       | Die Kontrollpunkte und Grade von Kurven erhöhen.                                                                |
| [Rebuild](/sketch/rebuild)                                         |                            | Vereinheitlichte Befehle zum Wiederaufbauen von Kurven und Flächen.                                             |
| [Rebuild Curve](/sketch/rebuild-curve)                             |                            | Durch Optimierung der Positionen der Kontrollpunkte wiederaufbauen.                                             |
| [Reverse](/sketch/reverse)                                         |                            | Vereinheitlichte Befehle zum Umkehren von Kurven und Flächen.                                                   |
| [Reverse Curve](/sketch/reverse-curve)                             |                            | Die Richtung einer Kurve umkehren.                                                                             |
| [Subdivide Curve](/sketch/subdivide-curve)                         | <kbd>Alt+S</kbd>         | Kontrollpunkte und Spannweiten von Kurven erhöhen.                                                               |
| [Slide](/sketch/slide)                                             |                            | Vereinheitlichter Befehl für Slide Curve CV, Slide Surface CV.                                                  |
| [Slide Curve CV](/sketch/slide-curve-cv)                           | <kbd>Shift+G</kbd>         | Verschieben Sie ein CV auf einer Oberfläche entlang des CV-Hulls.                                               |
| [Text](/sketch/text)                                               |                            | Kurven aus Text erstellen.                                                                                     |
| [Toggle Points](/sketch/toggle-points)                             |                            | Kontrollpunkte ein- oder ausblenden.                                                                           |
| **Project Commands**                                               |                           |                                                                                                            |
| [Alternative Duplicate](/sketch/alternative-duplicate)             | <kbd>Alt+D</kbd>       | Projektion der Kurven auf die aktuelle Konstruktionsfläche.                                                     |
| [Create Outline](/sketch/create-outline)                           |                          | Projektion der Umrisslinien der Objekte zur Erstellung einer 3D-Kurve.                                        |
| [Duplicate Curve and Project](/sketch/duplicate-curve-and-project) |                          | Duplizieren von Kurven und Projektieren.                                                                        |
| [Project](/sketch/project)                                         | <kbd>I</kbd>             | Vereinheitlichte Befehle für die Projektion von Körpern, Kurven und Kurven.                                   |
| [Project Body Body](/sketch/project-body-body)                     | <kbd>I</kbd>             | Erstellen einer neuen Kurve, die die Schnittlinie zwischen zwei Festkörpern oder Flächen darstellt.             |
| [Project Curve Body](/sketch/project-curve-body)                   | <kbd>I</kbd>             | Erstellen einer neuen Kurve durch Projektion einer Kurve auf einen Festkörper oder eine Fläche.                 |
| [Project Curve Curve](/sketch/project-curve-curve)                 | <kbd>I</kbd>             | Erstellen einer neuen Kurve durch Projektion der Schnittpunkte von Flächen, die aus zwei unabhängigen Kurven erzeugt wurden. |
| [Project Outline](/sketch/project-outline)                         | <kbd>Alt+D</kbd>       | Projektion der Umrisslinien der Festkörper oder Flächen.                                                       |
| **Join and Split Commands**                                        |                            |                                                                                                            |
| [Bridge](/sketch/bridge)                                           |                          | Vereinheitlichte Befehle für die Brücke von Kurven, Punkten und Kanten.                                        |
| [Bridge Curve](/sketch/bridge-curve)                               |                          | Zwei Kurven miteinander verbinden.                                                                               |
| [Bridge Edge](/sketch/bridge-edge)                                 |                          | Zwei Kanten miteinander verbinden.                                                                               |
| [Bridge Vertex](/sketch/bridge-vertex)                             |                          | Zwei Punkte miteinander verbinden.                                                                               |
| [Cut Curve](/sketch/cut-curve)                                     |                          | Eine Kurve schneiden.                                                                                          |
| [Join](/sketch/join)                                               | <kbd>J</kbd>             | Vereinheitlichte Befehle zum Verbinden von Kurven und Flächen.                                                |
| [Join Curves](/sketch/join-curves)                                 | <kbd>J</kbd>             | Kurven an Verbindungspunkten verbinden.                                                                         |
| [Split Segment](/sketch/split-segment)                             |                          | Kurve in zwei Segmente teilen.                                                                                  |
| [Trim](/sketch/trim)                                               | <kbd>T</kbd>             | Eine Kurve trimmen.                                                                                           |
| [Unjoin](/sketch/unjoin)                                           | <kbd>Alt+J</kbd>       | Vereinheitlichte Befehle zum Trennen von Kurven, Flächen und Schalen.                                         |
| [Unjoin Curves](/sketch/unjoin-curve)                              | <kbd>Alt+J</kbd>       | Getrennte Kurven wieder verbinden.                                                                             |
| [Insert Knot](/sketch/insert-knot)                                 |                        | Fügen Sie einen Kontrollpunkt auf einer Spline-Kurve hinzu.                                                       |
| **Vertex Commands**                                                |                          |                                                                                                            |
| [Convert Vertex](/sketch/convert-vertex)                           |                          | Punkt in Kontrollpunkt umwandeln.                                                                               |
| [Delete Redundant Topology](/sketch/delete-redundant-topology)     |                          | Überflüssige Punkte löschen.                                                                                    |
