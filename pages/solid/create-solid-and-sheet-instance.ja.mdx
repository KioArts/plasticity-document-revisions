# ソリッド・シートインスタンスを作成

<Reference shortcut="" command="インスタンス作成" button="コマンドパレットで"  buttonIcon="button-instance"/>

Solid または Sheet オブジェクトの基になるジオメトリをコピーせずに、再利用可能な複製を作成します。インスタンスは変換データ（位置、ピボットポイント、回転、スケール）、マテリアル情報を保持し、元の Solid オブジェクトへの参照を持つため、マスター ジオメトリに加えた変更はすべてのインスタンスに即座に反映されます。

インスタンスは親オブジェクトの読み取り専用コピーであり、個別に編集できません。元のオブジェクトと区別しやすいように、表面エッジは中間から淡いグレーで表示されます。

<Callout>
1. **Create Instance** コマンドを使用すると、ファスナー、パネル、繰り返し使用される構造部材など、多数の同一部品が必要な場合でも、モデルを軽量かつ保守しやすく保てます。  
2. インスタンスは **[Curve Array](/common/curve-array)**、**[Radial Array](/common/radial-array)**、**[Rectangular Array](/common/rectangular-array)** といった配列コマンドで利用できます。  
3. インスタンスは **[Mirror](/common/mirror)** コマンドでも使用でき、ライブ対称性が実現します。マスタージオメトリの変更はすべてのミラーインスタンスに自動的に反映されます。  
4. 現時点では作成したインスタンスに対して **[Place](/common/place)** および **[Copy with Placement](/common/copy-with-placement)** コマンドは使用できません。
</Callout>

## 基本的な使い方

1. 親オブジェクトとして使用したいソリッドまたはシートを選択します。  
2. コマンドパレットを開き、**インスタンス作成** と入力してコマンドを実行します。  
3. 作成されたインスタンスを目的の位置へ移動します。  
4. コマンドダイアログの **OK** をクリックするか、右クリックで確定します。

## インスタンスグループ

グループのインスタンス化は、オブジェクトのインスタンス化と同様の手順です。

グループが選択され、一次ジオメトリを含んでいる場合、グループのインスタンスが作成され、アウトライナー内の独立したカテゴリに配置されます（グループフォルダーの外側）。

<Center width="600px">
![Instancing a Group Part 1](/img/instance-group-solid.png)
</Center>

| 番号 | 説明 |
| :----: | ---- |
| 1 | コンポーネントが内部にあるメイングループ。 |
| 2 | インスタンス化される親オブジェクト。 |
| 3 | 「インスタンス」カテゴリの下に、メイングループの外側に配置されたインスタンス。 |

### 基本的な使い方

1. アウトライナーでインスタンス化したいグループを選択します。  
2. コマンドパレットで **インスタンス作成** を検索し、実行します。  
3. インスタンス化されたグループを移動します。  
4. コマンドダイアログの **OK** をクリックするか、右クリックで確定します。
