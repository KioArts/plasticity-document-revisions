# ソリッド・シートインスタンスを作成

<Reference shortcut="" command="インスタンス作成" button="コマンドパレットで"  buttonIcon="button-instance"/>

ソリッドまたはシートオブジェクトの再利用可能な複製を、元のジオメトリをコピーせずに作成します。インスタンスは変換データ（位置、ピボットポイント、回転、スケール）、マテリアルデータを保持し、元のソリッドオブジェクトを参照します。そのため、マスタージオメトリの変更はすべてのインスタンスに即座に反映されます。インスタンスは親オブジェクトの読み取り専用コピーであり、個別に編集できません。インスタンスは中〜薄灰色のエッジカラーで表示され、親オブジェクトと区別できます。

<Callout>
多数の同一部品（ねじ、パネル、繰り返し構造要素など）が必要で、モデルを軽量かつ保守しやすくしたい場合は **インスタンス作成** コマンドを使用してください。

インスタンスは以下の配列コマンドと連携します: [**カーブ配列**](/common/curve-array)、[**放射配列**](/common/radial-array)、[**矩形配列**](/common/rectangular-array)。  

さらに、[**ミラー**](/common/mirror) コマンドと組み合わせることでライブ対称が可能です。マスタージオメトリの変更は自動的にすべてのミラーインスタンスに反映されます。
</Callout>

## 基本的な使い方

1. 親オブジェクトとして使用したいソリッドまたはシートを選択します。  
2. コマンドパレットを開き、**インスタンス作成** と入力してコマンドを実行します。  
3. 作成されたインスタンスを目的の位置へ移動します。  
4. コマンドダイアログの **OK** をクリックするか、右クリックで確定します。

</Callout>

## インスタンスグループ

グループのインスタンス化は、オブジェクトのインスタンス化と同様の手順です。

グループが選択され、一次ジオメトリを含んでいる場合、グループのインスタンスが作成され、アウトライナー内の独立したカテゴリに配置されます（グループフォルダーの外側）。

![グループパーツのインスタンス化 1](/img/instance-group-solid.png)

| 番号 | 説明 |
| :----: | ---- |
| 1 | コンポーネントが内部にあるメイングループ。 |
| 2 | インスタンス化される親オブジェクト。 |
| 3 | 「インスタンス」カテゴリの下に、メイングループの外側に配置されたインスタンス。 |

### 基本的な使い方

1. アウトライナーでインスタンス化したいグループを選択します。  
2. コマンドパレットで **インスタンス作成** を検索し、実行します。  
3. インスタンス化されたグループを移動します。  
4. コマンドダイアログの **OK** をクリックするか、右クリックで確定します。
