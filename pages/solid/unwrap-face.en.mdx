## Description

Unwrap the surface of a selected face.

## Basic Usage

1. Select a face you want to unwrap.
2. Type **Unwrap** in the Command Palette.

<Callout type='info'>
  **Note:** When unwrapping a surface, the result is positioned at the center of the world XYZ origin.
</Callout>
