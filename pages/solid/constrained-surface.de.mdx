# Constrained Surface

<Reference shortcut="" command="Constrained Surface" button="" />

Erzeugen Sie eine Oberfläche, die durch einen Satz von Punkten verläuft, indem Sie auf sie klicken.

1. Verwenden Sie den **Constrained Surface**-Befehl in der Befehlspalette.
2. Klicken Sie auf die Punkte, durch die Sie verlaufen möchten.
3. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
    - **Tolerance:** Geben Sie die Toleranz der Oberfläche an.
    - **Angular tolerance:** Geben Sie die Winkeltoleranz der Oberfläche an.
    - **Optimize:**
        - **Performance:** Die Oberfläche wird schnell generiert.
        - **Smoothness:** Die Oberfläche wird glatt generiert.
    - <kbd>Cmd-Z</kbd>: Letzten Punkt rückgängig machen.
4. Bestätigen Sie mit OK im Befehlsdialog oder Rechtsklick.
