# Thicken

<Reference shortcut="" command="Thicken" button="Rechts von Command Bar" />

Fügen Sie einer ausgewählten Sheet eine Dicke hinzu. Die Dicke für jede Seite kann angegeben werden.<br />
Wenn Faces ausgewählt sind: siehe [**Thicken Face**](./thicken-face).

1. Wählen Sie das Sheet aus, das Sie verdicken möchten.
2. Klicken Sie auf den Knopf <ButtonIcon type="button-thicken" /> rechts von Command Bar.
3. Legen Sie die Optionen im Command Dialog und den Tastaturkürzeln fest.
    - **Thickness:** Geben Sie die Dicke für beide Seiten an.
    - <kbd>Tab</kbd> **Lock Distances** Gleichmäßige Dicke auf beiden Seiten.
    - <kbd>D</kbd> **Thickness** Bestimmen Sie die Dicke durch Bewegen des Cursors.
4. Bestätigen Sie mit OK im Command Dialog oder Rechtsklick.
