# Delete Face

<Reference shortcut="Shift + Backspace" command="Delete" button="" />

You can delete Faces.

## Basic Usage

1. Select the Faces you want to delete.
2. Press <kbd>Shift + Delete</kbd>.
3. the Faces are deleted.
