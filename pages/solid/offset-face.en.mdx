# Offset Face

You can Offset Faces.

1. Select the Faces you want to offset.
2. When a face is selected, the default command, **Offset Face**, is automatically executed.
3. Move the yellow dot to specify the distance.
4. Set the options in the command dialog and shortcut key.
     - **Distance:** Specify the distance.
     - **Degrees:** Specify the Angle.
     - **Grow:**
          - When Offset Face on any of the 3 booled inside faces. 
          - **Moving:** Move the whole wall when the offset gets to the outerwall.
          - **Fixed:** Stop at the outerwall.
          - **None:** Keep going by itself neither affecting or affected by the outerwall.
     - <kbd>D</kbd> **Distance** Specifies the distance by moving the cursor.
     - <kbd>A</kbd> **Angle adjacent** Specifies the Angle by moving the cursor.
     - <kbd>Tab</kbd> **Grow mode** Switches the grow mode.
5. Confirm with OK in the command dialog or right click.
