## Description

Create a pipe of any size from a Curve or an Edge. It can be used for boolean operations. Polygon or custom profile options are available.

## Basic Usage

1. Select the Curve or Edge you want to pipe. When you select multiple curves or edges, multiple pipes will be created.
2. Press <kbd>P</kbd>.
3. Drag the yellow dot to specify the section size and the yellow square to specify the pipe thickness. Drag the white circle to specify the rotation angle.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
     - <Kst>Command Dialog</Kst>
          - **Select target bodies:**
               - Same function as the Boolean command. See [**Boolean**](/solid/boolean) command for details.
          - **Select custom profile:** Select Regions, Curves or Faces to specify the custom profiles.
          - **Vertex count:** Specify the number of Vertices for the profile's polygon. If you specify 0, it will be circular.
          - **Diameter:** Specify the section size.
          - **Thickness:** Specify the pipe wall thickness.
          - **Angle:** Specify the rotation angle.
          - **Scale:** Specify the scale value on the ending point of the pipe. Click on the option dialog and manually enter your value or drag with your mouse from left to right. To reverse the ending point of the pipe, use [Reverse Command](/sketch/reverse).
          - **Distance:** Adjust the result length. Decreasing the value from 1.00 to 0 shortens the length. (Default: 1.00)
               - **Distance 1:** Adjust the beginning of the length.
               - **Distance 2:** Adjust the ending of the length.
     - <Kst>Keyboard Shortcuts</Kst>
          - <kbd>Shift + Q</kbd> Slice, <kbd>Q</kbd> Union, <kbd>W</kbd> Difference,  <kbd>B</kbd> New Body,  <kbd>T</kbd> Keep Tools
               - Same function as the Boolean command. See [**Boolean**](/solid/boolean) command for details.
          - <kbd>C</kbd> **Custom profile** Select the Regions, Curves or Faces to specify the custom profiles.
          - <kbd>D</kbd> **Section size** Specify the section size by moving the cursor.
          - <kbd>A</kbd> **Angle** Specify the rotation angle by moving the cursor.
          - <kbd>Shift + T</kbd> **Thickness** Specify the pipe wall thickness by moving the cursor.
5. Confirm with OK in the Command Dialog or right-click.

<Callout type="info">
**Note:**<br />
The **Pipe** command is *not* intended for creating dimensionally accurate pipes. Its purpose is to quickly generate visually appealing pipes or tubes.

Because it approximates angles, the resulting geometry is not suitable for precise plumbing, mechanical design, or any situation requiring exact dimensions. To create an accurate, fully controlled pipe, use the [**Sweep**](sweep) command instead.

</Callout>
