# Pipe

<Reference shortcut="P" command="Pipe" button="Right of Command Bar" />

You can create a pipe along a curve or edge.

1. Select the curve or edge you want to pipe.
2. Press <kbd>P</kbd>.
3. Set the options in the command dialog and Shortcut Key.
    <Option_Boolean />
    - **Select custom profile:** 
        - Click on the Regions, Curves or Faces to specify the custom profiles.
    - **Vertex count:**
        - Specify the number of vertices for the profile's polygon. If you specify 0, it will be circular.
    - **Section size:**
        - Specify the section size.
    - **Thickness:**
        - Specify the pipe thickness.
    - **Angle:**
        - Specify the rotation angle.
    - <kbd>C</kbd> **Custom profile**
        - Click on the Regions, Curves or Faces to specify the custom profiles.
    - <kbd>D</kbd> **Section size**
        - Specify the section size with mouse movement.
    - <kbd>A</kbd> **Angle**
        - Specify the rotation angle with mouse movement.
    - <kbd>Shift-T</kbd> **Thickness**
        - Specify the pipe thickness with mouse movement.
    - **Drag the yellow circle**
        - Specify the section size.
    - **Drag the yellow square**
        - Specify the pipe thickness.
    - **Drag the white circle**
        - Specify the rotation angle.
4. Confirm with OK in the command dialog or right click.
