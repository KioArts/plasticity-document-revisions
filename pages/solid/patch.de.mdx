## Patch-Erstellung

Sie können Patch-Oberflächen aus geschlossenen Kurven oder aus Löchern entlang von Kanten erzeugen. Für Löcher stehen Optionen für G0-, G1-, G2- sowie Ebenenkontinuität und Führungskurven zur Verfügung.

### Patch aus geschlossenen Kurven erstellen

Sie können eine Patch-Oberfläche aus geschlossenen Kurven erzeugen.

1. Wählen Sie alle geschlossenen Kurven aus, für die Sie einen Patch erstellen möchten.
2. Suchen Sie im **Suggested Commands Menu** oder in der **Command Palette** nach **Patch** und führen Sie den Befehl aus.
3. Nehmen Sie die Einstellungen im Befehlsdialog vor.
    - <Kst>Befehlsdialog</Kst>
        - **Toleranz:** Legt die Toleranz für den Patch fest.
4. Klicken Sie im Befehlsdialog auf „OK“ oder bestätigen Sie mit Rechtsklick.

## Patch für Löcher entlang von Kanten

Sie können Löcher entlang von Kanten mit einem Patch schließen.

1. Wählen Sie die Kanten aus, die das Loch umschließen.
2. Suchen Sie im **Suggested Commands Menu** oder in der **Command Palette** nach **Patch** und führen Sie den Befehl aus.
3. Nehmen Sie die Einstellungen im Befehlsdialog vor.
    - <Kst>Befehlsdialog</Kst>
        - **Kontinuität:**
            - **Ebene:** Legt die Oberflächenkontinuität auf Ebene fest.
            - **G0, G1, G2**
                - Legt die Oberflächenkontinuität fest ([Kontinuität – Kurven und Flächen](/cad-essentials/continuity-curve-and-surface)).
                - Durch Klicken auf die schwarzen Kreise kann die Kontinuität individuell festgelegt werden.
        - **Fläche:**
            - **Einzeln:**
            - **Minimal:**
            - **Qualität:**
        - **Toleranz:** Legt die Toleranz für den Patch fest.
4. Klicken Sie im Befehlsdialog auf „OK“ oder bestätigen Sie mit Rechtsklick.

## Erstellen eines Sheets mit Patch-Loch und Führungskurven

## Patch mit Führungskurven

Sie können Führungskurven verwenden, um ein Loch entlang einer Kante zu patchen.

1. Erstellen Sie eine Führungskurve.
2. Fügen Sie an jedem Punkt, an dem die Führungskurve die Oberfläche berührt, einen Isoparam hinzu. Verwenden Sie den Befehl [**Isoparam**](isoparam).
3. Wählen Sie die Kanten aus, die das Loch umschließen.
4. Suchen Sie im **Suggested Commands Menu** oder in der **Command Palette** nach **Patch** und führen Sie den Befehl aus.
5. Wählen Sie die Führungskurve aus.
6. Nehmen Sie die Einstellungen im Befehlsdialog vor.
    - Die Optionen entsprechen denen oben.
7. Klicken Sie im Befehlsdialog auf „OK“ oder bestätigen Sie mit Rechtsklick.

## Mehrere Löcher patchen

Sie können mehrere Löcher gleichzeitig patchen.

1. Wählen Sie das Sheet aus, das mehrere zu patchende Löcher enthält.
2. Suchen Sie im **Suggested Commands Menu** oder in der **Command Palette** nach **Patch** und führen Sie den Befehl aus.
3. Die Löcher werden gepatcht und das Ergebnis wird zu einem Solid.
<Callout type="warning">
**Hinweis:** Wenn ein Loch nicht gefüllt werden kann, schlägt der Befehl fehl.
</Callout>

## Sheet-Trimming mit Patch

Mit dem Patch-Tool können Sie ein Sheet trimmen.

1. Wählen Sie das zu trimmende Ziel-Sheet aus und blenden Sie es vorübergehend aus.
2. Wählen Sie das Basissheet aus und wählen Sie den Kantenloop des Lochs.
3. Wechseln Sie in den Body-Auswahlmodus, blenden Sie das Ziel-Sheet wieder ein und nehmen Sie die zusätzliche Auswahl zusammen mit dem Basissheet vor.
5. Suchen Sie im **Suggested Commands Menu** oder in der **Command Palette** nach **Patch** und führen Sie den Befehl aus.
6. Nehmen Sie die Einstellungen im Befehlsdialog vor.
    - <Kst>Befehlsdialog</Kst>
        - **Methode:**
            - **Trimmen zum Loch:** Trimmt das Ziel-Sheet zum Loch des Basissheets.
            - **Trimmen zum Sheet:** Trimmt das Ziel-Sheet entlang des Randabschnitts des Basissheets.
        - **Toleranz:** Legt die Toleranz für den Patch fest.
