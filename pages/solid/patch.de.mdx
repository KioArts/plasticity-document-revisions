## Beschreibung

Erstellt eine Patch‑Fläche aus geschlossenen Kurven oder Löchern entlang von Kanten. Für Löcher stehen G0-, G1-, G2- und planare Stetigkeitsoptionen sowie Führungskurven zur Verfügung.

## Patch aus Kurven erstellen

Sie können eine Patch‑Fläche aus geschlossenen Kurven erzeugen.

1. Wählen Sie die geschlossenen Kurven aus, die gepatcht werden sollen.
2. Suchen Sie **Patch** im Menü „Vorgeschlagene Befehle“ oder in der Befehlspalette und führen Sie ihn aus.
3. Legen Sie die Optionen im Befehlsdialog fest.
    - <Kst>Befehlsdialog</Kst>
        - **Toleranz:** Gibt die Toleranz für den Patch an.
4. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

## Löcher entlang von Kanten patchen

Sie können Löcher entlang von Kanten patchen.

1. Wählen Sie die Kanten aus, die das Loch umschließen.
2. Suchen Sie **Patch** im Menü „Vorgeschlagene Befehle“ oder in der Befehlspalette und führen Sie ihn aus.
3. Legen Sie die Optionen im Befehlsdialog fest.
    - <Kst>Befehlsdialog</Kst>
        - **Stetigkeit:**
            - **Planar:** Legt die Stetigkeit der Fläche als planar fest.
            - **G0, G1, G2**
                - Legt die [Stetigkeit der Fläche](/cad-essentials/continuity-curve-and-surface) fest.
                - Die Stetigkeit kann individuell durch Klicken auf die schwarzen Kreise auf der Fläche eingestellt werden.
        - **Flächen:**
            - **Einzeln:**
            - **Minimal:**
            - **Qualität:**
        - **Toleranz:** Gibt die Toleranz für den Patch an.
4. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

## Blechfläche mit Patch‑Loch erstellen

Sie können einen neuen Blechkörper erstellen, indem Sie Kanten eines Volumenkörpers auswählen.

1. Wählen Sie die Kanten eines Volumenkörpers aus.
2. Suchen Sie **Patch** im Menü „Vorgeschlagene Befehle“ oder in der Befehlspalette und führen Sie ihn aus.
3. Legen Sie die Optionen im Befehlsdialog fest.
    - <Kst>Befehlsdialog</Kst>
        - **Toleranz:** Gibt die Toleranz für den Patch an.
4. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

## Patch mit Führungskurven

Sie können Löcher entlang von Kanten mit Führungskurven patchen.

1. Erstellen Sie die Führungskurven.
2. Fügen Sie an jedem Punkt, an dem die Führungskurve die Fläche berührt, einen Isoparam ein. Verwenden Sie den Befehl [**Isoparam**](isoparam).
3. Wählen Sie die Kanten aus, die das Loch umschließen.
4. Suchen Sie **Patch** im Menü „Vorgeschlagene Befehle“ oder in der Befehlspalette und führen Sie ihn aus.
5. Wählen Sie die Führungskurven aus.
6. Legen Sie die Optionen im Befehlsdialog fest.
    - Optionen sind identisch wie oben.
7. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

## Mehrere Löcher patchen

Sie können mehrere Löcher gleichzeitig patchen.

1. Wählen Sie die Bleche mit mehreren Löchern aus, die gepatcht werden sollen.
2. Suchen Sie **Patch** im Menü „Vorgeschlagene Befehle“ oder in der Befehlspalette und führen Sie ihn aus.
3. Die Löcher werden gepatcht und zu Volumenkörpern geschlossen.
<Callout type="warning">
**Hinweis:** Wenn ein Loch nicht gefüllt werden kann, schlägt der Befehl fehl.
</Callout>

## Blech mit Patch trimmen

Sie können ein Blech mit dem Patch‑Werkzeug trimmen.

1. Wählen Sie das Zielblech aus, das Sie trimmen möchten, und blenden Sie es vorübergehend aus.
2. Wählen Sie das Basisblech aus und erstellen Sie eine Kantenloop‑Auswahl um das Loch.
3. Wechseln Sie in den Körperauswahlmodus, blenden Sie das Zielblech wieder ein und fügen Sie es der Auswahl hinzu.
5. Suchen Sie **Patch** im Menü „Vorgeschlagene Befehle“ oder in der Befehlspalette und führen Sie ihn aus.
6. Legen Sie die Optionen im Befehlsdialog fest.
    - <Kst>Befehlsdialog</Kst>
        - **Methode:**
            - **Trim to hole:** Trimmt das Zielblech zum Loch des Basisblechs.
            - **Trim to sheet:** Trimmt das Zielblech, sodass es mit dem Endabschnitt des Basisblechs übereinstimmt.
        - **Toleranz:** Gibt die Toleranz für den Patch an.
