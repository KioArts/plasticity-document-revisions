# Revolve

<Reference shortcut="" command="Revolve" button="Right of Command Bar" />

You can revolve regions or curves to create solids or sheets.

1. Select the region or curve to revolve.
2. Click the button.
3. Specify the revolution axis. 
   - Revolve along the axis defined by the first click as the start point and the second click as the endpoint.
   - <kbd>Shift-T</kbd> **Thickness:**
   - <kbd>A</kbd> **Angle:** Move the mouse to specify the rotation angle.
5. Confirm with OK in the command dialog or right-click.
