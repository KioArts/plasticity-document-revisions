# Unjoin

<Reference shortcut="Alt + J" command="Unjoin" button="In der Befehlsleiste" buttonIcon='button-join' />

Der Unjoin-Befehl wird basierend auf dem ausgewählten Objekt ausgeführt:
   - **Unjoin-Kurven:** Wenn Kurven ausgewählt sind, siehe [**Unjoin-Kurven**](/sketch/unjoin-curve) im Abschnitt „Sketch-Befehle“.
   - **Unjoin-Flächen:** Wenn Flächen ausgewählt sind, siehe [**Unjoin-Flächen**](/solid/unjoin-faces).
   - **Unjoin-Schalen:** Wenn Festkörper oder Flächen ausgewählt sind, siehe [**Unjoin-Schalen**](/solid/unjoin-shells).
