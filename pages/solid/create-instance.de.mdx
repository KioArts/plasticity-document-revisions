# Create Instance

Der Befehl **Create Instance** wird basierend auf dem ausgewählten Objekt ausgeführt:
     - **Create Solid and Sheet Instance:** Wenn Solids oder Sheets ausgewählt sind, siehe [**Create Solid and Sheet Instance**](create-solid-and-sheet-instance).
     - **Create Curve Instance:** Wenn Kurven ausgewählt sind, siehe [**Create Curve Instance**](/sketch/create-curve-instance).
