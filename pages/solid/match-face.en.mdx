# Match Face

You can replace a faces with another face.

1. Select faces to match.
2. Click the button <ButtonIcon type="button-match-face" />.
3. Select the surface to replace with.
4. Set the options in the Command Dialog.
    - **Grow:**
        - When the concave inner face is offseted.
        - **Moving:** Move the whole wall when the offset gets to the outerwall.
        - **Fixed:** Stop at the outerwall.
        - **None:** Keep going by itself neither affecting or affected by the outerwall.
    - **Side:** 
        - **Front:** When enabled, the face will be matched to the front face of the selected surface.
5. Confirm with OK in the Command Dialog or right-click.
