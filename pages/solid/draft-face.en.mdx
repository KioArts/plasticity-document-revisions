# Draft Face

You can draft a Face.

1. Select the Face as a reference plane.
2. Click the button <ButtonIcon type="button-draft-face" /> on the Right of Command Bar.
3. Select Faces to draft.
4. Drag the white circle to specify the angle of draft.
5. Set the options in the Command Dialog and Keyboard Shortcuts.
    - **Degrees:** Specify the angle of draft.
    - **Grow:**
        - When the concave inner Face is drafted.
        - **Moving:** Extend the outerwall when the drafted Face gets to the outerwall.
        - **Fixed:** Stop at the outerwall.
        - **None:** Keep going by itself neither affecting or affected by the outerwall.
    - <kbd>A</kbd> **Angle adjacent** Specify the angle by moving the cursor.
6. Confirm with OK in the Command Dialog or right-click.
