# Match Face

<Reference shortcut="" command="Match Face" button="推奨コマンドメニューで" buttonIcon="button-match-face"/>

フェイスを他のフェイスに置き換えることができます。

## 基本的な使い方

1. 置き換えたいフェイスを選択します。
2. 推奨コマンドメニュー ([Suggested Commands](/plasticity-essentials/command-bar#suggested-commands)) 内の <ButtonIcon type="button-match-face" /> ボタンをクリックしてください。
3. 置き換え先のサーフェイスを選択します。
4. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Grow:**
            - 凹面の内側のフェイスを合わせるとき。
            - **Moving:** 外壁に到達したら、壁全体を移動します。
            - **Fixed:** 外壁で停止します。
            - **None:** 外壁の影響を受けずに、フェイスが進み続けます。
        - **Side:** 
            - **Front:** 有効にすると、選択された面の前面に一致します。
5. コマンドダイアログでOKをクリックするか、右クリックで確定します。
