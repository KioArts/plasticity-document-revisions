# Create Instance

<Reference shortcut="" command="Create Instance" button="In Command Palette" buttonIcon="button-instance" />

The *Create Instance* command runs based on the selected object:
     - **Create Solid and Sheet Instance:** Solids or Sheets are selected, see [**Create Solid and Sheet instance**](create-solid-and-sheet-instance).
     - **Create Curve Instance:** If Curves are selected, see [**Create Curve Instance**](/sketch/create-curve-instance).
