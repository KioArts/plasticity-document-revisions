# Slide

<Reference shortcut="Shift + G" command="Raise Degree" button="Right of Command Bar" buttonIcon="button-raise-degree-curve-face"/>

Unified command for **Slide Curve CV**, **Slide Surface CV**. The command is run based on the selected object.
  - When CVs on Curves are selected: see [**Slide Curve CV**](/sketch/slide-curve-cv).
  - When CVs on Solids are selected: see [**Slide Surface CV**](/solid/slide-surface-cv).
