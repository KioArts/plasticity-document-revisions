## Description

A Nominal Surface is a feature that heals holes on a surface area by generating additional spans in a hidden manner when working with visible surface CVs, allowing for easy direct surface editing. The Remove Nominal Surface command displays the additional hidden spans on the surface area. 

## Basic Usage 

1. Select a face you want to reveal hidden CVs.
2. Use the **Remove Nominal Surface** command in the Command Palette.

<Callout type="info">
**Tip:**    Use the command [Rebuild](/solid/rebuild) command in the Command Palette to rebuild the surface area and remove unnecessary spans of the surface.
</Callout>
