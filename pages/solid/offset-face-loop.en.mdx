## Description

Offset the outline edges of selected Faces by specifying the distance to create new Edges. Multiple Faces create new Individual Edges or a combined Edge.

## Basic Usage

1. Select the Faces you want to offset.
2. Press <kbd>O</kbd>.
3. Move the mouse to specify the distance.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - <Kst>Command Dialog</Kst>
      - **Distance:** Specify the distance.
      **Gap fill:**
            - **Round:** Fills convex corners with a rounded curve.
            - **Linear:** Fills convex corners with straight lines, but the resulting edges may be fragmented rather than continuous.
            - **Natural:** Fills convex corners with straight lines while maintaining continuous edges.
            <img src="/img/offset-gap-fill.svg" width="450" />
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>V</kbd> **Gap fill:** Cycle through gap fill options.
      - <kbd>S</kbd> **Lock distances:** Offset with equal distance on both sides.
      - <kbd>D</kbd> **Distance:** Specify the distance with mouse movement.
      - <kbd>I</kbd> **Individual:** When multiple Faces are selected, turning on this option offsets the Faces in a combined form.
5. Confirm with OK in the Command Dialog or right-click.
