# Isoparam

You can add Edges to a Face of a Solid or Sheet.

1. Press <kbd>Ctrl-R</kbd>.
2. Place the mouse cursor on the position where you want to create the edge.
3. Set the options in the Shortcut Key.
   - <kbd>Tab</kbd> **U/V direction:** Toggle the direction of the Edge to create.
   - <kbd>Shift</kbd>-Wheel up **Add cut:** Increases the number of Edges.
   - <kbd>Shift</kbd>-Wheel down **Subtract cut:** Reduces the number of Edges.
4. Confirm with right click.
