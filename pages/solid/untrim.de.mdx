# Untrim

<Reference shortcut="Alt + T" command="Untrim" button="" />

Erstellen Sie ein Sheet basierend auf der ursprünglichen Form des ausgewählten Face.

## Grundlegende Nutzung

1. Wählen Sie das Face aus, das Sie rückschneiden möchten.
2. Drücken Sie <kbd>Alt + T</kbd>.
- <Kst>Befehlsdialog</Kst>
    - **Keep edges:** Wählen Sie aus, ob eingeprägte Linien eingefügt werden sollen, um die Kanten der beschnittenen Fläche zu erhalten.
- <Kst>Tastenkombinationen</Kst>
    - <kbd>Tab</kbd> **Keep edges:** Schaltet die Option zum Beibehalten der Kanten der beschnittenen Fläche um.
3. Ein Sheet basierend auf der ursprünglichen Form wird erstellt.
