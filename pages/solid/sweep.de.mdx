# Sweep

<Reference shortcut="Umschalt + P" command="Sweep" button="Im vorgeschlagenen Befehlsmenü" buttonIcon="button-sweep" />

Das Kehren von Faces oder Regions erzeugt Solids, während das Kehren von Curves Sheets produziert. Bestimmen Sie den Verdrehungswinkel und wählen Sie optional mehrere Kehrführungen aus.

## Grundlegende Nutzung

1. Wählen Sie die Regions, Faces oder Curves, die gekehrt werden sollen.
2. Drücken Sie <kbd>Umschalt + P</kbd>.
3. Wählen Sie die Curve aus, die als Pfad für das Kehren dienen soll.
4. Legen Sie die Optionen im Command Dialog und den Tastaturkürzeln fest.
   - <Kst>Befehlsdialog</Kst>
      - **Twist:** Bestimmen Sie den Verdrehungswinkel in Grad.
      - **Scale:** Geben Sie den Skalierungswert am Endpunkt des Rohrs an. Klicken Sie auf den Optionsdialog und geben Sie Ihren Wert manuell ein oder ziehen Sie mit der Maus von links nach rechts.
      - **Alignment:** 
         - **Parallel:** Kehrt parallel zum Pfad, unabhängig von der Orientierung.
         - **Normal:** Kehrt in einem Winkel, der mit dem Pfad ausgerichtet ist.
      - **Distance:**
         - Stellen Sie die Länge des Ergebnisses ein. Eine Verringerung des Werts von 1,00 auf 0 verkürzt die Länge. (Standardwert: 1,00)
   - <Kst>Advanced Dialog</Kst>
      - **Vereinfachen:** Geben Sie an, ob beschnittene planare Abschnitte erzeugt oder die einheitliche Geometrie beibehalten werden soll. (Standard: AUS)
      - **Ecktyp:**
         - Gehrung: Einen externen spitzen Eckpunkt haben.
         - Rund: Einen externen runden Eckpunkt haben.
   - <Kst>Befehlsdialog</Kst>
      - <kbd>D</kbd> **Twist:** Bestimmen Sie den Twist mit Mausbewegung.
5. Wählen Sie optional Kehrführungen aus.
      1. Stellen Sie sicher, dass **Select guides:** im Command Dialog hervorgehoben ist.
      2. Wählen Sie die Curves aus, die als Führungen für das Kehren dienen sollen.
         - Verwenden Sie <kbd>Umschalt</kbd> + Klick, um mehrere Führungen auszuwählen.
         - Verwenden Sie <kbd>Strg</kbd> + Klick zum Abwählen.
         - Die Reihenfolge, in der Sie die Führungen auswählen, beeinflusst die Form des Kehrens.
      3. Legen Sie die Optionen im Command Dialog fest.
         - <Kst>Befehlsdialog</Kst>
            - **Guide method:**
               - **Point:** Der Querschnitt des gekehrten Körpers wird gedreht und skaliert, sodass die Punkte, die ursprünglich mit der Führung und dem Pfad übereinstimmen, immer in Kontakt bleiben.
               - **Chord:**  Der Querschnitt wird gedreht, aber nicht skaliert.
               - **Curve:** Der Querschnitt wird gedreht, aber nicht skaliert. Die Führung und der Pfad müssen das Profil berühren. Das Profil bleibt mit beiden Führungsdrähten in Kontakt, aber nur einer der Berührungspunkte befindet sich an einem festen Punkt auf dem Profil; der andere kann sich entlang des Profils bewegen.
               Hinweis: Das Profil muss groß genug sein, um die größte Entfernung zwischen den beiden Führungsdrähten zu überbrücken.
6. Bestätigen Sie mit OK im Command Dialog oder Rechtsklick.
7. Das erstellte Objekt hängt von der ursprünglichen Auswahl ab.
   - Wenn Sie Regions und Faces auswählen, wird ein Solid erstellt.
   - Wenn Sie eine Curve auswählen, wird ein Sheet erstellt.
