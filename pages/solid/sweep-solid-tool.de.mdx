## Sweep Solid Tool

<Reference shortcut="" command="Sweep Tool" button="" />

Das Sweepen eines Solids entlang einer Kurve als Referenzkörper erzeugt ein einzigartiges Solid.

### Grundlegende Verwendung

1. Aktivieren Sie den Befehl, indem Sie „Sweep Tool“ in der Befehlsleiste suchen.
2. Wählen Sie das Solid aus, das Sie sweepen möchten.
3. Wählen Sie die Kurve, die als Pfad für den Sweep dient.
4. Stellen Sie die Optionen im Befehlsdialog und mit den Tastenkürzeln ein.
   - <Kst>Tastenkürzel</Kst>
      - <kbd>S</kbd> **Skalierung:** Geben Sie den Skalierungswert an.
      - <kbd>A</kbd> **Winkel:** Geben Sie den Winkelwert an.
      - <kbd>D</kbd> **Offset:** Geben Sie den Offset-Wert an.
5. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
