# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="" />

This command automatically calls the following command depending on the selected object.

- When curves are selected: [**"Offset Planar Curve"**](../sketch/offset-curve#offset-planar-curve) in Sketch's **"Offset Curve"**.
- When Regions are selected: [**"Offset Region"**](../sketch/offset-curve#offset-region) in Sketch's **"Offset Curve"**.
- When Faces are selected: see **"Offset Face Loop"**] described below.
- When Edges are selected: see **"Offset Edge"**] described below.
