# Unjoin Faces [#unjoin-faces]

<Reference shortcut="Alt + J" command="Unjoin" button="In Suggested Commands Menu"  buttonIcon='button-unjoin' />

Separate selected Faces from Solid or Sheet.

## Basic Usage

1. Select the Faces you want to separate.
2. Press <kbd>Alt + J</kbd>.
3. The Faces are separated.
