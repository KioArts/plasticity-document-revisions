# Boolean（ブーリアン）

<Reference shortcut="Q" command="Boolean" button="コマンドバー左" />

ソリッドまたはシートをブーリアンできます。
{/* 
1. ショートカット<kbd>Q</kbd>を押すか、ボタンをクリックします。
2. ターゲットとツールを指定します。
   - **Select target bodies**: ブーリアン対象となるソリッドまたはシートを選択します。
   - **Select tool bodies**: ツールとなるソリッドまたはシートを選択します。
4. ブーリアン操作を指定します。
   - <kbd>G</kbd> **Move**
        - ツールを移動できます。
   - <kbd>R</kbd> **Rotate**
        -ツールを回転できます。
        - <kbd>R</kbd> **Screen space**
            - 回転軸をもとに戻します。
   - <kbd>S</kbd> **Scale**
        -ツールを拡大縮小できます。
        - <kbd>S</kbd> **Uniform**
            - X、Y、Z軸方向に同じだけ拡大縮小します。
   - そのほかのオプションは **ブーリアンオプション** 🚧と **ソリッド編集の共通オプション** 🚧を参照してください。
5. コマンドダイアログのOKまたは右クリックで確定します。
 */}
