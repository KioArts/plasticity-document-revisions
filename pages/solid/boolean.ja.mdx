# Boolean（ブーリアンする）

<Reference shortcut="Q" command="Boolean" button="Left of Command Bar" />

ソリッドとシートの組み合わせでブーリアンを行うことができます。

1. <kbd>Q</kbd>を押すか、ボタン <ButtonIcon type="button-boolean" /> をクリックします。
2. ターゲットとツールを選択します。
   - **Select target bodies:** ブーリアン対象となるソリッドまたはシートを選択します。
   - **Select tool bodies:**  ブーリアンのツールとなるソリッドまたはシートを選択します。
3. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - **Keep Tools:** ブーリアンが実行され、ツールのオブジェクトはそのまま残ります。
   - <kbd>Shift + Q</kbd> **Slice:** ブーリアン対象との交点にソリッドを作成し、余りのソリッドも残ります。
   - <kbd>Q</kbd> **Union:** ブーリアン対象と結合します。
   - <kbd>W</kbd> **Difference:** ブーリアン対象から切り取ります。
   - <kbd>Shift + E</kbd> **Intersect:** ブーリアン対象との交点にソリッドを作成します。
   - <kbd>T</kbd> **Keep Tool:** ブーリアンが実行され、ツールのオブジェクトはそのまま残ります。
   - <kbd>G</kbd> **Move:** Moveコマンドと同じ操作でツールを移動します。
   - <kbd>R</kbd> **Rotate:** Rotateコマンドと同じ操作でツールを回転します。
   - <kbd>S</kbd> **Scale:** Scaleツールと同じ操作でツールを拡大縮小します。
   - <kbd>X</kbd> **X-axis:** X軸方向に限定します。
   - <kbd>Y</kbd> **Y-axis:** Y軸方向に限定します。
   - <kbd>Z</kbd> **Z-axis:** Z軸方向に限定します。
   - <kbd>Shift + X</kbd> **X plane:** X平面に限定します。
   - <kbd>Shift + Y</kbd> **Y plane:** Y軸平面に限定します。
   - <kbd>Shift + Z</kbd> **Z plane:** Z軸平面に限定します。
4. コマンドダイアログでOKをクリックするか、右クリックで確定します。
