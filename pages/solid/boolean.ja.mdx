# Boolean（ブーリアンする）

<Reference shortcut="Q" command="Boolean" button="コマンドバーで " buttonIcon="button-boolean"/>

ソリッドとシートの組み合わせでブーリアンを行うことができます。

## 基本的な使い方

1. <kbd>Q</kbd> を押してコマンドを実行します。
2. ターゲットとツールを選択します。
   - <Kst>コマンドダイアログ</Kst>
      - **Select target bodies:** 
         - このボタンがハイライトされているとき、ブーリアンのターゲットとして使用するソリッドまたはシートを選択できます。
         - ターゲットを変更するには、このボタンをもう一度クリックして、ソリッドまたはシートを選択します。
    - **Select tool bodies:**  
         - このボタンがハイライトされているとき、ブーリアンのツールとして使用するソリッドまたはシートを選択できます。
         - ツールを変更するには、このボタンをもう一度クリックして、ソリッドまたはシートを選択します。
3. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>Command Dialog</Kst>
      - **Keep Tools:** ツールオブジェクトをそのまま残してブーリアン演算を実行します。
      - **Opetation:**
         - **Union:** ターゲットとツールオブジェクトを結合して1つのソリッドにします。
         - **Difference:** ツールオブジェクトをターゲットから削除します。
         - **Intersect:** ターゲットとツールの交差部分のみをソリッドとして作成します。
         - **Slice:** ターゲットとの交差部分を分割してソリッドを作成し、残りの部分も保持します。
         - **Region:** ソリッドやシートの集合体内で閉じた領域からソリッドを作成します。
         - 操作による結果の違いについては、[通常のブーリアン演算](#normal-boolean-operations)のセクションを参照してください。
      - **Advanced:**
         - **Target Material:** ブーリアン演算中にターゲットオブジェクトをソリッドとして扱うか、シートとして扱うかを定義します。この設定は、特に「差分」や「交差」などの操作で、ツールとの相互作用に影響します。
            - **Default:** オブジェクトの元の状態（ソリッドまたはシート）を使用します。
            - **Empty:** オブジェクトを空の殻として扱います。
            - **Inside:** ツールの法線方向に基づいて、内側からソリッドとして扱います。
            - **Outside:** ツールの法線方向に基づいて、外側からソリッドとして扱います。
         - **Tool Material:** Controls how the tool object behaves in the Boolean operation. You can choose to treat it as a solid or a sheet, affecting how it interacts with parts of the target.
            - **Default:** オブジェクトの元の状態を使用します。
            - **Empty:** オブジェクトを空の殻として扱います。
            - **Inside:** ターゲットの法線方向に基づいて、内側からソリッドとして扱います。
            - **Outside:** Treats the body as a solid object from the outside based on the normal direction of Target object.
         - 操作による結果の違いについては、[高度なブーリアン演算の違い](#advanced-boolean-operations-differences)のセクションを参照してください。
         - <kbd>Shift + Q</kbd> **Slice:** 上記と同じです。
         - <kbd>Q</kbd> **Union:**  上記と同じです。
         - <kbd>W</kbd> **Difference:**  上記と同じです。
         - <kbd>Shift + E</kbd> **Intersect:**  上記と同じです。
         - <kbd>T</kbd> **Keep Tool:**  上記と同じです。
         - <kbd>G</kbd> **Move:** Moveコマンドと同じ操作でツールを移動します。
         - <kbd>R</kbd> **Rotate:** Rotateコマンドと同じ操作でツールを回転します。
         - <kbd>S</kbd> **Scale:** Scaleツールと同じ操作でツールを拡大縮小します。
         - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
         - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z-plane:** X, Y, Z平面に拘束します。
4. コマンドダイアログでOKをクリックするか、右クリックで確定します。
