# Refillet Face

<Reference shortcut="" command="Refillet Face" button="" />

You can redo the fillet without having to select individual fillet surfaces.

1. Select all surfaces, including the one you want to redo the fillet.
2. Use **Refillet Face** in the Command Palette.
3. Move the yellow dot to specify the radius of the fillet.
4. Set the options by shortcut key.
    - <kbd>D</kbd> **Distance:** Specify the distance by moving the cursor.
5. Confirm with a right-click.
