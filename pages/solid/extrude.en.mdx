# Extrude

<Reference shortcut="E" command="Extrude" button="Right of Command Bar" />

Extruding faces or regions generates solids, while extruding Curves produces sheets.

1. Press <kbd>E</kbd> or click the button <ButtonIcon type="button-extrude" />.
2. Select the faces, regions, or curves to extrude. 
3. Move the yellow dot to specify the distance.
4. Set the options in the command dialog and shortcut key.
- **Select target bodies:** Select the solid or sheet to be used as the boolean target.
      - <kbd>Shift-Q</kbd> **Slice:** Creates a solid at the intersections with the Boolean target and preserves the remaining solids.
      - <kbd>Q</kbd> **Union:** Joins the boolean target and tool together.
      - <kbd>W</kbd> **Difference:** Removes the tool from the boolean target.
      - <kbd>Shift-E</kbd> **Intersect:** Creates a solid at the intersection of the boolean target and tool.
      - <kbd>B</kbd> **New Body:** Creates a new solid without performing Boolean operations.
      - <kbd>T</kbd> **Keep Tool:** Performs Boolean operations while keeping the tool objects intact.
   - **Distance:** Specify the distance and the opposite distance.
   - **Degrees:** Specify the angle of extrusion.
   - **Thickness:** Specify the wall thickness.
   - **Keep tools:** Same as <kbd>T</kbd>
   - <kbd>F</kbd> **Freestyle** Specify the starting point and the endpoint, determining the direction and distance of Extrude.
   - <kbd>V</kbd> **Pivot** Specify the position of the Gizmo. 
   - <kbd>D</kbd> **Distance 1** Specify the distance by moving the cursor.
   - <kbd>A</kbd> **Angle 1** Specify the angle of extrusion by moving the cursor.
   - <kbd>Shift-T</kbd> **Thickness** Specify the wall thickness by moving the cursor.
   - <kbd>Tab</kbd> **Lock distances** Extrude with equal distance on both sides.
5. Confirm with OK in the command dialog or right-click.
