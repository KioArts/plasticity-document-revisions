# Unjoin

<Reference shortcut="Alt + J" command="Unjoin" button="In Command Bar" buttonIcon='button-join' />

The Unjoin command runs based on the selected object:
     - **Unjoin Curves:** If Curves are selected, see [**Unjoin Curves**](/sketch/unjoin-curve) in the Sketch Commands section.
     - **Unjoin Faces:** If Faces are selected, see [**Unjoin Faces**](/solid/unjoin-faces).
     - **Unjoin Shells:** If Solids or Sheets are selected, see [**Unjoin Shells**](/solid/unjoin-shells).
