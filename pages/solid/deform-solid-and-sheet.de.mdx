# Deform Solid and Sheet

<Reference shortcut="" command="Deform" button="" />

Wickeln Sie vorhandene Solids- und Sheet-Objekte auf eine regelmäßige oder unregelmäßige Oberfläche.

## Grundlegende Nutzung

1. Wählen Sie die Objekte aus, die Sie umhüllen möchten.
2. Geben Sie **Deform** in die Befehlsleiste ein.
3. Wählen Sie eine Referenzfläche als Neutralebene aus.
4. Wählen Sie die Zielfläche aus, auf die die Objekte deformiert werden sollen.
5. Stellen Sie die Optionen im Befehlsdialog ein.
    - <Kst>Befehlsdialog</Kst>
          - **Skalierung:** Skalierung in **Z,Y,X Wert**
          - **Offset:** Offset in **Z,Y,X Wert**
          - **Flip:** 
                - **Spiegeln:** Spiegeln Sie die deformierten Objekte horizontal zur Oberfläche.
                - **UV:** Drehen Sie die U- und V-Ausrichtung der deformierten Objekte um.
                - **Normal:** Drehen Sie die Normalen der deformierten Objekte um.
          - **Neuabmischen:** Mischen Sie die deformierten Objekte neu. Berechnet die Fasen Ihrer Objekte neu und übersetzt sie erneut, um unerwünschte Verformungen zu entfernen.
          - **Werkzeuge behalten:** Führen Sie die Deform-Operation durch, während die Werkzeugobjekte intakt bleiben.
6. Bestätigen Sie mit OK im Befehlsdialog oder durch Rechtsklick.

<Callout type="info" emoji="ℹ️">
Scale und Offset verwenden ein **UVN**-Koordinatensystem anstelle eines **XYZ**-Koordinatensystems, wenn der parametrische Wert angepasst wird. Dies ermöglicht es Ihnen, die Ausrichtung der deformierten Objekte an einem bestimmten Punkt in Bezug auf die Zieloberfläche festzulegen. Für weitere Informationen zum UVN-Koordinatensystem klicken Sie [hier](/cad-essentials/uvn-coordinate-system).
</Callout>

### Verwendung von Deform mit dem Unwrap Face‑Befehl

Um unerwünschte Verformungen oder ungenaue Platzierungen zu vermeiden, verwenden Sie den Befehl **Unwrap Face** aus der Befehlspalette auf dem Zielobjekt. Dadurch wird eine korrekt abgebildete Oberfläche erzeugt, die anschließend als genaue Referenzfläche verwendet werden kann.

> **Hinweis:** Dies funktioniert nur bei offenen Flächen, die weder periodisch noch geschlossen sind. Wenn die Fläche periodisch oder geschlossen ist, lesen Sie bitte das folgende Kapitel – [Verformung geschlossener Flächen](/solid/deform-solid-and-sheet#surface-deformation-on-closed-surfaces).

<VideoPlayer fileName="deform-solid-unwrap"/>

### Verwendung von Deform auf geschlossenen Flächen

Das Verformen von Objekten auf geschlossenen (periodischen) Flächen wie z. B. einer Zylinderfläche erfordert entweder das Fragmentieren der Fläche durch Einfügen isoparametrischer Linien oder eine Aufteilung des zu verformenden Objekts in Teile (falls möglich). Dies ist notwendig, da sich das Objekt beim Umwickeln einer geschlossenen Fläche selbst schneiden kann, was zu Verbindungs- und Kontinuitätsproblemen führt.
