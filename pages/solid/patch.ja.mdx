## カーブからパッチを作成する

閉じたカーブからパッチを作成することができます。

1. パッチを作成したい閉じたカーブを選択します。
2. コマンドバーの右側にある <ButtonIcon type="button-patch" /> ボタンをクリックします。
3. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Tolernace:** パッチの許容差を指定します。
4. コマンドダイアログで OK をクリックするか、右クリックして確定します。

## エッジに沿って穴をパッチする

エッジに沿って穴をパッチすることができます。

1. 穴を囲むエッジを選択します。
2. コマンドバーの右側にある <ButtonIcon type="button-patch" /> ボタンをクリックします。
3. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Continuity:**
            - **Planar:** サーフェスの連続性を平面として指定します。
            - **G0, G1, G2**
                - [サーフェスの連続性](/cad-essentials/continuity-curve-and-surface)を指定します。
                - サーフェス上の黒い丸をクリックすることで、個別に連続性を指定することができます。
        - **Faces:**
            - **Single:**
            - **Minimal:**
            - **Quality:**
        - **Tolernace:** パッチの許容差を指定します。
4. コマンドダイアログで OK をクリックするか、右クリックして確定します。

<VideoPlayer fileName="patch-2"/>

## パッチホールでシートを作成する

ソリッドのエッジを選択して、新しいシートボディを作成できます。

1. ソリッドのエッジを選択します。
2. ソリッドのエッジを選択します。
3. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **トレランス:** パッチのトレランスを指定します。
4. コマンドダイアログで OK をクリックするか、右クリックして確認します。

<VideoPlayer fileName="patch-6"/>

## ガイドカーブでパッチする

ガイドカーブを使って、エッジに沿って穴をパッチすることができます。

1. ガイドカーブを作成します。
2. ガイドカーブが接触する点に、[**Isoparam**](isoparam)コマンドでアイソパラムを追加します。
3. 穴を囲むエッジを選択します。
4. コマンドバーの右側にある <ButtonIcon type="button-patch" /> ボタンをクリックします。
5. ガイドカーブを選択します。
6. コマンドダイアログでオプションを設定します。
    - オプションは上記と同じです。
7. コマンドダイアログで OK をクリックするか、右クリックして確定します。

<VideoPlayer fileName="patch-3"/>

## 複数の穴をパッチする

複数の穴を一度にパッチすることができます。

1. パッチしたい複数の穴があるシートを選択します。
2. コマンドバーの右側にある <ButtonIcon type="button-patch" /> ボタンをクリックします。
3. 穴がパッチされて、ソリッドになります。

<Callout emoji="❗">
**注意**<br />
パッチできない穴があると、コマンドは失敗します。
</Callout>

## パッチでシートをトリムする

パッチツールを使用してシートをトリムすることができます。

1. 選択したいトリム対象のシートを選択し、一時的に非表示にします。
2. ベースシートを選択し、穴のある部分でエッジループ選択を行います。
3. ボディ選択モードに入り、トリム対象のシートを再表示させ、追加選択を行います。
5. コマンドバーの右側にある<ButtonIcon type="button-patch"/>ボタンをクリックします。
6. コマンドダイアログでオプションを設定します。 
    - <Kst>Command Dialog</Kst>
        - **Method:**
            - **Trim to hole:** 対象シートをベースシートの穴にトリムします。
            - **Trim to sheet:** 対象シートをベースシートの端部に合わせてトリムします。
        - **Tolerance:** パッチの公差を指定します。
