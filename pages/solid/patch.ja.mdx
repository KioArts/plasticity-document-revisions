## 説明

閉じたカーブ、またはエッジに沿った穴からパッチサーフェスを作成します。穴の場合、G0・G1・G2・平面の連続性オプションやガイドカーブが使用できます。

## カーブからパッチを作成する

閉じたカーブからパッチサーフェスを作成できます。

1. パッチしたい閉じたカーブを選択します。
2. 提案コマンドメニューまたはコマンドパレットで **Patch** を検索し、実行します。
3. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **許容差:** パッチの許容差を指定します。
4. コマンドダイアログの OK または右クリックで確定します。

## エッジに沿った穴をパッチする

エッジに沿った穴をパッチできます。

1. 穴を囲むエッジを選択します。
2. 提案コマンドメニューまたはコマンドパレットで **Patch** を検索し、実行します。
3. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **連続性:**
            - **平面:** サーフェスの連続性を平面として指定します。
            - **G0, G1, G2**
                - [サーフェスの連続性](/cad-essentials/continuity-curve-and-surface) を指定します。
                - サーフェス上の黒い円をクリックして個別に設定できます。
        - **フェース:**
            - **シングル:**
            - **ミニマル:**
            - **クオリティ:**
        - **許容差:** パッチの許容差を指定します。
4. コマンドダイアログの OK または右クリックで確定します。

## パッチ穴でシートを作成する

ソリッドのエッジを選択して新しいシートボディを作成できます。

1. ソリッドのエッジを選択します。
2. 提案コマンドメニューまたはコマンドパレットで **Patch** を検索し、実行します。
3. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **許容差:** パッチの許容差を指定します。
4. コマンドダイアログの OK または右クリックで確定します。

<VideoPlayer fileName="patch-6"/>

## ガイドカーブを使用したパッチ

ガイドカーブを使用してエッジに沿った穴をパッチできます。

1. ガイドカーブを作成します。
2. ガイドカーブがサーフェスに触れる各点にアイソパラメータを追加します。 [**Isoparam**](isoparam) コマンドを使用します。
3. 穴を囲むエッジを選択します。
4. 提案コマンドメニューまたはコマンドパレットで **Patch** を検索し、実行します。
5. ガイドカーブを選択します。
6. コマンドダイアログでオプションを設定します。
    - オプションは上記と同じです。
7. コマンドダイアログの OK または右クリックで確定します。

<VideoPlayer fileName="patch-3"/>

## 複数の穴をパッチする

複数の穴を一度にパッチできます。

1. パッチしたい複数の穴を持つシートを選択します。
2. 提案コマンドメニューまたはコマンドパレットで **Patch** を検索し、実行します。
3. 穴がパッチされ、ソリッドとして閉じられます。
<Callout type="warning">
**注意:** 埋められない穴がある場合、コマンドは失敗します。
</Callout>

<VideoPlayer fileName="patch-4"/>

## パッチでシートをトリムする

パッチツールを使用してシートをトリムできます。

1. トリムしたいターゲットシートを選択し、一時的に非表示にします。
2. ベースシートを選択し、穴の周囲でエッジループ選択を行います。
3. ボディ選択モードに入り、ターゲットシートを再表示して追加選択します。
5. 提案コマンドメニューまたはコマンドパレットで **Patch** を検索し、実行します。
6. コマンドダイアログでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **方法:**
            - **Trim to hole:** ターゲットシートをベースシートの穴に合わせてトリムします。
            - **Trim to sheet:** ターゲットシートをベースシートの端部に合わせてトリムします。
        - **許容差:** パッチの許容差を指定します。
