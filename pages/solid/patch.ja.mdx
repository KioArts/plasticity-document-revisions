## パッチ作成

閉じた曲線またはエッジ沿いの穴からパッチ表面を作成できます。穴の場合、G0、G1、G2、および平面連続性のオプションとガイド曲線が利用可能です。

### 閉じた曲線からのパッチ作成

閉じた曲線からパッチ表面を作成できます。

1. パッチを作成したい閉じた曲線をすべて選択します。
2. 「Suggested Commands Menu」または「Command Palette」で **Patch** を検索し、実行します。
3. コマンドダイアログの設定を行います。
    - <Kst>コマンドダイアログ</Kst>
        - **許容範囲:** パッチの許容範囲を指定します。
4. コマンドダイアログの「OK」をクリックするか、右クリックして確認します。

## エッジ沿いの穴パッチ

エッジ沿いに穴をパッチ作成できます。

1. 穴を囲むエッジを選択します。
2. 「Suggested Commands Menu」または「Command Palette」で **Patch** を検索し、実行します。
3. コマンドダイアログの設定を行います。
    - <Kst>コマンドダイアログ</Kst>
        - **連続性:**
            - **平面:** 表面の連続性を平面に指定します。
            - **G0, G1, G2**
                - 表面の連続性を指定します ([連続性 - 曲線と表面](/cad-essentials/continuity-curve-and-surface))。
                - 黒い円をクリックすることで、個別に連続性を指定できます。
        - **面:**
            - **単一:**
            - **最小限:**
            - **品質:**
        - **許容範囲:** パッチの許容範囲を指定します。
4. コマンドダイアログの「OK」をクリックするか、右クリックして確認します。

## パッチ穴とガイド曲線を使用したシート作成

既存のソリッドのエッジを選択することで、新しいシートボディを作成できます。

1. ソリッドのエッジを選択します。
2. 「Suggested Commands Menu」または「Command Palette」で **Patch** を検索し、実行します。
3. コマンドダイアログの設定を行います。
    - <Kst>コマンドダイアログ</Kst>
        - **許容範囲:** パッチの許容範囲を指定します。
4. コマンドダイアログの「OK」をクリックするか、右クリックして確認します。

## ガイド曲線を使用したパッチ

ガイド曲線を活用して穴をエッジ沿いにパッチ作成できます。

1. ガイド曲線を作成します。
2. ガイド曲線が表面に接する各点において、 isoparam を追加します。 [**Isoparam**](isoparam) コマンドを使用します。
3. 穴を囲むエッジを選択します。
4. 「Suggested Commands Menu」または「Command Palette」で **Patch** を検索し、実行します。
5. ガイド曲線を選択します。
6. コマンドダイアログの設定を行います。
    - オプションは上記と同じです。
7. コマンドダイアログの「OK」をクリックするか、右クリックして確認します。

## 複数の穴パッチ

複数の穴をまとめてパッチ作成できます。

1. パッチを作成したい複数の穴のあるシートを選択します。
2. 「Suggested Commands Menu」または「Command Palette」で **Patch** を検索し、実行します。
3. 穴がパッチされ、ソリッド化されます。
<Callout type="warning">
**注意:** 埋められない穴がある場合、コマンドは失敗します。
</Callout>

## パッチでシートトリミング

パッチツールを使用してシートをトリミングできます。

1. トリミングしたいターゲットシートを選択し、一時的に非表示にします。
2. ベースシートを選択し、穴があるエッジループの選択を行います。
3. ボディ選択モードに入り、ターゲットシートを表示して、ベースシートとの付加的選択を行います。
5. 「Suggested Commands Menu」または「Command Palette」で **Patch** を検索し、実行します。
6. コマンドダイアログの設定を行います。
    - <Kst>コマンドダイアログ</Kst>
        - **方法:**
            - **穴にトリミング:** ベースシートの穴にターゲットシートをトリミングします。
            - **シートにトリミング:** ベースシートの端セクションに合わせてターゲットシートをトリミングします。
        - **許容範囲:** パッチの許容範囲を指定します。
