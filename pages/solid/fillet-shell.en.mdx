## Description

Fillet or chamfer surface edges with options for conic, G2 continuous, or constant width profiles. Supports variable distance and range limiting.

## Basic Usage

1. Select the Edges to fillet or chamfer.
2. When one or more Edges are selected, the default command, **Fillet**, is automatically executed.
3. Alternatively, press the shortcut key <kbd>B</kbd> to execute.
4. Move the yellow dot to specify the distance of the fillet or chamfer. 
    - Positive movement:  results in a fillet.
    - Negative movement: results in a chamfer.
    <VideoPlayer fileName="fillet-solid-0"/>
5. Optionally, to select more Edges, <kbd>Ctrl</kbd> + click on the Edge. To deselect, <kbd>Ctrl</kbd> + click again.
6. Set the options in the Command Dialog and Keyboard Shortcuts. See [**Fillet options**](#fillet-options) and [**Chamfer options**](#chamfer-options) below.
7. Confirm with OK in the Command Dialog or right-click.

## Fillet options

### Command Dialog

**Distance:** 
    - To fillet an Edge, enter a positive value. To chamfer an Edge, enter a negative value
<Callout type="info">
**Tip:**    To copy a fillet radius, click on the source fillet, but only if the two Distance values are the same and the shape is a Conic.
</Callout>
<VideoPlayer fileName="fillet-solid-copyfillet"/>

**Shape:**
    - **Conic:** Produces a natural, smooth transition using a conic section curve.
    - **Chordal:** Maintains a constant chord length across the fillet, ensuring visual consistency regardless of the angles being filleted.
    - **G2:** Generates a fillet that is curvature continuous with both adjacent surfaces, ideal for complex or high-quality surface work. See [Surface Continuity](/cad-essentials/continuity-curve-and-surface).
    - **Full:** Creates a fillet that is tangent to all three neighboring faces (left, center, and right), resulting in a fully rounded shape across the selected face.
    
    <VideoPlayer fileName="fillet-shell-shape"/>

**Tension:**
    - Use the Tension dialog to control the intensity of the fillet's appearance. This setting applies to all shape types, excluding **Full Shape**.
    <VideoPlayer fileName="fillet-solid-tension"/>

**Y-Blend:** 
    - The “Y-blend” is a special kind of blend or fillet that occurs when three surfaces meet at a common point, forming a “Y” shape. Enabling **Attempt**, you can create better topology  in Y-blend and avoid undesired fillet results.
    <VideoPlayer fileName="fillet-solid-yblend"/>

**Selection:** 
    - Disable **Add tangent Edges** to limit fillets to selected Edges only.
    - Shortcut <kbd>T</kbd> is acceptable.
    <VideoPlayer fileName="fillet-sold-selection"/>

### Shortcuts

    - <kbd>A</kbd> **Chamfer angle:** Specify the chamfer angle by moving the cursor, or adjust the two distance values directly.
    - <kbd>D</kbd> **Fillet distance:** Specify the fillet distance by moving the cursor.
    - <kbd>C</kbd> **Chamfer distance:** Specify the chamfer distance by moving the cursor.
    - <kbd>V</kbd> **Add variable point:** Add a variable point to the fillet.
    - <kbd>L</kbd> **Add limit point:** Click on the fillet centerline to set a range limit. Click again to reverse direction. Up to two limit points can be set. Requires [**Object Snaps**](/plasticity-essentials/plasticity-interface/snap) to be enabled.
    - <kbd>T</kbd> **Toggle tangent edges:** Limit fillets to selected edges only.
    - <kbd>Ctrl</kbd> **Modify selection:** Add or remove edges from the fillet selection.

## Chamfer options
- **Shape:**
    - **Offset:** Default chamfer.
    - **Apex:** Reduce the displacement of the border of the chamfer surface.
- Others are the same as fillet options.
