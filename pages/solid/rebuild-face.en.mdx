## Description

Rebuild the surface of a face by adjusting spline count, degree, spans, and CV layout.

## Basic Usage

1. Select the face or faces to rebuild.
2. Execute the **Rebuild** command from the Command Palette.
3. Set the options in the Command Dialog.
    - <Kst>Command Dialog</Kst>
    - **Method:**
        - **Explicit control:** *(Available only on Studio License)*
            - **Degree:** Specify the U and V degree value.
            - **Spans:** Specify the U and V spans value.
            - **Extend:** Specify the U and V extend value.
            - **Shrink:** Shrink projecting rows and columns of CVs to fit more closely to the boundary edges of the trimmed surface.
        - **Tolerance:**
            - **Tolerance:** Specify the tolerance of surface CV count. Lower value results in large number of surface CVs and higher value results in small number of CVs. This determines the surface output quality.
            - **Extend:** Specify the U and V extend value.
            - **Shrink:** Shrink projecting rows and columns of CVs to fit more closely to the boundary edges of the trimmed surface.
5. Confirm with OK in the Command Dialog or right-click.
