## 説明

2つ以上のプロファイルカーブとエッジを使用してサーフェスを作成します。形状はガイドカーブまたはエッジで制御できます。G0、G1、G2の連続性とテンションをサポートします。

## カーブとエッジをロフトする

カーブとエッジは任意の組み合わせでロフトできます。

### 2つのカーブをロフトする

カーブをロフトする際は、**リミットポイントを追加** と **単純化** オプションを指定できます。

1. ロフトする2つのカーブを選択します。
2. <kbd>L</kbd> を押します。
3. コマンドダイアログおよびキーボードショートカットでオプションを設定します。
    - <Kst>Command Dialog</Kst>
        - **単純化:** トリムされた平面断面を生成するか、均一なジオメトリを維持するかを指定します。（デフォルト：**オン**）
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>L</kbd> **リミットポイントを追加:** ロフトする範囲を指定します。
            - [**オブジェクトスナップ**](/plasticity-essentials/plasticity-interface/snap) が有効になっていることを確認してください。
            - エッジをクリックしてリミットポイントを追加します。
            - 方向を反転するには、リミットポイントを再度クリックします。
            - 各カーブに最大2つのリミットポイントを設定できます。
4. コマンドダイアログで OK をクリックするか、右クリックで確定します。

<VideoPlayer fileName="loft-curve-and-curve"/>

### 2つのエッジをロフトする

エッジをロフトする際は、**連続性**、**テンション**、**リミットポイントを追加** オプションを指定できます。

1. ロフトする2つのエッジを選択します。
2. <kbd>L</kbd> を押します。
3. コマンドダイアログおよびキーボードショートカットでオプションを設定します。
    - <Kst>Command Dialog</Kst>
        - **連続性**: **G0, G1, G2**
            - [サーフェスの連続性](/cad-essentials/continuity-curve-and-surface) を指定します。
            - サーフェス上の黒い円をクリックすることで、連続性を個別に指定できます。
        - **テンション**: 各エッジのテンションを指定します。
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>Tab</kbd> **連続性を切り替え:** 連続性を切り替えます。
        - <kbd>D</kbd> **テンション:** カーソルを移動して、両方のエッジのテンションを同時に指定できます。
        - <kbd>L</kbd> **リミットポイントを追加:** ロフトする範囲を指定します。**2つのカーブをロフトする** と同様です。
4. コマンドダイアログで OK をクリックするか、右クリックで確定します。

<VideoPlayer fileName="loft-edge-and-edge"/>

### カーブとエッジをロフトする

カーブとエッジは任意の組み合わせでロフトできます。手順と機能は上記の **2つのカーブをロフトする** および **2つのエッジをロフトする** と同じです。

## 複数のカーブをロフトする

複数のカーブを一度にロフトできます。<kbd>Shift</kbd> + クリックで複数選択、<kbd>Ctrl</kbd> + クリックで選択解除します。手順と機能は **2つのカーブをロフトする** と同じです。エッジは始点と終点にも指定できます。

<Callout type="warning">
**注意:** プロファイルとガイドは、サーフェスが流れる順番に選択する必要があります。
</Callout>

<VideoPlayer fileName="loft-multiple-curves"/>

## ガイドを使用したロフト

1つ以上のガイドを使用してロフトの形状を制御できます。カーブとエッジの任意の組み合わせをガイドとして指定できます。ロフトされたサーフェスはトリムできます。

1. ロフトする2つのカーブを選択します。
2. <kbd>L</kbd> を押します。
3. ガイドカーブまたはエッジを選択します。<kbd>Shift</kbd> + クリックで複数選択、<kbd>Ctrl</kbd> + クリックで選択解除します。
4. コマンドダイアログおよびキーボードショートカットでオプションを設定します。
    - <Kst>Command Dialog</Kst>
            - **重なりをトリム:**
            - **ガイド:** ガイド側のロフトサーフェスをトリムします。
            - **プロファイル:** プロファイル側のロフトサーフェスをトリムします。
    - その他のオプションは上記の **2つのカーブをロフトする** および **2つのエッジをロフトする** と同じです。
5. コマンドダイアログで OK をクリックするか、右クリックで確定します。

<VideoPlayer fileName="loft-curve-with-guide"/>

## 連続ロフト

エッジを1つだけ選択してロフトした場合、隣接する2つのエッジをガイドとしてロフトできます。この機能を連続して実行することで、2つのシートを簡単に結合できます。

## ループロフト

3つ以上のカーブをロフトして、ループ状のシートを作成できます。

1. ロフトする3つ以上のカーブを選択します。
2. <kbd>L</kbd> を押します。
3. コマンドダイアログで **ループ** をクリックします。
4. キーボードショートカットでオプションを設定します。
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>L</kbd> **リミットポイントを追加:** ロフトする範囲を指定します。**「2つのカーブをロフトする」** と同様です。
4. コマンドダイアログで OK をクリックするか、右クリックで確定します。

## 頂点をロフトする

2つの頂点をロフトすると、2つのカーブをつなぐことができます。機能は [**頂点をブリッジ**](/sketch/bridge#bridge-vertex) と同じです。

## トラブルシューティングのヒント

<Callout type="info">
**ロフトが成功しない場合の考えられる原因**
1. カーブが複雑すぎる。カーブを再作成することが解決策になることが多いです。
2. ガイドカーブがプロファイルに接触していない。各接続に青緑色の点があることを確認してください（交差を示します）。正確な接触を確保するには、[**オブジェクトスナップ**](/plasticity-essentials/plasticity-interface/snap) を参照してください。
3. 複雑な形状を一度にロフトしようとしている。複数のロフトで作成したシンプルなサブセクション形状でモデルを設計してください。ジョインや追加のロフトを使ったブリッジカーブでサーフェスを接続してください。
4. ガイドカーブが多すぎる、または少なすぎる。練習が上達への近道です。プロファイルとガイドが多すぎると、ロフトがうまくいかない場合があります。プロファイルとガイドが近すぎる場合も同様の問題が発生します。
5. エッジの代わりにカーブを選択している。エッジとカーブで使用できるオプションが異なります。
6. カーブが正しい順序で選択されていない。プロファイルとガイドは、サーフェスが流れる順番に選択する必要があります。
7. 失敗したロフトは、実際には失敗ではないことがあります。通常、ガイドを選択するにつれてロフトが成長するのが見えます。しかし、次のガイドを選択したときにサーフェスが消えることがあり、失敗したように見えます。次のガイドに進み続けると、すべてのサーフェスが突然現れることがあります。Parasolidがすべての入力を受け取ってから問題を解決する場合があるようです。
8. プロファイルとガイドの選択順序を逆にする。通常はプロファイルを先に選択し、次にガイドを選択します。ロフトが失敗した場合は、「ガイド」を先に選択してから「プロファイル」を選択してみてください。逆に思えますが、これで解決することがあります。
</Callout>
