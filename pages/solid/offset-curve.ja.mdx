# Offset Curve（カーブをオフセットする）

<Reference shortcut="O" command="Offset Curve" button="推奨コマンドメニューで" buttonIcon="button-offset-curve"/>

The Offset Curve command runs based on the selected object:
   - **Offset Planar Curve:** If Curves are selected, see [**Offset Planar Curve**](/sketch/offset-planar-curve) in the Sketch Commands section.
   - **Offset Region:** If Regions are selected, see [**Offset Region**](/sketch/offset-region) in the Sketch Commands section.
   - **Offset Face Loop:** If Faces are selected, see [**Offset Face Loop**](offset-face-loop).
   - **Offset Edge:** If Edges are selected, see [**Offset Edge**](offset-edge).
