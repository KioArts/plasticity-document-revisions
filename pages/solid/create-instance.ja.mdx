# Create Instance

**Create Instance**コマンドは、選択したオブジェクトに基づいて動作します：
     - **Create Solid and Sheet Instance:** ソリッドまたはシートが選択されている場合は、[**Create Solid and Sheet Instance**](create-solid-and-sheet-instance)を参照してください。
     - **Create Curve Instance:** カーブが選択されている場合は、[**Create Curve Instance**](/sketch/create-curve-instance)を参照してください。
