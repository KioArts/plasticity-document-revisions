# Measure Curvature

<Reference shortcut="" command="Measure Curvature" button=""/>

The Measure Curvature command runs based on the selected object:
     - **Measure Surface Curvature:** If Surface Edges are selected, see [Measure Surface Curvature](/solid/measure-surface-curvature).
     - **Measure Curve Curvature:** If Curves are selected, see [Measure Curve Curvature](/sketch/measure-curve-curvature) in the Sketch Commands section.
