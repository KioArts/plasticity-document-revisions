## Offset Curve

{/* <Reference shortcut="O" command="Offset Curve" button="Im Menü „Vorgeschlagene Befehle“"  buttonIcon="button-offset-curve"/> */}

Der **Offset Curve**‑Befehl wird basierend auf dem ausgewählten Objekt ausgeführt:
   - **Offset Planar Curve:** Wenn Kurven ausgewählt sind, siehe [**Offset Planar Curve**](/sketch/offset-planar-curve) im Abschnitt „Sketch Commands“.
   - **Offset Region:** Wenn Regionen ausgewählt sind, siehe [**Offset Region**](/sketch/offset-region) im Abschnitt „Sketch Commands“.
   - **Offset Face Loop:** Wenn Flächen ausgewählt sind, siehe [**Offset Face Loop**](offset-face-loop).
   - **Offset Edge:** Wenn Kanten ausgewählt sind, siehe [**Offset Edge**](offset-edge).
