# Isoparam (エッジを加える)

ソリッドとシートのフェイスにエッジを加えることができます。

1. <kbd>Ctrl-R</kbd> を押します。
2. エッジを作成する位置にマウスカーソルを合わせます。
3. ショートカットキーでオプションを設定します。
   - <kbd>Tab</kbd> **U/V direction:** エッジの方向を切り替えます。
   - <kbd>Shift</kbd>-Wheel up **Add cut:** エッジの数を増やします。
   - <kbd>Shift</kbd>-Wheel down **Subtract cut:** エッジの数を減らします。
4. 右クリックで確定します。
