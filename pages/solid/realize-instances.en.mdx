# Realize Instances

<Reference shortcut="" command="Realize Instances" button="Command Palette" buttonIcon="button-instance" />

The Realize Instances command runs based on the selected object:
     - **Realize Solid and Sheet Instances** Solids or Sheets are selected, see [**Realize Solid and Sheet Instances**](realize-solid-and-sheet-instances).
     - **Realize Curve Instances:** If Curves are selected, see [**Realize Curve instances**](/sketch/realize-curve-instances).
