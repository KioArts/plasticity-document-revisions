## Beschreibung

Erstellt eine Oberfläche aus zwei oder mehr Profilkurven und Kanten. Die Form kann durch Führungskurven oder Kanten gesteuert werden. Unterstützt G0-, G1-, G2-Stetigkeit und Spannung.

## Kurven und Kanten loften

Sie können beliebige Kombinationen aus Kurven und Kanten loften.

### Zwei Kurven loften

Beim Loften von Kurven können die Optionen **Grenzpunkt hinzufügen** und **Vereinfachen** angegeben werden.

1. Wählen Sie zwei Kurven aus, die Sie loften möchten.
2. Drücken Sie <kbd>L</kbd>.
3. Legen Sie die Optionen im Befehlsdialog und über Tastaturkürzel fest.
    - <Kst>Command Dialog</Kst>
        - **Vereinfachen:** Gibt an, ob getrimmt planare Querschnitte erzeugt oder eine einheitliche Geometrie beibehalten werden soll. (Standard: **EIN**)
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>L</kbd> **Grenzpunkt hinzufügen:** Legt den Bereich für den Loft fest.
            - Stellen Sie sicher, dass [**Objektfang**](/plasticity-essentials/plasticity-interface/snap) aktiviert ist.
            - Klicken Sie auf die Kante, um einen Grenzpunkt hinzuzufügen.
            - Um die Richtung umzukehren, klicken Sie erneut auf den Grenzpunkt.
            - Auf jeder Kurve können bis zu zwei Grenzpunkte gesetzt werden.
4. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

### Zwei Kanten loften

Beim Loften von Kanten können die Optionen **Stetigkeit**, **Spannung** und **Grenzpunkt hinzufügen** angegeben werden.

1. Wählen Sie zwei Kanten aus, die Sie loften möchten.
2. Drücken Sie <kbd>L</kbd>.
3. Legen Sie die Optionen im Befehlsdialog und über Tastaturkürzel fest.
    - <Kst>Command Dialog</Kst>
        - **Stetigkeit**: **G0, G1, G2**
            - Geben Sie die [Stetigkeit der Oberfläche](/cad-essentials/continuity-curve-and-surface) an.
            - Die Stetigkeit kann individuell durch Klicken auf die schwarzen Kreise auf der Oberfläche festgelegt werden.
        - **Spannung**: Gibt die Spannung jeder Kante an.
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>Tab</kbd> **Stetigkeit wechseln:** Stetigkeit umschalten.
        - <kbd>D</kbd> **Spannung:** Durch Bewegen des Cursors können Sie die Spannung für beide Kanten gleichzeitig festlegen.
        - <kbd>L</kbd> **Grenzpunkt hinzufügen:** Legt den Bereich für den Loft fest. Wie bei **Zwei Kurven loften**.
4. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

<VideoPlayer fileName="loft-edge-and-edge"/>

### Kurve und Kante loften

Kurven und Kanten können in beliebiger Kombination gelofte werden. Ablauf und Funktionen entsprechen den oben beschriebenen Abschnitten **Zwei Kurven loften** und **Zwei Kanten loften**.

## Mehrere Kurven loften

Sie können mehrere Kurven gleichzeitig loften. <kbd>Shift</kbd> + Klick zum Mehrfachauswählen, <kbd>Ctrl</kbd> + Klick zum Abwählen. Ablauf und Funktionen entsprechen **Zwei Kurven loften**. Kanten können auch an Start- und Endpunkten angegeben werden.

<Callout type="warning">
**Hinweis:** Profile und Führungen müssen in der Reihenfolge ausgewählt werden, in der die Oberflächen verlaufen.
</Callout>

<VideoPlayer fileName="loft-multiple-curves"/>

## Loften mit einer Führung

Eine oder mehrere Führungen können verwendet werden, um die Form des Lofts zu steuern. Beliebige Kombinationen aus Kurven und Kanten können als Führungen angegeben werden. Die gelofte Oberfläche kann getrimmt werden.

1. Wählen Sie zwei Kurven aus, die Sie loften möchten.
2. Drücken Sie <kbd>L</kbd>.
3. Wählen Sie Führungskurven oder Kanten aus. <kbd>Shift</kbd> + Klick zum Mehrfachauswählen, <kbd>Ctrl</kbd> + Klick zum Abwählen.
4. Legen Sie die Optionen im Befehlsdialog und über Tastaturkürzel fest.
    - <Kst>Command Dialog</Kst>
            - **Überlappung trimmen:**
            - **Führungen:** Die gelofte Oberfläche auf der Führungsseite wird getrimmt.
            - **Profile:** Die gelofte Oberfläche auf der Profilseite wird getrimmt.
    - Die weiteren Optionen entsprechen den oben beschriebenen Abschnitten **Zwei Kurven loften** und **Zwei Kanten loften**.
5. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

## Kontinuierliches Loften

Wenn nur eine Kante ausgewählt und gelofte wird, können zwei angrenzende Kanten als Führungen gelofte werden. Diese Funktion kann nacheinander ausgeführt werden, um zwei Sheets einfach miteinander zu verbinden.

## Schleifen-Loft

Sie können drei oder mehr Kurven loften, um ein geschlossenes Sheet zu erstellen.

1. Wählen Sie drei oder mehr Kurven aus, die Sie loften möchten.
2. Drücken Sie <kbd>L</kbd>.
3. Klicken Sie im Befehlsdialog auf **Schleife**.
4. Legen Sie die Optionen über Tastaturkürzel fest.
    - <Kst>Keyboard Shortcuts</Kst>
        - <kbd>L</kbd> **Grenzpunkt hinzufügen:** Legt den Bereich für den Loft fest. Wie bei **„Zwei Kurven loften"**.
4. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

## Regionen loften

Sie können Regionen loften, um ein Solid zu erstellen.

1. Wählen Sie zwei oder mehr Regionen aus, die Sie loften möchten.
2. Drücken Sie <kbd>L</kbd>.
3. Wählen Sie optional Führungskurven aus, um die Form des Lofts zu steuern.
4. Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.

## Eckpunkte loften

Das Loften zweier Eckpunkte ermöglicht das Verbinden zweier Kurven. Die Funktionalität entspricht [**Eckpunkt überbrücken**](/sketch/bridge#bridge-vertex).

## Tipps zur Fehlerbehebung

<Callout type="info">
**Mögliche Ursachen, wenn der Loft nicht gelingt**
1. Kurven sind zu komplex. Das Neuerstellen der Kurven ist oft die Lösung.
2. Führungskurven berühren die Profile nicht. Prüfen Sie, ob jede Verbindung einen blau-grünen Punkt aufweist, der eine Schnittmenge anzeigt. Um eine präzise Berührung sicherzustellen, siehe [**Objektfang**](/plasticity-essentials/plasticity-interface/snap).
3. Versuch, eine komplexe Form in einem einzigen Loft zu erstellen. Entwerfen Sie das Modell mit einfacheren Teilformen, die aus mehreren Lofts bestehen. Verbinden Sie Oberflächen mit Joins oder überbrückenden Kurven mit zusätzlichen Lofts.
4. Zu viele oder zu wenige Führungskurven. Übung hilft hier. Lofts haben Schwierigkeiten, wenn zu viele Profile und Führungen verwendet werden. Das gleiche Problem tritt auf, wenn Profile und Führungen zu nahe beieinander liegen.
5. Kurven statt Kanten auswählen. Die verfügbaren Optionen unterscheiden sich für Kanten und Kurven.
6. Kurven wurden nicht in der richtigen Reihenfolge ausgewählt. Profile und Führungen müssen in der Reihenfolge ausgewählt werden, in der die Oberflächen verlaufen.
7. Ein fehlgeschlagener Loft ist manchmal kein Fehler. Im Allgemeinen wächst der Loft sichtbar, wenn Führungen ausgewählt werden. Jedoch können Oberflächen beim Auswählen der nächsten Führung verschwinden, was wie ein Fehler wirkt. Fahren Sie mit den nächsten Führungen fort – alle Oberflächen können plötzlich erscheinen. Es scheint, als ob Parasolid manchmal alle Eingaben benötigt, bevor es das Problem löst.
8. Reihenfolge der Auswahl von Profilen und Führungen umkehren. Normalerweise werden zuerst die Profile und dann die Führungen ausgewählt. Wenn ein Loft fehlschlägt, versuchen Sie, zuerst die „Führungen" und dann die „Profile" auszuwählen. Dies kann funktionieren, auch wenn es umgekehrt erscheint.
</Callout>
