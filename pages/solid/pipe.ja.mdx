# Pipe（パイプを作成）

<Reference shortcut="P" command="Pipe" button="コマンドバー右" />

カーブやエッジに沿ったパイプを作成できます。

1. パイプにしたいカーブやエッジを選択します。
2. <kbd>P</kbd> を押します。
3. 黄色い点をドラッグして断面サイズを、黄色い四角をドラッグしてパイプの厚みを指定します。白い丸をドラッグして回転角度を指定します。
4. コマンドダイアログやショートカットキーでオプションを設定します。
     - **Select target bodies:** ブーリアンのターゲットにするソリッドやシートを選択します。
          - <kbd>Shift-Q</kbd> **Slice:** ブーリアン対象との交点にソリッドを作成し、余りのソリッドも残ります。
          - <kbd>Q</kbd> **Union:** ブーリアン対象と結合します。
          - <kbd>W</kbd> **Difference:** ブーリアン対象から切り取ります。
          - <kbd>Shift-E</kbd> **Intersect:** ブーリアン対象との交点にソリッドを作成します。
          - <kbd>B</kbd> **New Body:** 新規にソリッドを作成します。ブーリアンは実行されません。
          - <kbd>T</kbd> **Keep Tool:** ブーリアンが実行され、ツールのオブジェクトはそのまま残ります。
     - **Select custom profile:** カスタムプロファイルにするリージョン、カーブ、フェースを選択します。
     - **Vertex count:** プロファイルの多角形の頂点数を指定します。0を指定すると円形になります。
     - **Section size:** 断面サイズを指定します。
     - **Thickness:** パイプの厚みを指定します。
     - **Angle:** 回転角度を指定します。
     - <kbd>C</kbd> **Custom profile** カスタムプロファイルにするリージョン、カーブ、フェースを選択します。
     - <kbd>D</kbd> **Section size** 断面サイズをカーソルを動かして指定します。
     - <kbd>A</kbd> **Angle** 回転角度をカーソルを動かして指定します。
     - <kbd>Shift-T</kbd> **Thickness** パイプの厚みをカーソルを動かして指定します。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
