# Square 

<Reference shortcut="" command="Square" button="" />

Square は、UV の次数とスパンを明示的に制御できるパラメトリック機能を備えた、Xnurbs ベースのコマンドで、高品質な四角形の面を作成することができます。

## 基本的な使い方

1. Square コマンドで使用したい境界エッジを選択します。
2. コマンドパレットに **Square** と入力します。
3. コマンドダイアログでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
   - **連続性:** **G0, G1, G2**
   - **平坦度:** XNurbs と同様に、このオプションは生成された面の平坦度と品質に影響します。値が高いほど高品質な面が生成され、値が低いほど低品質な面が生成されます。
   - **次数:** UV の次数を指定します。
   - **スパン: ** UV のスパン値を指定します。
   - **境界条件:** UV フローの条件を指定します。
        - **自然:** UV フローに要件はありません。
        - **標準:** UV アイソカーブを相互に直角に保ちます。
        - **次:** 境界条件が比較的平坦な場合、UV の垂直フローを生成します。
   - **CV を一致:** 境界エッジの CV を一致させます。
   - **ウェイト:** 曲線やソリッドボディのエッジに対する入力制約を指定し、一致させます。低い値ではこれらの制約を無視し、望ましくない UV フロー接続が発生し、高い値では制約が尊重され、指定された **次数** と スパン に基づいて望ましい UV フロー接続が生成されます。
   - **スケール:** スケールオプションを使用すると、境界エッジに表示される赤い線のサイズを制御できます。これらの線はエッジ偏差プロットを表し、作成されたサーフェスの境界エッジと挿入された入力との距離を示します。
6. コマンドダイアログで OK をクリックするか、右クリックして確認します。
