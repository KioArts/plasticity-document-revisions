# Extend

<Reference shortcut="" command="Extend" button="" />

The Extend command runs based on the selected object:
     - **Extend Curve:** If Curves are selected, see [Extend Curve](extend-curve) in the Sketch Commands section.
     - **Extend Sheet:** If outside Edges of the Sheet are selected, see [Extend Sheet](/solid/extend-sheet) in the Solid Commands section.
