# Find Boundary Edges

<Reference shortcut="" command="Find Boundary Edges" button="" />


Select the boundary Edges of an open area on a selected Sheet.

## Basic Usage

1. Select a Sheet you want to find boundary Edges.
2. Type in **"Find Boundary Edges"** in the Command Palette and click on it.
3. Edges of the Sheet opening are selected.
