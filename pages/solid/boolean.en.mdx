# Boolean

<Reference shortcut="Q" command="Boolean" button="Left of Command Bar" />

You can boolean operations on solids or sheets.
{/* 
1. Press <kbd>Q</kbd> or click the button.
2. Specify the target and tool.
   - **Select target bodies**: Select the solid or sheet to be used as the boolean target.
   - **Select tool bodies**: Select the solid or sheet to be used as the boolean tool.
3. Specify the boolean operation.
   - <kbd>G</kbd> **Move**
      - You can  move the tool.
   - <kbd>R</kbd> **Rotate**
      - You can  rotate the tool.
      - <kbd>R</kbd> **Screen space**
         - Resets the rotation axis.
   - <kbd>S</kbd> **Scale**
      - You can  scale the tool.
      - <kbd>S</kbd> **Uniform**
         - Scales uniformly in the X, Y, and Z directions.
   - Other options: Refer to **Solid Editing Boolean options** and **Solid Editing Common options**.(Both under construction.)
4. Confirm with OK in the command dialog or right-click.
 */}
