# Boolean

<Reference shortcut="Q" command="Boolean" button="Left of Command Bar" />

You can boolean operations on any combination of solids and sheets.

1. Press <kbd>Q</kbd> or click the button <ButtonIcon type="button-boolean" /> .
2. Select the target and the tool.
   - **Select target bodies:** Select the solid or sheet to be used as the boolean target.
   - **Select tool bodies:** Select the solid or sheet to be used as the boolean tool.
3. Set the options in the command dialog and shortcut key.
   - **Keep Tools:** Performs Boolean operations while keeping the tool objects intact.
   - <kbd>Shift-Q</kbd> **Slice:** Creates a solid at the intersections with the Boolean target and preserves the remaining solids.
   - <kbd>Q</kbd> **Union:** Joins the boolean target and tool together.
   - <kbd>W</kbd> **Difference:** Removes the tool from the boolean target.
   - <kbd>Shift-E</kbd> **Intersect:** Creates a solid at the intersection of the boolean target and tool.
   - <kbd>T</kbd> **Keep Tool:** Performs Boolean operations while keeping the tool objects intact.
   - <kbd>G</kbd> **Move:** Move the tool in the same way as the **Move** command.
   - <kbd>R</kbd> **Rotate:** Ratate the tool in the same way as the **Rotate** command.
   - <kbd>S</kbd> **Scale:** Scale the tool in the same way as the **Scale** command.
   - <kbd>X</kbd> **X-axis:** Limit to the X-axis direction.
   - <kbd>Y</kbd> **Y-axis:** Limit to the Y-axis direction.
   - <kbd>Z</kbd> **Z-axis:** Limit to the Z-axis direction.
   - <kbd>Shift-X</kbd> **X plane:** Limit to the X-plane direction.
   - <kbd>Shift-Y</kbd> **Y plane:** Limit to the Y-plane direction.
   - <kbd>Shift-Z</kbd> **Z plane:** Limit to the Z-plane direction.
4. Confirm with OK in the command dialog or right-click.
