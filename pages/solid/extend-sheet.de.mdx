# Extend Sheet [#extend-sheet]

<Reference shortcut="" command="Extend Sheet" button="" />

Verlängern Sie einen oder mehrere äußere Edges des Sheet, indem Sie die Entfernung angeben. Wählen Sie aus Verlängerungsformen wie Liner, Soft und Reflective.

1. Wählen Sie einen oder mehrere äußere Edges des Sheet aus, den Sie verlängern möchten.
2. Wenn die äußeren Edges ausgewählt sind, wird der Standardbefehl **Extend Sheet** automatisch ausgeführt.
3. Bewegen Sie den gelben Punkt, um die Entfernung anzugeben.
4. Legen Sie die Optionen im Befehlsdialog fest.
    - **Shape:** **Liner** /  **Soft** /  **Reflective**
        - Beziehen Sie sich auf das Video, um die Unterschiede zwischen jeder Option zu sehen.
    - **Distance:** Geben Sie die Entfernung zur Verlängerung an.
    - **Modify:** Wenn aktiviert, wird das Sheet zu einem einzelnen Sheet.
5. Bestätigen Sie mit OK im Befehlsdialog oder Rechtsklick.
