## Beschreibung

Sie können Flächen löschen.

## Grundlegende Verwendung

1. Wählen Sie die Flächen aus, die Sie löschen möchten.
2. Drücken Sie <kbd>Shift + X / Delete / Backspace</kbd>.
3. Die Flächen werden gelöscht.
