# Fillet

<Reference shortcut="B" command="Fillet" button="" />

The Fillet command runs based on the selected object:
     - **Fillet Curve:** If Curves are selected, see [**Fillet Curve**](/sketch/fillet-curve) in the Sketch Commands section.
     - **Fillet Vertex:** If verticies are selected, see [**Fillet Vertex**](/sketch/fillet-vertex) in the Sketch Commands section.
     - **Fillet Shell:** If Edges are selected, see [**Fillet Shell**](fillet-shell).
