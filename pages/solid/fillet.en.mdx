# Fillet

<Reference shortcut="B" command="Fillet" button="" />

This command automatically calls the following command depending on the selected object.
- When curves or verticies are selected: see Sketch's [**"Fillet Curves and Vertices"**](../sketch/fillet) command.
- When edges are selected: see [**"Fillet Shell"**](../solid/fillet) described below.
