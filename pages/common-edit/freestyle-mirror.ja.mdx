# Freestyle Mirror（自由軸でミラー）

<Reference shortcut="" command="Freestyle Mirror" button="" />

カーブ、ソリッド、シートを自由な軸でミラーできます。

1. ミラーしたいオブジェクトを選択します。
2. ショートカットキー`F`を押してコマンドパレットを開き、**Freestyle Mirror**と入力して選択します。
3. ミラー方向を指定します。
   - 1回目のクリックを始点、2回目のクリックを終点としてミラーの軸を指定します。
   - **Cut**
      - ミラー対象をミラー平面で分割してからミラーします。
   - `Q` **Union(merge)**
      - ミラーした2つを結合します。
4. コマンドダイアログのOKまたは右クリックで確定します。
