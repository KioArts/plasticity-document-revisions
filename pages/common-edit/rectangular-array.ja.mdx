# Rectangular Array（長方形配列で複製）

カーブ、ソリッド、シートを長方形配列で複製できます

<Reference shortcut="" command="Rectangular Array" button="コマンドバー右" />

1. 複製したいオブジェクトを選択します。
2. コマンドバーのボタンをクリックします。
3. 複製プレビューが表示されます。
4. 複製したい方向をクリックします。
   - **Distance 1**
      - 距離を指定します。
   - **Number**
      - 複製する数を指定します。`Shift`+マウスホイールの回転でも指定できます。
5. \<オプション\> もうひとつの方向にも配列で複製したいときは、ダイアログの**Select endpoint 2**をクリックします。
    1. 複製したい方向をクリックします。
         - **Distance 2**: 距離を指定します。
         - **Repeat**: 複製する回数を指定します。`Shift`+マウスホイールの回転でも指定できます。
6. コマンドダイアログのOKまたは右クリックで確定します。
