# Move

<Reference shortcut="G" command="Move" button="コマンドバー左" />

カーブ、ポイント、CV、ソリッド、シート、フェイス、エッジを移動できます。

1. 移動したいオブジェクトを選択します。
2. ショートカット `G` を押すか、ボタンをクリックします。
3. 表示された移動ギズモを使用して移動します。
   - `F` **Freestyle**
      - 1回目のクリックを始点、2回目のクリックを終点として、移動の方向と距離を指定します。
   - その他のオプションについては、[共通オプション](common-options)を参照してください。
4. コマンドダイアログのOKまたは右クリックで確定します。
