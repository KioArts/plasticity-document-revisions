# Rotate

<Reference shortcut="R" command="Rotate" button="Left of Command Bar" />

You can rotate: curves, multiple points, multiple  CVs, solids, sheets, faces.

1. Select the objects you want to rotate.
2. Press the shortcut `R` or click the button.
3. Use the displayed Rotation Gizmo to rotate.
   - `F` **Freestyle**
      - Specify the starting point with the first click and the ending point with the next click.
   - `W` **World space**
      - Releases the rotation restrictions.
   - Other options: Refer to [Common options](common-options) .
4. Confirm with OK in the command dialog or right click.
