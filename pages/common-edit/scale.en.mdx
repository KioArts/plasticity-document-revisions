# Scale

<Reference shortcut="S" command="Scale" button="Left of Command Bar" />

You can scale: curves, multiple points, multiple CVs, solids, sheets.

1. Select the objects you want to scale.
2. Press the shortcut `S` or click the button.
3. Use the displayed Scale Gizmo to resize.
   - `S` **Uniform**
      - Scales uniformly in the X, Y, and Z directions.
   - `F` **Freestyle**
      - The first click specifies the starting point for scaling, the next specifies the end point.
   - `W` **World space**
      - Releases the scaling restrictions.
   - Other options: Refer to [Common options](common-options) .
4. Confirm with OK in the command dialog or right click.
