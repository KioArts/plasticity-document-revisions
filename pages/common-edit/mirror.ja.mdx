# Mirror（ミラー）

<Reference shortcut="Alt-X" command="Mirror" button="Left of Command Bar" />

カーブ、ソリッド、シート、フェイスをミラーできます。

1. ミラーしたいオブジェクトを選択します。
2. ショートカットキー `Alt-X` を押すか、ボタンをクリックします。
3. 表示されたミラーギズモを使ってミラーします。
4. ミラーの方向とオプションを指定します。
   - `X` **Position X**
      - X軸の正の方向にミラーします。
   - `Y` **Position Y**
      - Y軸の正の方向にミラーします。
   - `Z` **Position Z**
      - Z軸の正の方向にミラーします。
   - `Shift-X` **Negative X**
      - X軸の負の方向にミラーします。
   - `Shift-Y` **Negative Y**
      - Y軸の負の方向にミラーします。
   - `Shift-Z` **Negative Z**
      - Z軸の負の方向にミラーします。
   - `F` **Freestyle**
      - 1回目のクリックを始点、2回目のクリックを終点としてミラーの軸を指定します。
   - `Q` **Union(merge)**
      - ミラーした2つを結合します。
   - **Cut**
      - ミラー対象をミラー平面で分割してからミラーします。
5. コマンドダイアログのOKまたは右クリックで確定します。
