# Radial Array（円形状配列で複製）

カーブ、ソリッド、シートを円形状配列で複製できます

<Reference shortcut="" command="Radial Array" button="コマンドバー右" />

1. 複製したいオブジェクトを選択します。
2. コマンドバーのボタンをクリックします。
3. 中心にしたい点をクリックします。
4. 複製プレビューが表示されます。
   - **Degrees**
      - 複製する角度を指定します。
   - **Number**
      - 複製する数を指定します。`Shift`+マウスホイールの回転でも指定できます。
   - **Repeat**
      - 放射状に繰り返す回数を指定します。
   - **Repeat distance**
      - 放射状に繰り返す間隔を指定します。
5. コマンドダイアログのOKまたは右クリックで確定します。
