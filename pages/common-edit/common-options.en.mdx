# Common options

<Callout emoji="">This section applies to both sketches and solids.</Callout>

These options can be specified using keyboard shortcuts while editing.

### Keybord shortcuts

- `X` **X axis**
    - Limit to the X-axis direction.
- `Y` **Y axis**
    - Limit to the Y-axis direction.
- `Z` **Z axis**
    - Limit to the Z-axis direction.
- `Shift-X` **X plane**
    - Limit to the X-plane direction.
- `Shift-Y` **Y plane**
    - Limit to the Y-plane direction.
- `Shift-Z` **Z plane**
    - Limit to the Z-plane direction.
- `G` **Screen space**
    - Limit to the visible plane.
- `V` **Pivot**
    - Click to specify the position of the Gizmo.
- `W` **Worldspace**
    - Remove restrictions such as direction of movement.
