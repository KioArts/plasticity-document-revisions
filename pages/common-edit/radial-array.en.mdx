# Radial Array

<Reference shortcut="" command="Radial Array" button="Right of Command Bar" />

You can duplicate curves, solids, sheets in a radial array.

1. Select the object to duplicate.
2. Click on the button.
3. Click on the point you want to set as the center.
4. The duplication preview will be displayed.
5. Specify the options:
   - **Degrees**
      - Specify the angle for duplication.
   - **Number**
      - Specify the number of duplications. You can also use `Shift`+Mouse wheel to adjust.
   - **Repeat**
      - Specify the number of repetitions in a radial pattern.
   - **Repeat distance**
      - Specify the distance between each repetition in a radial pattern.
6. Confirm with OK in the command dialog or right click.
