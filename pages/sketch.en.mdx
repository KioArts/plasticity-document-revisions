# Sketch Commands

| Command Name                                                      | Shortcut                   | Note                                                                                                       |
| ----------------------------------------------------------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------- |
| [Deform](/sketch/deform)                                           |                            | Unified command for Deform Curve, Deform Solid and Sheet.                                                   |
| [Deform Curve](/sketch/deform-sketch)                              |                            | Wrap existing curves onto a different surface.                                                              |
| [Extend](/sketch/extend)                                           |                            | Unified command for Extend Curve, Extend Sheet.                                                             |
| [Extend Curve](/sketch/extend-curve)                               |                            | Extend a Curve.                                                                                             |
| [Fillet](/sketch/fillet)                                           | <kbd>B</kbd>               | Unified command for Fillet Curve, Fillet Vertex, Fillet Shell.                                              |
| [Fillet Curve](/sketch/fillet-curve)                               | <kbd>B</kbd>               | Fillet or chamfer between two Curve segments.                                                               |
| [Fillet Vertex](/sketch/fillet-vertex)                             | <kbd>B</kbd>               | Fillet or chamfer connected vertices.                                                                       |
| [Offset Curve](/sketch/offset-curve)                               | <kbd>O</kbd>               | Unified command for Offset Planar Curve, Offset Region, Offset Face Loop, Offset Edge.                      |
| [Offset Planar Curve](/sketch/offset-planar-curve)                 | <kbd>O</kbd>               | Offset a planar Curve.                                                                                      |
| [Offset Region](/sketch/offset-region)                             | <kbd>O</kbd>               | Offset the outline curves of a Region.                                                                      |
| [Offset Vertex](/sketch/offset-vertex)                             | <kbd>O</kbd>               | Insert new vertices along the curve on both sides of a vertex.                                              |
| [Raise Degree](/sketch/raise-degree)                               | <kbd>Shift+S</kbd>       | Unified command for Raise Curve Degree, Raise Surface Degree.                                               |
| [Raise Curve Degree](/sketch/raise-curve-degree)                   | <kbd>Shift+S</kbd>       | Increase Control-Vertices and Degrees of Curves.                                                            |
| [Rebuild](/sketch/rebuild)                                         |                            | Unified command for Rebuild Curve, Rebuild Face.                                                            |
| [Rebuild Curve](/sketch/rebuild-curve)                             |                            | Rebuilt by optimizing the positions of the Control Vertices.                                                |
| [Reverse](/sketch/reverse)                                         |                            | Unified command for Reverse Curve, Reverse Sheet.                                                           |
| [Reverse Curve](/sketch/reverse-curve)                             |                            | Inverse the direction of a curve.                                                                           |
| [Subdivide Curve](/sketch/subdivide-curve)                         | <kbd>Alt+S</kbd>         | Increase Control-Vertices and Spans of Curves.                                                              |
| [Slide](/sketch/slide)                                             |                            | Unified command for Slide Curve CV, Slide Surface CV.                                                       |
| [Slide Curve CV](/sketch/slide-curve-cv)                           | <kbd>Shift+G</kbd>         | Slide a CV on a Curve along the CV Hull.                                                                    |
| [Text](/sketch/text)                                               |                            | Create Curves from text.                                                                                    |
| [Toggle Points](/sketch/toggle-points)                             |                            | Hide or Unhide Control-Vertex points.                                                                       |
| [Measure Curvature](/sketch/measure-curvature)                    |                            | Unified command for Measure Curvature.                                                                            |
| [Measure Curve Curvature](/sketch/measure-curve-curvature)         |                            | Measure curvature across curves.                                                                            |
| **Project Commands**                                               |                           |                                                                                                            |
| [Alternative Duplicate](/sketch/alternative-duplicate)             | <kbd>Alt+D</kbd>       | Project the Curves onto the current construction plane.                                                     |
| [Create Outline](/sketch/create-outline)                           |                          | Project the outline of the objects to create a 3D Curve.                                                    |
| [Duplicate Curve and Project](/sketch/duplicate-curve-and-project) |                          | Duplicate Curve And Project.                                                                                |
| [Project](/sketch/project)                                         | <kbd>I</kbd>             | Unified command for Project Body Body, Project Curve Body, Project Curve Curve.                             |
| [Project Body Body](/sketch/project-body-body)                     | <kbd>I</kbd>             | Create a new Curve representing the intersection line between two Solids or Sheets.                         |
| [Project Curve Body](/sketch/project-curve-body)                   | <kbd>I</kbd>             | Create a new Curve by projecting a Curve onto a Solid or Sheet.                                             |
| [Project Curve Curve](/sketch/project-curve-curve)                 | <kbd>I</kbd>             | Create a new Curve by projecting the intersection points of surfaces generated from two independent Curves. |
| [Project Outline](/sketch/project-outline)                         | <kbd>Alt+D</kbd>       | Project the outline of the Solids or Sheets.                                                                |
| **Join and Split Commands**                                        |                          |                                                                                                            |
| [Bridge](/sketch/bridge)                                           |                          | Unified command for Bridge Curve, Bridge Vertex, Bridge Edge.                                               |
| [Bridge Curve](/sketch/bridge-curve)                               |                          | Bridges two Curves.                                                                                         |
| [Bridge Edge](/sketch/bridge-edge)                                 |                          | Bridges two Edges.                                                                                          |
| [Bridge Vertex](/sketch/bridge-vertex)                             |                          | Bridges two Vertices.                                                                                       |
| [Cut Curve](/sketch/cut-curve)                                     |                          | Cut Curve.                                                                                                  |
| [Join](/sketch/join)                                               | <kbd>J</kbd>             | Unified command for Join Curves, Join Sheets.                                                               |
| [Join Curves](/sketch/join-curves)                                 | <kbd>J</kbd>             | Join Curves at connecting points.                                                                           |
| [Split Segment](/sketch/split-segment)                             |                          | Split a Curve into two segments.                                                                            |
| [Trim](/sketch/trim)                                               | <kbd>T</kbd>             | Trim Curve.                                                                                                 |
| [Unjoin](/sketch/unjoin)                                           | <kbd>Alt+J</kbd>       | Unified command for Unjoin Curves, Unjoin Faces, Unjoin Shells.                                             |
| [Unjoin Curves](/sketch/unjoin-curve)                              | <kbd>Alt+J</kbd>       | Separate joined Curves.                                                                                     |
| [Instance](/sketch/instance)                                 |                         | Unified command for Instance Curve, Instance Solid and Sheet.                                                                              |
| [Instance Curve](/sketch/instance-curve)                                 |                         | Duplicate multiple uneditable objects.                                                                              |
| [Insert Knot](/sketch/insert-knot)                                 |                         | Insert a CV on a spline curve.                                                                              |
| **Vertex Commands**                                                |                          |                                                                                                            |
| [Convert Vertex](/sketch/convert-vertex)                           |                          | Convert Vertices to Control-Vertices.                                                                       |
| [Delete Redundant Topology](/sketch/delete-redundant-topology)     |                          | Remove excess points.                                                                                       |
