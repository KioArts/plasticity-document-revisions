# Bedienung des 3D‑Viewports

## Bedienung mit der Maus

| Aktion                                              | Maus                                                     |
| --------------------------------------------------- | -------------------------------------------------------- |
| Ansicht drehen                                      | Mit der mittleren Maustaste ziehen (Scrollrad-Taste).   |
| Ansicht verschieben                                 | Mit der rechten Maustaste ziehen.                       |
| In festen Schritten zoomen                          | Mausrad scrollen.                                       |
| Kontinuierlich hinein- oder herauszoomen            | <kbd>Strg</kbd> + Ziehen mit der mittleren Maustaste.   |
| Um die Weltachsen und Diagonalen herum orbitieren   | Ziehen mit der mittleren Maustaste + <kbd>Alt</kbd>     |
| Beim Drehen in orthografische Projektion einrasten  | <kbd>Alt</kbd> + Ziehen mit der mittleren Maustaste.    |
| Bildschirm auf Cursorposition zentrieren            | <kbd>Alt</kbd> + Klick mit der mittleren Maustaste.     |

Sie können die Maus auch an andere Anwendungen anpassen. Siehe [Einstellungen](preferences) für weitere Informationen.

## Bedienung mit Tastaturkürzeln

### Ändern des Blickwinkels

Sie können die Blickrichtung über den Nummernblock anpassen.

| Blickrichtung                                        | Shortcut                |
| ---------------------------------------------------- | ----------------------- |
| Vorderansicht                                        | <kbd>Num.1</kbd>        |
| Rechte Ansicht                                       | <kbd>Num.3</kbd>        |
| Draufsicht                                           | <kbd>Num.7</kbd>        |
| Rückansicht                                          | <kbd>Strg + Num.1</kbd> |
| Linke Ansicht                                        | <kbd>Strg + Num.3</kbd> |
| Unteransicht                                         | <kbd>Strg + Num.7</kbd> |
|                                                      |                         |
| Wechsel zwischen Perspektive und <br />Orthoprojektion | <kbd>Num.5</kbd>     |

# Bedienung mit Befehlen

### Ändern der Kameraposition

Verwenden Sie Orbit‑Befehle, um die Kamera um einen festen Zielpunkt zu bewegen.

| Befehl                 | Aktion                                                     |
| ---------------------- | ---------------------------------------------------------- |
| Orbit: dolly in        | Bewegt die Kamera näher an das Ziel.                      |
| Orbit: dolly out       | Bewegt die Kamera weiter vom Ziel weg.                    |
| Orbit: pan down        | Verschiebt die Ansicht nach unten, ohne den Winkel zu ändern. |
| Orbit: pan left        | Verschiebt die Ansicht nach links, ohne den Winkel zu ändern. |
| Orbit: pan right       | Verschiebt die Ansicht nach rechts, ohne den Winkel zu ändern. |
| Orbit: pan up          | Verschiebt die Ansicht nach oben, ohne den Winkel zu ändern. |
| Orbit: rotate down     | Neigt die Kamera um das Orbit‑Ziel nach unten.            |
| Orbit: rotate left     | Dreht die Kamera um das Orbit‑Ziel nach links.            |
| Orbit: rotate right    | Dreht die Kamera um das Orbit‑Ziel nach rechts.           |
| Orbit: rotate up       | Neigt die Kamera um das Orbit‑Ziel nach oben.             |
