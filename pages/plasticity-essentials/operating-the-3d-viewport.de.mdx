# Bedienung des 3D-Viewports

## Bedienung mit der Maus

| Vorgang                                           | Maus                                                     |
| ------------------------------------------------- | -------------------------------------------------------- |
| Ansicht drehen                                   | Ziehen mit der mittleren Maustaste (Scrollrad-Taste).   |
| Ansicht verschieben                              | Ziehen mit der rechten Maustaste.                       |
| In festen Schritten ein- oder auszoomen          | Mausrad scrollen.                                       |
| Stufenlos ein- oder auszoomen                    | <kbd>Ctrl</kbd> + Mittlere Maustaste ziehen.            |
| Beim Umkreisen an Weltachsen und Diagonalen ausrichten | Mittlere Maustaste ziehen + <kbd>Alt</kbd> |
| Während der Drehung zur orthografischen Ansicht ausrichten | <kbd>Alt</kbd> + Mittlere Maustaste ziehen.             |
| Bildschirm auf die Cursorposition zentrieren     | <kbd>Alt</kbd> + Mittlere Maustaste klicken.            |


Sie können die Maus auch ähnlich wie in anderen Anwendungen einrichten. Weitere Informationen finden Sie unter [Preferences](preferences).

## Bedienung mit Tastenkombinationen

### Ändern des Betrachtungswinkels

Sie können die Betrachtungsrichtung mit dem Ziffernblock anpassen.

| Betrachtungsrichtung                                                    | Tastenkombination                |
| ----------------------------------------------------------------------- | -------------------------------- |
| Frontansicht                                                            | <kbd>Ziffernblock 1</kbd>        |
| Rechte Ansicht                                                          | <kbd>Ziffernblock 3</kbd>        |
| Obere Ansicht                                                           | <kbd>Ziffernblock 7</kbd>        |
| Rückansicht                                                             | <kbd>Strg + Ziffernblock 1</kbd> |
| Linke Ansicht                                                           | <kbd>Strg + Ziffernblock 3</kbd> |
| Untere Ansicht                                                          | <kbd>Strg + Ziffernblock 7</kbd> |
|                                                                         |                                  |
| Zwischen perspektivischer und <br />orthografischer Projektion wechseln | <kbd>Ziffernblock 5</kbd>        |
