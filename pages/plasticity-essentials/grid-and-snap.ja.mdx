# Grid and Snap

## Grid[#grid]

- 1. グリッドはアクティブな作業平面に表示されます。グリッドの単位とサイズは[環境設定](/plasticity-essentials/preferences)で指定できます。
- 2. スケッチやマウスドラッグでオブジェクトの編集をしているとき、マウスカーソルをグリッドにスナップできます。
- 3. グリッドスナップのオン/オフは、画面右上の**Snap to grid**をクリックしてください。その下の値を変更するとグリッドのサイズが変わります。
- 4. グリッドスナップを一時的にオフにするには、<kbd>Ctrl</kbd>キーを押しながらドラッグしてください。
- 5. コマンドパレットで実行することもできます。
    - **Grid: toggle grid snapping:** グリッドスナップのオン/オフ
    - **Viewport: decrease grid size:** グリッドのサイズを小さくする
    - **Viewport: increase grid size:** グリッドのサイズを大きくする
    
## メッシュと画像のスナップ[#mesh-and-image-snap]

- 対応するメッシュと画像には、限定的なスナッピング機能を使用できます。

   | タイプ                | スナップ対象 | ヒントヘルプ |
   | ------------------- | ----------- | -------- |
   | メッシュ (OBJ, STL, FBX, 3MF)| 頂点/面 | 頂点/面|
   | 画像 (.PNG, .JPG, .JPEG)| 角 | 角 | 

## Object snap[#object-snap]

- 1. スケッチやマウスドラッグでオブジェクトの編集をしているとき、マウスカーソルをオブジェクトにスナップできます。
- 2. オブジェクトスナップをオン/オフするには、画面右上の**Enable snapping**をクリックしてください。スナップの対象を指定するには、その下のボタン <ButtonIcon type="button-types-of-snapping" /> をクリックして表示された中から選択してください。
- 3. オブジェクトの編集中にスナップしたいときは、<kbd>Ctrl</kbd>キーを押しながらドラッグしてください。スケッチを描くときは、<kbd>Ctrl</kbd>キーは不要です。
- 4. スナップを一時的に無効にするには、<kbd>Shift + X</kbd> キーを押してください。これにより、現在あるスナップが有効になっている場合でも、異なる種類のスナップを一時的に無効にできます。目的のスナップタイプに達するまでボタンを繰り返し押すことができます。
- 5. スナップの対象は、制御点、カーブ、リージョン、エッジ、フェイスを指定できます。スナップの対象のスナップポイントには小さな黒い点が表示されます。
- 6. スナップ対象をポイントすると、チップヘルプが表示されます。そのときクリックすると正確にスナップされます。

    | スナップ対象 | チップヘルプ                                                                                                                                                                       |
    | ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | 制御点       | CV（制御点）                                                                                                                                                                       |
    | カーブ       | Curve（カーブ）, Beginning（始点）, End（終点）, Middle（中間点）, Closest（最も近い点）, Intersection（交点）, X, Y, Z, XY, YZ, ZX, Center（中心点）, 1/4, 3/4, Perpendicular（接点）, Tangent（垂直点）, Tan/Tan（両端が接する点） |
    | リージョン   | Region（リージョン）, Center（中心点）                                                                                                                                             |
    | エッジ       | Edge（エッジ）, End（端点）, Middle（中間点）                                                                                                                                      |
    | フェイス     | Face（フェイス）, Center（中心点）                                                                                                                                                 |

- コマンドパレットで実行することもできます。
    - **Snaps: toggle snap snapping** オブジェクトスナップのオン/オフ
