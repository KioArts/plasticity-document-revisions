# 構築平面

カーブはアクティブな構築平面上に作成されます。
構築平面は XY 平面、YZ 平面、XZ 平面のほか、新規に作成できます。

### XY 平面、YZ 平面、XZ 平面を使用する

- <kbd>テンキー 7</kbd> : XY 平面を使用。上から見た視線になります。
- <kbd>テンキー 3</kbd> : YZ 平面を使用。右から見た視線になります。
- <kbd>テンキー 1</kbd> : XZ 平面を使用。正面から見た視線になります。

### 1. 平面に合わせた構築平面を作成する

ソリッドまたはシートの平面のフェイスに合わせて構築平面を作成できます。

1. 構築平面にしたいフェイスまたはリージョンを選択します。
2. <kbd>スペース</kbd>を押します。
3. 構築平面が作成されます。
4. 作成した構築平面に正対する視線になります。

<VideoPlayer fileName="construction-plane-1"/>

視線を変更したくないときは、<kbd>スペース</kbd>の代わりに<kbd>Shift + スペース</kbd>を押します。

### 2. 面とエッジの選択による、表面に垂直に整列した構築平面の作成

面とエッジを選択することで、表面に垂直に整列した構築平面を作成できます。

1. 使用したい面とエッジを選択します。
2. <kbd>スペース</kbd> キーを押します。
3. 構築平面が作成されます。
4. ビューが作成された構築平面に整列します。

- ビューを変更したくない場合は、<kbd>Shift + スペース</kbd> を押してください。
<VideoPlayer fileName="construction-plane-6"/>

<Callout type="info" emoji="💡">
**Tip**
   面の選択はビューの整列の基準となり、エッジの選択はビューの方向を定めるガイドとして機能します。
</Callout>

### 3. 対向する平行面および領域からミッドプレーン構造平面を作成する

対向する平行面の平均距離位置に基づいて構造平面を作成し、ミッドプレーン構造が得られます。

1. ミッドプレーン構造平面として使用したい対向する面または領域を少なくとも2つ選択します。
2. <kbd>Space</kbd> を押します。
3. ミッドプレーン構造平面が作成されます。
4. 作成された構造平面に合わせてビューが調整されます。

<VideoPlayer fileName="construction-plane-2"/> 

ビューを変更したくない場合は、<kbd>Space</kbd> の代わりに <kbd>Shift + Space</kbd> を押してください。

### 4. 複数の頂点から構築平面を作成する

3つ以上の点をもとに構築平面を作成できます。

1. 構築平面にしたい頂点を3つ以上選択します。
2. <kbd>スペース</kbd>を押します。
3. 3つのときは正確な位置の構築平面が作成され、4つ以上のときはならした位置に作成されます。
4. 作成した構築平面に正対する視線になります。

<VideoPlayer fileName="construction-plane-3"/>

視線を変更したくないときは、<kbd>スペース</kbd>の代わりに<kbd>Shift + スペース</kbd>を押します。
また、スケッチのリージョンを選択して、これに合わせて構築平面を作成することもできます。

### 5. 見えている視線に合わせた構築平面を作成する

作業エリアに見えている視線に合わせて構築平面を作成できます。

#### 5.1 ワールド原点を通る構築平面を作成する

1. 構築平面にしたい視線にします。
2. <kbd>Ctrl + Space</kbd>>を押します。
3. 構築平面が作成されます。

<VideoPlayer fileName="construction-plane-4"/> 

#### 5.2 選択した点を通る構築平面を作成する

1. 構築平面にしたい視線にします。
2. <kbd>Ctrl + Shift + Space</kbd>>を押します。
3. 作成する構築平面の原点をクリックします。
4. 構築平面が作成されます。

<VideoPlayer fileName="construction-plane-5"/>

### 6. 構築平面の保存と利用

作成した構築平面を保存して後から利用できます。

#### 6.1 構築平面の保存

1. 画面上部の **Temporary construction plane** をクリックします。  
2. **Custom plane 0** がOutlinerに追加され、「Planes」に分類され、画面右側の「Added Planes」メニューに表示されます。  

![Outliner Planes](/img/construction-plane-saved.png)    

![Added Planes](/img/construction-plane-menu.png)  

#### 6.2 保存した構築平面の利用

1. 画面右の**Custom plane 0**をクリックします。
2. 保存した構築平面がアクティブになります。

- **Custom plane 0**をダブルクリックすると、保存した構築平面がアクティブになり、正対する視線になります。

#### 6.3 コンストラクションプレーンの名前変更

1. アウトライナーで保存されたコンストラクションプレーンをダブルクリックします。
2. コンストラクションプレーンの名前を変更し、Enterキーを押すか、3D ビューポートで左クリックします。
