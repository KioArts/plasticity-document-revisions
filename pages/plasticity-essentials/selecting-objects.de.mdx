# Objekte auswählen

Wählen Sie die gewünschten Objekte mit der Maus oder Tastatur aus.

## Auswahl mit der Maus

| Ziel         | Mausaktion                   | Auswahl                                                                     |
| ------------ | ---------------------------- | --------------------------------------------------------------------------- |
| Alle Objekte | Klick                        | Das Objekt auswählen.                                                       |
| Alle Objekte | Ziehen von rechts nach links | Wählt Objekte aus, die die grüne Auswahl berühren.                          |
| Alle Objekte | Ziehen von links nach rechts | Wählt Objekte vollständig aus, die in der violetten Auswahl enthalten sind. |

### Objektspezifische Auswahl

| Mausaktion                    | Ziel              | Auswahl                                         |
| ----------------------------- | ----------------- | ----------------------------------------------- |
| <kbd>Alt</kbd> + Klick        | Edge              | Edge-Schleifen auswählen.                       |
| <kbd>Strg + Alt</kbd> + Klick | Edge              | Edge-Ringe auswählen.                           |
| <kbd>Alt</kbd> + Klick        | Curve             | Alle Curves auf der gleichen Ebene auswählen.   |
| <kbd>Alt</kbd> + Klick        | Region            | Alle Regions auf der gleichen Ebene auswählen.  |
| <kbd>Alt</kbd> + Klick        | Face              | Alle zugehörigen Faces auswählen.               |
| <kbd>Strg + Alt</kbd> + Klick | Face              | Alle parallelen Faces auswählen.                |
| <kbd>Strg + Alt</kbd> + Klick | Cylindercal Face  | Alle parallelen zylindrischen Faces auswählen.  |
| <kbd>Alt</kbd> + Klick        | Solid/Sheet/Empty | Alle Solid/Sheet/Empty in der Gruppe auswählen. |

## Auswahl mit Tastenkombinationen

| Befehlsname       | Tastenkombination              | Hinweis                                                    |
| ----------------- | ------------------------------ | ---------------------------------------------------------- |
| Select All        | <kbd>A</kbd>                   | Alle Objekte auswählen.                                    |
| Deselect All      | <kbd>Escape</kbd>              | Auswahl aufheben.                                          |
| Invert Selection  | <kbd>Alt + A</kbd>             | Auswahl umkehren.                                          |
| Select Adjacent   | <kbd>Strg + Umschalt + =</kbd> | Angrenzende Objekte zu den aktuell ausgewählten auswählen. |

## Auswahl mit Befehlen
| Befehlsname      | Beschreibung                                                    |
| ----------------- | ------------------------------------------------------- |
| Select All Curves                 | Wählt alle Kurven im sichtbaren Ansichtsfenster aus.  |
| Select All Sheets                 | Wählt alle Flächenkörper im sichtbaren Ansichtsfenster aus.  |
| Select All Solids                 | Wählt alle Volumenkörper im sichtbaren Ansichtsfenster aus.  |
| Select All Measruments            | Wählt alle Messobjekte im sichtbaren Ansichtsfenster aus.  |

### Auswahl aus den aktuell ausgewählten Objekten

| Befehlname                    | Tastenkürzel        | Hinweis                                                                       |
| ----------------------------- | ------------------- | ----------------------------------------------------------------------------- |
| Auswahl: In Punkte umwandeln   | <kbd>Ctrl + 1</kbd> | Wählt nur die Punkte der derzeit ausgewählten Kurven aus.                    |
| Auswahl: In Kanten umwandeln   | <kbd>Ctrl + 2</kbd> | Wählt nur die Kanten der derzeit ausgewählten Soliden, Blätter, Flächen und Regionen aus. |
| Auswahl: In Flächen umwandeln  | <kbd>Ctrl + 3</kbd> | Wählt nur die Flächen der derzeit ausgewählten Soliden und Blätter aus.      |
| Auswahl: In Solide umwandeln   | <kbd>Ctrl + 4</kbd> | Wählt nur die Soliden oder Blätter der derzeit ausgewählten Flächen und Kanten aus. |
| Auswahl: In Gruppen umwandeln  | <kbd>Ctrl + 5</kbd> | Wählt alle Objekte innerhalb der Gruppe der ausgewählten Objekte aus.       |
