## 1. Skizze[#sketch]

![curve-vertex-controlvertex](/img/curve-vertex-controlvertex.png)

### 1. Kurve [#curve]
- Eine Linie oder Kurve, die aus Vertices und Kontrollpunkten besteht, wird als Kurve bezeichnet.
- Zum Auswählen in den [**Kanten-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Mehrere Kurven können verbunden und als eine einzelne Kurve behandelt werden.
- Wird in Blau angezeigt und wird gelb, wenn sie ausgewählt ist.

### 2. Vertex [#vertex]
- Die Punkte an beiden Enden einer Kurve werden als Vertices bezeichnet.
- Zum Auswählen in den [**Kontrollpunkt-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Werden als violette Punkte angezeigt und werden weiß, wenn sie ausgewählt sind.
- Wenn mehrere Vertices überlappen, wird der Punkt türkis.
    ![overlapped-vertices](/img/overlapped-vertices.png)

### 3. Kontrollpunkt [#control-vertex]
- Punkte, die die Form einer Kurve definieren, werden als Kontrollpunkte bezeichnet.
- Zum Auswählen in den [**Kontrollpunkt-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Kleiner als Vertices, werden als violette Punkte angezeigt und werden weiß, wenn sie ausgewählt sind.
- Vertices und Kontrollpunkte sind durch weiße Linien verbunden.
- Das Hinzufügen von Kontrollpunkten ermöglicht komplexere Kurven.

### 4. Region [#region]
- Ein von Kurven eingeschlossener Bereich wird als Region bezeichnet.
- Zum Auswählen in den [**Flächen-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Wird in Dunkelblau angezeigt und wird hellblau, wenn sie ausgewählt ist. Regionen werden nach etwa 700 Kurven pro Ebene nicht mehr angezeigt, da das Rendering rechenintensiv wird.
- Das Auswählen einer Region aktiviert automatisch den Befehl [**Extrudieren**](/solid/extrude).
- Das Löschen von Regionen löscht die Kurven, die die Region begrenzen.
![region](/img/region.png)

## 2. Festkörper[#solid]

### 1. Festkörper [#solid-1]
- Ein geschlossener Volumenkörper aus mehreren Flächen ohne Löcher wird als Festkörper bezeichnet.
- Zum Auswählen in den [**Körper-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Wird in Grau angezeigt, mit Umrissen die gelb werden, wenn er ausgewählt ist.
![Solid](/img/solid.png)

### 2. Fläche [#sheet]
- Ein offener Körper aus Flächen mit Löchern wird als Fläche bezeichnet.
- Das Entfernen eines Teils der Oberfläche eines Festkörpers verwandelt ihn in einen offenen Körper, eine Fläche.
- Zum Auswählen in den [**Körper-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Wird in Grau angezeigt, mit Umrissen die gelb werden, wenn sie ausgewählt ist.
![Sheet](/img/sheet.png)

### 3. Seite [#face]
- Jede ebene oder gekrümmte Oberfläche, die Festkörper und Flächen bildet, wird als Seite bezeichnet.
- Zum Auswählen in den [**Flächen-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Wird in Grau angezeigt und wird hellgelb, wenn sie ausgewählt ist.
- Das Auswählen einer Seite aktiviert automatisch den Befehl [**Seite versetzen**](/solid/offset-face).
![Face](/img/face.png)

### 4. Kante [#edge]
- Die Grenzlinien, die Seiten umgeben, werden als Kanten bezeichnet.
- Zum Auswählen in den [**Kanten-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Wird als schwarze Linien angezeigt und wird gelb, wenn sie ausgewählt ist.
- Das Auswählen einer Kante, an der zwei Seiten aufeinandertreffen, aktiviert automatisch den Befehl [**Verrundung**](/solid/fillet). Das Auswählen einer offenen Kante auf einer Fläche aktiviert automatisch den Befehl [**Fläche erweitern**](/solid/extend-sheet).
![Edge](/img/edge.png)

## 3. Bild und Netz[#image-and-mesh]

### 1. Bild [#image]
- Bilder, die mit der Importfunktion importiert wurden, werden als Bilder bezeichnet.
- Zum Auswählen in den [**Körper-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Umrisse werden gelb, wenn sie ausgewählt sind.

### 2. Netz [#mesh]
- 3D-Daten wie STL und FBX, die mit der Importfunktion importiert wurden, werden als Netze bezeichnet.
- Zum Auswählen in den [**Körper-Auswahlmodus**](/plasticity-essentials/plasticity-interface/selection-mode) wechseln.
- Umrisse werden gelb, wenn sie ausgewählt sind.
