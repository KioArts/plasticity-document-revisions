# Objekttypen

## Skizze

### Curve
Linie oder Curve bestehend aus Vertices und Control-Vertices.
Um auszuwählen, stellen Sie den [**Edge Selection Mode**](./selection-mode) ein.
Mehrere Curves können zu einer verbunden werden.
Es wird als blaue Linie in der Skizze angezeigt. Wenn Sie den Cursor darauf richten, wird es weiß und bei Auswahl gelb.

![Curve](/img/object-name-curve.png)

### Vertex
Ein Punkt, der an beiden Enden einer Curve existiert und die Position der Curve bestimmt.
Um es anzuzeigen und auszuwählen, stellen Sie den [**Control-Point Selection Mode**](./selection-mode) ein.
Es erscheint als lila Punkt in der Skizze. Es wird weiß, wenn Sie den Cursor darauf richten und weiß bleibt, wenn Sie es auswählen.
Wenn mehrere Vertices genau an derselben Position liegen, werden sie durch einen blau-grünen Punkt angezeigt.
Beachten Sie, dass dies nur anzeigt, dass die Vertices in der gleichen Position sind und nichts mit der Anwesenheit oder Abwesenheit einer Verbindung zu tun hat.

![Vertex](/img/object-name-vertex.png)

### Control-Vertex (CV)
Ein Punkt, der in der Mitte einer Curve existiert und die Form der Curve bestimmt.
Um es anzuzeigen und auszuwählen, stellen Sie den [**Control-Point Selection Mode**](./selection-mode) ein.
Es erscheint als lila Punkt, der kleiner ist als der Vertex in der Skizze. Es wird weiß, wenn Sie den Cursor darauf richten und weiß bleibt, wenn Sie es auswählen.
Vertices und Control-Vertices sind durch eine weiße Linie verbunden, um ihre Beziehung zur Curve leichter zu erkennen. Durch Hinzufügen weiterer Control-Vertices können komplexe Curves ausgedrückt werden.

![Control-Vertex](/img/object-name-cv.png)

### Region
Der Bereich wird von Curves umgeben. Um auszuwählen, stellen Sie den [**Face Selection Mode**](./selection-mode) ein.
Es erscheint als blauer Bereich in der Skizze. Wenn Sie den Cursor auf die Region richten, wird der Standardbefehl [**Extrude**](../solid/extrude) automatisch ausgeführt.
<Callout emoji="❗">
**Hinweis**<br />
Das Ergebnis des Extrude-Befehls unterscheidet sich, wenn eine geschlossene Curve ausgewählt ist und wenn eine Region ausgewählt ist: Eine extrudierte Curve wird zu einem Sheet und eine extrudierte Region zu einem Solid.
</Callout>
![Region](/img/object-name-region.png)

## Solid

### Solid
Ein vollständig geschlossenes dreidimensionales Objekt ohne Lücken in der Oberfläche. Um auszuwählen, stellen Sie den [**Body Selection Mode**](./selection-mode) ein. Wenn Sie einige Faces löschen, wird es zu einem Sheet. Es wird grau im 3D-Viewport angezeigt und die Kontur wird weiß, wenn Sie den Cursor darauf richten und gelb, wenn Sie es auswählen.

![Solid](/img/object-name-solid.png)

### Sheet
Eine offene Oberfläche, die aus einer oder mehreren Faces besteht. Um es auszuwählen, stellen Sie den [**Body Selection Mode**](./selection-mode) ein. Es wird grau im 3D-Viewport angezeigt und die Kontur wird weiß, wenn Sie den Cursor darauf richten und gelb, wenn Sie es auswählen.

![Sheet](/img/object-name-sheet.png)

### Face
Die ebenen oder gekrümmten Oberflächen, die Sheets und Solids bilden. Um auszuwählen, stellen Sie den [**Face Selection Mode**](./selection-mode) ein. Es wird grau im 3D-Viewport angezeigt, es hellt auf, wenn Sie den Cursor darauf richten und wird gelb, wenn Sie es auswählen. Wenn ein Face ausgewählt ist, wird der Standardbefehl [**Offset Face**](../solid/offset-face) automatisch ausgeführt.

![Face](/img/object-name-face.png)

### Edge
Der Rand eines Face.
Um auszuwählen, stellen Sie den [**Edge Selection Mode**](./selection-mode) ein. Er wird als schwarze Linie angezeigt, die weiß wird, wenn der Cursor darauf zeigt und gelb wird, wenn sie ausgewählt ist. Wenn der Edge des Grenzbereichs zwischen zwei Faces ausgewählt ist, wird der Standardbefehl [Fillet](../solid/fillet) automatisch ausgeführt, und wenn der offene Edge eines Sheet ausgewählt ist, wird der Standardbefehl [**Extend Sheet**](../solid/extend#extend-sheet) automatisch ausgeführt.

![Edge](/img/object-name-edge.png)
