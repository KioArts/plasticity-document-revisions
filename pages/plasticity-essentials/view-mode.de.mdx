# Ansichtsmodus

## Ansichtsmodus umschalten

Der Ansichtsmodus kann durch Klicken auf die Schaltfläche oben rechts auf dem Bildschirm oder durch die Verwendung der Tastenkombination umgeschaltet werden.

![Render-Modus](/img/render-mode.png)

| Funktion                                                                                   | Tastenkombination           | Button | Befehlname                          |
| ------------------------------------------------------------------------------------------ | -------------------------- | ------ | ----------------------------------- |
| Schnittanalyse umschalten                                                                  |                            | (1)    | Schnittanalyse                      |
| Orthografische Kamera umschalten                                                           | <kbd>Num. 5</kbd>          | (2)    | Ansicht: Orthografische Kamera umschalten |
| Overlay-Anzeige umschalten<br/>(Schaltet das Raster ein oder aus.)                        | <kbd>Alt+Shift+Z</kbd>     | (3)    | Ansicht: Overlays umschalten        |
| X-Ray-Modus umschalten<br/>(Macht verborgene Linien sichtbar und auswählbar.)             | <kbd>Alt + Z</kbd>         | (4)    | Ansicht: X-Ray-Modus umschalten     |
| Render-Modus umschalten<br/>(Wenn aktiviert, werden die Materialeinstellungen reflektiert.) |                            | (5)    |                                     |
