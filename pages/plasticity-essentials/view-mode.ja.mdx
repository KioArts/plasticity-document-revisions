# ビューモード

## ビューモードの切り替え

ビューモードは、画面右上のボタンをクリックするか、キーボードショートカットを使用して切り替えることができます。

![レンダーモード](/img/render-mode.png)

| 機能                                                                                      | ショートカット              | ボタン | コマンド名                          |
| ----------------------------------------------------------------------------------------- | -------------------------- | ------ | ------------------------------------ |
| セクション解析の切り替え                                                                  |                            | (1)    | セクション解析                      |
| 正投影カメラの切り替え                                                                    | <kbd>Num. 5</kbd>          | (2)    | ビューポート: 正投影カメラの切り替え |
| オーバーレイ表示の切り替え<br/>(グリッド表示のオン/オフを切り替えます。)                  | <kbd>Alt+Shift+Z</kbd>     | (3)    | ビューポート: オーバーレイの切り替え |
| X線モードの切り替え<br/>(隠れた線を可視化し、選択可能にします。)                          | <kbd>Alt + Z</kbd>         | (4)    | ビューポート: X線モードの切り替え    |
| レンダーモードの切り替え<br/>(オンにすると、マテリアル設定が反映されます。)               |                            | (5)    |                                      |
