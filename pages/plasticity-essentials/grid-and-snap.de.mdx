# Gitter und Fang

## Gitter[#grid]

- 1. Das Gitter wird auf der aktiven Konstruktionsebene angezeigt. Sie können die Gittereinheit und -größe in den [preferences](/plasticity-essentials/preferences) festlegen.
- 2. Sie können das Fang am Gitter verwenden, um Ihren Cursor beim Zeichnen oder Verschieben von Objekten am Gitter auszurichten.
- 3. Um das Fang am Gitter umzuschalten, klicken Sie in der oberen rechten Ecke des Fensters auf **Snap to grid**. Um die Gittergröße festzulegen, ändern Sie den Wert unter **Snap to grid**.
- 4. Um das Fang am Gitter vorübergehend zu deaktivieren, halten Sie <kbd>Strg</kbd> während des Ziehens gedrückt.
- 5. Sie können das Fang am Gitter im Befehlspalette ausführen; **Grid: toggle grid snapping**, **Viewport: decrease grid size**, **Viewport: increase grid size**.

## Mesh und Bild-Snap[#mesh-and-image-snap]

- Sie können begrenzte Snapping-Funktionen für die jeweiligen Meshes und Bilder verwenden.

   | Typ                | Snap-Ziel | Tipp-Hilfe |

   | Mesh (OBJ, STL, FBX, 3MF)| Scheitelpunkt/Fläche | Scheitelpunkt/Fläche|
   | Bild (.PNG, .JPG, .JPEG)| Ecke | Ecke | 

## Objekt Fang[#object-snap]

- 1. Sie können das Einrasten verwenden, um Ihren Cursor beim Zeichnen oder Verschieben von Objekten an Merkmale auszurichten.
- 2. Um das Einrasten von Objekten umzuschalten, klicken Sie in der oberen rechten Ecke des Fensters auf **Enable snapping**. Um verschiedene Einrasttypen umzuschalten, klicken Sie auf <ButtonIcon type="button-types-of-snapping" /> und passen Sie die Optionen an.
- 3. Um das Einrasten vorübergehend zu aktivieren, halten Sie die <kbd>Strg</kbd> Taste gedrückt. Beim Zeichnen von Kurven wird das Einrasten automatisch aktiviert.
- 4. Um das Einrasten vorübergehend zu deaktivieren, drücken Sie die Taste <kbd>Shift + X</kbd>. Dadurch können Sie verschiedene Arten des Einrastens vorübergehend ausschalten, wenn Sie gerade an eine eingerastet sind. Sie können die Taste wiederholt drücken, bis Sie den gewünschten Einrasttyp erreicht haben.
- 5. Sie können an Punkten verschiedener Objekttypen ausrichten: Control-Vertex, Curve, Region, Edge, and Face. Ein Fangziel erscheint als kleiner schwarzer Punkt.
- 6. Ein Hinweistext erscheint, wenn Sie auf ein Fangziel zeigen. Klicken Sie, wenn der Hinweistext sichtbar ist, um genau auszurichten.

    | Fangziel       | Hinweistext                                                                                                                  |
    | -------------- | ---------------------------------------------------------------------------------------------------------------------------- |
    | Control-Vertex | CV                                                                                                                           |
    | Curve          | Curve, Beginning, End, Middle, Closest, Intersection, X, Y, Z, XY, YZ, ZX, Center, 1/4, 3/4, Perpendicular, Tangent, Tan/Tan |
    | Region         | Region, Center                                                                                                               |
    | Edge           | Edge, End, Middle                                                                                                            |
    | Face           | Face, Center                                                                                                                 |

- Sie können das Einrasten von Objekten im Befehlspalette ausführen; **Snaps: toggle snap snapping**.
