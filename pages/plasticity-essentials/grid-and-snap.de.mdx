# Gitter und Fang

## Gitter[#grid]

- Das Gitter wird auf der aktiven Konstruktionsebene angezeigt. Sie können die Gittereinheit und -größe in den [Einstellungen](../start/preferences) festlegen.
- Sie können das Einrasten im Gitter verwenden, um Ihren Cursor beim Zeichnen oder Verschieben von Objekten am Gitter auszurichten.
- Um das Einrasten im Gitter umzuschalten, klicken Sie in der oberen rechten Ecke des Fensters auf **Am Gitter ausrichten**. Um die Gittergröße festzulegen, ändern Sie den Wert unter **Am Gitter ausrichten**.
- Um das Einrasten im Gitter vorübergehend zu deaktivieren, halten Sie <kbd>Strg</kbd> während des Ziehens gedrückt.
- Sie können das Einrasten im Gitter im Befehlspalette ausführen; **Gitter: Einrasten im Gitter umschalten**, **Ansicht: Gittergröße verkleinern**, **Ansicht: Gittergröße vergrößern**.

## Objekt Einrasten[#object-snap]

- Sie können das Einrasten verwenden, um Ihren Cursor beim Zeichnen oder Verschieben von Objekten an Merkmale auszurichten.
- Um das Einrasten von Objekten umzuschalten, klicken Sie in der oberen rechten Ecke des Fensters auf **Einrasten aktivieren**. Um verschiedene Einrasttypen umzuschalten, klicken Sie auf <ButtonIcon type="button-types-of-snapping" /> und passen Sie die Optionen an.
- Um das Einrasten vorübergehend zu aktivieren, halten Sie die <kbd>Strg</kbd> Taste gedrückt. Beim Zeichnen von Kurven wird das Einrasten automatisch aktiviert.
- Sie können an Punkten verschiedener Objekttypen ausrichten: Control-Vertex, Curve, Region, Edge, and Face. Ein Einrastziel erscheint als kleiner schwarzer Punkt.
- Ein Hinweistext erscheint, wenn Sie auf ein Einrastziel zeigen. Klicken Sie, wenn der Hinweistext sichtbar ist, um genau auszurichten.

    | Einrastziel    | Hinweistext                                                                                          |
    | -------------- | ---------------------------------------------------------------------------------------------------- |
    | Control-Vertex | CV                                                                                                   |
    | Curve          | Curve, Beginning, End, Middle, Closest, Intersection, X, Y, Z, XY, YZ, ZX, Center, 1/4, 3/4, Tan/Tan |
    | Region         | Region, Center                                                                                       |
    | Edge           | Edge, End, Middle                                                                                    |
    | Face           | Face, Center                                                                                         |

- Sie können das Einrasten von Objekten im Befehlspalette ausführen; **Snaps: Einrasten umschalten**.
