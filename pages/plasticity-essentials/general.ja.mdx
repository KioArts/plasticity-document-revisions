## 一般的な機能

| コマンド名                | ショートカット                                    | ノート                                                                           |
| ------------------------- | ----------------------------------------------- | -------------------------------------------------------------------------------- |
| コピー                    | <kbd>Ctrl + C</kbd>                             |                                                                                  |
| 貼り付け                  | <kbd>Ctrl + V</kbd>                             |                                                                                  |
| 元に戻す                  | <kbd>Ctrl + Z</kbd>                             |                                                                                  |
| やり直す                  | <kbd>Ctrl + Shift + Z</kbd>                     |                                                                                  |
| 削除                      | <kbd>Shift+X</kbd> または <kbd>Shift+Delete</kbd> または <kbd>Shift+Backspace</kbd> | 選択したオブジェクトを削除します。グループを削除すると、そのグループとその子要素も削除されます。 |
| コントロールポイントの削除 |                                                 | 選択したコントロールポイントを削除します。                                         |
| エッジの削除               |                                                 | 選択したエッジを削除します。                                                      |
| 面の削除                  | <kbd>Shift+X</kbd> または <kbd>Shift+Delete</kbd> または <kbd>Shift+Backspace</kbd> | 選択した面を削除します。                                                           |
| 解消                      | <kbd>Backspace</kbd> または <kbd>X</kbd>        | 選択したオブジェクトを削除します。グループを解消すると、そのグループは削除されますが、アイテムは残ります。 |
| 面の解消                  |                                                 | 冗長な面を削除します。                                                            |
| 編集: 最後のコマンドを繰り返す | <kbd>Shift+R</kbd>                            | 最後のコマンドを繰り返します。                                                     |
| サイドバー: 左側の表示/非表示 | <kbd>Ctrl+B</kbd>                             | 左側サイドバーの表示/非表示を切り替えます。                                        |
| サイドバー: 右側の表示/非表示 | <kbd>Ctrl + Shift + B</kbd>                   | 右側サイドバーの表示/非表示を切り替えます。                                        |
