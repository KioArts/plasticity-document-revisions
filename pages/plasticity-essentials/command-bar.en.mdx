# Command Bar

## Command Bar
- The command bar is located at the bottom of the screen.
![Command Bar](/img/command-bar.png)

- The command bar contains common commands such as: 

| Command | Shortcut |
| ------- | -------- |
| [Search Commands](/plasticity-essentials/command-palette) | <kbd>F</kbd> |
| [Move](/common/move) | <kbd>G</kbd> |
| [Rotate](/common/rotate) | <kbd>R</kbd> |
| [Scale](/common/scale) | <kbd>S</kbd> |
| [Duplicate Object](/common/duplicate) | <kbd>Shift + D</kbd> |
| [Boolean](/solid/boolean) | <kbd>Q</kbd> |
| [Rectangular Array](/common/rectangular-array) | - |
| [Radial Array](/common/radial-array) | - |
| Suggested Commands | <kbd>Shift + F</kbd>> |


Move, Rotate, Scale, Duplicate Object, Boolean, Rectangular Array, Radial Array and Suggested Commands.

- The command bar includes **Suggested Commands**, a window that highlights commands and tools based on your selection - whether it's a Solid, Sheet, Curve, Region, or Point. To expand the window, click the icon or press <kbd>Shift + F</kbd>.

![Suggested Commands](/img/suggested-commands.png)
