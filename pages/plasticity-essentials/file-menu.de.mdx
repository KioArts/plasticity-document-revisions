# P-Menü  

Das P-Menü kann durch Klicken auf das "P" in der oberen linken Ecke geöffnet werden.  

![P-Menü](/img/file-menu.png)  

| Befehlsname            | Tastenkombination           | Funktion                                                                                                                                                                                         |
| ---------------------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Letzte Dateien        |                             | Zeigt zuletzt gespeicherte Dateien an. Beim Darüberfahren wird eine Vorschau mit Miniaturansichten der Dateien und deren Inhalt angezeigt. Dateien können per Drag & Drop in die aktuelle Plasticity-Sitzung gezogen werden. |
| Neue Datei            | <kbd>Ctrl+N</kbd>         | Löscht die Szene und öffnet ein neues Dokument.                                                                                                                                                  |
| Neues Fenster         |                             | Startet Plasticity und öffnet ein neues Fenster. Die Akzentfarbe des neuen Fensters wird zufällig geändert, um es vom Hauptfenster zu unterscheiden. Kopieren und Einfügen zwischen Fenstern ist möglich. |
| Öffnen               | <kbd>Ctrl+O</kbd>         | Öffnet eine Plasticity-Datei.                                                                                                                                                                    |
| Import/Anhängen       | <kbd>Ctrl+Shift+O</kbd> | Importiert Dateien. Siehe [hier](import-export#file-import) für importierbare Formate.                                                                                                          |
| Speichern            | <kbd>Ctrl+S</kbd>         | Speichert das aktuelle Dokument.                                                                                                                                                                 |
| Speichern unter      | <kbd>Ctrl+Shift+S</kbd> | Speichert das Dokument unter einem neuen Namen.                                                                                                                                                   |
| Exportieren          | <kbd>Ctrl+Shift+E</kbd> | Exportiert Dateien. Siehe [hier](import-export#file-export) für exportierbare Formate.                                                                                                          |
| Schnellversion speichern |                         | Speichert eine Kopie der Datei mit einem Zeitstempel im Dateinamen.                                                                                                                              |
| Als Startszene speichern |                         | Speichert das aktuelle Dokument als Startszene, die beim nächsten Start automatisch geöffnet wird.                                                                                                |
| Einstellungen        | <kbd>Ctrl+,(Komma)</kbd> | Siehe [Einstellungen](preferences).                                                                                                                                                            |
| Lizenzen verwalten   |                             | Verwalten von Lizenz-Token und Geräten.                                                                                                                                                          |
| Plasticity-Version   |                             | Zeigt die installierte Version von Plasticity an.                                                                                                                                                |
| Beenden             | <kbd>Ctrl+Q</kbd>         | Beendet Plasticity.                                                                                                                                                                             |
