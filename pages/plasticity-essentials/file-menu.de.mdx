# P-Menü

Das P‑Menü kann geöffnet werden, indem man oben links auf das **P** klickt.

![P-Menü](/img/file-menu.png)

| Befehlsname                     | Tastenkürzel                | Funktion                                                                                                                                                                                        |
| --------------------------------| ----------------------------| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Zuletzt verwendete Dateien     |                            | Zeigt zuletzt gespeicherte Dateien an, an denen gearbeitet wurde. Beim Darüberfahren erscheint ein Vorschaubild mit Miniaturansichten der Dateien und deren Inhalt. Dateien können per Drag‑&‑Drop in die aktuelle Arbeitssitzung im Plasticity‑Projekt gezogen werden. |
| Neue Datei                      | <kbd>Strg+N</kbd>           | Löscht die Szene und öffnet ein neues Dokument.                                                                                                                                                 |
| Neues Fenster                   |                            | Startet Plasticity und öffnet ein neues Fenster. Die Akzentfarbe des neuen Fensters wird zufällig geändert, um es vom Hauptfenster zu unterscheiden. Kopieren und Einfügen zwischen Fenstern ist ebenfalls möglich. |
| Öffnen                          | <kbd>Strg+O</kbd>           | Öffnet eine Plasticity‑Datei.                                                                                                                                                                   |
| Importieren/Anhängen            | <kbd>Strg+Umschalt+O</kbd>  | Importiert Dateien. Siehe [hier](import-export#file-import) für unterstützte Formate.                                                                                                            |
| Speichern                       | <kbd>Strg+S</kbd>           | Speichert das aktuelle Dokument.                                                                                                                                                                |
| Speichern unter…                | <kbd>Strg+Umschalt+S</kbd>  | Speichert das Dokument unter einem neuen Namen.                                                                                                                                                 |
| Exportieren                     | <kbd>Strg+Umschalt+E</kbd>  | Exportiert Dateien. Siehe [hier](import-export#file-export) für exportierbare Formate.                                                                                                           |
| Schnellspeicher-Version         |                            | Legt eine Kopie der Datei mit Zeitstempel im Dateinamen an.                                                                                                                                     |
| Als Startszene speichern        |                            | Speichert das aktuelle Dokument als Startszene, die beim nächsten Start automatisch geladen wird.                                                                                               |
| Einstellungen                   | <kbd>Strg+,</kbd>           | Siehe [Einstellungen](preferences).                                                                                                                                                             |
| Lizenzen verwalten              |                            | Verwalten von Lizenz‑Tokens und Geräten.                                                                                                                                                        |
| Aktuelle Version 00.0.0         |                            | Aktuelle Version von Plasticity.                                                                                                                                                                 |
| Auf Plasticity Share veröffentlichen |                        | Veröffentlichung im Web. Siehe [hier](/common/publish-to-plasticity-share)                                                                                                                       |
| Meine geteilten Dokumente verwalten |                         | Hochgeladene (.plasticity‑)Dokumente auf Plasticity Share verwalten. Siehe [hier](/common/publish-to-plasticity-share)                                                                          |
| Beenden                         | <kbd>Strg+Q</kbd>           | Schließt Plasticity.                                                                                                                                                                            |
