# 3Dビューポートの操作

## マウスによる操作

| 操作                                             | マウス操作                                               |
| ------------------------------------------------ | -------------------------------------------------------- |
| ビューを回転する                                 | 中ボタン（スクロールホイールボタン）をドラッグする。     |
| ビューをパンする                                 | 右ボタンをドラッグする。                                 |
| 固定の増減でズームイン・ズームアウトする         | マウスホイールをスクロールする。                         |
| 連続的にズームイン・ズームアウトする             | <kbd>Ctrl</kbd> + 中ボタンをドラッグする。              |
| 回転時にワールド軸とその対角線にスナップする     | 中ボタンをドラッグ + <kbd>Alt</kbd> |
| 回転時に直交投影にスナップする                   | <kbd>Alt</kbd> + 中ボタンをドラッグする。               |
| カーソル位置を中心に画面を移動する               | <kbd>Alt</kbd> + 中ボタンをクリックする。               |

他のアプリケーションと同様のマウスにすることもできます。詳しくは[環境設定](preferences)を参照してください。

## キーボードショートカットによる操作

### ビューのアングルを変える

テンキーでビューの方向を調整できます。

| ビューの方向                 | ショートカット          |
| ---------------------------- | ----------------------- |
| 前から                       | <kbd>Num.1</kbd>        |
| 右から                       | <kbd>Num.3</kbd>        |
| 上から                       | <kbd>Num.7</kbd>        |
| 後ろから                     | <kbd>Ctrl + Num.1</kbd> |
| 左から                       | <kbd>Ctrl + Num.3</kbd> |
| 下から                       | <kbd>Ctrl + Num.7</kbd> |
|                              |                         |
| 透視投影と正投影を切り替える | <kbd>Num.5</kbd>        |
