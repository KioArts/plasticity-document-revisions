# 3D ビューポートの操作

## マウスでの操作

| 操作内容                                               | マウス操作                                                |
| ------------------------------------------------------ | --------------------------------------------------------- |
| ビューを回転する                                       | 中ボタン（ホイールボタン）をドラッグ                     |
| ビューをパンする                                       | 右ボタンをドラッグ                                       |
| 一定量でズームイン／アウトする                         | マウスホイールをスクロール                               |
| 連続的にズームイン／アウトする                         | <kbd>Ctrl</kbd> + 中ボタンドラッグ                       |
| ワールド軸およびその対角方向にスナップしながら回転する | 中ボタンドラッグ + <kbd>Alt</kbd>                        |
| 回転中に正投影へスナップする                           | <kbd>Alt</kbd> + 中ボタンドラッグ                        |
| カーソル位置を中心に画面を再配置する                   | <kbd>Alt</kbd> + 中ボタンクリック                        |

マウス操作を他のアプリケーションに近づけることもできます。詳細は [設定](preferences) を参照してください。

## キーボードショートカットでの操作

### 視点角度の変更

テンキーを使用して視点方向を調整できます。

| 視点方向                                               | ショートカット          |
| ------------------------------------------------------ | ------------------------ |
| 前面ビュー                                             | <kbd>Num.1</kbd>         |
| 右面ビュー                                             | <kbd>Num.3</kbd>         |
| 上面ビュー                                             | <kbd>Num.7</kbd>         |
| 背面ビュー                                             | <kbd>Ctrl + Num.1</kbd>  |
| 左面ビュー                                             | <kbd>Ctrl + Num.3</kbd>  |
| 底面ビュー                                             | <kbd>Ctrl + Num.7</kbd>  |
|                                                        |                          |
| 透視投影と<br />正投影の切り替え                       | <kbd>Num.5</kbd>         |

# コマンドでの操作

### カメラ位置の変更

固定ターゲット点を中心にカメラを移動するために、オービットコマンドを使用します。

| コマンド               | 操作内容                                                   |
| ---------------------- | ---------------------------------------------------------- |
| Orbit: dolly in        | カメラをターゲットに近づける                               |
| Orbit: dolly out       | カメラをターゲットから遠ざける                             |
| Orbit: pan down        | 角度を変えずにビューを下方向へ移動                         |
| Orbit: pan left        | 角度を変えずにビューを左方向へ移動                         |
| Orbit: pan right       | 角度を変えずにビューを右方向へ移動                         |
| Orbit: pan up          | 角度を変えずにビューを上方向へ移動                         |
| Orbit: rotate down     | ターゲットを中心にカメラを下方向へ傾ける                   |
| Orbit: rotate left     | ターゲットを中心にカメラを左方向へ回転                     |
| Orbit: rotate right    | ターゲットを中心にカメラを右方向へ回転                     |
| Orbit: rotate up       | ターゲットを中心にカメラを上方向へ傾ける                   |
