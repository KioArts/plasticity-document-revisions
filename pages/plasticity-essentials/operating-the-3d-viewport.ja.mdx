# 3Dビューポートの操作

## マウスによる操作

| 操作                                              | マウス                                                       |
| -------------------------------------------------- | -------------------------------------------------------- |
| ビューを回転する                                     | 中ボタン（スクロールホイールボタン）をドラッグする。              |
| ビューをパンする                                    | 右ボタンをドラッグする。                                     |
| ズームインまたはズームアウトする                    | マウスホイールをスクロールする。                               |
| ワールド軸およびその間の対角線にスナップしながら回転する | 中ボタンをドラッグ + <kbd>Alt</kbd>                     |
| 回転中に正投影にスナップする                        | <kbd>Alt</kbd> + 中ボタンドラッグ。                        |
| カーソル位置を中心に画面を中央揃えにする              | <kbd>Alt</kbd> + 中ボタンクリック。                         |


他のアプリケーションと同様のマウスにすることもできます。詳しくは[環境設定](preferences)を参照してください。

## キーボードショートカットによる操作

### ビューのアングルを変える

テンキーでビューの方向を調整できます。

| ビューの方向                 | ショートカット          |
| ---------------------------- | ----------------------- |
| 前から                       | <kbd>Num.1</kbd>        |
| 右から                       | <kbd>Num.3</kbd>        |
| 上から                       | <kbd>Num.7</kbd>        |
| 後ろから                     | <kbd>Ctrl + Num.1</kbd> |
| 左から                       | <kbd>Ctrl + Num.3</kbd> |
| 下から                       | <kbd>Ctrl + Num.7</kbd> |
|                              |                         |
| 透視投影と正投影を切り替える | <kbd>Num.5</kbd>        |
