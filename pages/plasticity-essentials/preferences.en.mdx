# Preferences

To open the Preferences window, click the P menu in the top left corner and select **Preferences**.
You need to restart Plasticity when you change the settings.

## Navigation

### Rotation Mode

- **Tumble:** 
    - Rotates the view around a fixed vertical axis, keeping the up direction locked. Moving the mouse left/right rotates horizontally, and up/down rotates vertically but the view cannot roll or flip upside down. (Default: **Selected**)

- **Arcball:** 
    - Rotates the view freely in all three axes with no axis constraints. The mouse movement is mapped to the surface of an imaginary sphere around the scene, allowing full 360 degree rotation in any direction.

### Navigation Settings

- **Zoom to cursor**:
    - When enabled, zooms in at the cursor position. (Default: **Enabled**)
- **Rotate around cursor**:
    - When enabled, the view rotates around the cursor position. (Default: **Enabled**)
- **Zoom vertical**:
    - When enabled, zooms in/out with vertical dragging using <kbd>Ctrl + Middle Button</kbd>. (Default: **Enabled**)
- **Invert zoom direction**:
    - When enabled, inverts the zoom direction. (Default: **Disabled**)
- **Invert wheel direction**:
    - When enabled, inverts the mouse wheel direction. (Default: **Disabled**)

### Navigation Presets

- You can choose presets for the same controls as other 3D software.

## Appearance

Customize colors for the UI and viewport, and adjust the size of control points, vertices, edges, and curve segments.

### UI Colors

- **Accent Colors:** Set the primary colors for the Plasticity UI. Changes apply in real time.
- **Supporting Colors:** Set the secondary colors for the UI. Affects active group highlights, connected CVs, and related elements.
- **Neutral Colors:** Set the background colors for menus and toolbars.
- **Viewport Colors:** Set the viewport background color.
  - **Choose Custom Color:** Specify a custom color.
- **Grid Colors:** Set the grid color.
  - **Choose Custom Color:** Specify a custom color.
- **Measurement Color:** Set the color for measurement lines.
  - **Choose Custom Color:** Specify a custom color.

### Properties

- **Control Points:** Specify the size of control points on curves.
- **Vertices:** Specify the size of vertices on curves.
- **Edges:** Specify the size of edges on sheets and solids.
- **Curve Segments:** Specify the thickness of curve lines.
- **UI Standard Spacing:** Specify the padding and spacing of UI elements such as toolbars and menus.
- **Zoom Factor:** Specify the zoom sensitivity.

### Ambient Occlusion (AO)

Adjust the **Ambient Occlusion** appearance in the viewport using the **Radius (1)** and **Falloff (2)** settings. The Radius value is a percentage of your current length unit. For example, with meters, a value of 10 converts to 0.1 meters. Radius ranges from 0 to 1000 and Falloff from 0.0 to 10.0 (defaults: 15 and 0.5).

## Server

- Configure the server to be used with the Blender Bridge.
- The default is Host: localhost, Port: 8980. Match these values with the settings in the Blender addon.

## Web Publishing

Turn on or off [**Publish to Plasticity Share**](/common/publish-to-plasticity-share) command. (Default: Off)

## SpaceMouse

- These are the settings for using the 3Dconnexion SpaceMouse.
- The SpaceMouse Compact, SpaceMouse Pro, and SpaceMouse Enterprise have been confirmed to work on Windows and macOS. Operation on Linux is unconfirmed.
- Control Pan Speed, Rotate Speed, and Zoom Speed.

## General

- Automatic invoke commands like offset, extrude, and fillet, based on the current selection.
    - When enabled, automatically calls commands such as offset, extrude, and fillet based on the current selection (Default: **Enabled**).

## Advanced

- You can enable or disable the last active session on startup in Plasticity. (Default: **Enabled**)
