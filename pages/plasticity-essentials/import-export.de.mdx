# Import/Export

Sie können Import- und Exportoperationen über die Optionen **"Importieren/Anhängen"** und **"Exportieren"** im P-Menü durchführen. Das Ziehen und Ablegen von Dateien in den 3D-Viewport ist ebenfalls möglich. <br/>

Die folgenden Dateiformate werden unterstützt:

## Importieren [#file-import]

| Dateiformat                 | Indie-Lizenz | Studio-Lizenz |                                                                  |
| --------------------------- | :------------: | :-------------: | ---------------------------------------------------------------- |
| Plasticity (*.plasticity)   | ✓           | ✓            |                                                                  |
| STEP (*.stp; *.step)        | ✓           | ✓            |                                                                  |
| ParaSolid (*.x_t; *.x_b)    | ✓           | ✓            |                                                                  |
| Wavefront OBJ (*.obj)       | ✓           | ✓            |                                                                  |
| FBX (*.FBX)                 | ✓           | ✓            |                                                                  |
| STL (*.stl)                 | ✓           | ✓            |                                                                  |
| Vector Image (*.SVG)         | ✓            | ✓             |                                                      |
| Bild (*.png; *.jpg; *.jpeg) | ✓           | ✓            | Weitere Details finden Sie [hier](importing-reference-images). |
| 3MF (*.3mf)                  |               | ✓             |                                                      |
| IGES (*.igs; *.iges)        |              | ✓            |                                                                  |
| SAT (*sat)                  |              | ✓            |                                                                  |
| Rhino (*.3dm)               |              | ✓            |                                                                  |
| DXF/DWG (*.dxf; *.dwg)       |              | ✓            | Nur auf Windows unterstützt                            |

## Exportieren [#file-export]

| Dateiformat                     | Indie‑Lizenz | Studio‑Lizenz | Eigenständiger Befehl im Befehls‑Palette | Hinweis |
|---------------------------------|:------------:|:-------------:|-------------------------------------------|---------|
| Plasticity (*.plasticity)       | ✓            | ✓             | –                                         | –       |
| STEP (*.stp; *.step)            | ✓            | ✓             | Export STEP                               | –       |
| PNG (*.png)                     | ✓            | ✓             | –                                         | Erfasst einen Screenshot des aktuellen Viewports, wobei UI‑Elemente ausgeblendet werden. Zugriff über **P‑Menü → Export** oder mit <kbd>Ctrl + Shift + E</kbd>. |
| Parasolid (*.x_t; *.x_b)        | ✓            | ✓             | Export Parasolid                          | –       |
| Wavefront OBJ (*.obj)           | ✓            | ✓             | Export OBJ                                | Details siehe [hier](export-obj). |
| STL (*.stl)                     | ✓            | ✓             | Export STL                                | –       |
| Vektorbild (*.svg)              | ✓            | ✓             | Export SVG                                | Verfügbar ab Version 25.3.0 Beta. |
| 3MF (*.3mf)                     | ✓            | ✓             | Export 3MF                                | –       |
| IGES (*.igs; *.iges)            | –            | ✓             | Export IGES                               | –       |
| SAT (*.sat)                     | –            | ✓             | Export SAT                                | –       |
