## Allgemeine Funktionen

| Befehlname                | Tastenkürzel                                      | Hinweis                                                                           |
| ------------------------- | ------------------------------------------------- | --------------------------------------------------------------------------------- |
| Kopieren                  | <kbd>Ctrl + C</kbd>                               |                                                                                   |
| Einfügen                  | <kbd>Ctrl + V</kbd>                               |                                                                                   |
| Rückgängig                | <kbd>Ctrl + Z</kbd>                               |                                                                                   |
| Wiederherstellen          | <kbd>Ctrl + Shift + Z</kbd>                       |                                                                                   |
| Löschen                   | <kbd>Shift+X</kbd> oder <kbd>Shift+Delete</kbd> oder <kbd>Shift+Backspace</kbd> | Löscht ausgewählte Objekte. Beim Löschen einer Gruppe werden die Gruppe und ihre Kinder gelöscht. |
| Steuerpunkt löschen        |                                                   | Löscht ausgewählte Steuerpunkte.                                                   |
| Kante löschen              |                                                   | Löscht ausgewählte Kanten.                                                        |
| Fläche löschen             | <kbd>Shift+X</kbd> oder <kbd>Shift+Delete</kbd> oder <kbd>Shift+Backspace</kbd> | Löscht ausgewählte Flächen.                                                        |
| Auflösen                   | <kbd>Backspace</kbd> oder <kbd>X</kbd>            | Löscht ausgewählte Objekte. Beim Auflösen einer Gruppe wird die Gruppe entfernt, aber ihre Elemente bleiben erhalten. |
| Fläche auflösen            |                                                   | Löscht ausgewählte redundante Flächen.                                             |
| Bearbeiten: Letzten Befehl wiederholen | <kbd>Shift+R</kbd>                            | Wiederholt den letzten Befehl.                                                     |
| Seitenleiste: Links ein/aus | <kbd>Ctrl+B</kbd>                                 | Umschalten der Anzeige der linken Seitenleiste.                                    |
| Seitenleiste: Rechts ein/aus | <kbd>Ctrl + Shift + B</kbd>                       | Umschalten der Anzeige der rechten Seitenleiste.                                   |
