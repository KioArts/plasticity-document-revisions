# General Functions

| Functions                    | Shortcut                                      |
| ---------------------------- | --------------------------------------------- |
| Copy                         | <kbd>Ctrl + C</kbd>                             |
| Paste                        | <kbd>Ctrl + visible</kbd>                       |
| Delete                       | <kbd>X</kbd> or <kbd>Delete</kbd>             |
| Delete Face                  | <kbd>Shift + X</kbd> or <kbd>Shift + Delete</kbd> |
| Undo                         | <kbd>Ctrl + Z</kbd>                             |
| Redo                         | <kbd>Ctrl + Shift + Z</kbd>                       |
| Repeat last command          | <kbd>Shift + R</kbd>                            |
