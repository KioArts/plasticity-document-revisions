## Fokus und Isolierung

## Objekte fokussieren

Dieser Befehl vergrößert das ausgewählte Objekt so, dass es den gesamten Viewport ausfüllt.

{/* <Reference shortcut="/ (Slash)" command="Viewport: Focus" button="" /> */}

<br/>

<CommandReference
  command="Viewport: focus"
  shortcut="/"
  button="In Command Palette"
  buttonIcon="/img/button-viewport-focus.png"
  tier="both"
/>

##### Verwendung

1. Wählen Sie die Objekte aus, auf die Sie fokussieren möchten.
2. Drücken Sie <kbd>/</kbd>, um den Befehl auszuführen.
3. Die ausgewählten Objekte werden vergrößert und füllen den gesamten Viewport aus.

## Objekte isolieren

Dieser Befehl fokussiert auf das ausgewählte Objekt, indem es ausschließlich auf dem Bildschirm angezeigt wird und alle anderen Objekte vorübergehend ausgeblendet werden. Dieser Befehl unterstützt auch hierarchische Isolierung.

{/* <Reference shortcut=". (Period)" command="Isolate" button="" /> */}

<br/>

<CommandReference
  command="Isolate"
  shortcut="."
  button="In Command Palette"
  buttonIcon="/img/button-isolate.png"
  tier="both"
/>

##### Verwendung

1. Wählen Sie die Objekte aus, auf die Sie fokussieren möchten.
2. Öffnen Sie die Befehlspalette und geben Sie „Isolate" ein, oder drücken Sie <kbd>.</kbd>, um den Befehl auszuführen.
3. Nur die ausgewählten Objekte werden auf dem Bildschirm angezeigt.

##### Hierarchische Isolierung

1. Nach der Ausführung des Isolate-Befehls können Sie weitere Objekte auswählen, auf die Sie fokussieren möchten.
2. Führen Sie den Isolate-Befehl erneut aus, und die Anzeige wird auf die neu ausgewählten Objekte eingeschränkt.
3. Dieser Vorgang kann über mehrere Isolierungsebenen hinweg wiederholt werden.
4. Um die aktuelle Isolierungsebene zu verlassen, drücken Sie <kbd>.</kbd> oder klicken Sie auf die Schaltfläche **Isolated** oben in der Mitte des Bildschirms, wenn kein Objekt ausgewählt ist.

##### Indikator und Ebenenanzahl

Ein Indikator oben auf dem Bildschirm zeigt an, dass der Isolate-Modus aktiv ist. Eine Zahl neben dem Indikator gibt an, wie viele Isolierungsebenen angewendet wurden. Wird beispielsweise „2 Isolated" angezeigt, bedeutet dies, dass Sie sich in der zweiten Isolierungsebene befinden.

## Isolierung von Objekten aufheben

Dieser Befehl blendet alle zuvor isolierten Objekte im Viewport wieder ein. Er kehrt nicht zur vorherigen hierarchischen Isolierungsebene zurück, sodass alle Objekte sichtbar sind.

{/* <Reference shortcut="Alt + . (Period)" command="Unisolate" button="" /> */}

<br/>

<CommandReference
  command="Unisolate"
  shortcut="Alt + ."
  button="In Command Palette"
  buttonIcon="/img/button-unisolate.png"
  tier="both"
/>

##### Verwendung

1. Öffnen Sie die Befehlspalette und geben Sie „Unisolate" ein, oder drücken Sie <kbd>Alt + .</kbd>, um den Befehl auszuführen.
2. Nach der Ausführung des Unisolate-Befehls sind alle sichtbaren Objekte im Viewport wieder zu sehen.

## Objekte zentrieren

Dieser Befehl zentriert ein bestimmtes Objekt in der Mitte des Bildschirms.

{/* <Reference shortcut="Space" command="Viewport: navigate to selection" button="" /> */}

<br/>

<CommandReference
  command="Viewport: navigate to selection"
  shortcut="Space"
  button="In Command Palette"
  buttonIcon="/img/button-navigate-to-selection.png"
  tier="both"
/>

##### Verwendung

1. Wählen Sie das Objekt aus, das Sie zentrieren möchten.
2. Drücken Sie <kbd>Space</kbd>, um den Befehl auszuführen.
3. Das ausgewählte Objekt wird in der Mitte des Bildschirms angezeigt.
