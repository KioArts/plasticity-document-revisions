# OBJ のエクスポート

1. エクスポートしたいソリッドまたはシートを選択します。
   - 何も選択されていない場合、表示されているすべてのソリッドとシートがエクスポートされます。
2. **P-メニュー** > **エクスポート/名前を付けて保存...** をクリックします。
3. 名前を付けて保存ダイアログが表示されます。
   - ファイル名を入力します。
   - ファイル形式として **Wavefront OBJ (*.obj)** を選択します。
4. **OBJ エクスポート** ダイアログが表示されます。オプションを設定します。

   | オプション | 概要 | 詳細 | 推奨値 |
   |---|---|---|---|
   | **トポロジー** | エクスポートされるポリゴン形状 | **Tris** – 三角形のみ。<br /> **Quads** – 四角形と三角形。<br /> **Ngons** – 四角形や三角形を含む多角形。 | |
   | **ワイヤーフレーム表示** | エクスポートされるメッシュのワイヤーフレームを表示 | トポロジーや密度を確認し、エクスポート前に表面品質を把握するのに便利。 | |
   | **密度** | 全体のポリゴン密度 | 他の設定を自動調整する唯一の値。<br /> 低い値：ポリゴンが少なく、形状が粗い。<br /> 高い値：ポリゴンが多く、詳細が増える。 | |
   | **最小幅** | ポリゴンの最小サイズ | 小さなフィレットなどで細かいポリゴンが密集するのを防ぐために増やします。 | |
   | **最大幅** | ポリゴンの最大サイズ | 非常に大きなポリゴンを避けたり、ポリゴンサイズを均一にするために減らします。 | |
   | **エッジ平面許容差** | エッジの最大許容偏差（cm） | 面の平面許容差に似ていますが、面の中心よりもエッジや境界に強く影響します。 | |
   | **面平面許容差** | 最大許容偏差（cm） | 低い値：ポリゴンが増え、詳細が増える。<br /> 高い値：ポリゴンが減り、形状が粗くなる。<br /> ポリゴン数を制御する最も効果的な方法のひとつ。 | Low poly: `0.01` – High poly: `0.0001` <br /> `0.001` より小さい値が必要な場合は、最大幅の縮小を検討してください。 |
   | **エッジ角度許容差** | エッジの最大角度偏差（ラジアン） | 面角度許容差に似ていますが、エッジや境界に強く影響します。 | Low poly: `0.35` – High poly: `0.15` |
   | **面角度許容差** | 最大角度偏差（ラジアン） | 低い値：曲面が忠実に変換され、ポリゴンが増える。<br /> 高い値：曲面が粗く変換され、ポリゴンが減る。 | Low poly: `0.35` – High poly: `0.15` |

   **詳細設定**

   | オプション | 概要 | 詳細 | 推奨値 |
   |---|---|---|---|
   | **カーブ最大長** | カーブセグメントの最大許容長さ | | |
   | **平面角度** | 平面許容差と最大幅の比率の最大値 | 四角形やポリゴンを増やすのに有効。 | |
   | **小さなフィーチャーを削除** | エクスポート時に小さな表面ディテールを削除 | バウンディングボックスの閾値を定義し、それより小さいディテールを省略。High‑poly → Low‑poly のワークフローに最適。 | |
   | **Inflect** | 表面品質を維持 | 曲面が凸から凹へ変化する部分に詳細を追加。 | |
   | **放射状サーフェスの簡略化** | 半径形状を簡略化してエッジフローを改善 | 球、円柱、トーラス、円錐を共通軸方向に揃え、ファセット化時のエッジフローを予測しやすくする。 | |
   | **凸 Ngons のみ** | 凹 Ngons を防ぎ、クリーンなトポロジーを維持 | Ngons エクスポート時のみ使用可能。必要に応じて追加面を挿入し、すべての Ngons を凸に強制。信頼性の高いトポロジーと一貫した法線を実現。 | |
   | **スケール** | エクスポートスケール倍率 | すべての座標に倍率を適用し、ターゲットアプリの単位系に合わせる。 | |
   | **アップ軸** | エクスポートされるジオメトリの向き | ターゲットアプリの慣習に合わせてアップ軸を設定。 | |
5. OK をクリックします。
6. OBJ ファイルがエクスポートされます。
