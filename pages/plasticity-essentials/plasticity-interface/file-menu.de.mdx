# P-Menü

Das P-Menü kann durch Klicken auf das P oben links geöffnet werden.

![P-Menü](/img/file-menu.png)

| Befehlsname | Tastenkürzel | Funktion |
| --- | --- | --- |
| Zuletzt geöffnete Dateien | | Zeigt zuletzt gespeicherte Dateien an, an denen gearbeitet wurde. Beim Darüberfahren mit der Maus wird ein Vorschaufenster mit Miniaturansichten der Dateien und deren Inhalt angezeigt. Dateien können per Drag-and-Drop in die aktuelle Arbeitssitzung des Plasticity-Projekts gezogen werden. |
| Neue Datei | <kbd>Ctrl+N</kbd> | Leert die Szene und öffnet ein neues Dokument. |
| Neues Fenster | | Startet Plasticity und öffnet ein neues Fenster. Die Akzentfarbe des neuen Fensters wird zufällig geändert, um es vom primären Fenster zu unterscheiden. Kopieren und Einfügen zwischen Fenstern ist ebenfalls möglich. |
| Öffnen | <kbd>Ctrl+O</kbd> | Öffnet eine Plasticity-Datei. |
| Importieren/Anhängen | <kbd>Ctrl+Shift+O</kbd> | Importiert Dateien. Importierbare Formate sind [hier](/plasticity-essentials/import-export) beschrieben. |
| Speichern | <kbd>Ctrl+S</kbd> | Speichert das aktuelle Dokument. |
| Speichern unter | <kbd>Ctrl+Shift+S</kbd> | Speichert das Dokument unter einem neuen Namen. |
| Exportieren | <kbd>Ctrl+Shift+E</kbd> | Exportiert Dateien. Exportierbare Formate sind [hier](/plasticity-essentials/import-export) beschrieben. |
| Schnellspeicherversion | | Speichert eine Kopie der Datei mit einem an den Dateinamen angehängten Zeitstempel. |
| Als Startszene speichern | | Speichert das aktuelle Dokument als Startszene, die beim nächsten Start automatisch geöffnet wird. |
| Einstellungen | <kbd>Ctrl+,(Komma)</kbd> | Siehe [Einstellungen](/plasticity-essentials/preferences). |
| Lizenzen verwalten | | Verwaltet Lizenz-Token und Geräte. |
| Aktuelle Version | | Aktuelle Version von Plasticity. |
| Auf Plasticity Share veröffentlichen | | Im Web veröffentlichen. Siehe [hier](/common/publish-to-plasticity-share). |
| Meine geteilten Dokumente verwalten | | Hochgeladene (.plasticity) Dokumente auf Plasticity Share verwalten. Siehe [hier](/common/publish-to-plasticity-share). |
| Beenden | <kbd>Ctrl+Q</kbd> | Beendet Plasticity. |
