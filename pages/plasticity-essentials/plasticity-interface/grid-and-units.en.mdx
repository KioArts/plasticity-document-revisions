# Grid and Units

![Grid and Units](/img/grid-and-units.png)

## Grid[#grid]

**Snap to grid:** You can use grid snapping to snap your cursor to the grid while drawing or moving objects. To specify the grid size, change the value under **Snap to grid**.
    - 1. The grid appears on the active construction plane. Set the grid unit in the **Units** section of the **Length** window, and the grid size in the **Grid size** field of the **Grid** section.
    - 2. Grid snapping can also be toggled via the Command Palette: **Grid: toggle grid snapping**, **Grid: decrease grid size**, **Grid: increase grid size**.
    - 3. To temporarily disable grid snapping, hold <kbd>Ctrl</kbd> while dragging.
    - 4. To customize the grid color, go to [Preferences](/plasticity-essentials/preferences). 
       
**Grid size:** Specifies the overall size of the grid.  
**Line every:** Specifies the spacing interval for the thick lines in the grid.
    - 1. Units are displayed while drawing curves, and you can press <kbd>Tab</kbd> to enter precise measurements.

## Units[#units]

**Length:** Specifies the units (Millimeter, Centimeter, Meter, Inch, Foot) for the viewport.   
**Mass:** Specifies a unit (Gram, Kilogram, Ounce, or Pound) to calculate the object's mass. The result appears in the **Selection Menu** in bottom right.
