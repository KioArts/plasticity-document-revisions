# Pメニュー

Pメニューは左上のPをクリックすることで開くことができます。

![Pメニュー](/img/file-menu.png)

| コマンド名 | ショートカット | 機能 |
| --- | --- | --- |
| 最近使ったファイル | | 最近保存して作業したファイルを表示する。マウスを重ねるとファイルの内容のサムネイル付きプレビューウィンドウが表示される。ファイルはPlasticityプロジェクトの現在の作業セッションにドラッグ＆ドロップすることができる。 |
| 新規ファイル | <kbd>Ctrl+N</kbd> | シーンをクリアして新しいドキュメントを開く。 |
| 新しいウィンドウ | | Plasticityを起動して新しいウィンドウを開く。新しいウィンドウのアクセントカラーはプライマリウィンドウと区別するためにランダムに変更される。ウィンドウ間でのコピー＆ペーストも可能。 |
| 開く | <kbd>Ctrl+O</kbd> | Plasticityファイルを開く。 |
| インポート/追加 | <kbd>Ctrl+Shift+O</kbd> | ファイルをインポートする。インポート可能な形式は[こちら](/plasticity-essentials/import-export)を参照。 |
| 保存 | <kbd>Ctrl+S</kbd> | 現在のドキュメントを保存する。 |
| 名前を付けて保存 | <kbd>Ctrl+Shift+S</kbd> | ドキュメントを新しい名前で保存する。 |
| エクスポート | <kbd>Ctrl+Shift+E</kbd> | ファイルをエクスポートする。エクスポート可能な形式は[こちら](/plasticity-essentials/import-export)を参照。 |
| クイック保存バージョン | | タイムスタンプをファイル名に付加してファイルのコピーを保存する。 |
| スタートアップシーンとして保存 | | 現在のドキュメントをスタートアップシーンとして保存し、次回起動時に自動的に開かれる。 |
| 環境設定 | <kbd>Ctrl+,(カンマ)</kbd> | [設定](/plasticity-essentials/preferences)を参照。 |
| ライセンスの管理 | | ライセンストークンとデバイスを管理する。 |
| 現在のバージョン | | Plasticityの現在のバージョン。 |
| Plasticity Shareに公開 | | ウェブに公開する。[こちら](/common/publish-to-plasticity-share)を参照。 |
| 共有ドキュメントの管理 | | Plasticity Shareにアップロードされた(.plasticity)ドキュメントを管理する。[こちら](/common/publish-to-plasticity-share)を参照。 |
| 終了 | <kbd>Ctrl+Q</kbd> | Plasticityを終了する。 |
