## Befehlsleiste

![Befehlsleiste](/img/command-bar.png)
- Die Befehlsleiste befindet sich am unteren Bildschirmrand und enthält häufig verwendete Befehle:

| Befehl                        | Tastenkürzel   |
| ----------------------------- | -------------- |
| [Search Commands](/plasticity-essentials/plasticity-interface/command-palette) | <kbd>F</kbd> |
| [Move](/common/move) | <kbd>G</kbd> |
| [Rotate](/common/rotate) | <kbd>R</kbd> |
| [Scale](/common/scale) | <kbd>S</kbd> |
| [Duplicate Object](/common/duplicate) | <kbd>Shift + D</kbd> |
| [Boolean](/solid/boolean) | <kbd>Q</kbd> |
| [Rectangular Array](/common/rectangular-array) | - |
| [Radial Array](/common/radial-array) | - |
| [Suggested Commands](/plasticity-essentials/plasticity-interface/command-bar#suggested-commands) | <kbd>Shift + F</kbd> |

## Vorgeschlagene Befehle
- Die Befehlsleiste enthält das Fenster **Vorgeschlagene Befehle**, das Befehle und Werkzeuge basierend auf Ihrer Auswahl hervorhebt - ob es sich um ein Solid, Sheet, Curve, Region oder Point handelt. Um das Fenster zu erweitern, klicken Sie auf die <ButtonIcon type="button-suggested-commands-dots"/> Schaltfläche oder drücken Sie <kbd>Shift + F</kbd>.
