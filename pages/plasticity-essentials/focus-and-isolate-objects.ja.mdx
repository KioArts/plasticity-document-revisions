## フォーカスとアイソレート

## オブジェクトをフォーカス

このコマンドは、選択したオブジェクトをビューポート全体に拡大表示します。

{/* <Reference shortcut="/ (Slash)" command="Viewport: Focus" button="" /> */}

<br/>

<CommandReference
  command="Viewport: focus"
  shortcut="/"
  button="In Command Palette"
  buttonIcon="/img/button-viewport-focus.png"
  tier="both"
/>

##### 使い方

1. フォーカスしたいオブジェクトを選択します。
2. <kbd>/</kbd> を押してコマンドを実行します。
3. 選択したオブジェクトがビューポート全体に拡大表示されます。

## オブジェクトをアイソレート

このコマンドは、選択したオブジェクトのみを画面に表示し、他のすべてのオブジェクトを一時的に非表示にすることで、選択オブジェクトにフォーカスします。このコマンドは階層的なアイソレートにも対応しています。

{/* <Reference shortcut=". (Period)" command="Isolate" button="" /> */}

<br/>

<CommandReference
  command="Isolate"
  shortcut="."
  button="In Command Palette"
  buttonIcon="/img/button-isolate.png"
  tier="both"
/>

##### 使い方

1. フォーカスしたいオブジェクトを選択します。
2. コマンドパレットを開いて「Isolate」と入力するか、<kbd>.</kbd> を押してコマンドを実行します。
3. 選択したオブジェクトのみが画面に表示されます。

##### 階層的なアイソレート

1. Isolateコマンドを実行した後、さらにフォーカスしたいオブジェクトを選択できます。
2. 再度Isolateコマンドを実行すると、新たに選択したオブジェクトのみに表示が絞り込まれます。
3. このプロセスを繰り返すことで、複数の階層でアイソレートを適用できます。
4. 現在のアイソレート階層を終了するには、オブジェクトが選択されていない状態で <kbd>.</kbd> を押すか、画面上部中央の **Isolated** ボタンをクリックします。

##### インジケーターと階層数

画面上部のインジケーターはアイソレートモードが有効であることを示します。インジケーターの隣に表示される数字は、適用されているアイソレートの階層数を示します。たとえば「2 Isolated」と表示されている場合、2番目のアイソレート階層にいることを意味します。

## オブジェクトのアイソレートを解除

このコマンドは、ビューポートでアイソレートされていたすべてのオブジェクトを再表示します。前の階層的アイソレートには戻らず、すべてのオブジェクトが表示された状態になります。

{/* <Reference shortcut="Alt + . (Period)" command="Unisolate" button="" /> */}

<br/>

<CommandReference
  command="Unisolate"
  shortcut="Alt + ."
  button="In Command Palette"
  buttonIcon="/img/button-unisolate.png"
  tier="both"
/>

##### 使い方

1. コマンドパレットを開いて「Unisolate」と入力するか、<kbd>Alt + .</kbd> を押してコマンドを実行します。
2. Unisolateコマンドを実行すると、ビューポートに表示されているすべてのオブジェクトを確認できます。

<VideoPlayer fileName="isolate-unisolate"/>

## オブジェクトをセンタリング

このコマンドは、特定のオブジェクトを画面の中央に表示します。

{/* <Reference shortcut="Space" command="Viewport: navigate to selection" button="" /> */}

<br/>

<CommandReference
  command="Viewport: navigate to selection"
  shortcut="Space"
  button="In Command Palette"
  buttonIcon="/img/button-navigate-to-selection.png"
  tier="both"
/>

##### 使い方

1. センタリングしたいオブジェクトを選択します。
2. <kbd>Space</kbd> を押してコマンドを実行します。
3. 選択したオブジェクトが画面の中央に表示されます。
