# スナップ

## メッシュおよび画像スナップ[#mesh-and-image-snap]

- メッシュおよび画像に対して、限定的なスナップ機能を使用できます。

   | タイプ                     | スナップ対象  | ヒント |
   | ------------------- | ----------- | -------- |
   | メッシュ (OBJ, STL, FBX, 3MF) | 頂点 / 面 | 頂点 / 面 |
   | 画像 (.PNG, .JPG, .JPEG) | 角 | 角 |

## オブジェクトスナップ[#object-snap]

![オブジェクトスナップ](/img/object-snap.png)

- 1. スナップ機能を使用すると、オブジェクトの描画や移動時にカーソルを特定の要素にスナップできます。
- 2. オブジェクトスナップを切り替えるには、右サイドバーの「スナップ」ウィンドウ内のトグルボタンをクリックしてください。異なる種類のスナップを切り替えるには、「オブジェクトスナップ」セクションを開き、オプションを調整します。
- 3. 一時的にスナップを有効にするには、<kbd>Ctrl</kbd>キーを押しながら操作します。曲線を描画するときは、自動的にスナップが有効になります。
- 4. 一時的にスナップを無効にするには、<kbd>Shift + X</kbd>キーを押してください。現在アクティブなスナップの種類を一時的に無効化できます。希望のスナップタイプに切り替わるまで、ボタンを繰り返し押してください。  
- 5. スナップは、さまざまな種類のオブジェクトに適用できます。対象は以下のとおりです：コントロール頂点、曲線、領域、エッジ、面、測定点。スナップターゲットは小さな黒い点として表示されます。
- 6. スナップターゲットにカーソルを合わせると、ヒントが表示されます。ヒントが表示された状態でクリックすると、正確にスナップできます。

    | スナップ対象       | ヒント                                                                                                                       |
    | -------------- | ---------------------------------------------------------------------------------------------------------------------------- |
    | コントロール頂点 | CV                                                                                                                           |
    | 曲線           | 曲線、始点、終点、中点、最寄り、交点、X, Y, Z, XY, YZ, ZX, 中心、1/4, 3/4, 垂直、接線、接線/接線 |
    | 領域           | 領域、中心                                                                                                                     |
    | エッジ         | エッジ、終点、中点                                                                                                             |
    | 面            | 面、中点                                                                                                                       |
    | 測定点        | 測定                                                                                                                           |

- コマンドパレットで **Snaps: スナップの切り替え** を実行できます。
