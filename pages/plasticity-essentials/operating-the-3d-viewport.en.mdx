# Operating the 3D Viewport

## Operating with the Mouse

| Operation                                          | Mouse                                                    |
| -------------------------------------------------- | -------------------------------------------------------- |
| To rotate the view                                 | Drag with the Middle Mouse Button (scroll wheel button). |
| To pan the view                                    | Drag with the Right Mouse Button.                        |
| To zoom in or out in fixed increments              | Scroll the mouse wheel.                                  |
| To continuously zoom in or out                     | <kbd>Ctrl</kbd> + Middle Mouse Button drag.              |
| To orbit while snapping to the world axes and diagonals between them | Middle Mouse Button drag + <kbd>Alt</kbd> |
| To snap to orthographic projection during rotation | <kbd>Alt</kbd> + Middle Mouse Button drag.               |
| To center the screen around the cursor position    | <kbd>Alt</kbd> + Middle Mouse Button click.              |

You can also make the mouse similar to other applications. See [Preferences](preferences) for more information.

## Operating with Keyboard Shortcuts

### Changing the Viewing Angle

You can adjust the viewing direction using the number pad.

| Viewing direction                                      | Shortcut                |
| ------------------------------------------------------ | ----------------------- |
| Front view                                             | <kbd>Num.1</kbd>        |
| Right view                                             | <kbd>Num.3</kbd>        |
| Top view                                               | <kbd>Num.7</kbd>        |
| Back view                                              | <kbd>Ctrl + Num.1</kbd> |
| Left view                                              | <kbd>Ctrl + Num.3</kbd> |
| Bottom view                                            | <kbd>Ctrl + Num.7</kbd> |
|                                                        |                         |
| Switch between perspective and <br />orthographic projection | <kbd>Num.5</kbd>  |

# Operating with Commands

### Changing the Camera Position

Use orbit commands to reposition the camera around a fixed target point.

| Command | Operation |
| ------- | --------- |
| Orbit: dolly in | Moves the camera closer to the target. |
| Orbit: dolly out | Moves the camera further from the target. |
| Orbit: pan down | Shifts the camera view downward without changing its angle. |
| Orbit: pan left | Shifts the camera view left without changing its angle. |
| Orbit: pan right | Shifts the camera view right without changing its angle. |
| Orbit: pan up | Shifts the camera view upward without changing its angle. |
| Orbit: rotate down | Tilts the camera downward around the orbit target. |
| Orbit: rotate left | Rotates the camera left around the orbit target. |
| Orbit: rotate right | Rotates the camera right around the orbit target. |
| Orbit: rotate up | Tilts the camera upward around the orbit target. |
