# インポート/エクスポート

**「インポート/追加」** と **「エクスポート」** オプションを使用して、Pメニューからインポートおよびエクスポート操作を行うことができます。3Dビューポートにファイルをドラッグ＆ドロップすることも可能です。<br/>

次のファイル形式をサポートしています。

## インポート [#file-import]

| ファイル形式                 | Indieライセンス | Studioライセンス |                                                                  |
| ---------------------------- | --------------- | ---------------- | ---------------------------------------------------------------- |
| plasticity (*.plasticity)    | ✓              | ✓               |                                                                  |
| STEP (*.stp; *.step)         | ✓              | ✓               |                                                                  |
| ParaSolid (*.x_t; *.x_b)     | ✓              | ✓               |                                                                  |
| Wavefront OBJ (*.obj)        | ✓              | ✓               |                                                                  |
| FBX (*.FBX)                  | ✓              | ✓               |                                                                  |
| STL (*.stl)                  | ✓              | ✓               |                                                                  |
| Vector Image (*.SVG)         | ✓            | ✓             |                                                      |
| Image (*.png; *.jpg; *.jpeg) | ✓              | ✓               | 詳細は[こちら](importing-reference-images)を参照してください。 |
| 3MF (*.3mf)                  |               | ✓             |                                                      |
| IGES (*.igs; *.iges)         |                 | ✓               |                                                                  |
| SAT (*sat)                   |                 | ✓               |                                                                  |
| Rhino (*.3dm)                |                 | ✓               |                                                                  |
| DXF/DWG (*.dxf; *.dwg)        |                 | ✓               | Windowsのみサポート                                       |

## エクスポート [#file-export]

| ファイル形式               | Indieライセンス | Studioライセンス |                                                  |
| -------------------------- | --------------- | ---------------- | ------------------------------------------------ |
| Plasticity  (*.plasticity) | ✓            | ✓             |                                        |
| STEP  (*.stp; *.step)      | ✓            | ✓             | コマンドパレットで単独コマンドとして利用可能 – 「Export STEP」|
| PNG  | ✓            | ✓             |  現在のビューポートのスクリーンショットを撮影し、UI要素を非表示にします。                                      |
| ParaSolid  (*.x_t; *.x_b)  | ✓            | ✓             |                                        |
| Wavefront OBJ (*.obj)      | ✓            | ✓             | For details, see [here](export-obj). |
| STL  (*.stl)               | ✓            | ✓             |                                        |
| 3MF (*.3mf)                | ✓            | ✓             |                                        |
| IGES  (*.igs; *.iges)      |               | ✓             |                                        |
| SAT  (*sat)                |               | ✓             | コマンドパレットで単独コマンドとして利用可能 – 「Export SAT」|
