# インポート/エクスポート

**「インポート/追加」** と **「エクスポート」** オプションを使用して、Pメニューからインポートおよびエクスポート操作を行うことができます。3Dビューポートにファイルをドラッグ＆ドロップすることも可能です。<br/>

次のファイル形式をサポートしています。

## インポート [#file-import]

| ファイル形式                 | Indieライセンス | Studioライセンス |                                                                  |
| ---------------------------- | :---------------: | :----------------: | ---------------------------------------------------------------- |
| plasticity (*.plasticity)    | ✓              | ✓               |                                                                  |
| STEP (*.stp; *.step)         | ✓              | ✓               |                                                                  |
| ParaSolid (*.x_t; *.x_b)     | ✓              | ✓               |                                                                  |
| Wavefront OBJ (*.obj)        | ✓              | ✓               |                                                                  |
| FBX (*.FBX)                  | ✓              | ✓               |                                                                  |
| STL (*.stl)                  | ✓              | ✓               |                                                                  |
| Vector Image (*.SVG)         | ✓            | ✓             |                                                      |
| Image (*.png; *.jpg; *.jpeg) | ✓              | ✓               | 詳細は[こちら](importing-reference-images)を参照してください。 |
| 3MF (*.3mf)                  |               | ✓             |                                                      |
| IGES (*.igs; *.iges)         |                 | ✓               |                                                                  |
| SAT (*sat)                   |                 | ✓               |                                                                  |
| Rhino (*.3dm)                |                 | ✓               |                                                                  |
| DXF/DWG (*.dxf; *.dwg)        |                 | ✓               | Windowsのみサポート                                       |

## エクスポート [#file-export]

| ファイル形式                         | Indie ライセンス | Studio ライセンス | コマンドパレットのスタンドアロンコマンド | 備考 |
|------------------------------------|:----------------:|:-----------------:|-------------------------------------------|------|
| Plasticity (*.plasticity)          | ✓                | ✓                 | –                                         | –    |
| STEP (*.stp; *.step)               | ✓                | ✓                 | Export STEP                               | –    |
| PNG (*.png)                        | ✓                | ✓                 | –                                         | UI 要素を非表示にした状態で現在のビューポートのスクリーンショットを取得します。**P‑Menu → Export** または <kbd>Ctrl + Shift + E</kbd> でアクセスできます。 |
| Parasolid (*.x_t; *.x_b)           | ✓                | ✓                 | Export Parasolid                          | –    |
| Wavefront OBJ (*.obj)              | ✓                | ✓                 | Export OBJ                                | 詳細は [こちら](export-obj) をご覧ください。 |
| STL (*.stl)                        | ✓                | ✓                 | Export STL                                | –    |
| ベクター画像 (*.svg)                | ✓                | ✓                 | Export SVG                                | バージョン 25.3.0 ベータ以降で利用可能です。 |
| 3MF (*.3mf)                        | ✓                | ✓                 | Export 3MF                                | –    |
| IGES (*.igs; *.iges)               | –                | ✓                 | Export IGES                               | –    |
| SAT (*.sat)                        | –                | ✓                 | Export SAT                                | –    |
