## OBJ exportieren

1. Wählen Sie den Solid- oder Sheet-Körper aus, den Sie exportieren möchten.
   - Wenn nichts ausgewählt ist, werden alle angezeigten Solids und Sheets exportiert.
2. Klicken Sie auf **P-Menü** > **Exportieren/Speichern unter...**.
3. Ein Speichern-unter-Dialog erscheint.
   - Geben Sie den Dateinamen ein.
   - Wählen Sie **Wavefront OBJ (*.obj)** als Dateityp.
4. Der Dialog **OBJ exportieren** erscheint. Konfigurieren Sie die Optionen.

   | Option | Übersicht | Details | Empfohlener Wert |
   |---|---|---|---|
   | **Topologie** | Form der exportierten Polygone | **Tris** – nur Dreiecke. <br /> **Quads** – Vierecke und Dreiecke. <br /> **Ngons** – Polygone mit mehreren Seiten, einschließlich Quads und Tris. | |
   | **Drahtgitter anzeigen** | Zeigt das Drahtgitter des exportierten Meshes an | Nützlich zur Vorschau von Topologie und Dichte, um die Oberflächenqualität vor dem Export einzuschätzen. | |
   | **Dichte** | Gesamte Polygonanzahl | Der einzige Wert, der andere Einstellungen automatisch anpasst. <br /> Niedriger Wert: weniger Polygone, gröbere Form. <br /> Hoher Wert: mehr Polygone, mehr Details. | |
   | **Minimale Breite** | Minimale Polygongröße | Erhöhen, um eine Überfüllung kleiner Polygone in Bereichen wie kleinen Fasen zu vermeiden. | |
   | **Maximale Breite** | Maximale Polygongröße | Verringern, um extrem große Polygone zu vermeiden oder die Polygongröße gleichmäßiger zu machen. | |
   | **Kanten-Ebenentoleranz** | Maximal zulässige Abweichung von Kanten (in cm) | Ähnlich wie Flächen-Ebenentoleranz, betrifft jedoch Kanten und Flächenränder stärker als die Flächenmitte. | |
   | **Flächen-Ebenentoleranz** | Maximal zulässige Abweichung (in cm) | Niedriger Wert: mehr Polygone, mehr Details. <br /> Höherer Wert: weniger Polygone, gröbere Form. <br /> Eine der effektivsten Methoden zur Kontrolle der Polygonanzahl. | Low poly: `0.01` – High poly: `0.0001` <br /> Wenn ein Wert kleiner als `0.001` benötigt wird, erwägen Sie stattdessen die Reduzierung der maximalen Breite. |
   | **Kanten-Winkeltoleranz** | Maximal zulässige Winkelabweichung von Kanten (in Radiant) | Ähnlich wie Flächen-Winkeltoleranz, betrifft jedoch Kanten und Flächenränder stärker. | Low poly: `0.35` – High poly: `0.15` |
   | **Flächen-Winkeltoleranz** | Maximal zulässige Winkelabweichung (in Radiant) | Niedriger Wert: gekrümmte Flächen werden detailgetreu konvertiert, mehr Polygone. <br /> Höherer Wert: grobere Konvertierung, weniger Polygone. | Low poly: `0.35` – High poly: `0.15` |

   **Erweitert**

   | Option | Übersicht | Details | Empfohlener Wert |
   |---|---|---|---|
   | **Maximale Kurvenlänge** | Maximal zulässige Länge eines Kurvensegments | | |
   | **Ebenenwinkel** | Maximal zulässiges Verhältnis zwischen Ebenentoleranz und maximaler Breite | Nützlich, um mehr Vierecke oder Polygone zu erhalten. | |
   | **Kleine Features entfernen** | Entfernt winzige Oberflächendetails beim Export | Definiert eine Bounding-Box-Schwelle — Details kleiner als dieser Wert werden ausgelassen. Ideal für High‑Poly‑zu‑Low‑Poly‑Workflows. | |
   | **Inflect** | Erhält die Oberflächenqualität | Fügt Details in Bereichen hinzu, in denen gekrümmte Flächen von konvex zu konkav übergehen. | |
   | **Radiale Flächen vereinfachen** | Vereinfacht Radiengeometrie für besseren Edge‑Flow | Richtet Kugeln, Zylinder, Tori und Kegel entlang einer gemeinsamen Achse aus, sodass Edge‑Flow‑Linien vorhersehbarer verlaufen. | |
   | **Nur konvexe Ngons** | Verhindert konkave Ngons für sauberere Topologie | Nur verfügbar beim Export von Ngons. Erzwingt konvexe Ngons und fügt bei Bedarf zusätzliche Flächen ein. Liefert zuverlässige Topologie und konsistente Normalen bei leicht erhöhtem Ngon‑Count. | |
   | **Skalierung** | Export-Skalierungsfaktor | Wendet einen Multiplikator auf alle exportierten Koordinaten an, um die Einheit des Zielprogramms zu berücksichtigen. | |
   | **Up-Achse** | Orientierung der exportierten Geometrie | Legt die Up-Achse fest, um die Konventionen der Zielanwendung zu erfüllen. | |
5. Klicken Sie auf OK.
6. Die OBJ-Datei wird exportiert.
