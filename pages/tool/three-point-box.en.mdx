# Three-Point Box

<Reference shortcut='' command='Three-Point Box' button='Toolbar' />

You can  create a rectangular box defined by three points.

1. Click the button <ButtonIcon type="button-three-point-box" /> on the toolbar.
2. Click at the start point of one side of the rectangle.
   - <kbd>X</kbd>, <kbd>Y</kbd>, <kbd>Z</kbd> **X, Y, Z-axis:** [Limit to the coordinate axis direction](./sketching-tips#limit-to-the-coordinate-axis-direction).
3. Click at the endpoint of that side of the rectangle.
4. Click at the third point of the rectangle.
5. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Select target bodies**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - These options have the same function as the Boolean command. See [**Boolean**](../solid/boolean) command for details.
   - **Width:** Specify the width.
   - **Length:** Specify the Length.
   - **Height:** Specify the height.
6. Move the cursor and click at the desired height.
7. A rectangular box is created.
