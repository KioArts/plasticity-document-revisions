# Center Box

<Reference shortcut='' command='Center Box' button='Toolbar' />

Allows you to create a rectangular box from a specified center point.

1. Click the button <ButtonIcon type="button-center-box" /> on the toolbar.
2. Click at the center point of the base surface.
   - <kbd>C</kbd> **Center/Corner:** Switches to Corner Box.
   - <kbd>S</kbd> **Square:** Limits to a square.
3. Click at the end point of the perimeter.
4. Move the cursor and click at the desired height.
5. Set the options in the command dialog and Shortcut Key.
   <Boolean_Options />
   - **Width:** Specifies the width.
   - **Length:** Specifies the Length.
   - **Height:** Specifies the height.
   - <kbd>H</kbd> **Height:** Moves the cursor to specify the height.
6. Confirm with OK in the command dialog or right click.
7. A rectangular box is created.
