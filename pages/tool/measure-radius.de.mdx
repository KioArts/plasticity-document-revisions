## Beschreibung

Misst den Radius einer Kante einer Volumenfläche oder einer Kurve (Skizze) und erstellt eine Radius‑Annotation.

## Grundlegende Verwendung

1. Öffnen Sie die Befehlspalette, geben Sie **Measure Radius** ein und führen Sie den Befehl aus.  
2. Wählen Sie den Objekttyp, den Sie messen möchten:
    | Typ | Hinweis |
    | ---- | ---- |
    | Solid | Klicken Sie auf die Flächenkante, deren Radius Sie messen möchten. |
    | Curve | Klicken Sie auf die Skizze, deren Radius Sie messen möchten. |
3. Bewegen Sie die Maus im Viewport, um die Länge der Beschriftungslinie festzulegen, und klicken Sie mit der linken oder rechten Maustaste zur Bestätigung.  
4. Der gemessene Wert erscheint neben dem **R**‑Symbol und kennzeichnet den Radius. Die Messung wird außerdem im **Outliner** unter **Measurements** hinzugefügt.
