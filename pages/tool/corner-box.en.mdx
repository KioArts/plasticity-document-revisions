# Corner Box

<Reference shortcut='Shift-B' command='Corner Box' button='Toolbar' />

Allows you to create a rectangular box from a specified two points on a diagonal line.

1. Click the button <ButtonIcon type="button-corner-box" />ã€€on the toolbar.
2. Click at the start point of the diagonal line on the base surface.
   - <kbd>C</kbd> **Center/Corner**
      - Switches to Center Box.
   - <kbd>S</kbd> **Square**
      - Limits to a square.
3. Click at the end point of the diagonal line.
4. Move the cursor and click at the desired height.
5. Set the options in the command dialog and Shortcut Key.
   - **Select target bodies**
      - Specifies the boolean targets.
   - **Width**
      - Specifies the width.
   - **Length**
      - Specifies the Length.
   - **Height**
      - Specifies the height.
   <Option_Boolean />
   - <kbd>H</kbd> **Height**
      - Moves the cursor to specify the height.
6. Click OK in the command dialog or right-click to confirm.
7. A rectangular box is created.
