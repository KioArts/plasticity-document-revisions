# Center Point Arc（中心からの円弧）

<Reference shortcut='' command='Center Point Arc' button='ツールバー' />

中心と円周上の2点で指定した円弧を作成できます。

1. ツールバーのボタン <ButtonIcon type="button-center-point-arc" /> をクリックします。
2. 円の中心位置でクリックします。
3. カーソルを移動して直径の最初の位置でクリックします。
   - <kbd>X</kbd>, <kbd>Y</kbd>, <kbd>Z</kbd> **X, Y, Z-axis:** [座標軸方向に制限します](./sketching-tips#limit-to-the-coordinate-axis-direction)。
4. カーソルを移動して円周上の2点目でクリックします。
5. 円弧が作成されます。
