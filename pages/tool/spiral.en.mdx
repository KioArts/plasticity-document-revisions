# Spiral

<Reference shortcut='' command='Spiral' button='Toolbar' />

You can create a spiral.

1. Click the button <ButtonIcon type="button-spiral" /> on the toolbar.
2. Click at the start point of the spiral axis.
3. Move the cursor and click at the end point of the spiral axis.
   <Option_X />
   <Option_Y />
   <Option_Z />
4. Move the cursor and click at the point for the spiral diameter.
5. Set the options in the command dialog and Shortcut Key.
   - **Turns**
      - Specifies the number of rotations.
   - **Radius**
      - Specifies the diameter.
   - **Angle OR Pitch**
      - Specifies the angle or pitch. Each value is linked.
   - **Handedness**
      - Reverses the direction of rotation.
   - <kbd>A</kbd> **Angle**
      - Specifies the angle with mouse movement.
   - <kbd>D</kbd> **Length**
      - Specifies the length of the axis with mouse movement.
   - <kbd>R</kbd> **Radius**
      - Specifies the diameter with mouse movement.
6. Confirm with OK in the command dialog or right-click.
7. A spiral is created.
