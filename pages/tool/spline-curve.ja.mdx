# Spline Curve（スプラインカーブ）

<Reference shortcut="" command="Spline Curve" button="ツールバー" />

スプラインカーブを作成できます。

1. ツールバーのボタン <ButtonIcon type="button-spline-curve" /> をクリックします。
2. スプラインカーブの最初の位置でクリックします。
3. 次の位置でクリックを繰り返して、スプラインカーブを作成します。
   <Option_K />
   - <kbd>Tab</kbd> **Control /  Through-point**
      - Control Point Curveに切り替えます。
   - <kbd>Ctrl-Z</kbd> **Undo**
      - 直前のクリックを取り消します。
   <Option_X />
   <Option_Y />
   <Option_Z />
4. コマンドダイアログのOKまたは右クリックで確定します。
5. スプラインカーブが作成されます。
