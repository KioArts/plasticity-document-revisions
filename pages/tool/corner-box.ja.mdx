## 説明

底面の対角線と高さを指定して、長方形のボックスを作成できます。

## 基本的な使い方

1. ツールバーの <ButtonIcon type="button-corner-box" /> ボタンをクリックします。
2. 底面の一つの角をクリックして指定します。
3. カーソルを動かして底面のサイズを調整し、クリックして確定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Tab</kbd>: 寸法を直接入力できます。詳細は[こちら](/tool/sketching-essentials#specify-dimensions-numerically)。
      - <kbd>C</kbd> **Center/Corner:** コーナーボックスに切り替えます。
      - <kbd>S</kbd> **Square:** 底面を正方形に制限します。
4. カーソルを動かして高さを調整し、クリックして確定します。
   - <Kst>コマンドダイアログ</Kst>
      - **対象ボディを選択**:
         - Boolean コマンドと同じ動作です。詳細は [**Boolean**](/solid/boolean) を参照してください。
      - **幅:** 幅を指定します。
      - **長さ:** 長さを指定します。
      - **高さ:** 高さを指定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
         - Boolean コマンドと同じ動作です。詳細は [**Boolean**](/solid/boolean) を参照してください。
      - <kbd>H</kbd> **Height:** カーソル移動で高さを調整します。
5. 黄色いドットを動かして形状を調整することもできます。
6. コマンドダイアログの OK をクリックするか、右クリックして確定します。
7. 長方形のボックスが作成されます。
