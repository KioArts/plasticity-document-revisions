## 説明

ソリッドのエッジまたはカーブ（スケッチ）の半径を測定し、半径注記を作成します。

## 基本的な使い方

1. コマンドパレットを開き、**Measure Radius** と入力して実行します。  
2. 測定するオブジェクトの種類を選択します:
    | 種類 | 説明 |
    | ---- | ---- |
    | Solid | 測定したいサーフェスエッジをクリックします。 |
    | Curve | 測定したいスケッチをクリックします。 |
3. ビューポート内でマウスを動かして注記ラインの長さを調整し、左クリックまたは右クリックで確定します。  
4. 測定値は **R** 記号の横に表示され、半径であることを示します。また、測定結果は **Outliner** の **Measurements** に追加されます。
