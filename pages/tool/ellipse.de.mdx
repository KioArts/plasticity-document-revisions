# Ellipse

<Reference shortcut="" command="Ellipse" button="Werkzeugleiste" buttonIcon='button-ellipse' />

Sie können eine Ellipse erstellen, indem Sie das Zentrum sowie den Radius der großen oder kleinen Achse angeben.

## Grundlegende Verwendung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-ellipse" /> in der Werkzeugleiste.  
2. Bestimmen Sie das Zentrum, um den Radius der großen oder kleinen Achse zu beginnen.  
3. Bewegen Sie den Cursor, um die Größe der Ellipse anzupassen, und klicken Sie zur Bestätigung.  
   - <Kst>Tastaturkürzel</Kst>  
      - <kbd>Tab</kbd>: Ermöglicht die direkte Eingabe von Maßen. Einzelheiten finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).  
      - <kbd>K</kbd> **Knife**: Bitte lesen Sie [Verwendung des Messer‑Modus](/tool/sketching-essentials#use-knife-mode).  
      - <kbd>X</kbd>/<kbd>Y</kbd> **X/Y‑Achse**: [Auf X‑ bzw. Y‑Achse einschränken](/tool/sketching-essentials#constrain-to-an-axis).  
4. Klicken Sie im Befehlsdialog auf **OK** oder führen Sie einen Rechtsklick aus, um zu bestätigen.  
5. Die Ellipse wird erstellt.
