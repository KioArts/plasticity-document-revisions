# Spline Curve

<Reference shortcut="" command="Spline Curve" button="Toolbar" />

You can create a spline curve.

1. Click the button <ButtonIcon type="button-spline-curve" /> on the toolbar.
2. Click at the start point of the spline curve.
3. Repeat clicking at the next point to create the spline curve.
   <Option_K />
   - <kbd>Tab</kbd> **Control / Through-point**
      - Switches to Control Point Curve.
   - <kbd>Ctrl-Z</kbd> **Undo**
      - Cancel the last click.
   <Option_X />
   <Option_Y />
   <Option_Z />
4. Confirm with OK in the command dialog or right click.
5. A spline curve is created.
