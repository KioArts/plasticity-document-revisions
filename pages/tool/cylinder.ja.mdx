## 説明

底面の円と高さを指定して、シリンダー（円柱）を作成できます。

## 基本的な使い方

1. ツールバーの <ButtonIcon type="button-cylinder" /> ボタンをクリックします。
2. 円の中心位置をクリックして指定します。
3. カーソルを動かして底面円の大きさを調整し、クリックして確定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Tab</kbd>: 寸法を直接入力できます。詳細は[こちら](/tool/sketching-essentials#specify-dimensions-numerically)。
      - <kbd>V</kbd> **Vertical/Horizontal:** 作業平面に対する傾きを変更します。
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z 軸:** [X/Y/Z 軸に拘束](/tool/sketching-essentials#constrain-to-an-axis)。
4. カーソルを動かして高さを調整し、クリックして確定します。
   - <Kst>コマンドダイアログ</Kst>
      - **対象ボディを選択**
         - Boolean コマンドと同じ動作です。詳細は [**Boolean**](/solid/boolean) を参照してください。
      - **半径:** 半径を指定します。
      - **高さ:** 高さを指定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
         - Boolean コマンドと同じ動作です。詳細は [**Boolean**](/solid/boolean) を参照してください。
      - <kbd>F</kbd> **Radius:** カーソル移動で半径を調整します。
      - <kbd>D</kbd> **Height:** カーソル移動で高さを調整します。
5. 黄色いドットを動かして形状を調整することもできます。
6. コマンドダイアログの OK をクリックするか、右クリックして確定します。
