# Cylinder（円柱）

<Reference shortcut='Shift-C' command='Cylinder' button='ツールバー' />

中心と直径で指定した円から円柱を作成できます。

1. ツールバーのボタン <ButtonIcon type="button-cylinder" /> をクリックします。
2. 円の中心位置でクリックします。
3. カーソルを移動して直径の位置でクリックします。
   - <kbd>V</kbd> **Vertical/ Horizontal:** Changes the inclination with respect to the construction plane.
   - <kbd>X</kbd> **X-axis:** X軸方向に限定します。
4. カーソルを動かして高さの位置でクリックします。
5. コマンドダイアログとショートカットキーでオプションを設定します。
   <Boolean_Options />
   - **Radius:** 直径を指定します。
   - **Height:** 高さを指定します。
   - <kbd>F</kbd> **Radius:** カーソルを動かして直径を指定します。
   - <kbd>D</kbd> **Height:** カーソルを動かして高さを指定します。
5. コマンドダイアログのOKまたは右クリックで確定します。
6. 円柱が作成されます。
