## Beschreibung

Sie können einen Kreis erstellen, indem Sie drei Punkte auf seinem Umfang festlegen.

## Grundlegende Verwendung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-three-point-circle" /> in der Werkzeugleiste.
2. Klicken Sie auf einen Punkt auf dem Kreisumfang.
3. Bewegen Sie den Cursor und klicken Sie auf einen zweiten Punkt auf dem Umfang.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht die direkte Eingabe von Maßen. Details finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>K</kbd> **Knife:** Siehe [Knife-Modus verwenden](/tool/sketching-essentials#use-knife-mode).
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Achse:** [Auf X-, Y- oder Z-Achse beschränken](/tool/sketching-essentials#constrain-to-an-axis).
4. Bewegen Sie den Cursor, um die Größe anzupassen, und klicken Sie zur Bestätigung.
5. Der Kreis wird erstellt.
