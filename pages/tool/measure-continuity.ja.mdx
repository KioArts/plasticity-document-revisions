## 説明

隣接するエッジが G0・G1・G2 のいずれの連続性を満たしているかを測定します（G3 は非対応）。  
このコマンドは **連続性コーム（Continuity Combs）** を生成し、Outliner の **Measurements** に追加されます。

## 基本的な使い方

1. 隣接して接している 2 つのサーフェスエッジを選択します。
2. コマンドパレットで *Measure Continuity* と入力します。
3. コマンドダイアログでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
         - **Continuity:** エッジ連続性の種類を確認します:
            - **G0:** 位置連続性をチェックし、位置のずれを Plasticity の現在の単位で表示します。
            - **G1:** 接線連続性をチェックし、隣接面の角度差（度）を表示します。
            - **G2:** 曲率連続性をチェックし、曲率のずれを正規化された割合（0.0〜1.0）で表示します。
         - **Continuity Markers:** [こちら](#continuity-markers)を参照してください。
   - <Kst>曲率ダイアログ</Kst> — [こちら](/plasticity-essentials/plasticity-interface/curvature)を参照してください。
        - **G0 Scale / G1 Scale / G2 Scale:** 
                - コームおよび偏差プロットの表示スケールを調整します。
        - **Show maxes:** 
                - 評価されたエッジに沿って数値偏差と連続性マーカーを表示します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>Tab</kbd> **Cycle:** 連続性オプションを順番に切り替えます。
4. コマンドダイアログの **OK** をクリックするか、右クリックで確定します。
5. G0/G1/G2 の連続性測定が作成され、**Outliner** の **Measurements** に追加されます。

## Continuity Markers[#continuity-markers]

連続性マーカーは、2 つの隣接面が選択した連続性レベルをどの程度満たしているかを素早く把握するための指標です。

- **G0（位置）** — 点が一致している。
- **G1（接線）** — 接線方向が一致している。
- **G2（曲率）** — 曲率が一致している。

各アイコンは、測定された偏差が許容範囲内か、例外ケースか、または連続性条件を満たしていないかを示します。

| Icon | 意味 |
| :---: | --- |
| <ButtonIcon type="button-green-check"/> | **許容範囲内。** 選択した連続性レベル（G0/G1/G2）を満たしており、偏差は許容範囲内です。 |
| <ButtonIcon type="button-yellow-check"/> | **G2 曲率連続性の特別ケース。** 偏差値 **1.0** は **「フラット」** として報告され、ある方向で曲率半径が無限大であることを意味します。これは **例外** でありエラーではありません。視覚的確認には [シェーダー](/plasticity-essentials/plasticity-interface/shader-mode) の使用を推奨します。 |
| <ButtonIcon type="button-yellow-warning"/> | **連続性レベル間の不一致。** 現在の連続性レベルは許容範囲内でも、下位レベルが失敗している場合があります（例：G1 は OK だが G0 が失敗）。 |
| <ButtonIcon type="button-red-cross"/> | **許容範囲外。** 選択した連続性レベルを許容偏差内で満たしていません。 |
