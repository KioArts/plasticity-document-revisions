# Tool Common options

These options can be specified using keyboard shortcut during the execution of the tool.

### Command dialog

   - **Width**
      - Specifies the width. 
   - **Height**
      - Specifies the height. 
   - **Length**
      - Specifies the length. 
   - **Radius**
      - Specifies the radius. 

### Keyboard shortcut

   - <kbd>K</kbd> **Knife**
      - Add edges to the face with the created curve.
   - <kbd>X</kbd> **X axis**
      - Limit to the X-axis direction.
   - <kbd>Y</kbd> **Y axis**
      - Limit to the Y-axis direction.
   - <kbd>Z</kbd> **Z axis**
      - Limit to the Z-axis direction.
