# Three-Point Circle（3点指定の円）

<Reference shortcut='' command='Three-Point Circle' button='ツールバー' />

円周上の3点で指定した円を作成できます。

1. ツールバーのボタン <ButtonIcon type="button-three-point-circle" />をクリックします。
2. 円周上の最初の位置でクリックします。
3. カーソルを移動して円周上の2点目でクリックします。
   - <kbd>K</kbd> **Knife:** [作成されるカーブの位置にエッジを追加します](./sketching-tips#add-edges-to-the-position-of-the-curve-to-be-created)。
   - <kbd>X</kbd>, <kbd>Y</kbd>, <kbd>Z</kbd> **X, Y, Z-axis:** [座標軸方向に制限します](./sketching-tips#limit-to-the-coordinate-axis-direction)。
4. カーソルを移動して円周上の3点目でクリックします。
5. 円が作成されます。
