## Beschreibung

Sie können einen Bogen erstellen, indem Sie drei Punkte auf seinem Umfang festlegen.

## Grundlegende Verwendung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-three-point-arc" /> in der Werkzeugleiste.
2. Klicken Sie auf die erste Position auf dem Umfang des Bogens.
3. Bewegen Sie den Cursor und klicken Sie auf den zweiten Punkt auf dem Umfang.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Achse:** [Auf X-, Y- oder Z-Achse beschränken](/tool/sketching-essentials#constrain-to-an-axis).
4. Bewegen Sie den Cursor und klicken Sie auf den dritten Punkt auf dem Umfang.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht die direkte Eingabe von Maßen. Details finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
5. Der Bogen wird erstellt.
