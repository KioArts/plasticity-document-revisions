# Three Point Box（3点指定の直方体）

<Reference shortcut='' command='Three-Point Box' button='ツールバー' />

3点を指定した四角形から直方体を作成できます。

1. ツールバーのボタンをクリックします。
2. 四角形の一辺の始点でクリックします。
   - オプションは[ソリッドツールの共通オプション](common-options)を参照してください。
3. 四角形の一辺の終点でクリックします。
4. 四角形の3点目でクリックします。
5. カーソルを動かして目的の高さでクリックします。
   - オプションは[ブーリアンオプション](boolean-options)を参照してください。
6. 直方体が作成されます。
