## Beschreibung

Sie können einen Kreis erstellen, der an drei Kurven tangential ist.

## Grundlegende Verwendung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-tangent-circle" /> in der Symbolleiste.
2. Klicken Sie auf die erste und zweite Kurve, an die der Kreis tangential sein soll. Optional können Sie auch die Tangente zur dritten Kante bestimmen.
3. Bewegen Sie den Cursor, um die Größe des Kreises anzupassen.
    - <Kst>Befehlsdialog</Kst>
        - **Erstes Kurvensegment auswählen:** Klicken Sie, um das erste Kurvensegment auszuwählen, wenn es hervorgehoben ist.
        - **Zweites Kurvensegment auswählen:** Klicken Sie, um das zweite Kurvensegment auszuwählen, wenn es hervorgehoben ist.
        - **Radius:** Geben Sie den Durchmesser an.
- <Kst>Tastaturkürzel</Kst>
  - <kbd>F</kbd> **Einfrieren** - Verhindert, dass sich der Tangentenkreis bewegt, sodass Sie den Mauszeiger für eine präzisere Platzierung neu positionieren können. Nützlich, wenn Sie die Führungsrichtung beim Ziehen des Tangentenkreises umkehren möchten.
  - <kbd>Tab</kbd> **Radius** - Führt Sie zum Radiusfeld, damit Sie einen Radiuswert eingeben können.
4. Klicken Sie im Befehlsdialog auf OK oder rechtsklicken Sie zur Bestätigung.
5. Der Kreis wird erstellt.
