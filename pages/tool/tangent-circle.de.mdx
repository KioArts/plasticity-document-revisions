# Tangent Circle 

<Reference shortcut="" command="Tangent Circle" button="Toolbar" buttonIcon='button-tangent-circle' />

Sie können einen Kreis erstellen, der tangential zu zwei Kurven verläuft.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-tangent-circle" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf die erste und zweite Kurve, an denen der Kreis tangential verlaufen soll. Optional können Sie die Tangente zu Ihrer dritten Kante finden.
3. Bewegen Sie den Cursor, um die Größe des Kreises anzupassen.
    - <Kst>Befehlsdialog</Kst>
        - **Select first curve segment:** Klicken Sie, um das erste Kurvensegment auszuwählen, wenn es hervorgehoben ist.
        - **Select second curve segment:** Klicken Sie, um das zweite Kurvensegment auszuwählen, wenn es hervorgehoben ist.
        - **Radius:** Durchmesser angeben.
    - <Kst>Tastaturkürzel</Kst>
  - <kbd>F</kbd> **Freeze** – Verhindert, dass sich der Tangenten‑Kreis bewegt, sodass Sie den Mauszeiger für eine präzisere Platzierung neu positionieren können. Nützlich, wenn Sie die Führungsrichtung beim Ziehen des Tangenten‑Kreises umkehren müssen.
  - <kbd>Tab</kbd> **Radius** – Öffnet das Radiuseingabefeld, damit Sie einen Radiuswert eingeben können.
4. Klicken Sie im Befehlsdialog auf OK oder rechtsklick zur Bestätigung.
5. Der Kreis wird erstellt.
