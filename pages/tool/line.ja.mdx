# Line（線分）

<Reference shortcut="Shift-A" command="Line" button="ツールバー" />

一連の線分を作成できます。

1. ツールバーのボタンをクリックします。
2. 線分の最初の位置でクリックします。
3. 次の位置でクリックを繰り返して、一連の線分を作成します。
   - <kbd>Tab</kbd> 
      - 長さと角度を数値で指定します。
   - <kbd>Ctrl-Z</kbd> **Undo**
      - 直前のクリックを取り消します。
   - そのほかのオプションは[スケッチツールの共通オプション](common-options)を参照してください。
4. コマンドダイアログのOKまたは右クリックで確定します。
5. 線分が作成されます。
