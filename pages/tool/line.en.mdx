# Line

<Reference shortcut="Shift-A" command="Line" button="Toolbar" />

You can create a series of connected line segments.

1. Click the button <ButtonIcon type="button-line" /> on the toolbar.
2. Click at the start point of the line segment.
3. Repeat clicking at the next point to create the series of line segments.
   - <kbd>Tab</kbd> Specify length and angle with a numerical value.
   - <kbd>K</kbd> **Knife:** Add edges to the face with the created curve.
   - <kbd>Ctrl-Z</kbd> **Undo:** Cancel the last click.
   - <kbd>X</kbd> **X-axis:** Limit to the X-axis direction.
   - <kbd>Y</kbd> **Y-axis:** Limit to the Y-axis direction.
   - <kbd>Z</kbd> **Z-axis:** Limit to the Z-axis direction.
4. Confirm with OK in the Command Dialog or right-click.
5. A line segment is created.
