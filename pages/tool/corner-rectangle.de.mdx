## Beschreibung

Sie können ein Rechteck erstellen, indem Sie seine Diagonale festlegen.

## Grundlegende Verwendung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-corner-rectangle" /> in der Symbolleiste.
2. Klicken Sie auf eine Ecke des Rechtecks.
3. Bewegen Sie den Cursor, um die Größe anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastaturkürzel</Kst>
      - <kbd>Tab</kbd>: Ermöglicht die direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>K</kbd> **Messer:** Weitere Informationen finden Sie unter [Messermodus verwenden](/tool/sketching-essentials#use-knife-mode).
      - <kbd>C</kbd> **Mitte / Ecke:** Wechselt zum Mittelpunktrechteck.
      - <kbd>S</kbd> **Quadrat:** Beschränkt auf eine quadratische Form.
4. Rechtsklicken Sie zur Bestätigung.
5. Das Rechteck wird erstellt.
