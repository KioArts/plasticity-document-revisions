# Kugel

<Reference shortcut='' command='Sphere' button='Werkzeugleiste' />

Erstellen Sie eine Kugel, indem Sie deren Mittelpunkt und Durchmesser angeben. Auch boolesche Operationen sind verfügbar.

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-sphere" /> in der Werkzeugleiste.
2. Klicken Sie auf den Punkt für das Zentrum der Kugel.
3. Bewegen Sie den Cursor und klicken Sie auf den gewünschten Radius.
4. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - **Select target bodies**, <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - Diese Optionen haben die gleiche Funktion wie der Befehl Boolean. Siehe [**Boolean**](../solid/boolean) Befehl für Details.
   - **Radius:** Geben Sie den Radius an.
   - <kbd>D</kbd> **Radius:** Bewegen Sie den Cursor, um den Radius anzugeben.
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Achse:** [Auf die X-, Y- oder Z-Achse beschränken](./sketching-essentials#constrain-to-an-axis).
5. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
6. Eine Kugel wird erstellt.
