# Krümmung umschalten

<Reference shortcut="" command="Toggle Curvature" button="Werkzeugleiste" buttonIcon='button-toggle-curvature' />

Der Befehl „Krümmung messen“ wird basierend auf dem ausgewählten Objekt ausgeführt:
- **Oberflächenkrümmung umschalten:** Wenn Oberflächenkanten ausgewählt sind, siehe [Oberflächenkrümmung umschalten](/solid/toggle-surface-curvature) im Abschnitt „Solid‑Befehle“.
- **Kurvenkrümmung umschalten:** Wenn Kurven ausgewählt sind, siehe [Kurvenkrümmung umschalten](/sketch/toggle-curve-curvature) im Abschnitt „Sketch‑Befehle“.
