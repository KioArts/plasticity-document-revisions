# Control Point Curve（制御点カーブ）

<Reference shortcut="" command="Control Point Curve" button="ツールバー" />

制御点カーブを作成できます。

1. ツールバーのボタン <ButtonIcon type="button-control-point-curve" /> をクリックします。
2. 制御点カーブの最初の位置でクリックします。
3. カーブの制御点が通過する点をクリックします。クリックを繰り返して制御点カーブを作成します。
   - <kbd>K</kbd> **Knife:** [作成されるカーブの位置にエッジを追加します](./sketching-tips#add-edges-to-the-position-of-the-curve-to-be-created)。
   - <kbd>Tab</kbd> **Control / Through-point:** Spline Curveに切り替えます。
   - <kbd>Ctrl + Z</kbd> **Undo:** 直前のクリックを取り消します。
   - <kbd>X</kbd>, <kbd>Y</kbd>, <kbd>Z</kbd> **X, Y, Z-axis:** [座標軸方向に制限します](./sketching-tips#limit-to-the-coordinate-axis-direction)。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
5. 制御点カーブが作成されます。
