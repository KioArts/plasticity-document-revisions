## 説明

コントロールポイントをクリックしてカーブを作成します。

## 基本的な使い方

1. ツールバーの <ButtonIcon type="button-control-point-curve" /> ボタンをクリックします。
2. カーブの開始点をクリックします。
3. コントロールポイントをクリックしてカーブを作成します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>K</kbd> **Knife:** [ナイフモードの使用](/tool/sketching-essentials#use-knife-mode) を参照。
      - <kbd>Tab</kbd> **コントロール / 通過点:** スプラインカーブに切り替えます。
      - <kbd>Ctrl + Z</kbd> **元に戻す:** 最後のクリックを取り消します。
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z 軸:** [軸に制限する](/tool/sketching-essentials#constrain-to-an-axis)。
4. 最後のコントロールポイントをクリックしたら、右クリックで確定します。
5. カーブが作成されます。

<Callout type="info">
**2 種類のカーブの違い**  
   - **Spline Curve:** カーブはクリックしたすべての点を通過します。  
   - **Control Point Curve:** クリックした点はコントロールポイントとなり、カーブ上に位置しない場合があります。
</Callout>
