## Beschreibung

Sie können ein Rechteck erstellen, indem Sie drei Punkte festlegen.

## Grundlegende Verwendung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-three-point-rectangle" /> in der Symbolleiste.
2. Klicken Sie auf eine Ecke des Rechtecks.
3. Klicken Sie auf einen weiteren Punkt des Rechtecks.
4. Bewegen Sie den Cursor, um die Größe anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastaturkürzel</Kst>
      - <kbd>Tab</kbd>: Ermöglicht die direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>K</kbd> **Messer:** Weitere Informationen finden Sie unter [Messermodus verwenden](/tool/sketching-essentials#use-knife-mode).
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Achse:** [Auf X-, Y-, Z-Achse beschränken](/tool/sketching-essentials#constrain-to-an-axis).
3. Rechtsklicken Sie zur Bestätigung.
5. Das Rechteck wird erstellt.
