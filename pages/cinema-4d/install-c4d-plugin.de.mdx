# Einrichtung von Cinema 4D Bridge

Um das Feature von Cinema 4D Bridge zu nutzen, müssen Sie das Plugin in Cinema 4D installieren und die Funktion innerhalb von Plasticity aktivieren.

## Cinema 4D Plugin installieren

1. Laden Sie das Cinema 4D Bridge Plugin aus dem offiziellen GitHub-Repository herunter. Hier ist der direkte Link zur neuesten Version:
    [plasticity-c4d-bridge.zip](https://github.com/ninsent/plasticity-c4d-plugin/releases/download/v1.1.0/PlasticityBridge_v1.1.0.zip)

2. Entpacken Sie das Archiv. Sie sollten einen Ordner mit dem Namen `PlasticityBridge` erhalten.

3. Kopieren Sie den Ordner `PlasticityBridge` in Ihr Cinema 4D-Plugin-Verzeichnis:
    - **Windows:** `C:\Program Files\Maxon Cinema 4D\plugins`
    - **macOS:** `/Applications/Maxon Cinema 4D/plugins`

4. Starten Sie Cinema 4D (neu). Das Plugin erscheint unter **Erweitern > Plasticity Bridge**.

    - Cinema 4D 2023 oder neuer ist erforderlich.

5. Durch Öffnen von **Erweitern > Plasticity Bridge** wird der Plasticity Bridge-Dialog angezeigt, der wie jedes andere Cinema 4D-Panel angedockt werden kann.

## Funktion in Plasticity aktivieren

1. Starten Sie Plasticity.

2. Öffnen Sie Einstellungen aus dem P-Menü, das sich oben links auf dem Bildschirm befindet.

3. Öffnen Sie auf der linken Seite Server und setzen Sie ein Häkchen bei Aktiviert.

Jetzt sind Sie bereit, das Feature von Cinema 4D Bridge zu nutzen.
