# Cinema 4D Bridgeのセットアップ

Cinema 4D Bridgeの機能を使うには、Cinema 4Dへのプラグインのインストールと、Plasticityでの機能の有効化が必要です。

## Cinema 4Dプラグインのインストール

1. 公式のGitHubリポジトリからCinema 4D Bridgeプラグインをダウンロードします。以下は最新バージョンへの直接のリンクです。:
    [plasticity-c4d-bridge.zip](https://github.com/ninsent/plasticity-c4d-plugin/releases/download/v1.1.0/PlasticityBridge_v1.1.0.zip)

2. アーカイブを解凍します。`PlasticityBridge` という名前のフォルダができます。

3. `PlasticityBridge` フォルダをCinema 4Dのプラグインディレクトリにコピーします:
    - **Windows:** `C:\Program Files\Maxon Cinema 4D\plugins`
    - **macOS:** `/Applications/Maxon Cinema 4D/plugins`

4. Cinema 4Dを起動(または再起動)します。プラグインは **拡張 > Plasticity Bridge** の下に表示されます。

    - Cinema 4D 2023 以降が必要です。

5. **拡張 > Plasticity Bridge** を開くと、Plasticity Bridgeダイアログが表示されます。このダイアログは他のCinema 4Dパネルと同様にドッキングできます。

## Plasticityでの機能の有効化

1. Plasticityを起動します。

2. 画面左上のPメニューからPreferencesを開きます。

3. 左側のServerを開き、Enabledにチェックリストを入れます。

以上で、Cinema 4D Bridgeの機能を使う準備が整いました。
