# Einführung in Cinema 4D Bridge

Die Cinema 4D Bridge ist ein Feature, das den schnellen Transfer von Mesh-Daten von Plasticity nach Cinema 4D ermöglicht. Dies wird als Cinema 4D Plugin realisiert. Sobald Sie verbunden sind, können Sie einfach auf Aktualisieren klicken, und die Daten werden sehr schnell geladen. Sie müssen sich keine Gedanken darüber machen, in eine temporäre Datei zu exportieren und alle Einstellungsoptionen richtig zu bekommen.

Selbst wenn Sie die Geometrie in Plasticity ändern, bleiben die in Cinema 4D eingestellten Materialien erhalten. Darüber hinaus wird die Gruppenstruktur in Plasticity als Null-Objekt-Gruppen in Cinema 4D wiedergegeben. In diesem Prozess wird die Hierarchie von Cinema 4D mit dem Plasticity Dokumentnamen als Identifikator erstellt. Daher können mehrere Plasticity-Dokumente in ein einziges Cinema 4D-Dokument integriert werden.

Es gibt auch eine Live-Link-Funktion. Der Live-Link ist ein wirklich cooles Feature; man kann sehen, wie Sachen in Echtzeit aktualisiert werden. Das Hauptarbeitspferd der Cinema 4D Bridge ist die Fähigkeit, Assets schnell reinzubekommen, so dass man sie mit Cinema 4D Materialien, Cinema 4D Beleuchtung, Raytracing sehen kann. Sie können Ihre Objekte in einer Szene mit der echten Kamera betrachten, die möglicherweise in einem Spiel oder in einem finalen Render verwendet wird, mit einem spezifischen FOV. Sie erhalten die volle Leistung von Cinema 4D, einschließlich dem Verknüpfen von Dingen zu Nulls, so dass Sie sie animieren, riggen und sehen können, ob Ihre mechanischen Teile zusammenpassen.

Der Workflow ist auch ein fantastisches Erlebnis. Nachdem Sie das Plasticity-Dokument in die Cinema 4D-Szene eingefügt haben, können Sie immer noch die Facetteneinstellungen ändern. Fast alle Facetting-Tools von Plasticity sind innerhalb von Cinema 4D verfügbar. Das bedeutet, dass Sie schnell die benötigte Polygon-Dichte oder den Edge-Flow für Ihr Modell erhalten können, was zu einem schnelleren Workflow führt.

Die Spezifikation von UV-Nähten und Schärfen in Cinema 4D wird in der aktuellen Version nicht beibehalten. Das Cinema 4D Plugin kommt jedoch mit Funktionen, die Randkanten automatisch als Auswahl-Tags speichern und Geometrie automatisch auswählen, die den Plasticity-Flächen oder -Kanten entspricht. Diese Features tragen auch zu einem schnelleren Workflow bei.
