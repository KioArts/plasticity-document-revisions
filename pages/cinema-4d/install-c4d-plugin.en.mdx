# Setup Cinema 4D Bridge

To utilize the Cinema 4D Bridge feature, it's necessary to install the plugin in Cinema 4D and enable the function within Plasticity.

## Installing the Cinema 4D Plugin

1. Download the Cinema 4D Bridge plugin from the official GitHub repository. Here is the direct link to the latest version:
    [plasticity-c4d-bridge.zip](https://github.com/ninsent/plasticity-c4d-plugin/releases/download/v1.1.0/PlasticityBridge_v1.1.0.zip)

2. Unzip the archive. You should end up with a folder named `PlasticityBridge`.

3. Copy the `PlasticityBridge` folder into your Cinema 4D plugins directory:
    - **Windows:** `C:\Program Files\Maxon Cinema 4D\plugins`
    - **macOS:** `/Applications/Maxon Cinema 4D/plugins`

4. Launch (or restart) Cinema 4D. The plugin will appear under **Extensions > Plasticity Bridge**.

    - Cinema 4D 2023 or later is required.

5. Opening **Extensions > Plasticity Bridge** will show the Plasticity Bridge dialog, which can be docked like any other Cinema 4D panel.

## Enabling Feature in Plasticity

1. Launch Plasticity.

2. Open Preferences from the P menu located at the top left of the screen.

3. On the left, open Server and check the box for Enabled.

Now, you are ready to use the Cinema 4D Bridge feature.
