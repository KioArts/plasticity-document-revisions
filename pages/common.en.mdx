### Transform

- Tools to move, rotate, scale, and mirror objects in your scene.

| Name | Description |
| :--- | :--- |
| [Move](/common/move) | Move objects to a new position in the scene. |
| [Rotate](/common/rotate) | Rotate objects around a defined axis or point. |
| [Scale](/common/scale) | Scale objects uniformly or along a specific axis. |
| [Mirror](/common/mirror) | Mirror objects across a defined plane or axis. |

### Array and Duplication

- Tools to duplicate and distribute objects across your scene.

| Name | Description |
| :--- | :--- |
| [Duplicate](/common/duplicate) | Create a copy of a selected object in place. |
| [Curve Array](/common/curve-array) | Distribute copies of an object along a curve. |
| [Radial Array](/common/radial-array) | Distribute copies of an object in a circular pattern around a center point. |
| [Rectangular Array](/common/rectangular-array) | Distribute copies of an object in a grid pattern. |
| [Copy with Placement](/common/copy-with-placement) | Copy an object and place it at a specified location. |
| [Paste with Placement](/common/paste-with-placement) | Paste a copied object at a specified location. |
| [Place](/common/place) | Distribute copy of an object and place them at a reference point. |

### Materials
