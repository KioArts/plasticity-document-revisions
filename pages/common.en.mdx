# Common Commands for Sketch and Solid

| Command Name                                         | Shortcut                    | Note                                       |
| ---------------------------------------------------- | --------------------------- | ------------------------------------------ |
| [Complete Edge](/common/complete-edge)               |                             | Extend the selected Edge until it intersects with another Edge.       |
| [Copy with Placement](/common/copy-with-placement)   | <kbd>Ctrl+Shift+C</kbd>     | Copy objects with a reference point.       |
| [Curve Array](/common/curve-array)                   |                             | Duplicate objects along a Curve.           |
| [Dimension](/common/dimension)                       | <kbd>=</kbd>                | Modify dimension values.                   |
| [Duplicate](/common/duplicate)                       | <kbd>Shift+D</kbd>          | Duplicate objects.                         |
| [Fork Material](/common/fork-material)               | <kbd>Shift+M</kbd>          | Duplicate and edit material for selected object.                         |
| [Measure](/common/measure)                           | <kbd>Ctrl+=</kbd>           | Measure dimension values.                  |
| [Measure Radius](/common/measure-radius)                 |  | Measure radial values.                  |
| [Mirror](/common/mirror)                             | <kbd>Alt+X</kbd>            | Mirror objects.                            |
| [Move](/common/move)                                 | <kbd>G</kbd>                | Move objects.                              |
| [Publish to Plasticity Share](/common/publish-to-plasticity-share) |                | Upload and share Plasticity documents via the Web.     |
| [Paste with Placement](/common/paste-with-placement) | <kbd>Ctrl+Shift+V</kbd>     | Paste objects with a reference point.      |
| [Place](/common/place)                               | <kbd>Ctrl+D</kbd>           | Duplicate objects specified axis.          |
| [Radial Array](/common/radial-array)                 |                             | Duplicate objects in a radial pattern.     |
| [Rectangular Array](/common/rectangular-array)       |                             | Duplicate objects in a rectangular pattern.|
| [Remove Material](/common/remove-material)           | <kbd>Alt+M</kbd>            | Remove material.                           |
| [Rotate](/common/rotate)                             | <kbd>R</kbd>                | Rotate objects.                            |
| [Scale](/common/scale)                               | <kbd>S</kbd>                | Scale objects.                             |
| [Section Analysis](/common/section-analysis)         |                             | Create a virtual slice through an object.  |
| [Set Material](/common/set-material)                 | <kbd>M</kbd>                | Set material.                              |
