### Transformieren

- Werkzeuge zum Bewegen, Drehen, Skalieren und Spiegeln von Objekten in deiner Szene.

| Name | Beschreibung |
| :--- | :--- |
| [Move](/common/move) | Bewege Objekte an eine neue Position in der Szene. |

| [Scale](/common/scale) | Skaliere Objekte gleichmäßig oder entlang einer bestimmten Achse. |
