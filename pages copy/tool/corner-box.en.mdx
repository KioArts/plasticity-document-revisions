# Corner Box

<Reference shortcut='Shift-B' command='Corner Box' button='Toolbar' />

You can  create a rectangular box from a specified two points on a diagonal line.

1. Click the button <ButtonIcon type="button-corner-box" />ã€€on the toolbar.
2. Click at the start point of the diagonal line on the base surface.
   - <kbd>C</kbd> **Center/Corner:** Switches to Center Box.
   - <kbd>S</kbd> **Square:** Limits to a square.
3. Click at the endpoint of the diagonal line.
4. Move the cursor and click at the desired height.
5. Set the options in the Command Dialog and shortcut key.
   <Boolean_Options />
   - **Width:** Specify the width.
   - **Length:** Specify the Length.
   - **Height:** Specify the height.
   - <kbd>H</kbd> **Height:** Moves the cursor to specify the height.
6. Confirm with OK in the Command Dialog or right-click.
7. A rectangular box is created.
