# Three Point Rectangle（3点指定の四角形）

<Reference shortcut='' command='Three Point Rectangle' button='ツールバー' />

3点を指定した四角形を作成できます。

1. ツールバーのボタン <ButtonIcon type="button-three-point-rectangle" /> をクリックします。
2. 四角形の一辺の始点でクリックします。
   - <kbd>Tab</kbd> 長さを数値で指定します。
   - <kbd>K</kbd> **Knife:** 作成されたカーブでフェイスにエッジを追加します。
   - <kbd>X</kbd> **X-axis:** X軸方向に限定します。
   - <kbd>Y</kbd> **Y-axis:** Y軸方向に限定します。
   - <kbd>Z</kbd> **Z-axis:** Z軸方向に限定します。
3. 四角形の一辺の終点でクリックします。
4. 四角形の3点目をクリックします。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
6. 四角形が作成されます。
