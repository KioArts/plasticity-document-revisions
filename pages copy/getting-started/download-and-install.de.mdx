# Download und Installation

Das Verfahren für den Download und die Installation variiert je nach Lizenztyp, den Sie besitzen. Bitte beziehen Sie sich auf die Anweisungen, die Ihrer Lizenz über die folgenden Links entsprechen:
- [Für Benutzer mit Studio-Lizenz](#studio-user)
- [Für Benutzer mit Indie-Lizenz](#indie-user)
- [Für Benutzer, die die kostenlose Testversion nutzen](#trial-user)

## Für Benutzer mit Studio-Lizenz[#studio-user]

Benutzer mit einer Studio-Lizenz können sowohl die Stable-Version, die stabil im Betrieb ist, als auch die Beta-Version, die experimentelle Funktionen enthält, verwenden.
Beide Versionen, Stable und Beta, können getrennt installiert und austauschbar verwendet werden.

Das gleiche Verfahren gilt sowohl für neue Installationen als auch für Updates. Eine Deinstallation der alten Version ist beim Aktualisieren nicht erforderlich.

#### Download und Installation
1. Öffnen Sie die offizielle GitHub-Release-Seite.
   - https://github.com/nkallen/plasticity/releases
2. Die neueste Version wird oben auf der Release-Seite angezeigt, vergangene Versionen sind darunter aufgelistet.
3. Die als "Latest" in grün gekennzeichnete Version neben der Versionsnummer ist die neueste Stable-Version.
   ![Stable](/img/download-latest.png)
4. Die als "Pre-release" in gelb gekennzeichnete Version neben der Versionsnummer ist die Beta-Version.
   ![Beta](/img/download-beta.png)
5. Sie können entweder die Stable- oder die Beta-Version oder beide installieren. Die Installationsreihenfolge ist egal.
6. Klicken Sie auf die Version, die Sie herunterladen möchten, und gehen Sie zum Dateilistebildschirm.
<Tabs items={['Windows', 'macOS', 'Linux']}>
   <Tabs.Tab>
   7. Klicken Sie nach dem Aufklappen von Assets auf "plasticity-*****.Setup.exe" aus der Dateiliste. Der Download beginnt.
   8. Nach Abschluss des Downloads öffnen Sie die heruntergeladene Datei.
   9. Das Installationsprogramm wird gestartet, folgen Sie den Anweisungen, um mit der Installation fortzufahren.
   </Tabs.Tab>
   <Tabs.Tab>
   7. Klicken Sie nach dem Aufklappen von Assets auf die für die Installation auf macOS geeignete Datei aus der Dateiliste. Der Download beginnt.
      - Für macOS (M1): Plastycity-darwin-arm64-*****.zip
      - Für macOS (Intel): Plastycity-darwin-x64-*****.zip
   8. Nach Abschluss des Downloads entpacken Sie die heruntergeladene Datei.
   9. Ziehen Sie das Anwendungspaket in den Ordner "Programme".
   </Tabs.Tab>
   <Tabs.Tab>
   7. Klicken Sie nach dem Aufklappen von Assets auf die geeignete Datei aus der Dateiliste. Der Download beginnt.
      - Für rpm: Plastycity-***.rpm
      - Für NuGet Package Manager: Plastycity-***.nupkg	
      - Für deb: Plastycity_*****.deb
   8. Nach Abschluss des Downloads fahren Sie wie gewohnt mit der Installation fort.
   </Tabs.Tab>
</Tabs>

Nach Abschluss der Installation fahren Sie mit der Lizenzaktivierung, dem Radial Menu und der Einrichtung der Blender Bridge fort.

#### Lizenzaktivierung

Die Lizenzaktivierung ist erforderlich, wenn Plasticity zum ersten Mal verwendet wird.

10. Starten Sie Plasticity.
11. Klicken Sie auf das P-Menü oben links und klicken Sie auf "Lizenzen verwalten".
   ![P-Menu](/img/p-menu.png)
12. Geben Sie den Token ein und klicken Sie auf "Token validieren".
13. Die Lizenzaktivierung ist abgeschlossen.

#### Radial Menu Einrichtung
14. Eine Einrichtung ist erforderlich, wenn Sie das Radial Menu verwenden. Wenn Sie das Radial Menu nicht verwenden, ist dieser Schritt nicht erforderlich.
15. Eine Beschreibung der Radial Menu-Funktionen und Einrichtungsanweisungen finden Sie [hier](../plasticity-essentials/radial-menu).

#### Blender Bridge Einrichtung
16. Eine Add-on-Installation ist erforderlich, wenn Sie die Blender Bridge verwenden. Wenn Sie die Blender Bridge nicht verwenden, ist dieser Schritt nicht erforderlich.
17. Eine Beschreibung der Blender Bridge-Funktionen und Anweisungen zur Add-on-Installation finden Sie [hier](../blender/introduction-blender-bridge).

Die Installation ist jetzt abgeschlossen.

## Für Benutzer, die die kostenlose Testversion nutzen[#trial-user]

Die kostenlose Testversion kann von der offiziellen Seite heruntergeladen werden.

1. Öffnen Sie die offizielle Seite.
   - https://www.plasticity.xyz/
2. Laden Sie die für Ihr Betriebssystem geeignete Datei herunter.

<Tabs items={['Windows', 'macOS', 'Linux']}>
   <Tabs.Tab>
   3. Nachdem der Download abgeschlossen ist, öffnen Sie die heruntergeladene Datei.
   4. Das Installationsprogramm startet, folgen Sie den Anweisungen, um die Installation fortzusetzen.
   </Tabs.Tab>
   <Tabs.Tab>
   3. Nachdem der Download abgeschlossen ist, entpacken Sie die heruntergeladene Datei.
   4. Ziehen Sie das Anwendungspaket in den Ordner "Programme".
   </Tabs.Tab>
   <Tabs.Tab>
   3. Nachdem der Download abgeschlossen ist, fahren Sie wie gewohnt mit der Installation fort.
   </Tabs.Tab>
</Tabs>

Nach Abschluss der Installation fahren Sie mit der Lizenzaktivierung, Radial Menu und Blender Bridge-Einrichtung fort.

#### Lizenzaktivierung
5. Starten Sie Plasticity.
6. Der Lizenzbildschirm wird angezeigt.
7. Klicken Sie auf "Kostenlose Testversion starten".
8. Geben Sie Ihre E-Mail-Adresse und Ihr Passwort ein und klicken Sie auf "Konto erstellen".
9. Ein Token für die Testversion wird an Ihre E-Mail-Adresse gesendet.
10. Geben Sie das Token ein und klicken Sie auf "Token überprüfen".
11. Schließen Sie den Lizenzbildschirm. Sie können jetzt die kostenlose Testversion verwenden.

#### Radial Menu-Einrichtung
12. Eine Einrichtung ist erforderlich, wenn Sie das Radial Menu verwenden. Wenn Sie das Radial Menu nicht verwenden, ist dieser Schritt nicht erforderlich.
13. Eine Beschreibung der Radial Menu-Funktionen und Einrichtungsanweisungen finden Sie [hier](../plasticity-essentials/radial-menu).

#### Blender Bridge-Einrichtung
14. Eine Add-On-Installation ist erforderlich, wenn Sie die Blender Bridge verwenden. Wenn Sie die Blender Bridge nicht verwenden, ist dieser Schritt nicht erforderlich.
15. Eine Beschreibung der Blender Bridge-Funktionen und Add-On-Installationsanweisungen finden Sie [hier](../blender/introduction-blender-bridge).

Die Installation ist jetzt abgeschlossen.
