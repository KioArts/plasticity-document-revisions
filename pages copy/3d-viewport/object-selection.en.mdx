# Object Selection

## Selection Modes

- You need to switch the selection mode according to the object you want to select.
- The selection mode can be toggled using the buttons in the upper left corner of the screen or keyboard shortcuts.

    ![Selection mode buttons](./img/selection-mode-buttons.png)
<div className="dl">
- Control point selection mode: 
   - Press <kbd>1</kbd> or click the button(1).
- Edge selection mode: 
   - Press <kbd>2</kbd> or click the button(2).
- Face selection mode: 
   - Press <kbd>3</kbd> or click the button(3).
- Solid selection mode: 
   - Press <kbd>4</kbd> or click the button(4).
- All selectable mode: 
   - Press <kbd>Tab</kbd>
<br />
- Multiple selection modes can be specified.
- Toggle only control point selection mode: 
   - Press <kbd>Shift - 1</kbd> or <kbd>Shift</kbd>-click the button(1).
- Toggle only edge selection mode: 
   - Press <kbd>Shift - 2</kbd> or <kbd>Shift</kbd>-click the button(2).
- Toggle only face selection mode: 
   - Press <kbd>Shift - 3</kbd> or <kbd>Shift</kbd>-click the button(3).
- Toggle only solid selection mode: 
   - Press <kbd>Shift - 4</kbd> or <kbd>Shift</kbd>-click the button(4).
</div>

## Selection

- You can select by clicking. Additionally, you can select by dragging the mouse or using shortcuts.

### Selecting with Mouse Dragging

- The items that can be selected differ based on the direction of the mouse drag.
<div className="dl">
- Dragging from the right towards the left: 
   - Everything that touches the green selection area is selected.
- Dragging from the left towards the right: 
   - Only those completely within the purple selection area are selected.
</div>
![Mouse Drag selection](./img/mouse-drag-selection.gif)

### Selecting with Shortcuts

<div className="dl">
- Select all: 
   - Press <kbd>A</kbd> or use **"Select all"** in the Command Palette.
- Select All Curves: 
   - Use **"Select All Curves"** in the Command Palette.
- Deselect all: 
   - Press <kbd>Esc</kbd> or use **"Deselect all"** in the Command Palette.
- Invert Selection: 
   - Press<kbd>Alt - A</kbd> or use **"Invert Selection"** in the Command Palette.
- Make selection unselectable: 
   - Use **"Lock Selected"** in the Command Palette.
<br />
- Select edges included in the selection: 
   - Press <kbd>Ctrl - 2</kbd> or use **"Selection: convert to edges"** in the Command Palette.
- Select faces included in the selection: 
   - Press <kbd>Ctrl - 3</kbd> or use **"Selection: convert to faces"** in the Command Palette.
- Select solids included in the selection: 
   - Press <kbd>Ctrl - 4</kbd> or use **"Selection: convert to solids"** in the Command Palette.

![Select objects included](./img/select-objects-included.gif)

<br />
- Add adjacent items to the selection: 
   - Press <kbd>Ctrl - Shift - =</kbd> or use **"Select Adjacent"** in the Command Palette.
- Select edge loops: 
   - <kbd>Alt</kbd>-click 
- Select edge rings: 
   - <kbd>Alt - Ctrl</kbd>-click
- Select all Curves, regions on the same plane: 
   - <kbd>Alt</kbd>-click 
- Select all parallel faces: 
   - <kbd>Alt - Ctrl</kbd>-click 
</div>
