# Fillet

<Reference shortcut="B" command="Fillet" button="Right of Command Bar" />

This command automatically calls the following command depending on the selected object.
   - When curves or verticies are selected: see **Fillet Curves and Vertices** described below.
   - When edges are selected: see [**Fillet**](../solid/fillet)  in the Solid Commands section.

## Fillet Curves and Vertices

You can fillet or chamfer curves and vertices at corners.

1. Select the Curves or vertices where you want to apply fillet or chamfer.
2. Press <kbd>B</kbd> or click the button <ButtonIcon type="button-fillet" /> on the Right of Command Bar.
3. Move the yellow dot to specify the radius of the fillet or chamfer. Depending on the direction of movement, it will either be filleted or chamfered.
4. Set the options in the Command Dialog and shortcut key.
   - **Distance:** Specify the Distance.
   - <kbd>D</kbd> **Fillet Distance:** Specify the fillet distance by moving the cursor.
   - <kbd>C</kbd> **Chanfer Distance:** Specify the chanfer distance by moving the cursor.
5. Confirm with OK in the Command Dialog or right-click.
