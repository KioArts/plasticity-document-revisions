# File Menu

ファイルメニューは左上の P をクリックして開きます。

![File Menu](/img/file-menu.png)

| Command Name              | Shortcut                  | Note                                                                               |
| ------------------------- | ------------------------- | ---------------------------------------------------------------------------------- |
| New                       | <kbd>Ctrl-N</kbd>         | シーンをクリアして、新しいドキュメントを開きます                                   |
| Open                      | <kbd>Ctrl-O</kbd>         |                                                                                    |
| Import/Append             | <kbd>Ctrl-Shift-O</kbd>   | [インポートできるファイル形式](#supported-import-file-formats) を参照してください  |
| Save                      | <kbd>Ctrl-S</kbd>         |                                                                                    |
| Export/Save as            | <kbd>Ctrl-Shift-S</kbd>   | [エクスポートできるファイル形式](#supported-export-file-formats)を参照してください |
| Quick save version        |                           | ファイル名の末尾にタイムスタンプを付け加えて別名保存します                         |
| Save as startup scene     |                           | 現在のドキュメントを起動時に開くようにします。                                     |
| Preferences               | <kbd>Ctrl-, (Comma)</kbd> | [設定](./preferences) を参照してください                                           |
| Manage licenses           |                           | ライセンストークンとデバイスの管理をします。                                       |
| Plasticity Version: 0.0.0 |                           | インストールされているPlasticityのバージョン                                       |
| Quit                      | <kbd>Ctrl-Q</kbd>         |                                                                                    |


## インポートできるファイル形式 [#supported-import-file-formats]
- **Indie and Studio** 
    - Plasticity file (*.plasticity)
    - STEP file (*.stp;*.step)
    - Parasolid file (*.x_t;*.x_b)
    - Wavefront OBJ file (*.obj)
    - Image files (*.png;*.jpg;*.jpeg)
- **Studio only**
    - IGES files (*.igs;*.iges) 
    - SAT files (*sat)
    - Rhino files (*.3dm) 

## エクスポートできるファイル形式 [#supported-export-file-formats]
- **Indie and Studio** 
    - Plasticity file (*.plasticity)
    - STEP file (*.stp;*.step)
    - Parasolid file (*.x_t;*.x_b)
    - [Wavefront OBJ file](./export-obj) (*.obj)
    - STL file (*.stl)
- **Studio only**
    - IGES file (*.igs;*.iges)
    - SAT file (*sat)
