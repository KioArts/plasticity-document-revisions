# Hollow（中空にする）

<Reference shortcut="" command="Hollow" button="コマンドバー右" />

ソリッドを中空にすることができます。

1. 中空にしたいソリッドのフェイスを選択します。
2. ボタン <ButtonIcon type="button-hollow" /> をクリックします。
3. 黄色い点を動かして壁の厚みを指定します。
4. コマンドダイアログでオプションを設定します。
  - **thickness:** 壁の厚みを指定します。
  - **Local:** 
  - <kbd>D</kbd> カーソルを動かして厚みを指定します。
5. コマンドダイアログでOKをクリックするか、右クリックで確定します。
