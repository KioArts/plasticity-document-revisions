# Find Boundary Edges

You can select edges of a sheet opening.

1. Select a sheet you want to find boundary edges.
2. Click the button <ButtonIcon type="button-find-boundary-edges" /> on the Right of Command Bar.
3. Edges of the sheet opening are selected.
